🏥 5 種長照服務類型對照

同一個失智長輩可能同時用 2-3 種(例:居家服務 + 日照 + 喘息),不是互斥。

類型地點適用費用(實付/月,長照給付後)申請
👵 居家服務(居服員) 家裡 輕度-中度失智 / 主照顧者白天工作 失能等級 2-8 級給付額度不同,自付 16% (一般戶) / 5% (中低收) / 0% (低收) 長照 1966 → CMS 評估
☀️ 日間照顧中心(日照) 機構(白天去晚上回) 輕度-中度失智 / 需要團體刺激 同上,失能等級越高給付越多 長照 1966
😌 喘息服務 居家式 / 機構式 / 日間 主照顧者需短期休息 每年額度有限制(失能等級 2-6 級 21 天,7-8 級 32 天) 長照 1966 / 須事先預約
🌏 外籍看護工 家裡(24 小時同住) 中度-重度失智 / 主照顧者無法 月薪約 21,000-25,000 + 食宿 + 仲介費(不在長照給付內) 巴氏量表評估 → 勞動部許可 → 仲介媒合
🏢 住宿機構 機構(住進去) 重度失智 / 主照顧者已撐不住 30,000-80,000+ / 月(類型差異大,部分有長照給付) 看 home-handbook 機構章節

⚠️ 第一個關鍵動作:撥 1966 申請評估

長照服務 99% 都從 1966 長照專線 → CMS 評估 開始。沒做評估 = 沒失能等級 = 沒給付額度。還沒做就先做這個。

📞 看申請流程 💰 看補助系統 💵 預算試算

📞 長照申請流程(0 到開始服務)

⏱ 5 分鐘版:撥 1966 → 縣市照管中心派員到府評估(免費)→ 取得失能等級(2-8 級)→ 照管專員擬「照顧計畫」→ 媒合服務提供者 → 開始服務。從打電話到開始服務約 2-4 週。

🔢 標準 SOP(7 步驟)

  1. 撥 1966 長照專線(免費) — 提供長輩姓名 / 身分證 / 戶籍地址 / 主照顧者聯絡
  2. 等照管中心電話約評估(1-2 週)
  3. 照管專員到府評估(1-1.5 小時)— 用 CMS(Care Management System,長照失能評估量表)+ 失智 CDR 量表
  4. 取得失能等級 — 2-8 級(8 級最重)。失智長輩多在 2-4 級
  5. 照管專員擬定「照顧計畫」 — 推薦服務組合 + 給付額度
  6. 家屬選擇服務提供者(居服員 / 日照中心 / 喘息) + 簽約
  7. 開始服務 — 服務提供者上門 / 接送

📋 評估前要準備的

  • 長輩 身分證 + 健保卡
  • 失智診斷證明書(神內 / 精神科開的,有 CDR 等級最好)
  • 重大傷病卡(若有,失智可申請)
  • 主照顧者描述清單 — 長輩日常自理狀況、行為問題、用藥、特殊需求 → 評估員依此打分

⚠️ 評估的小心機

實話實說 + 強調最壞那天。評估員看 1.5 小時,失智長輩可能那天「狀況好」(failsafe 機制 — 客人來會表現好)。主照顧者要主動補充「平常她的狀況比今天嚴重」。否則失能等級評低 = 給付少,未來要重評。

🔄 失能等級複評

狀況惡化(失智一定會)時,撥 1966 申請複評。等級提升 → 給付額度提升。每 6 個月可複評一次。

💰 長照補助系統

關鍵概念:長照 2.0 不是「政府幫你付錢」,是「政府給你額度,你選服務在額度內政府支付,超出自己付」。額度依失能等級。

🎁 其他補助 / 福利

  • 重大傷病卡(失智可申請) — 健保部分負擔減免
  • 身心障礙手冊 / 證明 — 額外稅務、停車、社會福利
  • 輔具補助 — 輪椅 / 氣墊床 / 防走失裝置 等(透過長照申請)
  • 無障礙環境改善 — 防跌扶手 / 浴室改造 補助 10,000 / 次,失能 4 級以上每年最高 40,000
  • 交通接送補助 — 失能 4 級以上,1,680-2,400 / 月(看就醫)

⚠️ 補助互斥

  • 長照給付 vs 外籍看護工 互斥:有外籍看護工不能領長照 BCS 給付(可申請喘息)
  • 住宿機構大部分自費:長照給付對住宿機構支援有限,要看機構類別

👵 居家照顧服務員(居服員)

📌 一句話

居服員到家裡來照顧長輩,主照顧者上班 / 喘息時段使用。失智輕-中度最常用。每次 1-3 小時。

🏷 3 種來源

來源特點找法
各縣市居家照顧服務中心(政府委託) 受過訓 + 政府監督 + 走長照給付。最推薦 1966 → 照管專員媒合
民間居服機構(連鎖 / 非營利) 同樣走長照給付,品質依業者 1966 / 各縣市社會局名單
個人接案(私人聘僱) 不在長照給付內(全自費)。彈性高但無監督 朋友介紹 / 社區公告

📋 居服員證照要求

  • 照顧服務員結業證書(衛福部認證,90 小時訓練 + 30 小時實習)
  • 單一級照顧服務員技術士證(更高標,含理論考試)
  • 失智症照護課程結業證明(失智專責服務需要)

👁 居服員面試 / 觀察 SOP(關鍵)

  1. 請對方自我介紹 + 照顧過幾位失智長輩(具體經驗 vs 紙上談兵)
  2. 問:長輩拒絕洗澡 / 拒絕服藥怎麼辦? — 聽具體經驗
  3. 問:長輩攻擊行為怎麼處理? — 看是否會「降溫」而非衝突
  4. 問:長輩走失過嗎?怎麼處理? — 看是否懂得通報
  5. 觀察跟長輩的互動 5 分鐘(直接讓他見長輩,看眼神 / 語氣 / 動作)
  6. 問願不願意拍照 / 寫紀錄給家屬看 — 透明度
  7. 問交通方式 + 是否準時 — 居服員遲到是常見問題
  8. 確認簽訂的「照顧計畫」內容 — 每次該做什麼,白紙黑字

📜 簽約該注意的

  • 每次服務時段 + 服務內容明列(避免「來了就坐著」)
  • 缺勤替補規則(居服員生病誰來?)
  • 異動通知期(機構換人要提前通知)
  • 監視器同意條款(可裝在公共空間)

🚨 居服員常見問題(主照顧者抱怨 Top 5)

  1. 遲到 / 早走
  2. 滑手機不照顧
  3. 「來了就坐著」沒做該做的活動
  4. 不寫紀錄 / 不拍照
  5. 對長輩沒耐心 / 講話大聲

看 🚨 SOP 沒做怎辦 知道怎麼正式投訴 + 換人。

☀️ 日間照顧中心(日照)

📌 一句話

白天去機構,晚上回家。像「老人版的幼兒園」。失智輕-中度最常用,提供團體刺激防退化。

🏷 2 種類型

類型差別適用
失智專責日照師資受過失智訓練 / 環境設計防走失 / 活動針對失智設計已確診失智的長輩
一般日間照顧混合長者 / 失能但未失智可參加失能但認知尚可

👁 日照參訪 SOP

  1. 第一印象:有沒有怪味?長輩們的神情(呆滯 vs 投入)?
  2. 師生比(法定 1:8 失智專責)
  3. 活動安排:一天時程表(認知活動 / 體能 / 用餐 / 午休 / 才藝)
  4. 環境:防走失設計 / 廁所數量 / 緊急按鈕 / 監視器
  5. 餐食:看當天午餐 + 點心
  6. 接送:有交通車?還是要自送?
  7. 主照顧者「家訪」(部分日照會做)
  8. 體驗日:多數日照可體驗 1-3 天

📋 失智長輩去日照前要準備

  • 失智診斷證明 + 身障證明(若有)
  • 用藥清單 + 過敏史
  • 照顧計畫書(從照管專員拿)
  • 備品包:換洗衣物 / 藥盒 / 尿布 / 緊急聯絡卡
  • 看 🏠 home-handbook 日照日 SOP 詳細

⚠️ 失智長輩日照適應期

頭 2-4 週可能拒絕去 / 鬧情緒。這是正常的(失智者對新環境敏感)。漸進入園:第 1 週半天 → 第 2 週全天 → 第 3 週起穩定。連續 6 週仍拒絕 → 跟日照討論策略。

😌 喘息服務

📌 一句話

讓主照顧者休息的服務。每年有額度。主照顧者一定要用,不用是浪費。

🏷 3 種喘息

類型地點適用情境
居家喘息家裡(居服員過來)主照顧者出門幾小時
日間喘息日照中心(短期送)主照顧者上班 / 看醫生
機構喘息住宿機構(短住 1-30 天)主照顧者旅行 / 自己住院 / 心力耗竭

📊 額度

  • 失能 2-6 級:21 天 / 年(換算約 32,340 元 / 年)
  • 失能 7-8 級:32 天 / 年(換算約 48,510 元 / 年)
  • 沒用會作廢,不能跨年

⚡ 怎麼預約

  1. 長照 1966 → 提出喘息需求
  2. 照管專員確認額度
  3. 家屬選喘息類型 + 服務提供者
  4. 居家 / 日間 → 較快(1 週);機構 → 要提早 1 個月排

🆘 緊急喘息(主照顧者突然倒下)

主照顧者車禍 / 急病住院 → 馬上撥 1966 申請緊急喘息。失智者可立刻轉到機構式喘息,通常 24-48 小時內安排。這是「我倒下了長輩怎辦」的核心 backup。

💡 主照顧者排喘息的時機

  • 每月固定 1-2 次居家喘息(規律,不靠突發)
  • 每季 1 次日間喘息(讓自己看牙醫 / 健檢 / 跟朋友聚)
  • 每年 1-2 次機構喘息(7 天以上,真的休息)

不要等「我快撐不住」才用 — 預先用,持續性比偶爾大量更可持續。

🌏 外籍看護工(俗稱外勞)

申請流程複雜 + 跟長照給付互斥。多用於中-重度失智 + 主照顧者完全無法自顧者。

📋 申請資格(巴氏量表評估)

  • 長輩需巴氏量表評估(由特約醫院醫師執行)
  • 評分 ≤ 35 分(完全依賴他人)= 符合資格
  • 失智症另有「特定身心障礙項目」可申請,不需巴氏量表
  • 重大傷病卡(失智可拿)有助申請

🔢 申請 SOP(8 步驟)

  1. 長輩到特約醫院做巴氏量表評估(掛號 + 評估費)
  2. 取得評估證明 / 失智診斷(若直接走失智路徑)
  3. 向勞動部申請聘僱許可(可線上)
  4. 取得「招募許可函」(1-2 個月)
  5. 找仲介公司 / 直聘聯合服務中心 — 直聘較便宜但麻煩
  6. 媒合外籍看護工(印尼 / 越南 / 菲律賓 / 泰國)
  7. 外籍看護工入境 + 體檢 + 報到
  8. 開始工作 + 雇主有訓練義務

💰 費用

月薪(雇主付)21,000-25,000 元(法定基本薪資以上)
勞健保 / 就業安定費約 4,000-5,000 / 月
食宿(家裡提供)—
仲介費(若用仲介)初次 60,000-100,000 + 月服務費 1,500-2,000
體檢 / 機票 / 簽證初次 30,000-50,000
總月成本約 30,000-35,000 / 月

⚠️ 重要注意

  • 跟長照給付互斥:有外籍看護工 → 失能 BCS 給付不能領(但可領喘息)
  • 第一次媒合風險高:語言 / 文化差異 / 對失智不熟。3 個月觀察期
  • 休假規定:每 7 天 1 天休假(雇主可選擇加班付雙倍工資)
  • 逃逸風險:仲介公司會處理但仍是主照顧者麻煩
  • 失智專責訓練:好仲介會提供失智照護訓練,要主動問

📚 哪裡查更多

  • 勞動部勞動力發展署 wda.gov.tw
  • 直聘聯合服務中心 0800-085-151(免仲介費)
  • 各縣市勞工局

🏢 住宿型機構

本章只給概要,完整參訪 SOP / 比較 / 紅旗看 home-handbook 長期規劃章節(已有完整覆蓋)。

🏷 機構類型(衛福部分類)

類型主要對象費用範圍
養護型失能但無重大醫療需求30,000-50,000 / 月
長期照護型失能 + 慢性病 + 醫療需求40,000-60,000 / 月
失智專責機構失智長輩(中-重度)50,000-80,000 / 月
護理之家需護理級照護50,000-90,000 / 月

⏰ 何時開始考慮機構

  • 主照顧者出現身體警訊(高血壓 / 心律不整 / 抑鬱)
  • 長輩夜間不睡 / 白天嗜睡導致全家失眠 1 個月以上
  • 長輩大小便失禁無法控管
  • 家中無 24 小時陪伴可能
  • 跟手足討論後同意

💵 預算試算

📋 觀察 / 面試 Checklist

👵 居服員每日觀察(SOP 對照)

  • ☐ 準時到 / 準時離(±10 分鐘)
  • ☐ 完成簽到表 + 工作紀錄
  • ☐ 拍照 / 寫紀錄給家屬
  • ☐ 執行所有「照顧計畫」上列的活動
  • ☐ 用藥「親見入口 + 確認吞下」(不是放著就走)
  • ☐ 餐食按時段(早 / 中 / 晚) + 確認份量
  • ☐ 跟長輩有實際互動(不是滑手機)
  • ☐ 有狀況立即通知家屬(不是等下班才說)

☀️ 日照中心參訪 Checklist

🧩 跨服務組合(怎麼搭配最好)

同一個失智長輩通常需要 2-3 種服務搭配,不是單一服務。

🏷 常見組合範例

📦 組合 A:居家服務 + 喘息(輕-中度)

  • 主照顧者上班:居服員每天上班時段 4 小時(15K 給付夠)
  • 主照顧者偶爾休假:每月 1-2 次居家喘息(週末)
  • 主照顧者旅行:每年 1 次機構喘息 7 天
  • 適用:輕-中度失智 / 主照顧者有工作 / 預算中等

📦 組合 B:日照 + 居家喘息(中度)

  • 白天:日照中心(週一-五,有交通車)
  • 晚上 / 週末:主照顧者顧
  • 主照顧者休假:每月居家喘息
  • 適用:中度失智 / 主照顧者有工作 / 長輩需團體刺激

📦 組合 C:外籍看護工 + 喘息(中-重度)

  • 24 小時:外籍看護工(月成本 30-35K)
  • 看護工休假日:主照顧者 / 親屬
  • 主照顧者休假:可申請喘息(BCS 不能領但喘息可)
  • 適用:中-重度失智 / 主照顧者完全無法自顧 / 預算寬

📦 組合 D:住宿機構(重度)

  • 長期住:失智專責機構
  • 家屬探視:每週 1-2 次
  • 適用:重度失智 / 行為問題嚴重 / 主照顧者已撐不住
  • 看 home-handbook 完整評估

⏰ 演進時程

失智一定到重度。從輕度開始就規劃下一階段:

  1. 確診後 0-1 年(輕度):組合 A(居家+喘息)— 主照顧者保留工作
  2. 1-3 年(中度):組合 B(日照)或進入 C(看護工)
  3. 3 年+(重度):評估 D(機構)或維持 C

  • 1955:外籍勞工申訴專線(免費,24 小時)
  • 113:婦幼保護專線(失智長輩受虐通報)

  • 1925:安心專線(主照顧者心理崩潰)

📋 投訴 SOP

  1. 留證據:照片 / 影片 / 文字紀錄(時間 / 地點 / 細節)
  2. 先跟機構主管反映(口頭 + 書面)
  3. 機構不處理 → 上層主管機關
  4. 嚴重 / 涉及違規 → 直接打 1966 / 衛生局 / 113(留下紀錄)
  5. 必要時換服務提供者(透過 1966 重新媒合)

⚠️ 常見「投訴沒效」的原因

  • 沒留證據 → 變成「你說我說」
  • 跳過機構主管 → 主管機關會先要求機構自處理
  • 太情緒 / 沒重點 → 對方逃避
  • 沒寫書面 → 沒有正式紀錄
