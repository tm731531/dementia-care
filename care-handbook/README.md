## 涵蓋的 5 種服務
1. **居家照顧服務**(居服員)
2. **日間照顧中心**(日照)
3. **喘息服務**(居家 / 日間 / 機構式)
4. **外籍看護工**
5. **住宿機構**(連 home-handbook/long-term)

## 互動工具
- 🧙 **Wizard**(3 個問題 → 個人化推薦)
- 💵 **預算試算**(失能等級 + 收入 → 各服務實付)
- 📋 **觀察 / 面試 Checklist**

## 為什麼做這個
親情無契約 — 親人接手照護幾乎都失敗。**手冊真正用途是給「有付錢的服務提供者」拿來做 SOP 對照工具**。care-handbook 在「找的階段」做出選對人選對機構的決策工具。
