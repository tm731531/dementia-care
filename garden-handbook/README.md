# 🌱 家庭園藝手冊

給在陽台、窗邊、室內小空間想種點東西的人。**涵蓋可食植物 + 觀葉 + 觀花 + 根莖類,共 25 種**。

**上位 mission**:[dementia-care repo](https://github.com/tm731531/dementia-care) 的「**讓整個家忙起來**」策略 — 用可食/觀賞植物讓家裡有**生長中的東西、需要每天互動的動作、可以採來吃的成果**,特別對有失智長輩的家庭,種植活動是 behavior redirection 的有效工具。

## 🎯 這本手冊能幫你做什麼

### 1. 看一個 plant 就知道全貌 — **5 秒總覽(5W1H)**
每個 plant 頁頂會自動顯示 6 行摘要:
- 👥 Who:誰適合種(新手/兒童/失智級別)
- 🌱 What:要買什麼(具體數量/規格)
- 📅 When:什麼時候種(月份 + 節氣)
- 📍 Where:哪裡買 / 哪裡種
- 💡 Why:一句話總結好處
- 🔧 How:大致流程

下面還有「適合情境」tag badges(寵物勿食、兒童可參與、花粉過敏、需深盆等)。

### 2. 每個 plant 10 個 section 完整 SOP
1. 💡 **why** — 為什麼值得種
2. 🛒 **obtain** — 取得方式(含 **📦 新手包 · 三步照順序** + **⚠️ Step 0 月份檢查**)
3. 🏺 **pot-soil** — 盆器與土壤
4. 🌳 **steps** — 種植步驟
5. 📋 **timing** — **完整時程表** · 月份 × 節氣雙軌警告 + Day-by-Day SOP
6. 🌾 **fertilizer** — 施肥 · 固體肥優先 · **盆邊每 10cm = 3 粒**速查
7. ✂️ **harvest** — 採收(or 觀葉的 🌱 繁殖與分株)
8. 🛡 **troubleshoot** — 失敗點與除錯
9. 🧠 **dementia** — **失智照護應用 Notes**(輕度/中度/重度三層,跟設定頁連動)
10. 📓 **log** — 實測紀錄

### 3. 🎯 規劃模式(設定頁開關)
- **關閉**(預設):看全部 25 plants — 探索/規劃明年
- **開啟**:首頁只顯示「**現在 + 未來 2 個月能下手** + ⭐ 你的最愛」
- 月份過了自動進出,不用記
- 已種的植物透過 ⭐ 最愛保留可見

### 4. 🛒 購物籃 · 一趟菜市場搞定
想同時種多個 plants?加進購物籃後自動合併:
- 📦 **按來源分組**:🏠 家裡 / 📦 蝦皮 / 🌷 花市 / 🏪 全聯 / 🥬 市場
- 🧮 **聰明合併**:培養土加總 · 肥料共用 1 包 · 工具 dedup
- 🗓 **採購 → 種植 → 採收 建議流程**:Day 0 備土 → Day 3 種 → Day 30 施肥 → Day 30+ 採
- 🔧 **基本工具組清單**:鏟子/耙子/澆水壺/手套/剪刀
- 📋 **匯出純文字** → 貼 LINE 或記事本列印

### 5. 📚 通用園藝知識頁
跨植物共通的東西:
- 🏺 挑盆器的 5 原則
- 💧 澆水:免洗筷檢測濕度
- ✂️ 修剪:什麼時候剪、怎麼剪
- ☀️ 陽光/空氣/水 三元素
- 🌾 **施肥**:SOP 化、固體肥優先、缺肥 vs 過肥對照表
- 📅 **24 節氣**:比月曆更準的種植指南(清明開工、立冬停工)

## 👥 誰適合看這本手冊

1. **照顧失智長輩的家庭**
   - 可食植物讓長輩「**每天有事做**」(高維護的地瓜葉/香草類)
   - 觀葉植物是「**照顧者累到不想顧時的後援**」(低維護的虎尾蘭/多肉)
   - 失智照護 notes **依級數連動**(輕度能自主 / 中度單步 / 重度感官陪伴)
2. **想在家種點東西的新手** — 沒經驗、沒時間、怕失敗 → 每個 plant 都有「新手包」
3. **想給小孩看植物長大的爸媽** — 向日葵 / 草莓 / 紅蘿蔔 都是兒童超愛
4. **忙到沒空顧植物但想要綠意** — 多肉 / 虎尾蘭 / 黃金葛 幾乎不用照顧

## 💾 本機資料(localStorage)

手冊不會上傳任何資料到雲端。存在你裝置的:
- ⭐ 最愛植物清單(規劃模式會優先顯示)
- 🛒 購物籃(跨 session 保留)
- 📝 每個植物的 memo(時間戳記錄)
- 💚 失智照護級數(輕度/中度/重度)
- 🎯 規劃模式 on/off

支援**匯出/匯入 JSON 備份**(⚙️ 設定頁),換裝置不丟資料。

Storage key: `gardenHandbookState`

## 🧰 技術

- **純前端單檔**:`index.html` 一個檔案搞定,無 framework、無 build step
- **離線可用**:所有資源 inline(SVG favicon、base64 圖片、字型系統 fallback)
- **零外部 CDN 依賴**:不連 Google Fonts、不連 analytics、一切請求都來自你自己的裝置
- **Emoji 相容性**:所有 plant emoji 只用 Unicode ≤ 10.0(2017 前),確保舊平板/舊手機也能顯示
