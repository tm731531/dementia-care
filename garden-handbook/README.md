# 🌱 家庭可食植物手冊

一本**家庭可食植物種植手冊**,給在陽台、窗邊、室內小空間想種點東西的人。

## 這是什麼

每個植物一個 entry,包含:

- 🪴 取得方式(扦插 / 種子 / 塊莖)
- 📋 種植步驟
- ⏳ 等待時間
- ☀️ 光線需求 / 適合地點
- 🌿 採收方式
- 🐛 失敗點與除錯

## 誰適合看

1. **想在家種點東西但是完全新手** —— 沒經驗、沒時間、怕失敗
2. **照顧失智長輩的家庭** —— 每個 entry 都附「失智照護應用 notes」,教你怎麼把種植變成長輩的 behavior redirection 療法
3. **想給小孩看植物長大的爸媽**

## 上位 mission:讓整個家忙起來

這本手冊是 [dementia-care repo](https://github.com/tm731531/dementia-care) 「讓整個家忙起來」策略的一部分。

家裡有東西在生長、有事要每天做、有成果可以收 —— 這不只是種菜,**是把家變成活的**。對失智長輩特別重要:

- **每天都有合理的理由去陽台**(如果植物耗水,例如地瓜葉)
- **採回來可以讓她剝老葉** → 延續「剝蒜頭工作站」的肌肉記憶路線
- **植物真的需要她,不是在應付她**

詳見 Tom 的 blog:
- [失智照顧半年的三個誤會](https://blog.tomting.com/2026/04/22/dementia-care-three-misunderstandings/)
- [失智長輩為什麼一直到處走](https://blog.tomting.com/2026/04/22/dementia-mother-finding-home/)
