# 🌱 家庭園藝手冊

一本**家庭園藝手冊**,給在陽台、窗邊、室內小空間想種點東西的人。**涵蓋可食植物 + 觀葉植物**。

## 這是什麼

每個植物一個 entry,包含:

- 🪴 取得方式
- 🪣 **盆器與土壤**(per-plant,每個植物不同 —— 黑土/紅土/砂土/培養土各有適合對象,追肥也不同)
- 📋 種植步驟
- ⏳ 等待時間 / ✂️ 採收或修剪
- 🐛 失敗點與除錯
- 🧠 **失智照護應用 Notes(3 級:輕度 / 中度 / 重度)** — 每個植物對不同照護階段有不同建議

還有一個獨立的 **📚 通用園藝知識** 頁,放跨植物共通的東西:盆器 5 原則、免洗筷檢測濕度、修剪時機、苔球法、懶人瓶。

## 誰適合看

1. **想在家種點東西的新手** —— 沒經驗、沒時間、怕失敗
2. **照顧失智長輩的家庭** —— 可食植物讓長輩「每天有事做」,觀葉植物是「照顧者累到不想顧時的後援」
3. **想給小孩看植物長大的爸媽** —— 水耕黃金葛就是家人共同儀式的好起點

## 上位 mission:讓整個家忙起來

這本手冊是 [dementia-care repo](https://github.com/tm731531/dementia-care)「讓整個家忙起來」策略的一部分。

這個 mission 底下有兩層:

- **High-maintenance 植物**(地瓜葉、香草類)→ 讓長輩**天天有事做**
- **Low-maintenance 植物**(虎尾蘭、黃金葛類)→ 給**照顧者的喘息後援**

兩層都服務同一個 mission,只是服務不同的人(可食服務長輩,觀葉服務照顧者自己)。

詳見相關 blog:
- [失智照顧半年的三個誤會](https://blog.tomting.com/2026/04/22/dementia-care-three-misunderstandings/)
- [失智長輩為什麼一直到處走](https://blog.tomting.com/2026/04/22/dementia-mother-finding-home/)

## 本機資料(localStorage)

手冊會把你的**最愛植物、memo 紀錄、照護級數**存在你裝置的 localStorage,**不會上傳到任何地方**。

- ⭐ 可隨時加入/取消最愛
- 📝 每個植物可寫多筆 memo(時間戳記錄)
- 💚 設定頁可切換照護級數(輕度/中度/重度)—— 失智應用建議會依此變
- 📥 可匯出備份(JSON),換裝置時可匯入還原

Storage key: `gardenHandbookState`
