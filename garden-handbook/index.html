照顧失智長輩的家庭特別適合種這些植物。每天澆水、採摘、剝老葉這些動作,是失智長輩保留「還有用」感覺的重要 ritual。手冊會告訴你怎麼把種植變成長輩的日常 behavior redirection。

上位 mission:讓整個家忙起來

這本手冊是 dementia-care repo 的一部分。這個 repo 的 mission 是「讓整個家忙起來」—— 用可食植物、結構化記錄、陪伴工具,讓有長輩、小孩的家,每天都有東西在發生、有事要做、有成果可以收。

家忙了,人就不會一直躺著。這比任何「治療」都更根本。

🍃 地瓜葉

難度很低(新手友善)
季節春夏最旺(4–9 月)
光線半日照以上
地點陽台 / 窗邊 / 戶外

為什麼新手適合從地瓜葉開始

  • 幾乎怎麼種都活 — 扦插成功率極高,是最皮厚的可食植物之一
  • 長得快 — 夏天 3–5 天就可以採一把
  • 一次買,吃整季 — 採收後不斷冒新葉,可以一直吃到秋冬
  • 莖葉都可食 — 不像某些菜只能吃葉子,地瓜葉連梗都能炒
  • 吃水量大 — 夏天要每天澆,所以每天都有理由去陽台(對失智照護特別重要,下面詳說)

取得方式

✅ 推薦:扦插(新手首選)

  • 網購枝條:搜「地瓜葉扦插」,一批通常 20 條左右,便宜
  • 市場買帶莖的地瓜葉:買一把回來,吃掉葉子,留下帶節的莖直接拿來插
  • 成功率 80% 以上。運送損耗高(買來的枝條常看起來半死),但只要有 1 條活就夠

⚠️ 不推薦新手:地瓜塊莖發芽

超市買的地瓜常被噴抑芽劑,可能一直不發芽。要泡水、要溫度、要耐心。本手冊作者 Tom 在 2026 年 2 月試了兩個月,全部失敗,才改用扦插。

如果你真的想試塊莖,建議到市場買非連鎖的農家地瓜,成功率高些。但如果目標是「種出可吃的葉子」而不是「挑戰塊莖」,請直接買枝條。

種植步驟

  1. 選盆 — 直徑 25 公分以上、深 20 公分以上。越大越好(地瓜葉根系會擴散)。盆底一定要有排水孔。
  2. 準備土 — 一般培養土就可以。講究的人可以先養土(加有機肥,擺幾天讓土活化)。
  3. 剪枝條 — 每條留 3–4 個節點(節點 = 葉子長出來的位置)。頂端葉子留 2–3 片,下面的葉子剝掉(避免蒸散水分)。
  4. 插進土 — 下方 2–3 個節點埋進土,露出 1–2 片葉子在外面。
  5. 第一次澆水澆透 — 從上澆到盆底有水流出。之後保持濕潤(不要積水)。
  6. 放在半日照以上的地方 — 斜照 OK,全日照更好。第一週如果太曬可以先放散光處,等發根再移到光好的地方。

等待時間

階段時間
插下去 → 冒第一片新葉1–2 週
冒新葉 → 首次可採收4–6 週
持續採收期整個夏天,可採到秋冬降溫

採收

  • 採 頂端 5–10 公分的嫩葉莖
  • 留根部與一部分莖 → 會從節點繼續長
  • 越採越多 — 定期採收反而刺激生長,放著不採才長得慢
  • 建議一週至少採 1–2 次,讓植株持續活躍

🧠 失智照護應用 Notes

地瓜葉對失智長輩是**極佳**的 behavior redirection 工具,因為它的特性剛好打中幾個痛點:

為什麼地瓜葉特別適合失智長輩

  • 耗水量大 → 每天需要澆水 → 長輩每天有合理的理由去陽台(如果她原本就有想往外走的衝動,這會把能量導去陽台)
  • 視覺回饋快 → 夏天幾天就有明顯新葉,她看到自己澆的有成果
  • 莖葉可剝 → 採回來可以請她剝老葉、去硬梗。這個動作跟「剝蒜頭」是同類型肌肉記憶,有廚房經驗的長輩會自然做
  • 讓她感覺自己有用 → 植物真的需要她,不是在應付她。這比任何「活動」都更根本

Setup 建議(如果家裡有失智長輩)

  • 放在長輩居住層的陽台(讓她隨時看得到、摸得到)
  • 水壺事先裝滿放盆旁 —— 不要讓她自己接水(她可能找不到水龍頭)
  • 把澆水加進她的日常 SOP —— 例如早餐後、午睡前,固定時段執行
  • 不要讓她看到你先澆 —— 如果她覺得你已經澆過,她就不會動
  • 採收後給她處理 —— 剝老葉、去硬梗、分菜裝袋,都是可做的「前置動作」

🤝 怎麼貢獻

這本手冊是 open source,歡迎你 fork 加自己家實測過的植物:

  1. Fork dementia-care repo
  2. 在 garden-handbook/index.html 加一個新的 <section class="plant-entry">
  3. 參考地瓜葉那個 entry 的結構,但欄位不強制 —— 你的植物不適合「失敗點表格」就不要放,但「失智照護應用 Notes」是本手冊的特色,盡量補
  4. 發 PR 過來,標題 feat(garden): 新增 X 植物 entry

不收商業連結、不收不可食植物、不收農藥推薦。主軸是「家庭可食 + 長輩照護友善」。
