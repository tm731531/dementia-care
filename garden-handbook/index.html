通用園藝知識

這裡放的是跨植物共通的知識。每個植物自己的盆器/土壤/追肥在該植物頁。

🏺 挑盆器的 5 原則

植物跟盆器有「黃金比例」大約是 1 : 1.68。盆器高度是 1,植物高度約 1.68 倍,最好看。

  1. 淺盆不配高樹
  2. 闊葉可搭大盆,方圓皆可
  3. 小盆換新盆,挑比原本大一號的
  4. 換盆看樹徑:樹徑高大,盆器也要相對大
  5. 不懂配色:挑水泥盆、白盆、北歐風花盆(百搭)

⚠️ 每個植物該用什麼土、多大盆、需不需要追肥(鉀/鈣/鎂肥)都不一樣

黑土、紅土、砂土、椰纖土、仙人掌土……各有適合對象。具體請看該植物的「盆器與土壤」section。

💧 澆水:免洗筷檢測濕度

澆不澆水,不要只看表土 —— 表土會被風吹日曬弄乾,裡面其實還濕。

方法:拿一根免洗筷插進土裡深一點,拔出來看:

  • 筷子沾黏泥土 → 裡面還濕,不用澆
  • 筷子乾淨 → 乾了,該澆

這是最簡單又準確的檢測方式。

✂️ 修剪:什麼時候剪、怎麼剪

「植物愈修愈美麗」,但時機很重要。

  • 涼爽時才修剪(原則) —— 太熱太冷都是壓力,剪了容易生病
  • 春、秋天適合大整形
  • 其他涼爽日子可以微整形(剪長的、剪亂的、剪太高的)
🌱 懶人照顧:苔球法 + 懶人瓶

苔球植物

把小盆栽整坨連土包進青苔片,微塑成圓形,用縫衣線綁固定。乾了浸水盆 1-2 分鐘即可。

適合植物:山蘇花、銀點秋海棠、旺旺樹、鐵線蕨 等耐陰、適合室內半日照的植物。

⚠️ 苔球不能直接日照

會變黃。放在光線好但不直曬的位置。

懶人瓶

利用虹吸原理設計的花器:

  • 下方是水,上面花盆底部預先置入吸水麻繩/棉片
  • 土乾了自動吸水,不乾就不吸
  • 看水位就知道該不該補
☀️ 三元素:陽光、空氣、水

陽光

  • 室內植物假日建議出去曬太陽
  • 但久居室內的植栽別突然搬出去大太陽下,會枯萎
  • 過渡期:先放靠窗邊、矮牆附近,間接日照

空氣

通風不良 → 病蟲害機會高。陽台、窗邊、有空氣流動的位置都比密閉角落好。

水

不要過度澆水。看上面的「免洗筷檢測」section。

🌾 施肥:缺肥 vs 過肥對照表

為什麼盆栽一定要施肥

地種的植物根系能擴展找養分,盆栽只有盆裡那一點土。澆水會慢慢把養分沖走,兩個月後土就「空了」。不施肥 = 植物活著但長不好:葉小、色淡、採量越來越少。

通用時機規則

  • 剛種下 / 剛換盆 → 不施(根怕燒)
  • 冒新葉、植物穩住 → 開始施
  • 生長期 → 規律施(夏天葉菜 7–10 天一次液肥;觀葉植物 2–4 週一次)
  • 冬季 / 太熱 / 太冷 → 休眠不施

⚠️ 缺肥 vs 過肥 — 兩者症狀很像,新手最容易誤判

看到葉黃就「再補肥」,結果其實是過肥在惡化。一定要分辨:

缺肥過肥
葉色均勻淡/黃,從老葉開始葉緣焦黃、捲曲,像燙傷
新葉越長越小異常肥大、顏色過深
土表正常白/灰色結晶(鹽積)
生長慢、僵看似旺但根可能已爛
補救補薄肥 + 有機質停肥 + 大量清水沖盆 3–5 次,嚴重換土

每種植物自己的缺肥/過肥具體症狀,看該植物頁的「🌾 施肥」section。

肥料三大類

  • 化學液肥(花寶 1/2/3 號):便宜有效,新手首選。稀釋後澆
  • 化學顆粒 / 緩釋肥(好康多、魔肥):放一次吃幾個月,懶人友善
  • 有機肥(魚精、豆餅、雞糞顆粒、堆肥):味道重但會養土,長期土質越來越好

NPK 三元素速查

元素功能哪種植物要重
N 氮長葉葉菜、觀葉 → 20-20-20 或 30-10-10
P 磷開花結果果菜、花卉 → 10-30-20
K 鉀根莖健壯、抗病根莖類、老化期 → 10-10-30

市售「花寶 1 號」= 20-20-20 均衡通用款,買一罐新手都夠用。

📌 施肥通則(不管什麼植物)

  • 新手一律包裝稀釋濃度再減半 — 濃了會燒,薄了再加就好
  • 澆根部土面,不澆葉心(葉心積水會爛)
  • 早晚施,不在大太陽下施(水分蒸發快,肥集中燒根)
  • 施肥後澆一次清水,沖掉葉面或盆底殘留
