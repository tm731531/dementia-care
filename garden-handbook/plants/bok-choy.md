# 🥬 青江菜 家庭種植指南

> 類別:**葉菜** · 難度:低-中(生長快、怕熱) · 季節:秋、冬、春(夏天抽苔)

## 🎯 5 秒總覽(5W1H)

- **👥 Who**:新手友善 · 兒童可參與
- **🌱 What**:青江菜種子 1 包 · 長條盆 50cm 或 淺盆 · 培養土 5L

## 🛒 購物清單

- **繁殖材料**:青江菜種子 1 包 (花市, NT$30–50)
- **盆器**:長條盆 50cm 或 淺盆 × 1 個 (NT$150–250)

- **肥料**:好康多 1 號 350g (多盆共用)

## 💡 為什麼值得從青江菜開始

- **台灣最家常的菜** — 幾乎每個人都吃過青江菜,採下來直接煮,成就感最直接

- **生長期短,reward 迴路快** — 播種後 25–30 天可摘外葉,40–60 天全株採收,新手不用等太久

- **錯開播種,連吃好幾個月** — 每隔 2–3 週播一批,9 月 – 隔年 3 月連續有菜

- **種子便宜、播種門檻低** — 一包種子幾十元可種整季,不需要扦插枝條,純撒種

- **外葉採、心葉繼續長** — 不像蘿蔔要等整株,可以邊吃邊讓它長

- **每天都有理由去陽台** — 疏苗、澆水、摘外葉,每個動作都簡單、安全,適合帶著長輩一起做

## 🛒 取得方式

> 

**🗓️ Step 0 · 先看月曆再動手**

**現在要是 9 月 – 隔年 3 月才開始播種。** 5–7 月(立夏-大暑)播種**整批抽苔開花** — 氣溫超過 25°C 青江菜進入抽苔模式,葉梗變老、苦澀不好吃,等於白種。如果現在是夏天,**等到白露(9 月上旬)後**再下手,現在就先備盆土。

> 

**📦 推薦的新手包 · 三步照順序來**

- **先備盆與土(這一步要比種子先到)** — 家附近花市買 **長條盆 40–50cm × 深 15–20cm + 培養土**(合計 NT$150–200),搬回家放好定位、土填 8 分滿、先澆一次水讓土服貼,加一把基肥有機質拌入土中更好。

- **再買種子** — 花市種子架或蝦皮搜尋「青江菜 種子」(一包約 NT$30–60,可種整季多批)

- **備齊當天立刻播種** — 種子均勻撒在土表 → 蓋薄土 0.5cm → 澆透水 → 放半日照以上

⚠️ **順序不能顛倒**:種子到了土還沒準備好,隨便擺一天發芽率就降。總預算 **NT$250 有找**,不用懂園藝 —— 但**月份要對**(見上面 Step 0)。

**其他取得管道**

| | 管道 | | 優點 | | 適合誰 | |

| | 花市種子架 | | 當場看包裝、可挑品種 | | 第一次種,想確認新鮮度 | |

| | 蝦皮/露天 | | 品種多、評價可參考 | | 想試台灣本土選育品種 | |

| | 花市幼苗 | | 省發芽等待期 | | 懶得等、或季節已晚 | |

## 🏺 盆器與土壤

**盆器**

- **長條盆(40–50cm)或淺盆** — 青江菜根系淺(約 15cm),不需要深盆

- 深度 **15–20 cm 足夠**

- 盆底**必須有排水孔**(積水直接爛根)

- 寬一點比深一點重要 — 同一盆可以種 4–6 株、拉開間距

**土壤**

- 一般**培養土即可**(市售蔬菜用培養土更好)

- 青江菜需氮多(葉菜),播種前可拌入一把有機質基肥(雞糞有機肥、骨粉或堆肥)

- 排水要好 — 黏土盆底可墊一層珍珠石避免積水

*施肥細節(時機、劑量、好康多用量)獨立成下面的「🌾 施肥」section。*

## 🌳 種植步驟

- **備盆填土** — 見上面「盆器與土壤」;土填 8 分滿,先澆一次讓土服貼

- **撒種** — 種子均勻撒在土表,間距 5–8 cm;或直接撒密一點,等發芽後疏苗

- **蓋薄土** — 撒完蓋 0.5 cm 細土,用手輕壓讓種子接觸土壤

- **澆透水** — 第一次澆到盆底出水,之後保持土表微濕(不積水)

- **放半日照以上** — 斜照 OK;直射更好;陰暗處葉梗會瘦弱

> 

**✂️ 疏苗 — 最關鍵的動作(不疏全部瘦弱)**

青江菜**一定要疏苗**。種得太密,全部搶陽光搶養分,每株都長不大。

- **第一次疏苗(Day 10)**:長出 2 片本葉時,每叢只留 1 株最強壯的,間距拉到 **3 cm**

- **第二次疏苗(Day 20)**:長出 4–5 片本葉時,間距再拉到 **10 cm**

- 疏掉的幼苗不用丟 — 可以移植到另一個盆,或直接吃掉(嫩苗很甜)

## 📋 完整時程表 · 播種到採收

**播種 → 發芽 → 疏苗 → 施肥 → 採收**,一張表涵蓋整個生長週期。**照做就對,不用觀察植物狀態**。

> 

**⚠️ 開工前先看月曆 / 節氣:挑對時間,不然全部抽苔**

青江菜**喜涼怕熱**,氣溫超過 25°C 就會抽苔開花 — 梗變硬苦澀,不好吃。**節氣比月份準**,兩套都列:

| | | | 月份 | | 節氣 | | 狀態 | |

| | ✅ 最佳 | | **9 – 3 月** | | **白露(9/8)→ 春分(3/20)** | | 涼爽,生長旺,**手冊所有時程以此為準** | |

| | 🟡 勉強 | | 8 月 / 4 月 | | 立秋(8/7) / 清明(4/4) | | 邊緣期,抽苔風險高,要選耐熱品種 | |

| | ❌ 絕對別種 | | **5 – 7 月** | | **立夏(5/5)→ 大暑(7/22)** | | 整批抽苔開花,等於白種 | |

如果現在是夏天,**不要播種** —— 先備盆土,等白露(9 月上旬)再下手。

## 🌾 施肥

**本 handbook 預設固體緩釋肥,不推液肥給新手** —— 不用稀釋、不用算濃度、撒一次吃整季、難燒根。青江菜是葉菜,**氮肥需求大**,但生長期短,一次施夠就行。

*施肥時機(Day 21)已經統整在上面「📋 完整時程表」,這個 section 專講 **用多少 · 用哪款 · 怎麼撒**。*

> 

**📐 不同盆大小 · 速查(好康多 1 號)**

| | 盆 | | 顆數 | |

| | 3 吋圓盆(Ø9cm) | | **3 粒** | |

| | 5 吋圓盆(Ø15cm) | | **5 粒** | |

| | 7 吋圓盆(Ø21cm) | | **8 粒** | |

| | **長方盆 50×20cm** | | **15 粒** | |

| | 大陽台盆槽 60cm+ | | **18–20 粒** | |

**原則**:寧可比這個少,不可比這個多。青江菜生長期才 40–60 天,**一次施夠就行,不用施第二次**。

**🛒 買哪一款**

- **好康多 1 號**(NPK 14-11-13)— 萬用款,葉菜/觀葉都適用

- 350g 小包約 NT$100,一包吃多盆多季

- 全聯園藝區、花市、iGarden 網購都有

- 替代品:魔肥、奧綠肥(類似的緩釋顆粒肥都 OK)

> 

**📱 記憶外包 — 播種當天把施肥日期設好**

不要依賴記憶、不要依賴觀察植物。播種當天就打開 Google 行事曆:

- 播種日 + 21 天 → 「撒 15 粒好康多(澆水)」

**決定一次做完,腦袋騰出來做別的事**。這是照護者 handbook 的核心哲學 —— 決定外包到行事曆/便利貼,不是記在腦裡。

**🧪 進階:想用液肥的路線(不建議新手)**

如果家裡已經有液肥、或想追求更快的葉色深綠:

- **花寶 1 號**(NPK 20-20-20),包裝稀釋比**再減半**(防燒)

- 疏苗後 3–5 天第一次,之後每 10 天一次澆根部

- 早晚施,不在中午大太陽下施;施完澆一次清水沖殘留

- 有機派:魚精水、豆餅水(黃豆泡水發酵 1 週)

## ✂️ 採收

**兩種採收方式**

- **外葉陸續採(Day 25–30 開始)** — 從最外圈摘 2–3 片老葉,讓心葉繼續展開;可延長採收期 1–2 週

- **全株採收(Day 40–60)** — 整株從基部剪斷或連根拔起;一次收清,盆子空出來再播下一批

**連續收穫策略**

- 每隔 2–3 週錯開播一批 — 9 月 – 隔年 3 月可連續 3–4 個月都有菜

- 一批全株採完 → 換掉 1/3 的舊土補充新培養土 → 立刻播下一批

- 採下來的外葉最好**當天炒**(青江菜放冰箱 2–3 天葉梗就軟)

> 

**🥬 採收時機判斷**

看菜梗基部直徑:

- **梗基部直徑 1.5 cm 以上**:外葉可採

- **整株 10–15 cm 高**:全株採收最佳時機

- **花苞冒出前趕快採**:一旦開始抽苔,葉梗變老苦澀,要立刻採收不要再等

## 🛡 失敗點與除錯

| | 症狀 | | 可能原因 | | 對策 | |

| | 葉背有小蟲、葉捲曲 | | 蚜蟲(常見於春秋) | | 強水柱沖葉背;手抓;避農藥(要吃的菜) | |

| | 葉片有孔洞、缺口 | | 菜蟲(紋白蝶幼蟲) | | 翻葉背找蟲、手抓;套網避免蝴蝶產卵 | |

| | 抽苔開花,梗變老 | | 氣溫超過 25°C、已過季 | | 趕快採收,苔花可食(水燙);避開 5–7 月種 | |

| | 葉梗細、葉片小 | | 疏苗不夠、太擠 | | 果斷拔掉多餘株,維持 10cm 間距 | |

| | 葉子發黃(整株) | | 缺氮肥或缺水 | | 確認土濕度;若已過 Day 21 補撒 5 粒好康多 | |

| | 根部爛黑 | | 積水、排水孔堵塞 | | 盆底墊珍珠石;減少澆水頻率 | |

| | 種子撒下去沒發芽 | | 種子太舊/覆土太厚/太乾 | | 覆土不超過 0.5cm;保持土表微濕但不積水;換新種子 | |

## 🧠 失智照護應用 Notes

### 輕度

> 

**📍 輕度:熟悉的家常菜,她還能主導**

青江菜是台灣最家常的菜,幾乎每個人都有幫廚記憶。她還能理解多步驟指令、自主走去陽台執行動作。

- 你口頭觸發就好:「今天來看看菜長多大了」、「外葉可以摘了,晚餐加一道」

- 她自己拔外葉、洗菜、擇菜、剝老葉 — 這些是幾十年的肌肉記憶,不需要重新教

- **「這是我種的」的價值感最重要** — 炒菜前讓她知道這是她種的

- 疏苗動作可以讓她決定:「這兩株哪個留?」— 給她做決定的機會

- **目標**:最小化你需要介入的動作。她的自主性最珍貴,不要剝奪

### 中度

> 

**📍 中度:你拔她揀,每步要你引導**

她能執行單一動作,但「走去陽台 + 找工具 + 判斷哪片採 + 採完回來」這種多步驟會卡住。

- 你去陽台摘好菜,放到她面前的洗菜盆裡 — 不要讓她自己找水龍頭

- 言語指令**單一動作一次**:「幫我把這片葉子剝掉」,不是「先洗再擇再切根」

- **「把爛的挑出來」** — 這個判斷動作簡單、清晰,失智早中期還能做,且有成就感

- 分揀、剝老葉、撕菜梗絲 — 這個動作跟剝蒜頭、剝地瓜葉是同類型,身體記憶殘留久

- 你切根洗淨,她做手的動作 — 你負責刀/火/水,她負責揀選

- 青江菜綠葉白梗對比強,視覺辨識容易,不需要猜哪邊是哪邊

### 重度

> 

**📍 重度:不要求動作,要求存在與感官**

不要求她完成任何動作。青江菜的價值從「活動」轉為**感官陪伴**。

- 帶她坐在盆旁邊,你澆給她看;把一片葉子放到她手上,讓她握著

- 青江菜**綠葉 × 白梗**對比強烈 — 視覺刺激好,不需要解說也能感受

- 洗菜聲音、葉子的脆聲 — 聲音刺激;讓她聞洗淨青菜的氣味

- 煮青江菜湯時讓她坐在旁邊,熟悉的食物香氣(嗅覺刺激)

- 吃的時候切細,熟悉的家常味道在嘴裡 — 這個比「她有沒有做」更珍貴

- 每天重複同樣時段、同樣路徑,**節奏感比「完成度」重要**

這個階段的價值:**她不需要「做」,她需要「在」這個熟悉的家常場景裡**。
