## 💡 為什麼薑是家庭必種的根莖類

- **廚房零廢棄起步** — 發了芽眼的廚房老薑直接種,成本幾乎為零

- **兩用:採葉 + 採塊** — 種植期間可剪嫩薑葉炒菜,秋末再挖出薑塊

- **嗅覺記憶超強** — 薑的辛香氣是台灣家庭廚房最深刻的氣味之一,對失智長輩有強烈喚起效果

- **挖薑像尋寶** — 秋末把整盆土挖開,一塊一塊找薑塊,是動手感最強的收穫儀式

- **用途廣泛** — 薑茶、煮雞湯、滷肉、薑汁撞奶,採回來馬上能用

- **越冬再發** — 冬天地上葉枯,但塊莖在土裡安全過冬,隔年春天自動再發,一次種多年

## 🛒 取得方式

> 

**🗓️ Step 0 · 先看月曆再動手**

**現在要是 3 月底(春分後)到 4 月中旬再開始種。** 薑塊需要地溫穩定在 **18°C 以上**才能發芽,冬天(11 月 – 2 月)種下去基本原地不動或爛掉。如果現在是冬天,**等到隔年 3 月底(春分)後**再下手 — 現在就先備深盆與土、讓薑塊在室內催芽。

**今年(2026)時間窗**:3/20 春分 → 4/4 清明最佳;清明後到立夏(5/5)勉強;立夏之後種下去當年採塊量會少很多。

> 

**📦 推薦的新手包 · 三步照順序來**

- **先備深盆與土(這一步要比薑塊先到)** — 花市買 **直徑 30 cm 以上、深至少 30 cm** 的塑膠盆(深是關鍵,薑塊往下長) + 一般培養土混少量粗砂,合計 NT$200–300。搬回家放好定位,土填 6 分滿備用。

- **取薑塊** — 最快:冰箱/廚房找發了芽眼的老薑;或市場買帶皮老薑(生機超市、傳統市場)NT$50–80/斤;或蝦皮搜尋「老薑 種薑」NT$100–150。**確認有白色或綠色芽眼才種得起來**。

- **催芽後種下** — 薑塊泡溫水 12 小時 → 濕紗布包住放陰涼處 3–5 天見白芽 → 芽眼朝上埋進盆,覆土 5–10 cm → 澆透

⚠️ **順序不能顛倒**:薑塊催芽後要馬上種,不能讓嫩芽在袋子裡悶。總預算 **NT$400 有找**,不含採塊前的漫長等待 —— 但這等待是「放著不管它長」的那種。

**取種來源比較**

| | 來源 | | 優點 | | 注意 | |

| | **廚房老薑** | | 免費、現成 | | 超市薑可能噴抑芽劑,確認有芽眼再種 | |

| | **傳統市場老薑** | | 多半未噴藥、芽眼大 | | 挑表皮皺、有突出芽眼的塊 | |

| | **生機超市有機薑** | | 無農藥疑慮 | | 價格較高 NT$80–120/斤 | |

| | **蝦皮「種薑」** | | 專門賣種用,芽眼多 | | 春季才大量有貨,冬天很少 | |

> 

**⚠️ 超市薑抑芽劑問題**

部分超市進口薑會噴「氯苯胺靈(CIPC)」抑芽劑延長保存。症狀:催芽兩週完全沒反應。如果失敗,換傳統市場或有機薑重試。老薑(薑皮厚、纖維多)比嫩薑更容易發芽。

## 🏺 盆器與土壤

**盆器 — 深是最重要的**

- **深度至少 30 公分**(這比直徑更重要,薑塊橫向膨大往下擠)

- 直徑建議 **30 公分以上**,一個薑塊需要一定空間水平伸展

- 如果有「長方型陽台槽」(深 30 cm,長 60–80 cm)是最理想的 — 可以種 2–3 個薑塊排列

- 盆底**必須有排水孔**,薑怕積水爛根

- 陶盆排水好但重(移動時注意);塑膠盆輕,夏天土乾快要多澆

**土壤**

- **排水良好**是首要條件 — 積水會爛薑塊,比缺水更危險

- 一般培養土 7 份 + 粗砂或珍珠石 3 份混合(增加透氣性)

- 有機質豐富的土更好 — 可加入少量有機肥(雞糞堆肥、腐葉土)先養土

- 避免黏重黑土(容易積水)

- 種前盆底鋪一層小石頭或碎磚塊,幫助排水

## 🌳 種植步驟

- **選薑塊** — 找有 **芽眼**(白色或淡綠色突起)的老薑塊,一塊 50–100g 就夠。可切塊,每塊保留至少 1–2 個芽眼,切口晾乾半天讓傷口結痂(防腐爛)。

- **催芽** — 薑塊泡溫水(40°C 左右)12 小時 → 撈出擦乾 → 用濕紙巾或濕紗布包住 → 放室內溫暖處(25°C)3–5 天直到見白芽冒出。這步驟讓發芽更快更整齊。

- **備盆** — 深盆填土至 **6 分滿**(留出 5–10 cm 的覆土空間 + 後期培土空間)

- **種下** — 薑塊**芽眼朝上**平放在土面,覆土 5–10 cm。不要埋太深,否則發芽更慢。多塊種植保持 15–20 cm 間距。

- **第一次澆水澆透** — 澆到盆底有水流出,之後等土面 3–4 cm 乾再澆(薑怕積水)

- **放半日照處** — 發芽前可稍陰,出芽後移到明亮散光或半日照位置。**避免夏天正午直射**(葉會曬傷變黃)

- **培土** — 薑苗長到 20–30 cm 時,在基部**補一次土**壓住薑塊,幫助橫向膨大。這步驟很多人忘記。

> 

**💡 催芽這步驟要不要做?**

不催芽直接種也可以,但等待時間更長(4–6 週才見芽)。春天氣溫不穩定時,沒催芽的薑塊在土裡悶太久容易爛。**推薦催芽**:3–5 天的準備換來 2–4 週更快發芽,值得。

## 📋 完整時程表 · 種塊到採收

**種塊 → 催芽 → 發芽 → 葉盛期 → 薑塊膨大 → 採收 → 越冬**,一張表涵蓋完整一年。**照做就對,薑大部分時間自己長**。

> 

**⚠️ 開工前先看月曆 / 節氣:挑對季節才有薑塊可挖**

薑是**長日照暖季作物**,需要春天種、秋天收。種太晚,秋天塊莖還小就要入冬;種太早,低地溫讓薑塊爛在土裡。**節氣比月份準**:

| | | | 月份 | | 節氣 | | 狀態 | |

| | ✅ 最佳 | | **3 月底 – 4 月初** | | **春分(3/20)→ 清明(4/4)** | | 地溫回升穩定,全年最佳種植窗口,**手冊所有時程以此為準** | |

| | 🟡 勉強 | | 4 月中 – 5 月初 | | 清明後 → 立夏(5/5) | | 可種,但採塊量略少;台灣南部勉強到穀雨(4/20) | |

| | ❌ 絕對別種 | | **11 – 2 月** | | **立冬(11/7)→ 雨水(2/19)** | | 地溫過低,薑塊不發芽或爛根。等來年春分再試 | |

## ✂️ 採收:葉與塊兩種模式

**模式 A · 採葉(種植期間隨時可採)**

- 薑葉可做**薑葉茶、煮魚去腥、滷肉鋪底**

- 採法:從植株外側剪下 **1–2 支完整葉莖**,保留中心新葉繼續生長

- 每次採不超過植株 1/3,以免影響地下塊莖膨大

- 嫩薑葉(新冒出的)香氣最濃

**模式 B · 採塊(11–12 月,一年一次的尋寶)**

- **採塊信號**:地上葉片 **大部分變黃、枯萎** → 代表養分已收回到塊莖

- 採前一週停止澆水,讓土乾一點,薑塊比較好找

- **採法**:整盆土倒在大布或報紙上 → 徒手在土裡翻找薑塊。這是最好的尋寶儀式。

- 找到的薑塊用軟刷輕輕刷掉泥土;新鮮薑塊香氣最飽滿

- **保留種薑**:保留最健康的一塊薑(帶多個芽眼),放室內陰涼處越冬,隔年春天繼續種

- 若嫌麻煩:直接把薑塊留在盆裡,停水越冬,春天自然再發

> 

**📐 期望採量**

一個 30 cm 深盆、一塊種薑,正常收成 **300–600g 薑塊**(視管理與季節)。種 2–3 塊可採 1–2 公斤,夠一個家庭半年廚房用量。第二年(留塊越冬)的收成通常比第一年更好。

## 🛡 失敗點與除錯

| | 症狀 | | 可能原因 | | 對策 | |

| | 催芽 2 週沒動靜 | | 抑芽劑 / 溫度不夠 / 薑塊太乾 | | 換傳統市場老薑;確認室溫 22°C+;紗布保持濕潤 | |

| | 種下 4 週沒出芽 | | 地溫不足 / 種得太深 / 抑芽 | | 確認是 3 月底後;覆土深度超過 10 cm 就翻淺一點 | |

## 🧠 失智照護應用 Notes

### 輕度

> 

**📍 輕度:廚房延伸 × 自主感維持**

她還能執行多步驟動作、理解「這個明年可以挖」的因果概念。薑的種植週期長,是建立**長期期待感**的好工具。

- **讓她選薑塊** — 帶她去市場或冰箱,請她挑「有芽眼的老薑」,她的廚房經驗在這裡完全派上用場

- **催芽觀察** — 每天掀開紗布看白芽有沒有長大:「你覺得今天長多少?」這是每日一個小儀式

- 澆水和培土可以讓她主導,你只需要口頭提醒:「該給薑喝水了」

- **用薑葉煮湯時請她剪** — 把薑葉採回廚房,讓她剪進鍋裡。採→煮→吃的完整迴路強化意義感

- **目標**:她是「這個薑的主人」。照顧者的角色是幫她維持這個身份,不是接手

### 中度

> 

**📍 中度:嗅覺記憶是最強的入口**

她的多步驟能力下降,但嗅覺通路保存最久。薑的辛香氣是台灣廚房的核心記憶,**聞到薑的味道,比任何語言都更能喚醒「我在家」的感覺**。

- **採薑葉環節**:你剪好一支薑葉,放到她手上讓她揉一揉聞一聞。這步驟不需要指令,嗅覺反應是自動的

- 澆水改成**你裝好水壺放在盆旁**,請她:「幫我澆一下這盆薑」— 單一動作,不要多步驟

- 挖薑時**你鬆土她找薑塊** — 把手伸進土裡摸到薑塊是強烈的觸覺驚喜,這個動作她通常做得到

- 採到薑塊後立刻問:「這個要做薑茶嗎?還是雞湯?」讓她做一個簡單決定,維持主體感

- **盆位置放她日常動線**(如廚房外走道),讓她每天路過都自然觸碰、聞到

這個階段的價值:**嗅覺記憶 > 語言記憶**。讓薑的氣味成為「今天有做有感覺」的錨點。

### 重度

> 

**📍 重度:不要求「做」,要求「存在」**

她的動作指令已難以執行,但**薑葉的觸感與氣味仍能引發反應** — 臉部表情放鬆、輕微嗅聞動作、偶爾呢喃。這些都是有效的感官陪伴。

- 你採一支薑葉輕揉後放在她鼻前 1–2 秒 — 觀察她的表情變化就夠了,不需要她說話

- 帶她坐在薑盆旁邊,你澆水給她看;聲音(水聲)+ 氣味(土 + 薑)+ 視覺(綠葉)三感同時刺激

- 挖薑時讓她的手被扶著輕輕碰一下薑塊 — 不需要她握住或做什麼,接觸就是參與

- 把新鮮薑塊放在她手心讓她握著,重量與溫度是具體的存在感

- **煮薑茶時讓她在旁邊** — 廚房的薑茶香氣可能是數十年記憶的載體,這個氣味值得讓她吸到

這個階段的價值:**薑的辛香是最不需要認知處理的感官刺激**。嗅覺連結杏仁核與海馬迴,是失智晚期最後留存的通路之一。每一次聞到薑,都是一次微小的「我還在這裡」。
