## 💡 為什麼值得在家種洋蔥

- **廚房裡最常用的香氣基底** — 炒菜、洋蔥湯、漢堡、滷肉幾乎都要它;從陽台直接採,比超市的新鮮一倍

- **兩種玩法自己選** — 長葉(像青蔥替代品,4–6 週可採)vs 等球莖(5–6 個月,採整顆洋蔥),同一棵植物,收益完全不同

- **廢物利用起步** — 廚房切洋蔥剩下的「發芽底部」直接插土,幾乎免費。長出來的葉子立刻可以炒菜

- **嗅覺刺激強烈** — 洋蔥味是失智照護最有效的感官錨點之一;切洋蔥、聞味道本身就是一個 ritual

- **台灣常見品種** — 黃洋蔥(香甜)、紫洋蔥(偏辛)、白洋蔥,花市和農會 10–11 月都有苗

- **照顧省力** — 根扎下去之後基本不用管,偶爾澆水就好。等待期長但期間幾乎不費心

## 🛒 取得方式

> 

**🗓️ Step 0 · 先看月曆再動手**

**現在要是 10 月–1 月才開始定植。** 洋蔥是**短日照作物**:春夏日照長,它只長葉不結球;秋冬短日照,才會向下結球莖。春夏(2–9 月)定植結果是:長出一把細葉、始終不結球、最後倒伏死亡。如果現在是春天,**先備盆土、等到 10 月寒露前後再下手**。

> 

**📦 推薦的新手包 · 三步照順序來**

- **先備盆與土(這一步要比苗先到)** — 花市買 **深度 30 公分以上的深盆**(或直徑 30cm+ 圓盆)+ 蔬菜培養土,合計 NT$150–250;土填 8 分滿、澆一次水讓土服貼

- **10–11 月去花市或農會買「洋蔥苗」** — 一束 10–20 株約 NT$50–100,挑根白莖硬的;大型花市(建國假日花市、台中公園花市)及各地農會 10 月中旬起開始有貨

- **買到當天立刻定植** — 每株間距 10–15 公分,球根微微露出土面,澆透水,放全日照處

⚠️ **順序不能顛倒**:洋蔥苗根系嬌嫩,離土後數小時內就要定植;先到苗才備土,等土期間苗在袋裡悶死。總預算 **NT$300–400 有找**。

> 

**⚠️ 廚房洋蔥發芽底部的正確期待**

切下帶根底部插土會長出**洋蔥葉**(可當青蔥用),但**不會再長成大球莖**。它能量已經在那顆洋蔥裡消耗殆盡。想要球莖,買苗或播種才是正途。發芽底部適合「先享受採葉的成就感」,不適合「等待採整顆洋蔥」。

## 🏺 盆器與土壤

**盆器**

- **深度最少 30 公分** — 球莖要往下長,淺盆會讓球被頂出土面、曬裂;這是洋蔥最常失敗的硬體原因

- 寬度 30 公分以上,可以種 4–6 株(每株間距 10–15 公分)

- 盆底**必須有排水孔** — 洋蔥根怕積水,尤其梅雨季(5–6 月正好採收期)容易爛根

- 長方盆(60×30×30 cm)最省空間,可種 8–10 株一排

**土壤**

- **蔬菜專用培養土**最方便,或一般培養土 + 少量有機質

- 排水要好:可以混 10–20% 珍珠石或粗沙避免積水

- 洋蔥喜微鹼(pH 6.0–7.0),一般培養土都在這範圍,不用特別調

- **定植前把土翻鬆、打碎大土塊** — 球莖在密實土裡長不開

## 🌳 種植步驟

- **備盆** — 深盆裝土 8 分滿、打碎土塊、先澆一次水讓土定型

- **定植洋蔥苗** — 每株間距 10–15 公分,用手指鑽一個洞,把球根尖端朝上插進去,深度讓球根頂部與土面齊平或微微露出 0.5 公分;埋太深球莖悶爛,太淺根系不穩

- **澆透第一次水** — 澆到盆底有水流出;往後土面 3 公分乾再澆

- **放全日照處** — 陽台南面或頂樓;一天至少 6 小時直射日照。日照不足葉細軟、球不成形

- **插標籤記日期** — 洋蔥從定植到採收要 5–6 個月,沒標籤很容易忘記是哪天種的

> 

**🔑 最關鍵的一個動作:球根露出土面**

洋蔥球莖在土裡要有空間向外膨脹。球根頂部**齊土面或微微露出 0.5 公分**是正確深度。很多人習慣「種深一點比較穩」,結果洋蔥被土悶住長不大。等球莖開始膨脹時(約 2–3 個月後),可以輕輕扒開周圍土讓球莖「出頭」。

## 📋 完整時程表 · 定植到採球莖

**定植 → 發根 → 長葉 → 結球 → 採收**,洋蔥是全程最長的家庭可食植物之一(5–6 個月)。好處是期間幾乎不用管。**照做就對。**

> 

**⚠️ 開工前先看月曆 / 節氣:挑對時間,不然只長葉不結球**

洋蔥是**短日照結球作物**:日照時數縮短(秋冬)時,把能量轉入地下結球;日照延長(春夏)時,只長地上葉。在台灣,秋冬定植 → 春天開始抽葉 → 初夏結球採收,是唯一可行的時程。**節氣比月份準**(溫度和日照雙重影響):

## 🌾 施肥

**本 handbook 預設固體緩釋肥** — 不用稀釋、不用算濃度、撒一次吃 3 個月、難燒根。洋蔥施肥有兩個重點:早期促葉,結球期促球不促葉。

**⚠️ 結球期(Day 90 後)停氮改鉀**

好康多 1 號(NPK 14-11-13)是全效肥,Day 30 第一次用很好。但 Day 90 之後球莖開始膨脹,**氮(N)太多會讓地上葉繼續猛長,把能量留在葉片而不是球莖**,結果葉多球小。

- **好康多 1 號**(NPK 14-11-13)— Day 30 首選,350g 約 NT$100

- **草木灰** — 台灣花市或農資店有售,天然高鉀,Day 90 結球期加一次

## ✂️ 採收

**玩法一:採葉(青蔥替代品,隨時可採)**

- 定植 4–6 週後,葉片長到 20 公分以上即可開始採

- 用剪刀從**距土面 5 公分處**剪下,留下基部讓它繼續長

- 洋蔥葉比青蔥香氣更濃,切碎炒蛋、撒湯都好用

- 採葉**不影響最終採球**,只要根還在、球莖還在發育,可以同時享受兩者

- ⚠️ 結球期(Day 90 後)採葉要節制 — 地上葉越多,光合作用越強,球才長得大。留 3–4 片葉不採,讓它繼續工作

**玩法二:採球莖(等 5–6 個月)**

- **採收訊號**:地上葉片**自然倒伏、垂下超過 2/3** — 不要人為把葉子折倒;等它自己倒才是真的成熟

- 倒伏後**再等 1–2 週**讓球皮乾硬再採,不要急

- 採收方法:雙手輕握球莖連根往上拔;鬆土再拔,不要硬拉傷球皮

- 採後放在**通風遮陰處曬 1–2 天**,讓外層薄皮完全風乾

- 存放:陰涼通風處可放 2–3 個月;放冰箱反而縮短保存(低溫讓澱粉糊化)

> 

**⚠️ 常見提早採收的錯誤**

看到球根頂出土就以為可以採 — **不行**。球莖頂出是正常現象(讓球曬太陽幫助外皮硬化),但要等地上葉倒伏才是真的成熟。提早採收的洋蔥球小、外皮薄、不耐放。

## 🛡 失敗點與除錯

| | 症狀 | | 可能原因 | | 對策 | |

| | 定植後葉片一直軟趴趴 | | 根系還沒扎穩(正常,2 週內)/ 積水爛根 | | 等 2 週觀察;若土濕透檢查排水孔是否堵塞 | |

| | 只長葉、始終不結球 | | 春夏種(日照太長)/ 品種問題 | | 最常見失敗!確認定植時間是 10–1 月;下次改秋冬種 | |

| | 球小(核桃大)就倒伏 | | 太晚定植(1 月後)/ 施肥不足 | | 當作「迷你洋蔥」採收;下次提早到 10–11 月定植 | |

| | 球根爛掉 | | 積水 / 排水孔堵塞 / 梅雨季未防水 | | 扒開土看根;腐爛部分剪掉,換新培養土;墊高盆底排水 | |

| | 葉尖焦黑 | | 日照過強燒葉 / 過肥 | | 輕微日燒不影響球莖;確認施肥量是否超過速查表 | |

| | 抽出花苞(抽薹) | | 冬天太冷 + 苗太小時低溫感應 / 種太早 | | 花苞立刻剪掉讓能量回球;下次買大苗、10–11 月定植 | |

| | 地上葉倒伏但球很小 | | 正常的小球 / 提早倒伏(病) | | 若外觀正常就採收曬乾;外皮若發黑有異味是病,丟棄 | |

> 

**⚠️ 抽薹(開花)是大警訊**

洋蔥一旦抽薹(從中心長出直挺的花梗),球莖幾乎停止膨脹,能量全轉到開花結籽。**發現花苞立刻剪掉**(從花梗基部整枝剪除),可以搶回部分球莖發育。原因通常是「苗太小遇到低溫春化」或「定植時間不對」。台灣秋冬種的洋蔥比較少這個問題;用種子自育的較常見。

## 🧠 失智照護應用 Notes

### 輕度

> 

**📍 輕度:強烈氣味做感官錨點,廚房連動**

輕度失智者嗅覺記憶保留較好,熟悉的廚房氣味常能觸發長期記憶。洋蔥的氣味幾乎人人有童年記憶。

- 讓她參與**切洋蔥**的動作(監督安全情況下):切洋蔥流眼淚的感受是具體的身體記憶,常能讓失智者說出「以前我都幫家裡切洋蔥」這種自發性回憶

- **每天請她去看一下洋蔥長多高了** — 葉片生長緩慢但可觀察,給她一個可以主導的小報告任務

- 採下洋蔥葉當天就用在晚餐 — 「這是我種的」的感覺在輕度階段最強,請她告訴其他家人今天的菜用了她種的蔥

- 洋蔥苗定植時請她一起種:用手指鑽洞、插苗的觸覺動作簡單、重複、有結構,完全在她能力範圍內

- **目標**:維持「我還是那個在廚房有貢獻的人」的自我感

### 中度

> 

**📍 中度:嗅覺刺激 + 單步動作**

中度失智者多步驟執行有困難,但嗅覺和觸覺感知往往比認知表現好。洋蔥強烈的氣味是很有效的感官介入工具。

- **聞洋蔥葉**:你採下一片洋蔥葉,用手搓出氣味,放到她鼻子前 — 不需要她做任何事,只要感受。這個感官刺激常能帶出情緒反應和喃喃自語式的記憶碎片

- 澆水動作**事先準備好**:小水壺裝滿放在洋蔥盆旁,帶她過去、給她水壺,一句話:「幫我澆一下」。不要要求多步驟

- 指令**一次一個動作**:不說「去澆水然後看看葉子有沒有長」,只說「幫我澆一下」

- 如果採了洋蔥葉要炒菜,讓她**坐在廚房旁邊聞炒洋蔥的香氣** — 不需要她操作;廚房香氣本身就是感官連結

這個階段的價值:**用她最熟悉的廚房香氣,維繫她與「家」「食物」的情感連結**。

### 重度

> 

**📍 重度:不要求動作,用氣味帶她回來**

重度失智者語言和動作退化,但嗅覺往往是所有感官中退化最慢的一個。洋蔥的氣味在這個階段反而有特殊價值。

- **搓葉聞香**:採一片洋蔥葉搓出氣味,放在她手邊或輕觸她的手背 — 觀察她的表情是否有反應,哪怕只是皺眉或微笑

- 炒洋蔥時把她推進廚房旁邊:鍋裡的洋蔥香是強烈的廚房記憶觸發器,對重度失智者也常有效

- 把採下的球莖放在她手上讓她握著:圓形、光滑、有重量的感觸是安撫性的感官輸入

- 每天重複同樣路徑(去陽台看洋蔥),**節奏感比「完成度」重要**

這個階段的價值:**嗅覺是通往情感記憶最後一條路。洋蔥不需要語言就能召喚廚房、家、溫暖。**
