# 🍅 小番茄 家庭種植指南

> 類別:**果菜** · 難度:中(要搭架、要授粉、要避病害) · 季節:春播秋採 / 秋播春採(台灣避夏天)

- **👥 Who**:兒童可參與(注意:進階(搭架/授粉) · 需深盆 30cm+)

## 🛒 購物清單

- **繁殖材料**:小番茄盆苗(新手首選) 1-2 株 (花市, NT$30–80)
- **盆器**:深盆 30cm+(根系深) × 1 個 (NT$150–250)
- **土壤**:一般培養土 10L (NT$100–150)
- **肥料**:好康多 1 號 + 花寶 2 號(開花期高磷) (多盆共用)
- **額外工具**:竹枝/支架(搭架), 剪刀, 毛刷或電動牙刷(授粉)

## 💡 為什麼值得在家種小番茄

- **視覺回饋超強** — 果實從綠轉黃轉紅,每天都在變色,是所有可食植物裡觀察感最強的一種

- **一株可採 2–3 個月** — 側枝不斷開花,採了還有,採收儀式感持續整季

- **可食可觀** — 紅色果實掛枝上是陽台最好看的場景,老人小孩都被吸引

- **香氣特殊** — 番茄葉有獨特青澀草香(不同於果實),是嗅覺刺激的好素材

- **失智照護的「紅色錨點」** — 紅果對失智長輩的視覺吸引力非常強,每天「去看有沒有紅的」是極好的 routine 觸發器

- **全家參與點高** — 澆水、搖花授粉、採收、清洗、裝盤,每個動作都有不同難度,適合不同能力的家人分工

## 🛒 取得方式

> 

**🗓️ Step 0 · 先看月曆再動手**

**盛夏(5–7 月)絕對不要種。**小番茄開花期若氣溫持續超過 35°C,花粉失活、落花不結果,整季白種。台灣最佳播種窗口只有兩個:

- **春播**:2–3 月(立春到驚蟄),5–6 月開始採果

- **秋播**:8–9 月(立秋到白露),11–12 月開始採果

如果現在是 5–7 月:等到 8 月立秋後再下手。這段時間先備好盆土和支架。

> 

**📦 推薦的新手包 · 三步照順序來**

- **先備盆、土、支架(這一步要比幼苗先到)** — 花市買 **30cm+ 深盆(深度至少 30cm)** + 富含有機質的蔬果培養土 + 120cm 長支柱 2 根,合計 NT$200–300,放好定位填土備用。

- **再去花市挑幼苗** — 春秋兩季花市都有,一株 NT$50–100。挑**枝幹挺、節間短、葉色深綠**的;避免已有開花的苗(定植衝擊會讓第一批花落光)。

- **買回當天立刻定植** — 輕脫盆勿破壞根系,種入壓實 → 澆透 → 立支柱 → 放全日照處

⚠️ **順序不能顛倒**:幼苗離開花市盆後不耐擺,若支架和盆土沒就位就先買苗,苗會在袋子裡悶傷。總預算 **NT$400 有找**,難點不在錢,在**月份要對**(見上面 Step 0)。

## 🏺 盆器與土壤

**盆器**

- 直徑 **30 公分以上**、深度 **30 公分以上**(根系深,小盆會矮化,結果量少)

- 盆底**必須有排水孔**(積水是番茄的死亡原因之一)

- 建議用陶盆或塑膠盆均可,避免金屬盆(夏天土溫過高)

**土壤**

- **蔬果專用培養土**或一般培養土 + 珍珠石(7:3)增加排水透氣

- 定植前混入少量**基礎緩釋肥(好康多 1 號)**作底肥

- 加一把**蛋殼粉**混入土中補鈣 — 預防尾腐病(果底發黑)的最省錢做法

**支架(必備)**

- 定植時就要立 — 事後再補容易傷根

- 最低 **120cm 竹支柱或鐵支柱**,植株會長超過 1 米

- 每長 20–30cm 就要用園藝夾或鬆綁綁一次

## 🌳 種植步驟

- **準備盆與土** — 大盆填培養土九分滿,混入蛋殼粉和少量基礎緩釋肥

- **定植幼苗** — 幼苗輕脫盆(勿破壞根系),種入正中央,根系周圍壓實不留氣穴

- **第一次澆水澆透** — 澆到盆底有水流出,確認土服貼

- **立支柱** — 距離植株 5cm 插入支柱,深度至少 30cm;用鬆綁將莖固定

- **放全日照處** — 每天至少 6 小時直射陽光,越多越好

> 

**✂️ 打側芽(小番茄高產的關鍵)**

「側芽」是長在主莖與葉柄夾角的嫩芽。如果不打,植株會長成大叢雜亂形,養分分散,結果少。

- 每週檢查一次,用手指**輕輕掰掉側芽**(別用剪刀,容易染病)

- 保留主莖和 1–2 根粗側枝即可

- **不打側芽 = 葉多果少**;這是新手最常忽略的步驟

## 🌾 施肥

**小番茄施肥分兩階段**:定植後用緩釋肥打底,開花期換高磷鉀肥促果。兩者方向不同,不可混淆。

*施肥時機(Day 14 / Day 35)已統整在「📋 完整時程表」,這個 section 專講 **用多少 · 用哪款 · 怎麼撒**。*

> 

**📐 定植後緩釋肥 · 速查(好康多 1 號)**

| | 盆 | | 顆數 | |

| | 3 吋圓盆(Ø9cm) | | **3 粒** | |

| | 5 吋圓盆(Ø15cm) | | **5 粒** | |

| | 7 吋圓盆(Ø21cm) | | **8 粒** | |

| | 30cm 深盆(番茄標配) | | **8–10 粒** | |

| | 長方盆 50×20cm | | **15 粒** | |

**原則**:寧少勿多。緩釋肥每 3 個月補一次,**開花後不補緩釋肥**,改用液肥。

**🌸 開花期追磷鉀肥(花寶 2 號)**

- **花寶 2 號**(NPK 10-30-20),高磷鉀促花促果

- 第一串花苞出現即開始,每 **10 天一次**

- 依包裝比例稀釋,澆根不澆葉心

- 採收結束後停用,換回好康多 1 號維持土力

**🥚 補鈣(防尾腐病)**

- 定植時混入**蛋殼粉**或市售鈣鎂肥

- 坐果後每 2–3 週在土面撒一次蛋殼粉,澆水讓鈣慢慢釋放

- 澆水節奏保持均勻 — 乾濕劇烈落差是尾腐病的主因

> 

**📱 記憶外包 — 定植當天就把整季施肥做完**

不要依賴記憶。定植當天設 Google 行事曆:

- 定植日 + 14 天 → 「撒好康多 8 粒(澆水)」

- 定植日 + 35 天 → 「換花寶 2 號液肥(見花就加)」

- 定植日 + 45 天起,每 10 天 → 「花寶 2 號」(重複提醒)

**整季的決定一次做完**。這是照護者 handbook 的核心哲學 —— 決定外包到行事曆,不是記在腦裡。

## ✂️ 採收

**採收時機**

- 果實**全紅**再採(半紅可以繼續掛枝上熟,不用急)

- **輕輕轉動**自然脫落,不要硬拔(硬拔容易傷枝、傷下一串花)

- 持續採收會促進新花新果 — **越採越多**是真的

> 

**🤝 手動授粉 · 小番茄高產的隱藏技巧**

戶外有風和蜜蜂幫忙,但陽台環境相對封閉。**每天輕搖花枝**可以明顯提高坐果率:

- 花開期每天上午 9–11 點(花粉最活躍時段)輕搖花梗 10–15 秒

- 或用電動牙刷(關掉水)輕觸花背,震動幫助花粉散落

- 也可以用小毛筆在花心輕輕點幾下

⚠️ **花開多卻不坐果,九成是沒授粉**。這步不難,但容易忽略。

**採後處理**

- 洗淨後直接吃或裝碗擺餐桌 — 自己種的特別甜

- 多的放冰箱冷藏 3–5 天

- 累積夠多時可以做番茄醬、番茄乾(曬乾/烤箱低溫)

## 🛡 失敗點與除錯

| | 症狀 | | 可能原因 | | 對策 | |

| | **開花多卻不結果** | | 日照不足 / 沒授粉 / 高溫落花 | | 移到光更好處;每天搖花枝;盛夏(> 35°C)是正常落花,換季再試 | |

| | **果底發黑凹陷(尾腐病)** | | 缺鈣 + 澆水不均 | | 加鈣鎂肥或撒蛋殼粉;澆水保節奏不要久旱突澆 | |

| | **果皮裂開** | | 久旱後突然大量澆水 | | 澆水保持節奏,土表乾就澆,別等完全乾透再補 | |

| | **葉面白粉** | | 白粉病(真菌) | | 加強通風;葉面噴稀釋牛奶(牛奶:水 = 1:10),持續一週 | |

| | **葉面黃褐斑、下葉先掉** | | 晚疫病 / 葉黴病(真菌) | | 立即剪除病葉;加強通風;避免葉面澆水;嚴重時換土換盆 | |

| | **葉片捲曲** | | 缺水 / 缺鎂 / 蟲害 | | 先確認土壤濕度;缺鎂補鎂肥;翻葉查蟲(蚜蟲、薊馬) | |

| | **植株徒長,葉多果少** | | 沒打側芽 / 日照不足 | | 立刻打側芽;確保每天 6 小時以上直射陽光 | |

| | **定植後萎蔫 3–5 天** | | 緩苗正常現象 | | 維持澆水,不要施肥,通常一週內自行恢復 | |

> 

**⚠️ 病害預防比治療容易**

小番茄最大敵人是**晚疫病和葉黴病**,台灣梅雨季尤其嚴重。預防措施:**1)** 不從頭頂澆水(水滴打葉面是病菌傳播路徑),改從盆緣澆根。**2)** 每天檢查下葉,有黃褐斑立刻剪掉。**3)** 陽台保持通風。發現太晚整株可能整季毀掉。

## 🧠 失智照護應用 Notes

### 輕度

> 

**📍 輕度:小番茄是全家最好的「共同語言」**

她還能理解多步驟、自主行動。小番茄的採收儀式可以做得最完整,而且三代都參與得進來。

- **每天的觀察任務**:「去看看有沒有紅的」— 有明確目的地、有明確回報,最不容易被拒絕

- 澆水、每天搖花枝授粉 — 她都能自己做,你只需要口頭觸發

- 採收讓她主導:你只說「今天幾顆紅了?」讓她告訴你答案

- 採收後請她**洗果子、分裝、挑壞的** — 延伸到廚房的多步驟活動

- **目標**:讓她覺得「這棵番茄是我在管的」,自主感比什麼都珍貴

這個階段的價值:**「今天幾顆紅的?」可以是三代共同的每日 moment**。

### 中度

> 

**📍 中度:以紅色果實做單一任務觸發**

她無法完整管理整株番茄,但**採收這個時刻**可以是她最有成就感的參與點。

- 你負責日常澆水、施肥、打側芽 — 她不需要知道這些

- **採收時請她一起**:你站旁邊,她負責「把紅的轉下來放碗裡」— 單一動作,難度低,成就感高

- 洗果子她能做(手放水中搖動、用濾籃倒水) — 重複手部動作,不需要判斷

- 搖花枝授粉動作簡單,**你帶她去、帶她搖**:「幫我搖一下這串花,讓它結果」

- **把決定外包到物理空間**:水壺事先裝滿擺盆旁,讓她不用找水龍頭

- 言語指令**一次一個動作**:「幫我採那顆紅的」而不是「去看看番茄、紅的採下來、帶回來」

這個階段的價值:**紅色果實是她最能「看懂」的任務訊號 — 紅了就採,不需要判斷**。

### 重度

> 

**📍 重度:紅色 + 香氣 + 味覺記憶的感官陪伴**

不要求動作。小番茄的視覺(紅色)、嗅覺(番茄葉特有青澀香)、味覺(番茄酸甜)是這個階段最有價值的三條感官路徑。

- 帶她到番茄盆旁坐著,你採給她看 — 慢慢的,讓她看到果實轉紅、脫落

- **讓她握著一顆紅番茄** — 圓潤光滑的觸感、紅色、重量,感官刺激豐富

- 把番茄葉揉一下放到她鼻前讓她聞 — 番茄葉的青草香是很強的嗅覺觸發,可能喚起記憶

- 採一顆熟透的紅番茄讓她吃 — **味覺記憶是最後退化的一種**,番茄甜酸可能是她這輩子熟悉的味道

- 每天重複同樣時段、同樣路徑去陽台 — 節奏感比完成度重要

這個階段的價值:**她不需要「種」,她需要「被帶去那裡」。番茄的紅色、香氣、味道是陪伴她的語言,不是任務**。
