## 🛒 購物清單

- **繁殖材料**:小黃瓜種子 或 盆苗 1 包/1-2 株 (花市, NT$30–100)
- **盆器**:深盆 30cm+ × 1 個 (NT$150–250)
- **土壤**:一般培養土 10L (NT$100–150)
- **肥料**:好康多 1 號 + 花寶 2 號(開花期) (多盆共用)
- **額外工具**:竹枝/支架(爬藤必備,1.5-2 公尺), 剪刀

## 💡 為什麼值得種小黃瓜

- **看得見成果** — 果實一天比一天大,每天量長度都不同,視覺上非常有成就感

- **多產** — 一株健康的小黃瓜可以採 10–20 條,一個夏天吃不完

- **從花到瓜只要 7–10 天** — 比等長葉的植物更快有收穫,維持動力

- **攀藤視覺變化豐富** — 每天爬高一點、纏住架子一段,比盆栽更有「生命感」

- **品種多** — 一般小黃瓜、水果小黃瓜(更甜更脆)、刺瓜(外皮有刺,口感清爽),新手從水果小黃瓜入門最成功率高

- **採完馬上可以吃** — 拍黃瓜、涼拌,從採到上桌 10 分鐘,自種的成就感很真實

## 🛒 取得方式

> 

**🗓️ Step 0 · 先看月曆再動手**

**現在要是 3–6 月才播種。** 台灣平地小黃瓜是春夏果菜,高溫才長得好,最佳播種窗口是**驚蟄(3/5)到夏至(6/21)**前後。秋冬(9 月以後)氣溫下降,小黃瓜成長停滯、開花少、果實畸形,不推薦新手在這個時段嘗試。

| | | | 月份 | | 節氣 | | 狀態 | |

| | ✅ 最佳 | | **3 – 5 月** | | **驚蟄(3/5)→ 立夏(5/5)** | | 氣溫穩升、日照充足、病蟲壓力低,**強烈推薦** | |

| | 🟡 勉強 | | 6 月 | | 芒種(6/5)前後 | | 仍可種,但梅雨+夏初高溫需注意白粉病;勤排水 | |

| | ❌ 絕對別播 | | **9 – 2 月** | | **白露(9/8)→ 雨水(2/19)** | | 低溫日短,開花少、果實畸形、植株弱,成功率極低 | |

如果現在是秋冬,**先備盆土、網路找好種子來源,等隔年 3 月清明前後再下手**。

> 

**📦 推薦的新手包 · 三步照順序來**

- **先備盆、土、支架(這些要比種子先到)** — 直徑 30 cm 以上深盆 + 培養土 + 珍珠石(1:4 混)+ 1.5–2 公尺竹竿或 Ω 形爬藤架。花市或蝦皮搜尋「爬藤架 陽台」,合計約 NT$300–400。

- **再買種子或小苗** — 蝦皮搜尋「水果小黃瓜種子」(NT$50–80,一包約 15–20 粒);或春天直接買花市穴盤苗(約 NT$30–50 一株,省發芽等待)。

- **到貨當天立刻播種/定植** — 種子直播入盆(見種植步驟),或穴盤苗帶土移入大盆,澆透水後放全日照處。

⚠️ **順序不能顛倒**:小苗根系細嫩,等你找到盆才種會傷根。總預算 **NT$400–500 有找**,含架子一季可採 10–20 條瓜。

**其他取得管道**

- **花市穴盤苗**(3–5 月春季最多):省去發芽等待,看得見苗才買,成功率最高

- **蝦皮 / 博客來農業館**:品種選擇多,可挑「水果小黃瓜」或「刺瓜」等特色品種

- **超市黃瓜留種**:台灣超市小黃瓜多為 F1 雜交,留種第二代性狀會分離,**不推薦新手**

## 🏺 盆器與土壤

**盆器**

- 直徑 **30 公分以上**、深 30 公分以上(根系深、肥需求大)

- 一株一大盆;或長方形陽台槽 60×25 cm 種 2 株,株距 30 cm

- 盆底**必須有排水孔**,小黃瓜怕積水爛根

- 材質不拘,夏天深色盆會吸熱,偏好淺色或白色

**土壤**

- **透氣排水最重要** — 培養土加 20% 珍珠石混勻,避免夏天暴雨積水

- 加一把椰纖(椰殼纖維)可增加保水又不悶根,有機質充足小黃瓜長更快

- 定植前可先拌入適量有機基肥(雞糞堆肥顆粒),讓養分緩慢釋放

**支架(必備,其他植物不需要)**

- 小黃瓜是攀藤植物,藤蔓可爬 1.5–2 公尺,**不搭架就倒伏、通風差、易生病**

- 推薦:竹竿 + 綁紮帶(最便宜)、Ω 形爬藤架(穩定好看)、A 字型竹架

- 架子定植當天就要插進盆,不要等到植株倒了才補

## 🌳 種植步驟

- **備盆備架** — 盆底放一層礫石或發泡煉石幫助排水,填土 8 分滿,插好支架

- **播種(種子路線)** — 每盆挖 1–2 個小洞,深約 1 公分,每洞放 1–2 粒種子,覆土輕拍,澆透水;出苗後留最強的 1 株,其餘剪掉(不要拔,傷根)

- **或定植(買苗路線)** — 穴盤苗帶土整株移入大盆中央,填土壓緊根部,澆透水

- **放全日照處** — 每天至少 6 小時直射陽光;日照不足會只長葉不開花

- **引藤上架** — 藤蔓每長 15–20 公分就用軟綁帶輕輕引導上架,不要讓它垂下

> 

**🌸 授粉要點(小黃瓜最常失敗的環節)**

小黃瓜同株開**雌花**和**雄花**兩種,沒有昆蟲幫忙,果實不會長大。陽台種植通常缺蜜蜂,**要人工授粉**:

- **雌花辨識**:花朵底部有一顆小小的「迷你瓜」(子房),開花時間約早上 6–10 點

- **雄花辨識**:花朵底部是細梗,沒有小瓜

- **授粉方法**:摘下雄花,把中心花粉輕碰雌花中心 2–3 下。或用小畫筆輕刷雄花再刷雌花

- **最佳時段**:早晨花剛開時授粉,中午以後花萎縮效果差

- **失敗徵兆**:未授粉的雌花底部小瓜會在 2–3 天後黃化掉落

**每天早晨 5 分鐘看一次花,是採到瓜的關鍵。**

**播種 → 發芽 → 定植 → 開花 → 首採 → 盛產**,一張表涵蓋整季。**照做就對,不用每天觀察猜狀態**。

## 🌾 施肥

**小黃瓜是重肥果菜**,比地瓜葉需要更多養分,尤其開花結果期間消耗很快。本 handbook 仍預設固體緩釋肥為主,開花後可選擇性加液肥追肥。

*施肥時機(Day 30 / Day 90)已統整在上面「📋 完整時程表」,這個 section 專講 **用多少 · 用哪款 · 怎麼撒**。*

>

| | 3 吋圓盆(Ø9cm) | | **3 粒** | |

| | 5 吋圓盆(Ø15cm) | | **5 粒** | |

| | 7 吋圓盆(Ø21cm) | | **8 粒** | |

| | **圓盆 Ø30cm(推薦最小盆)** | | **10 粒** | |

**原則**:寧可比這個少,不可比這個多。小黃瓜重肥但燒根也快。

**🛒 買哪一款**

- **好康多 1 號**(NPK 14-11-13)— 基礎施肥用,果菜適用

- **好康多 2 號**(高磷配方)— 有些園藝師開花結果期換用,促進座果

- 350g 小包約 NT$100,全聯園藝區、花市都有

> 

**📱 授粉提醒外包給手機,施肥提醒外包給行事曆**

播種當天就打開 Google 行事曆設兩個提醒:

- 播種日 + 30 天 → 「撒 10 粒好康多(澆水)」

- 播種日 + 90 天 → 「再撒 10 粒好康多」

**這一季的施肥決定當天就做完。腦袋騰出來去關注更重要的事**(例如每天早上授粉才是關鍵)。

## ✂️ 採收

- 果實長到 **15–20 公分**(水果小黃瓜 12–15 cm)即可採收;過晚採瓜皮變硬、味苦、種子老化

- 用剪刀在果梗處**剪斷**,留梗 1 cm;不要扭轉(會傷藤蔓接點)

- **越採越旺** — 一條瓜留太久會讓植株耗盡養分停止生新花

- 盛產期每 2–3 天巡一次,錯過會變老瓜

- 採完的瓜室溫可放 2–3 天,冰箱保鮮袋可放 1 週

> 

**🥒 採收後怎麼吃(快速上桌)**

- **拍黃瓜**:用刀背拍裂 → 鹽醃 10 分鐘 → 瀝水 → 拌蒜末 + 香油 + 醋,5 分鐘完成

- **涼拌黃瓜**:切薄片 → 鹽醃 → 加糖醋蒜

- **生吃**:水果小黃瓜直接洗淨吃,甜脆不輸水果

從陽台採回來到端上桌 10 分鐘,**這就是自種最真實的成就感**。

## 🛡 失敗點與除錯

| | 症狀 | | 可能原因 | | 對策 | |

| | 播種 10 天後沒發芽 | | 土太冷 / 種子太老 / 覆土太深 | | 確認氣溫 >18°C;種子覆土 1 cm 不要超過 2 cm;換新種子 | |

| | 只長葉不開花 | | 日照不足 / 氮肥過多 | | 移到每天 6 小時以上直射光處;停止施肥觀察 | |

| | 雌花底部小瓜黃化掉落 | | 未授粉或授粉失敗 | | 每天早上 6–10 點人工授粉;確認雄花已開(有花粉) | |

| | 瓜長到一半停止生長 | | 水分不足 / 養分不夠 / 授粉不完全 | | 增加澆水頻率;補施肥;確保授粉充足 | |

| | 瓜形彎曲或畸形 | | 授粉不均匀 / 溫度過高 / 缺水 | | 授粉要刷滿雌花花柱;高溫期早晨澆水降溫 | |

| | 葉片白色粉末(白粉病) | | 通風不良 + 濕氣重(梅雨期最常見) | | 修剪過密葉片;改善通風;噴小蘇打水(1g/500ml)或稀釋牛奶 | |

| | 葉片有小洞或蟲咬痕 | | 夜蛾幼蟲 / 蚜蟲 / 葉蟎 | | 夜間手電筒找蟲手抓;蚜蟲用水強力沖洗;避免農藥(要吃的) | |

| | 莖基部腐爛(猝倒病) | | 土太濕 + 真菌感染 | | 改善排水;感染部位剪除;嚴重整株拔除換土 | |

## 🧠 失智照護應用 Notes

### 輕度

> 

**📍 輕度:每天量瓜,建立儀式感**

她還能理解指令、走到陽台、記得昨天做了什麼。小黃瓜的每日生長變化讓照護活動有了**可量測的成果**。

- **每天早上量瓜長度,記到小本子** — 可用尺量,也可以用手比。她主導,你只提醒時間

- 授粉動作她可以自己來(用小畫筆刷花粉),動作細膩、有目的感

- 採收時讓她判斷「這條夠大了嗎?」— 保留決策自主性

- 採回來的瓜讓她準備拍黃瓜 — 用刀背拍、鹽醃、拌調味,整個流程她都能做

- **目標**:每天有一件「有成果可以看」的事。量瓜記錄本就是她的成就感來源

這個階段的價值:**維持她「我還是有在貢獻這個家」的感覺**。

### 中度

> 

**📍 中度:單一動作、感官參與**

她的多步驟能力下降,但單一動作和感官刺激仍然有效。小黃瓜提供豐富的視覺和觸覺刺激。

- **你引導去陽台,請她看今天有沒有新的花** — 簡單任務一次一步

- 看到雌花開,你準備好畫筆遞給她 — 她做刷花這個動作

- 瓜長大了摘下來放到她手上,讓她握著感受涼重感 — 觸覺刺激強

- 廚房備料時讓她拿刀背「拍」黃瓜 — 動作有力道、有聲音、有參與感

- **不要讓她做決策(採不採?夠大嗎?)**,把判斷部分由你先做好,給她執行

這個階段的價值:**她不需要「懂」種植,她需要「參與」這個每天發生的儀式**。

### 重度

> 

**📍 重度:存在、感官、陪伴**

動作和理解都有限,但感官和情緒仍然有反應。把她帶到藤架旁邊,讓她在那個空間裡就是目的。

- 帶她坐在藤架旁邊,你澆水、她看,不要求她做什麼

- 把剛採的瓜放到她手上 — 涼涼的重量感、黃瓜皮的清香,這些感官刺激直達

- 攀藤架的立體視覺(綠葉 + 黃花 + 懸掛的瓜)比一盆平鋪的葉菜更能吸引目光

- 讓她聞剛採的黃瓜 — 清香氣味對嗅覺記憶有獨特的觸發效果

- 每天重複同樣時段去陽台看瓜,**節奏感比「完成」更重要**

這個階段的價值:**她不需要「種」,她需要「在這個有生命的空間裡」。黃瓜的味道、重量、顏色 — 比任何話語都更直接到達她**。
