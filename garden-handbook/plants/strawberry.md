## 🛒 購物清單

- **繁殖材料**:草莓子苗(挑帶花芽的) 2-3 株 (花市, NT$50–120)
- **盆器**:長條盆 或 吊盆 × 1 個 (NT$150–300)
- **土壤**:一般培養土 10L (NT$100–150)
- **肥料**:好康多 1 號 + 花寶 2 號(開花/結果期) (多盆共用)
- **額外工具**:毛刷(人工授粉), 剪刀

## 💡 為什麼值得種草莓

- **紅色果實 + 甜味 = 強記憶刺激** — 草莓的外觀與氣味幾乎所有人都能立刻辨認,特別適合失智長輩的感官活動

- **採摘即食,reward 立刻到嘴裡** — 不用等、不用煮,看到紅的摘下來就吃

- **「尋寶」遊戲天然存在** — 每天去翻葉片找有沒有紅的,等待感強、參與動機高

- **白花與紅果視覺對比強** — 花期有白色小花,結果期紅綠對比鮮明,視覺刺激清楚,不挑視力

- **走莖自我繁殖** — 春天母株長出走莖(匍匐莖),一株變多株,越種越多

- **季節節奏清晰** — 秋種、冬花、春果、夏休,照顧者與長輩一起感受節氣變化

## 🛒 取得方式

> 

**🗓️ Step 0 · 先看月曆再動手**

**只有 9–11 月(白露–立冬)是正確的下手時機。** 夏天高溫(4–8 月)草莓植株會融解,幾乎必死;12 月種苗已來不及在最冷前扎根分化花芽,結果少。如果現在是春夏,**先備好盆土等秋天**,蝦皮賣家夏天通常停賣草莓苗,也是風向球。

> 

**📦 推薦的新手包 · 三步照順序來**

- **先備盆與土(這一步要比苗先到)** — 買 **直徑 20–30 cm 的長條盆或吊盆**(深 15–20 cm 即可,根系淺)+ **培養土混泥炭土 + 珍珠石**(比例約 6:2:2),合計 NT$200–300,填好、澆一次讓土服貼備用。

- **再下單草莓苗** — 10–11 月花市、園藝店或蝦皮搜尋「草莓苗 帶子株」,選葉色翠綠、莖粗壯、**無炭疽病紅斑**的苗,一株 NT$50–100

- **到苗當天立刻定植** — 脫盆輕抖舊土 → 種入盆 → **生長點(心)必須露出土面**(埋太深立刻爛)→ 澆透 → 放全日照陽台

⚠️ **順序不能顛倒**:苗買來若沒有盆土就定植,根系暴露或悶在袋裡一天就開始萎爛。總預算 **NT$400 有找**,種植節奏要對(見上面 Step 0)。

**其他取得管道**

| | 來源 | | 時機 | | 說明 | |

| | **花市幼苗**(主流) | | 10–11 月 | | 可選品種(豐香、章姬、桃薰),新手優選 | |

| | **走莖子苗** | | 春天(3–4 月) | | 朋友/母株長出走莖後剪段壓土,免費繁殖 | |

| | **蝦皮網購** | | 9–11 月 | | 搜尋「草莓苗 帶子株 台灣」,注意確認苗健康 | |

> 

**⚠️ 新手陷阱:炭疽病苗**

買苗時要仔細翻看葉片與莖基部,若有**圓形黑褐色凹陷斑點**就是炭疽病(Colletotrichum),帶回家會蔓延整盆。選健康苗比後期救苗容易百倍。

## 🏺 盆器與土壤

**盆器**

- **長條盆或吊盆** — 草莓根系淺(20 cm 以內),不需深盆

- 直徑 20–30 cm、深 15–20 cm 足夠

- **吊盆特別適合** — 走莖自然垂下美觀,果實不接觸土面避免腐爛

- 盆底**必須有排水孔** — 草莓怕積水,爛根極快

**土壤**

- **排水好的砂質壤土**,微酸性(pH 6.0–6.5)最佳

- 推薦配方:培養土(6)+ 泥炭土(2)+ 珍珠石(2)

- 不要用純培養土 — 太黏,夏季悶熱易爛根

- 如果土取得麻煩,花市「草莓專用土」也 OK

## 🌳 種植步驟

- **備盆備土** — 見上面「盆器與土壤」,填土至九分滿,先澆一次讓土服貼

- **定植幼苗** — 脫盆後輕輕撥散舊土,種入盆,**生長點(心部)要露出土面**,埋太深立刻爛

- **澆透水** — 第一次澆到盆底出水

- **放全日照處** — 早晨太陽為主,至少 6 小時直射,光不夠花少甜度低

- **每天觀察澆水** — 冬天 2–3 天一次,春天開花期土微乾才澆,不要積水

> 

**⚠️ 人工授粉 — 冬天小花不會自己授粉**

台灣冬天沒有蜜蜂幫忙授粉,室內或封閉陽台更嚴重。**開花後每天用小毛刷或棉花棒輕輕在花心轉一圈**,把花粉從雄蕊轉到雌蕊。授粉成功 → 果實飽滿;沒有授粉 → 果實小、畸形或不結。這是室內草莓最容易漏掉的一步。

## 📋 完整時程表 · 定植到盛產

**定植 → 花芽分化 → 開花 → 結果 → 盛產 → 夏季休眠**,一張表涵蓋整年。**照做就對,不用時時觀察植物狀態**。

> 

**⚠️ 開工前先看月曆 / 節氣:挑對時間,不然花少果少**

草莓**喜涼爽怕夏熱**,花芽分化需要低溫短日照(立冬前後自然到來),果實需要冬春陽光糖化。兩套時間軸都列:

| | | | 月份 | | 節氣 | | 狀態 | |

| | ✅ 最佳 | | **9–11 月** | | **白露(9/8)→ 立冬(11/7)** | | 苗有足夠時間扎根、自然低溫啟動花芽,**手冊所有時程以此為準** | |

| | 🟡 勉強 | | 8 月底 / 12 月 | | 處暑(8/23) / 大雪(12/7) | | 邊緣期,夏末太熱易萎、12 月太晚花芽少 | |

| | ❌ 絕對別種 | | **4–7 月** | | **清明(4/4)→ 大暑(7/22)** | | 高溫「溶解」植株,存活率接近零;即使活也不結果 | |

## 🌾 施肥

**本 handbook 預設固體緩釋肥,不推液肥給新手** —— 不用稀釋、不用算濃度、撒一次吃 3 個月、難燒根。草莓特殊規則:結果期**禁氮肥**(促葉不促果)。

> 

**⚠️ 草莓施肥特別規則**

- **結果期(1–4 月)禁止氮肥** — 氮肥讓葉猛長,花和果能量被搶走;只補磷(P)鉀(K)

- **花前(11–12 月)補一次磷鉀液肥** — 花寶 2 號或磷酸二氫鉀,按包裝稀釋後澆根部

- **果期補鉀肥 + 鈣肥** — 果實飽滿的關鍵,缺鈣果實容易爛尖

**🛒 買哪一款**

- **好康多 1 號**(NPK 14-11-13)— 定植後 30 天基礎底肥用

- **花寶 2 號**(NPK 10-30-20)— 高磷款,花前一個月噴一次促花

- 350g 小包約 NT$100,一包可用兩盆好幾季

- 全聯園藝區、花市、iGarden 網購都有

## ✂️ 採收

- 等到**整顆紅透**再摘 — 離枝後不會再熟,綠色摘下沒有甜度

- **連蒂一起摘下**,保持果實完整,避免破皮快速腐爛

- 新鮮採摘直接吃最好;放冰箱 1–2 天食用完畢

- 每天去翻葉片檢查 — 果實藏在葉下,每天找一次才不錯過

**走莖管理(採收期同步要做)**

- **盛產期(2–4 月)剪掉走莖** — 走莖耗母株體力,剪掉讓能量集中到果實

- **4 月後可留走莖讓子苗生根** — 把走莖前端的子苗壓到小盆土裡,3–4 週生根後剪斷與母株的連結,即為新苗

- 一株母株可繁殖 3–5 個子苗,秋天換季用

**日常維護(摘葉 + 剪老葉)**

- **每週摘老葉、黃葉、受病葉** — 讓植株通風、避免真菌

- 葉子擠密時摘一些讓陽光能照進果實

## 🛡 失敗點與除錯

| | 症狀 | | 可能原因 | | 對策 | |

| | 種下後葉子很快萎黃 | | 炭疽病、根系受傷或夏天定植 | | 確認季節對了;換健康苗 | |

| | 葉多花少或不開花 | | 氮肥太多 / 日照不足 / 花芽分化未完成 | | 停氮肥,改磷鉀;確認全日照;等到 12 月再觀察 | |

| | 開花但果實小或畸形 | | 沒有授粉 | | 每天毛刷在花心轉一圈;移到有風處也幫助授粉 | |

| | 果實爛尖 | | 缺鈣 / 土積水 | | 補鈣肥;確認盆底排水 | |

| | 葉面白粉 | | 白粉病(真菌) | | 通風、減少葉面淋濕;嚴重可用碳酸氫鉀液噴 | |

| | 葉子發黑斑 | | 炭疽病 / 葉斑病 | | 摘除病葉、停止淋濕葉面;嚴重整株換掉 | |

| | 有螞蟻在盆邊 | | 有蚜蟲(蟻共生) | | 找到葉背蚜蟲,水沖或手抹;消滅蚜蟲螞蟻自然離開 | |

| | 夏天整株萎爛 | | 高溫超過 35°C,正常休眠 | | 不用救;移遮蔭等降溫;或淘汰等秋天重種 | |

| | 生長點(心部)發黑 | | 種太深、悶熱潮濕 | | 重新種,心部必須露出土面 | |

## 🧠 失智照護應用 Notes

### 輕度

> 

**📍 輕度:祖孫共同記憶製造機**

草莓幾乎沒有長輩不喜歡,特別是跟小孩一起摘。秋天一起種苗、冬天等花、春天採果,完整的季節陪伴。

- **秋冬一起種幼苗** — 從無到有的等待,給她一個「我種的草莓」的所有感

- **每天一起去找有沒有紅的** — 尋寶儀式,時間感強,比「去散步」更有目的

- 她自己摘、自己吃 — 採摘即食的立即 reward,不需任何解釋

- 跟小孩一起摘 — 一個祖孫場景,紅果、笑聲、甜味,可以拍下來

- 她還能幫忙:澆水、摘老葉、拿毛刷授粉 — 讓她知道「這盆需要她」

這個階段的價值:**她是照顧者,不是被照顧者 — 草莓需要她每天來看**。

### 中度

> 

**📍 中度:她參與採與吃**

多步驟會卡住,但「看到紅的 → 摘 → 放嘴裡」這個流程她大概率可以完成。草莓的紅色非常好辨認。

- 你負責日常照顧、除葉、補肥、人工授粉

- **採收時牽她到盆邊,一起找紅的** — 不用她認識草莓,看到紅的就說「這個紅了」

- 小小的果實,抓握動作對她手指是**溫和的精細運動**

- 吃一顆的甜味 — 不需語言,味覺直達,幾乎所有人都有反應

- **白花與紅果的顏色對比強烈**,視覺刺激清楚,視力退化也能分辨

這個階段的價值:**把「採草莓」變成一個每天可以重複的短儀式**(對中度失智,重複比多樣重要)。

### 重度

> 

**📍 重度:紅與甜的簡單喜悅**

草莓可能比其他活動更容易讓重度失智長輩有反應 — 因為太鮮明。不需要她「做」任何事。

- 採一顆洗淨放她手中 — 觸感、重量、涼意,感官刺激

- 切半放嘴邊讓她嘗 — 甜味是幾乎所有人都保留最久的味覺記憶

- **紅與綠的視覺對比強烈**,即使視力退化也能看到紅色,視覺刺激清楚

- 帶她坐在盆旁邊,你澆水給她看、你授粉給她看 — 不要求她做,讓她「在場」

- 每天重複同樣時段、同樣路徑到盆旁 — 節奏感比完成度重要

這個階段的價值:**她不需要「種草莓」,她需要「紅色那顆甜的」這個記憶鎖點**。甜味、紅色、小小的圓 — 這些是幾乎最後才消失的感官。
