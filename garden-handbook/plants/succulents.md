## 💡 為什麼多肉植物適合照護者家庭

- **忘了澆反而活得好** — 多肉死因 90% 是澆太多,照護者忙到忘澆水,反而最適合

- **不需要每天互動** — 不像葉菜要每天澆,多肉一週看一眼就夠,不增加照護者負擔

- **視覺療癒、造型多樣** — 熊掌形、玫瑰形、柱狀,擺在桌上就是裝飾

- **葉插繁殖是「奇蹟時刻」** — 一片葉子放在土上,6-8 週後長出小植株,老人小孩都會驚嘆

- **觸感刺激豐富** — 有絨毛(熊童子)、有凸粒(月兔耳)、有光滑(桃蛋、石蓮花),觸覺刺激豐富

- **可以做組盆藝術** — 多種多肉合植成一盆,像佈置微型庭園,有創作感

## 🛒 取得方式

> 

**🗓️ Step 0 · 先看季節再動手**

多肉植物**四季都可以買**,但不同季節要注意:

**春秋(3–5 月、9–11 月)是入手最佳時機**,溫度 15–25°C,根系活躍、適應快。

**夏天(30°C+)買苗要注意**:運送途中悶熱容易傷苗,到手後放陰涼通風處緩苗 3–5 天,不要立刻曬太陽。

**冬天寒流期(10°C 以下)**:運送可能凍傷,等寒流過後再下單。

> 

**📦 推薦的新手包 · 三步照順序來**

- **先備盆與多肉專用土(這一步要比苗先到)** — 花市買 **3–4 吋小陶盆或塑膠盆(有排水孔)** + **多肉專用砂質土**(或培養土混粗砂/赤玉土 1:1),合計 NT$100–150。

- **再挑苗** — **花市 / 多肉專賣店**挑喜歡的品種,一盆 NT$50–200;或蝦皮搜尋「多肉植物 組合包」(NT$100–300 有多株)。

- **到手當天緩苗,隔天上盆** — 收到先放陰涼通風處 1 天,確認根系健康後上盆、填土、不要馬上澆水(等 3–5 天讓根系適應)。

⚠️ **順序不能顛倒**:苗到了沒有適合的土,根系悶在塑膠袋容易爛。總預算 **NT$200–400 有找**。

## 🏺 盆器與土壤

**盆器**

- **有排水孔**是最重要條件 — 積水是多肉最大死因

- 陶盆優於塑膠盆 — 陶盆透氣、排濕快,塑膠盆保水過久

- 尺寸比苗大一號即可 — 太大的盆土量多保水久,反而不好

- 組盆可用淺盆(高度 5–8cm)— 擺多種多肉在一起

**土壤**

- **多肉專用砂質土**(花市直接買「多肉仙人掌土」)是最方便選擇

- 自調方式:培養土 + 粗砂/赤玉土 = **1:1 混合**

- 關鍵原則:**快速排水、不積水** — 澆水後 30 分鐘土表應該回乾

- 不要用一般培養土直接種 — 保水太久,根系窒息

> 

**⚠️ 盆底鋪粗砂或碎磚塊**

盆底放一層粗砂、小石子、或破盆片,讓排水孔不被土堵住。這個步驟很多人跳過,結果積水爛根。

*施肥細節獨立成下面的「🌾 施肥」section。*

## 🌳 種植步驟

**路徑 A · 買苗直接上盆**

- **緩苗** — 收到苗先放陰涼通風處 1 天,讓它適應新環境

- **備土** — 盆底鋪粗砂,填入多肉專用土至 8 分滿

- **脫盆** — 輕拍塑膠原盆邊緣,連土球取出,不要拉莖

- **放入新盆** — 根系展開,填土固定,根頸(莖基部)與土面齊平

- **等待 3–5 天再澆水** — 讓根系修復後再給水;第一次澆透到盆底出水

- **放在明亮散光處** — 前 2 週避免直射強光,之後可移到日照充足處

**路徑 B · 葉插繁殖(多肉神技)**

- **取葉** — 捏住葉片基部,輕輕左右旋轉取下,確保葉基完整(有生長點才能發根)

- **晾傷口** — 平放在廚房紙巾上,傷口朝上,晾 1–2 天讓切口乾燥

- **放在土上** — **不要埋入土中**,直接平放在濕潤土面(葉尖朝上或水平皆可)

- **噴霧保濕** — 每 2–3 天對土面噴霧,保持微濕,不要淹水

- **等待 2–4 週** — 葉基部長出小根和新芽,母葉漸漸乾縮是正常現象

- **等小株長到 1–2cm 再移盆** — 母葉完全枯掉後可移除

> 

**💡 葉插成功率提升訣竅**

- 取葉時要取**完整的葉片**(含葉基),葉片斷掉的不會發根

- 多取幾片 — 成功率約 50–70%,多取保險

- 不要催(不要埋、不要大量澆水) — 急是葉插最大敵人

## 🌾 施肥

**多肉植物施肥量遠低於葉菜** — 多肉原生環境就是貧瘠乾燥地,過肥反而徒長(葉片拉長、顏色變淡、失去美感)。

*施肥時機(Day 30 / 每 3 個月)已在上面「📋 照顧節奏表」統整,這個 section 專講 **用多少 · 用哪款 · 怎麼撒**。*

> 

**📐 不同盆大小 · 速查(好康多 1 號)**

| | 5 吋圓盆(Ø15cm) | | **4–5 粒** | | 中型組盆 | |

**原則**:多肉寧可少肥,絕不多肥。少肥頂多長慢;多肥徒長變醜甚至根傷。

**🛒 買哪一款**

- **好康多 1 號**(NPK 14-11-13)— 萬用緩釋款,多肉也適用

- 或選**多肉專用緩釋肥**(花市有賣,NPK 比例低一些更安全)

- 350g 小包約 NT$100,一包用幾年

**📱 記憶外包 — 上盆當天就排好施肥行事曆**

打開 Google 行事曆:

- 上盆日 + 30 天 → 「撒 3 粒好康多(多肉盆)」

- 之後每 3 個月重複

## 🌱 繁殖與擴盆

**葉插繁殖 — 多肉最神奇的技能**

一片葉放在土上,兩個月後長出完整新植株。這個過程對失智長輩和小孩都是最直觀的「奇蹟」。

> 

**🌿 葉插全流程(超詳細版)**

- **選健康的葉片** — 飽滿、沒病斑,從母株中下部取(頂端嫩葉成功率低)

- **整葉取下** — 捏葉基部左右旋轉,確認連帶葉基(生長點)一起取下;葉片扯斷不會發根

- **晾 1–2 天** — 傷口在空氣中癒合形成保護層,不晾直接放土容易腐爛

- **平放土面** — 葉基朝下或水平放置在濕潤土面上,**不要埋入土中**

- **每 2–3 天噴霧** — 維持土面微濕,不要淹水;直接對葉片噴也 OK

- **等 2–4 週發根** — 葉基部長出粉紅小根,再過 2–4 週長出綠色小芽

- **等 6–8 週長新株** — 小株越來越大,母葉開始乾縮(正常,養分轉給小株)

- **3–4 個月獨立上盆** — 母葉完全枯掉後移除,小株已有完整根系可以獨立

**分株(側芽換盆)**

- 石蓮花、蘆薈等多肉會長**側芽(子株)**,長到 3–5cm 後可以切下來獨立

- 切下側芽 → 晾 1–2 天傷口 → 插入多肉土 → 等 1–2 週發根 → 正常照顧

- 子株比葉插快:通常 2–3 個月就能成為獨立成株

**組盆藝術 — 多肉的進階玩法**

- 用淺盆把 5–10 種不同多肉種在一起,像佈置一個微型庭園

- 選顏色有對比的品種:綠色石蓮 + 紅色虹之玉 + 粉色桃蛋 + 灰藍仙人掌

- 加上裝飾石頭或彩色礫石更好看

- 組盆適合送禮,也適合讓失智長輩一起動手「種種看」

| | 繁殖方式 | | 難度 | | 所需時間 | | 適合品種 | |

| | 葉插 | | ⭐⭐ 中等 | | 3–4 個月 | | 石蓮花、虹之玉、桃蛋 | |

| | 側芽分株 | | ⭐ 易 | | 2–3 個月 | | 蘆薈、石蓮花、仙人掌 | |

| | 直接買小苗 | | ⭐ 最易 | | 即時 | | 所有品種 | |

## 🛡 失敗點與除錯

> 

**⚠️ 多肉三大死因(90% 都是這三個)**

- **澆水過多** — 葉子積水、根系缺氧腐爛

- **通風不良** — 密閉空間悶熱悶濕,病菌孳生

- **冬天凍傷** — 台灣低海拔少見,但寒流來要搬室內

| | 症狀 | | 可能原因 | | 對策 | |

| | 葉片**軟爛、半透明、變色** | | 澆水過多,根爛了 | | 立刻停水,脫盆看根系;爛根剪掉,晾乾換新土重種 | |

| | 葉片**皺縮、乾癟** | | 缺水或根系失能 | | 先確認根:有根就澆透;無根就切莖晾乾重插 | |

| | 葉片**拉長徒長**、莖細 | | 光線不足 | | 移到窗邊或陽台,增加日照;已徒長可砍頭重新扦插 | |

| | 葉片顏色**全是綠色**,不轉紅 | | 日照不足或太涼 | | 移到全日照處;秋天氣溫下降自然會轉色 | |

| | **葉插沒反應**超過 4 週 | | 葉基不完整/太濕/太乾 | | 重取葉確認有完整葉基;土面微濕不要淹 | |

| | 有**蟲子**(介殼蟲、粉蝨) | | 通風不良、密植 | | 棉棒沾酒精擦除;嚴重可用稀釋肥皂水噴 | |

| | 盆底**長青苔或白黴** | | 土太濕、通風差 | | 停水讓土乾透;改善通風(移到有風的窗邊) | |

**🔍 判斷是缺水還是爛根(兩者症狀很像)**

<div style="padding:10px 0 0;font-size:14px;">

皺縮缺水 vs 根爛都會讓葉片軟,判斷方法:

- 輕輕拉一下葉片:爛根的植株葉片很容易脫落;缺水的緊

- 聞土壤氣味:爛根有酸臭味;正常土是泥土清香

- 脫盆觀察:爛根呈褐色/黑色/糊狀;健康根是白色/米色

**確定是爛根**:剪掉所有爛根,在通風處晾乾 2–3 天,換新多肉土重種。

## 🧠 失智照護應用 Notes

### 輕度

> 

**📍 輕度:她能主導,多肉是最低壓力的植物任務**

她還能理解指令、做多步驟動作。多肉的特點:沒有「忘了澆水就死」的壓力,這讓她不會因為忘記而有挫敗感。

- **每週一次澆水儀式** — 固定在週六下午,她主導,你在旁邊。不需要每天提醒

- 葉插過程讓她**自己取葉、自己放土面** — 步驟簡單,不怕做錯

- 觀察葉插發根的過程:「妳看,上週放的那片葉長根了」— 提供**持續的小驚喜**

- 讓她幫忙**命名品種**:「這個圓圓的叫什麼名字好?」— 啟動命名記憶和創意

- **組盆時讓她挑顏色搭配** — 保留她的審美自主性

這個階段的價值:**維持她「我有在照顧一個生命」的自我效能感**。多肉不會因為她忘了澆水而死,不讓她有挫敗感。

### 中度

> 

**📍 中度:動作還在,觀察和觸摸比操作更重要**

她可以執行單一動作,但「找水壺→ 澆水 → 放回」這種多步驟可能卡住。重點轉移到感官體驗。

- **澆水器事先裝好擺在盆旁** — 不讓她去找水龍頭,降低失敗機會

- 一次只說一個指令:「幫我摸摸看這個熊掌」、「你覺得這個葉子軟不軟?」

- **觸摸不同品種的多肉**:絨毛感(熊童子)、光滑感(桃蛋)、粗粒感(仙人掌)— 豐富觸覺刺激

- 葉插的小株用**放大鏡一起看**:「你看,這麼小一片葉長出這麼多根了」

- 固定位置、固定時間:多肉擺在她每天走過的地方,不要換位置

這個階段的價值:**不要求「完成任務」,要求「有感知、有互動」**。多肉的多樣觸感是很好的感官切入點。

### 重度

> 

**📍 重度:不要求動作,多肉是感官陪伴的媒介**

她的自主動作有限。多肉的價值從「照顧活動」轉為「感官刺激和存在陪伴」。

- 把盆栽移到她椅子旁邊,讓她**伸手就能摸到**

- 你照顧多肉的時候讓她在旁邊看 — 不要求她做什麼,讓她參與這個場景

- **把葉片放到她手上**:「妳摸摸看,這個是什麼感覺?」— 等待,不要催

- 葉插長出小芽時:**把盆放到她面前、靠近她的眼睛說**:「妳看,長出來了」— 不需要她回應,讓她看到

- 多肉的顏色(紅、粉、灰藍、綠)變化豐富,**放在窗邊讓光線穿過**,視覺刺激效果好

- 重複相同的互動節奏:每天同一時間、同一盆、同樣說同樣的話 — **節奏感比內容更重要**

這個階段的價值:**多肉不需要語言就能傳遞「這個還活著、在生長」的訊息。生命感知不需要認知能力**。仙人掌的刺、熊童子的絨毛、石蓮花的圓潤 — 這些觸覺比語言更直接。
