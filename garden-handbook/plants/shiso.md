## 💡 為什麼值得種紫蘇

- **一種兩用** — 紫葉(日式)和綠葉(韓式)口味不同,兩種都可食,選一種種,或兩種都買種子各一盆

- **廚房全才** — 醃梅子、包烤肉、剁碎拌豆腐、揉碎泡茶,一盆就把日式 + 韓式 + 台式料理都覆蓋了

- **香氣極強** — 手揉一片葉,整個陽台都是香的;比薄荷更厚重、更有層次,嗅覺刺激性強

- **每天都有理由互動** — 夏天每 5–7 天採一次嫩葉,有人要吃就有人要去陽台,進而帶動澆水 ritual

- **秋末自動留種** — 抽苔開花後種子自然落土,隔年春天往往自己長出來,不需重新買種

- **播種比扦插更穩** — 直播土面,7–14 天發芽,不需枝條取得,任何季節只要月份對就能下手

## 🛒 取得方式

> 

**🗓️ Step 0 · 先看月曆再動手**

**播種必須在 3–8 月。** 發芽適溫 18–23°C,冬天(立冬 11/7 起)低溫紫蘇會凍死,新苗更是撐不住。如果現在是冬天,**不要播種** — 先備盆土、把種子買好放冰箱冷藏保存,等隔年 3 月驚蟄(3/5)前後再下手。4–5 月清明穀雨之間是台灣播種黃金期,溫度穩、發芽整齊。

> 

**📦 推薦的新手包 · 三步照順序來**

- **先備盆與土(要比種子先到)** — 花市買 **20–25cm 圓盆或長方盆 + 疏鬆培養土**(合計 NT$150–200);土填 8 分滿、先澆透讓土服貼,表面整平。

- **買種子** — 蝦皮搜尋「[紫蘇 種子](https://shopee.tw/search?keyword=%E7%B4%AB%E8%98%87+%E7%A8%AE%E5%AD%90)」(1 包 NT$30–80,通常有幾百粒夠種整季);或花市直接買種子包,確認包裝有「紫葉紫蘇」或「青紫蘇/綠紫蘇」標示。

- **到手當天立刻撒種** — 種子**不覆土**(光照有助發芽),直接撒土面、用手輕壓讓種子接觸土壤 → 輕噴水保持濕潤 → 放半日照以上的地方

⚠️ **順序不能顛倒**:種子到手後若盆土還沒備好,就先用濕紙巾包起來放冰箱冷藏(可撐 2–3 天)。總預算 **NT$250 有找**。

> 

**🔴 紫葉 vs 綠葉,選哪個?**

**紫葉(日式紫蘇)** — 顏色鮮豔、香氣更濃,醃梅子用這種,顏色也是失智感官刺激的亮點。

**綠葉(青紫蘇/荏胡麻)** — 口味較溫和,韓式料理包烤肉用這種,炒菜更百搭。

*第一次種推薦**紫葉** — 視覺效果好,照護應用更豐富(見「🧠 失智照護 Notes」)。*

## 🏺 盆器與土壤

**盆器**

- 直徑 **20 公分以上**圓盆,或 40cm 以上長方盆

- 紫蘇植株長大後直徑可達 50–70cm、高 60–90cm,盆子夠大才不會限制生長

- 盆底**必須有排水孔** — 紫蘇喜水但怕積水,積水容易根腐

- 多株密植時用大長方盆(植株間距 25–30cm)

**土壤**

- 一般培養土即可,**疏鬆排水良好**是關鍵

- 可混入 20% 珍珠石或粗砂,避免夏天澆水頻繁後板結

- 紫蘇在弱酸性土壤(pH 5.5–6.5)生長最旺 — 一般培養土通常在這範圍,不需特別調整

- 播種前把土壤澆透,表面整平 — 種子撒下去才能均勻貼合土面

*施肥細節獨立成下面的「🌾 施肥」section。*

## 🌳 種植步驟

- **選盆** — 見上面「盆器與土壤」,土填 8 分滿、澆透整平

- **撒種子** — 種子**不覆土**;薄薄一層撒土面,種子之間留 2–3cm 間距(太密之後要疏苗)

- **輕壓接觸** — 用手掌輕輕壓讓種子與土壤接觸(不要深埋),幫助吸水發芽

- **噴水保濕** — 用噴霧罐輕噴,讓土面保持微濕;❌ 不要大水沖(種子會被沖走)

- **放半日照以上位置** — 陽台半日照 OK,全日照更好;❌ 室內光線不足發芽差

> 

**🌱 發芽期管理要點**

- 每天早晚各噴一次水,保持表土微濕(不可讓表土乾掉,發芽期最脆弱)

- 7–14 天出苗;若超過 14 天沒動靜,可能是種子太舊或溫度太低

- 出苗後等幼苗長出 2–3 對本葉才疏苗(拔掉最弱的,每穴留 1 株),間距保持 20–25cm

## 📋 完整時程表 · 播種到採收

**播種 → 發芽 → 施肥 → 採收 → 留種**,一張表涵蓋整季。**照做就對,不用猜植物狀態**。

> 

**⚠️ 開工前先看月曆 / 節氣:月份對,種什麼都贏**

紫蘇**喜溫暖、怕霜凍**,發芽適溫 18–23°C,冬天播種幾乎不發芽甚至爛種。兩套都列:

## ✂️ 採收與料理

**採收方式**

- 採**頂端嫩葉 2–4 片**,保留下方 4–5 對葉讓植株繼續長

- 每次採後 5–7 天又長出新嫩葉 — 盛夏時幾乎無限採

- **見花苞就掐掉**:一旦開花葉片會變小、香氣下降。不打算留種的話,把花序掐掉讓植株繼續產葉

- 採後放陰涼處、或泡冷水保鮮,1–2 天內用完風味最佳

**台灣家庭常見用途**

| | 用途 | | 說明 | |

| | **醃紫蘇梅** | | 每年 5–6 月青梅季,紫葉紫蘇醃梅子,梅子轉紅後香氣濃郁;台灣信義鄉、南投農家的傳統做法 | |

| | **包烤肉/生魚片** | | 整片葉當包菜(韓燒用法),或剪絲撒生魚片上去腥增香 | |

| | **紫蘇飲料** | | 葉片加薑、糖、檸檬汁煮成夏日消暑飲;或直接泡冷水成香草水 | |

| | **拌豆腐 / 涼拌** | | 切碎拌涼豆腐,淋醬油麻油,夏天快速料理 | |

| | **醬漬紫蘇** | | 葉片醬油浸漬 1–2 天,拌飯下粥都好吃(日式腌菜做法) | |

> 

**🌰 秋末留種 SOP**

- 抽苔後**停止採葉**,讓花序長完、結種莢

- 種莢從綠轉褐色、變硬時就可以剪(別等太久,熟透種子會自己落土)

- 剪下整條花穗,倒扣放進紙袋,室溫乾燥 7–10 天

- 輕搓花穗讓種子落袋,挑掉雜質 → 放進小夾鏈袋標上日期 → 冰箱冷藏(可保存 1 年)

## 🛡 失敗點與除錯

| | 症狀 | | 可能原因 | | 對策 | |

| | 播種 14 天沒出苗 | | 種子過舊 / 溫度太低 / 表土乾掉 | | 確認月份對(3–8 月);重播新種子;每天噴水保濕表土 | |

| | 幼苗徒長(細長倒伏) | | 光線不足 | | 移到更亮的地方;避免室內窗旁光線太弱 | |

| | 葉片顏色褪淡、不夠紫 | | 光線不足或缺肥 | | 移到全日照;確認已在 Day 30 施肥 | |

| | 葉片有蟲洞 | | 葉蟬 / 夜蛾幼蟲 | | 早晚翻葉背手抓;嚴重時皂液噴霧(食用葉記得採前洗淨) | |

## 🧠 失智照護應用 Notes

### 輕度

> 

**📍 輕度:讓她主導醃梅子這件事**

她還能理解指令、執行多步驟動作。紫蘇最特別的價值:醃梅子是台灣家庭女主人的記憶,讓她主導這件事能喚起「我還是那個會做醃梅的人」的自我感。

- **每年 5–6 月青梅季,讓她決定** — 「今年要不要醃梅子?」這個問題會自動帶出記憶和主導感

- 採葉動作她能完全獨立完成:你只需要說「幫我去採幾片紫蘇」

- 剪葉、疊葉、加鹽搓揉(去汁)這些動作跟她以前做料理的動作同類型,容易上手

- **盆栽放廚房窗邊或陽台入口**:每天進出都看得到,自然形成 check 植物的習慣

- 讓她決定「今天採幾片」,不要替她做決定

這個階段的價值:**紫蘇的香氣和醃梅的記憶,是「我還是那個媽媽」的身份確認**。

### 中度

> 

**📍 中度:香氣先行,動作跟上**

她的多步驟能力下降,但嗅覺和情感記憶通常比語言記憶更耐久。紫蘇特殊的香氣 — 紫蘇醛的濃郁薄荷感 — 往往能喚起情緒記憶,即使她說不出「這是什麼」。

- **先讓她聞**:採一片葉在她面前揉碎,讓香氣擴散 — 不要急著說「這是紫蘇」,讓她自己反應

- 如果她說「這個味道...」就跟上:「對,以前你都用這個醃梅子」— 不用糾正記憶,跟著走就好

- 採葉動作簡化成**單一指令**:「幫我摘一片」而不是「去採幾片回來放碗裡」

- **紫色葉片視覺刺激**:顏色比一般綠色植物更搶眼,即使她不記得植物名稱也會注意到

- 盆栽放在她日常動線顯眼處,讓視覺自然觸發接近動作

這個階段的價值:**香氣記憶比語言記憶更耐久,紫蘇的氣味是喚起「以前的她」最有效的鑰匙之一**。

### 重度

> 

**📍 重度:感官在,動作不強求**

語言溝通困難,自主動作受限。但嗅覺、觸覺、視覺的感官刺激仍然有效,甚至是這個階段最可靠的溝通管道。

- 帶她坐在盆旁,**摘一片葉放她手心**,讓她握著 — 不要求她做任何事,就這樣

- 你揉碎葉片湊近讓她聞 — 觀察臉部表情變化;有時候她說不出話但表情會變柔和

- **紫色葉片**:和一般綠色植物不同,顏色對比強,視覺上有吸引力,眼神會跟著轉過去

- 每天同一時段、同一位置的 ritual — 節奏感和空間固定比「她能做什麼動作」更重要

- 如果她還能握住葉子就讓她握;如果她會把葉子放嘴裡就全程陪同(紫蘇可食,但監督安全)

這個階段的價值:**她不需要「做」,她需要那個熟悉的香氣和顏色在旁邊。紫蘇的氣味比任何語言都更能讓她感到安心**。
