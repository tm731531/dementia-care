## 💡 為什麼黃金葛是「家人分工」代表植物

- **幾乎不會死** — 低光、忘澆水、水耕土耕都能活,是最接近「放著就長」的植物

- **NASA 空氣淨化榜** — 1989 年 NASA Clean Air Study 列入,可吸附甲醛、苯、一氧化碳(雖然效果有限,但有象徵意義)

- **水耕觀根** — 玻璃瓶養,全家每天能看到白色氣根在水裡飄,有生命感

- **水耕換水是最低門檻的家務** — 小孩 3 歲就能拿瓶子去廁所換水,培養責任感

- **長大可以爬、可以吊** — 藤蔓垂掛書架或攀上棚架,不佔桌面空間

- **扦插繁殖超簡單** — 一根插水裡一週長根,可以不斷分株送人或在家裡到處種

## 🛒 取得方式

> 

**🗓️ Step 0 · 四季可買,但夏天買最好適應**

黃金葛四季皆可種,但有個選時機的訣竅:

| | | | 月份 | | 節氣 | | 建議 | |

| | ✅ 最佳 | | **4 – 9 月** | | **清明(4/4)→ 白露(9/8)** | | 氣溫高、新根發得快、適應期短 | |

| | 🟡 勉強 | | 10 / 3 月 | | 寒露(10/8) / 驚蟄(3/5) | | 還可以,適應期拉長一點 | |

| | ❌ 注意 | | **11 – 2 月冷氣房** | | **立冬(11/7)→ 雨水(2/19)** | | 冷氣吹太多會讓葉子捲曲發黃;放遠離出風口的位置 | |

黃金葛不怕冬天,但**台灣室內冬天的冷氣直吹**比低溫本身更傷。冬天買回來放室內明亮處、遠離冷氣出風口就 OK。

> 

**📦 推薦的新手包 · 三步照順序來**

- **先備容器(水耕路線)或盆+土(土耕路線)** — 決定你要走哪條:水耕備玻璃瓶/透明罐(家裡就有);土耕花市買 15cm 以上盆 + 排水培養土(約 NT$100–150)

- **取得枝條或幼株** — 花市 / IKEA / 大賣場園藝區一盆 NT$80–150;或跟朋友要一根藤插水瓶,完全免費

- **到手當天立刻定位** — 水耕:剪掉下節葉子插進裝水玻璃瓶,放餐桌或書桌明亮處;土耕:鬆土、插進去、澆透

⚠️ **順序不能顛倒**:枝條先到但瓶子/盆還沒準備好,插水也要讓節點浸到水。總預算 **水耕 NT$0–100(家裡有瓶子就免費)、土耕 NT$200 有找**。

**其他取得管道**

| | 管道 | | 價格 | | 適合 | |

| | 花市 / 園藝行 | | NT$80–200 | | 現買現看品質,有人幫你挑健康的 | |

| | IKEA / 大賣場 | | NT$99–199 | | 順便帶回,不用特別跑一趟 | |

| | 蝦皮「黃金葛 扦插苗」 | | NT$50–100 | | 宅配方便,挑評價高的賣家 | |

| | 朋友分株 | | 免費 | | 最穩,活株帶回比扦插苗更好 | |

| | 超市/菜市場(少見) | | – | | 偶爾有,通常是土耕小盆 | |

**挑株要點**

- 葉子**亮綠、斑紋清楚**,不要有黑斑或爛葉

- 莖**結實不軟塌**

- 盆土**沒有腐臭味**(爛根信號)

## 🏺 盆器與土壤(或水耕容器)

**水耕路線(家庭分工首選)**

- 玻璃瓶、透明罐、寬口花瓶、廢棄醬菜瓶都可

- **透明容器的好處**:根系清晰可見,每天觀察根的生長是視覺刺激來源(對失智照護很有用)

- 水量:淹過根節 2–3 cm;不要淹到葉子

- 換水頻率:夏天每週、冬天 10 天;水渾濁隨時換

**土耕路線**

- 盆器:直徑 15 cm 以上,盆底**必須有排水孔**

- 土壤:一般培養土即可;想排水更好可以混入 20% 珍珠石

- 吊盆/長條盆都可以,藤蔓夠長時吊掛很好看

*施肥細節(好康多用量、時機)獨立成下面的「🌾 施肥」section。*

## 🌳 種植步驟

**水耕步驟**

- **剪枝條** — 從母株剪下一段有 2–3 個節點的藤,剪口斜剪

- **去下節葉** — 最下面 1–2 片葉子剪掉,只留上面 1–2 片(泡水的葉子會爛)

- **插進水瓶** — 節點浸在水裡;葉子在水面上

- **放明亮處** — 避免強光直曬;餐桌邊窗旁最佳

- **等 1–2 週長白根** — 不需要任何動作,只要看著

> 

**💡 水耕轉土耕(選做)**

水耕根長到 5 cm 以上,可以移到土裡種。注意:**水耕根跟土耕根構造不同**,移植後會有 1–2 週適應期,葉子可能稍微蔫一下再恢復。新手可以直接一路水耕,不必轉土。

**土耕步驟**

- **備盆** — 盆底鋪一層碎石或濾網防土流失

- **填土 8 分滿** — 先澆一次水讓土服貼

- **插入枝條** — 下方 1–2 個節點埋進土,壓實周圍土壤

- **澆透** — 澆到盆底排水孔流出水

- **放室內明亮處** — 前 2 週避免強光,讓根系建立

## 🌾 施肥

**本 handbook 預設固體緩釋肥,不推液肥給新手** —— 不用稀釋、不用算濃度、撒一次吃 3–6 個月、難燒根。

*施肥時機已統整在上面「📋 照顧節奏表」,這個 section 專講 **用多少・用哪款・怎麼撒**。*

> 

**📐 好康多 1 號 · 不同盆大小速查**

| | 盆 | | 顆數 | |

| | 3 吋圓盆(Ø9cm) | | **3 粒** | |

| | 5 吋圓盆(Ø15cm) | | **5 粒** | |

| | 7 吋圓盆(Ø21cm) | | **8 粒** | |

| | 長方盆 50×20cm | | **15 粒** | |

**原則**:寧可比這個少,不可比這個多。黃金葛耐貧瘠,過肥反而燒根。

**🛒 買哪一款**

- **好康多 1 號**(NPK 14-11-13)— 萬用款,觀葉植物適用

- 350g 小包約 NT$100,小盆用幾年不完

- 全聯園藝區、花市、iGarden 網購都有

- 替代品:魔肥、奧綠肥(類似緩釋顆粒肥都 OK)

**🗓️ 施肥時機**

- 扦插/種下後 **30 天**第一次(讓根先穩定再施肥)

- 之後每 **3–6 個月**一次(緩釋肥效果持久)

- 冬天(11–3 月)**停肥**(植物半休眠,肥料吸不了反而鹽積)

**🧪 進階:水耕液肥路線(不建議新手)**

<div style="padding:10px 0 0;font-size:14px;">

如果你走水耕路線,可以用水耕液肥:

- 一般水耕肥或花寶 2 號(NPK 20-20-20),稀釋比**照包裝再少一半**

- 夏天每月一次,冬天兩個月一次

- 換水時順便加,不用另外處理

**黃金葛水耕長期不施肥也不死**,只是長得慢一點。液肥只是加速。

## 🌱 扦插與分株

黃金葛是**最容易繁殖的植物之一**。剪一段藤插水裡就會長根,不需要任何特殊技術。

**扦插步驟(水中發根法)**

- **選健康藤段** — 至少 2 個節點,有 1–2 片葉子

- **剪口斜剪** — 接觸水面面積大一點,發根快

- **摘掉下節葉子** — 入水的葉子會腐爛,只留水面上的葉子

- **插進水瓶** — 節點浸水中;放明亮散光處

- **等 7–14 天** — 白色氣根從節點長出,到 3–5 cm 長就可以移土或繼續水耕

> 

**⚠️ 水耕根轉土耕要有心理準備**

水耕根是為適應水環境長出來的,土耕根構造不同。移植後:

• 前 1–2 週葉子可能輕微蔫垂(正常適應期)

• 保持土壤略濕(不要太乾,讓水耕根轉型)

• **不要因為葉子蔫就猛澆水** —— 過澆才是真正的危險

**大株換盆/分株時機**

- 根從盆底排水孔長出來 → 該換大一號的盆了

- 春天換盆最佳(清明 4/4 前後);冬天避免換盆

- 換盆時可以趁機把長藤剪短,剪下來的都可以插水繁殖

- 一盆變多盆,可以放在家裡不同角落,或送給朋友

> 

**🎁 黃金葛當伴手禮**

剪一根藤插進裝水的小玻璃瓶,貼一張小便條寫「照顧方式:每週換水就好」—— 成本幾乎零,但對收到的人來說是一份會活著的禮物。特別適合送給同樣在照顧長輩的朋友。

## 🛡 失敗點與除錯

| | 症狀 | | 可能原因 | | 對策 | |

| | 葉子整片發黃、掉落 | | 澆水過多 / 土太濕 / 爛根 | | 停澆讓土乾透;嚴重的話拔出來看根,爛的剪掉換土 | |

| | 新葉葉尖發黑 | | 澆太多或施肥過濃 | | 暫停施肥;澆一次大量清水沖掉鹽分 | |

## 🧠 失智照護應用 Notes

### 輕度

> 

**📍 輕度:一起來家庭小儀式**

她還能理解指令、自主參與。黃金葛水耕可以是**家人每週共同的小儀式**。

- 水耕放餐桌上 —— 每天吃飯都能看到,不需要特別「去找」這件事

- 換水可以一起做:她拿瓶子、你倒水、小孩加液肥。三步驟、三個人,互不打擾

- 剪一根藤讓她插在新瓶裡 —— 她能看到「這一株是從我那瓶分出去的」,成就感

- **擦葉動作**:給她一塊濕布,請她把葉子擦亮。動作單純不需要決策,可以完整做完

- 這個階段的價值:**維持她「我還是這個家的照顧者」的自我感**

### 中度

> 

**📍 中度:小孩接棒,長輩見證**

長輩難以主導多步驟任務,但能**在旁邊看、摸葉子、聞水味**。這時候家裡的小孩可以成為主力。

- **讓小孩負責換水**(水耕的話)—— 責任感 + 不會弄壞 + 讓你喘口氣

- 奶奶坐在餐桌旁看小孩做,偶爾摸葉子。不要求她做什麼,存在就好

- 每次換水的對話可以是:「奶奶你看,弟弟幫黃金葛換水了」—— 她參與了,即使只是見證

- 水耕透明容器的白色氣根:**視覺刺激**,每天不一樣,「今天根又長了一點」

- **三代共同圍繞一株植物**:長輩從「照顧者」轉為「見證者」,不是「被照顧的人」

這個階段的價值:**讓照護不是你一個人扛的事**。

### 重度

> 

**📍 重度:一個你不孤單的提醒**

長輩已經很少主動互動。黃金葛在這個階段的意義,**有一半是給你自己的**。

- 放在長輩視線容易落到的地方 —— 靜態綠意,不需要任何語言

- 偶爾帶她的手摸摸葉子,不說話;葉面光滑、略有涼意的觸感,是一種感官陪伴

- 水耕容器裡白色飄動的根 —— 如果她的眼睛還在,這是一個不要求回應的視覺焦點

- 小孩繼續換水 —— 這個家的照護**不是只有你一個人在撐**

- 當你太累,小孩可以接手這件事 —— 他不會弄壞,這是你可以真正放手的一件事

這份「綠意還在、小孩也在參與」的接續感,對照顧者的孤立感是真實的支撐。


## 📓 實測紀錄

> 

**冬季冷氣直吹測試(觀察中)** — 冬天室內冷氣出風口旁邊的盆,斑紋確實比窗邊的淡,葉尖偶有焦枯。移到窗邊後 2–3 週恢復。**教訓:冬天擺位比澆水更重要**。已寫進上面「💡 為什麼」和「🛡 失敗點」section。

> 

水耕路線(餐桌玻璃瓶)實測中。一根藤 Day 8 長出第一條白根,Day 14 有 4–5 條根,長度 2–3 cm。全家吃飯都會主動去看「今天根又長了多少」—— 這個小小的生命感比預期更有效果。

奶奶在餐桌旁坐下時偶爾會指著瓶子說「這個長根了」,雖然她已經叫不出「黃金葛」這三個字。*她認得這件事,即使叫不出名字。*
