## 🛒 購物清單

- **盆器**:長條盆 50cm × 1 個 (NT$150–250)

- **肥料**:好康多 1 號 350g (多盆共用)

## 💡 為什麼值得在家種香菜

- **台灣家常必備香草** — 湯麵、煎餃、滷肉飯、薑母鴨都要它;超市一把 NT$15–25,但自採永遠比買的鮮

- **嗅覺記憶最強的植物之一** — 香菜腺體的香氣極具穿透力,是對失智長輩嗅覺刺激效果最好的廚用香草之一

- **盆子不用大** — 一個 20cm 圓盆或 2 尺長方盆就夠一家人採 2–3 輪,窗邊就能種

- **採收週期短** — 播種後 40–50 天首採,秋冬持續採 2–3 輪不斷

- **動作豐富** — 搓種子催芽、播種、疏苗、採葉、洗葉,每個動作都可以讓長輩參與

- **一次播一批,整個冬季吃** — 不像夏季菜兩週就結束,秋冬氣候穩,採收期拉長

## 🛒 取得方式

> 

**🗓️ Step 0 · 先看月曆再動手**

**播種視窗:10 月 – 隔年 2 月(寒露到雨水)**。香菜性喜冷涼,最適生長溫度 15–22°C。春天播種看起來發芽了,但一到 4–5 月氣溫回升就會**迅速抽苔開花,葉子變苦變少**。夏天(6–9 月)幾乎沒辦法種 — 30°C 以上發芽率低、苗細弱、隨時抽苔。**現在如果是 3 月以後,等到寒露(10 月上旬)再下手**,等待期間先備好盆土和種子。

> 

**📦 推薦的新手包 · 三步照順序來**

- **先備盆與土(這一步要比種子先到)** — 花市或網購 **20–25cm 圓盆或 2 尺長方盆 + 一般蔬菜培養土**(合計 NT$150–200);土填 8 分滿、澆透、讓土面稍乾再播種。

- **買香菜種子(芫荽種子)** — 蝦皮搜尋「[芫荽 種子](https://shopee.tw/search?keyword=%E8%8A%AB%E8%8D%BD+%E7%A8%AE%E5%AD%90)」或花市、農會超市、種苗行(一包 NT$30–60,可播 2–3 盆)

- **播種前一天催芽** — 種子浸水 24 小時 + 手指搓破硬殼(見下方步驟)→ 隔天濾乾再播;不催芽的話發芽率和速度都差很多

⚠️ **月份要對**:種子很便宜,但播錯季節白等一個月然後失敗。總預算 **NT$250 有找**。

> 

**⚠️ 超市香菜留種子:成功率低**

超市買來的香菜束通常是嫩株,種子未熟。能播的要帶有褐色成熟種球(雙懸果),綠色嫩果發芽率極低。想自己留種,等自家盆栽開花結果後採熟透的再用。

## 🏺 盆器與土壤

**盆器**

- 深度 **15 公分以上**即可(香菜根系不深)

- 寬度:圓盆 **20–25cm** 或 **2 尺長方盆(約 60×20cm)**

- 盆底**必須有排水孔**;香菜怕積水,積水 3 天就爛根

- 可播較密,疏苗留間距 5–8cm/株

**土壤**

- **一般蔬菜培養土即可**(帶有機質,排水適中)

- 不需要特別鬆、也不需要重黏土;一般花市培養土不加改良直接用

- 如果土偏黏,可混入少量粗砂或珍珠石讓排水更好

- 播種前把土面耙平、壓稍實,撒種子比較均勻

## 🌳 種植步驟

> 

**🔑 香菜最關鍵的一步:催芽**

香菜種子學名叫「雙懸果」— 外殼是一個硬殼裡包兩粒種子。這個硬殼**不搓破,兩粒都很難發芽**;就算發芽也慢(2–3 週起跳)。催芽有兩個動作缺一不可:

- **浸水 24 小時** — 讓硬殼充分吸水軟化

- **搓破或輕壓裂硬殼** — 手指搓、掌心輕壓、或兩個碗底對夾輕搓;不用搓到粉碎,有裂縫就好

催芽後發芽時間縮短到 **7–10 天**(未催芽要 14–21 天,且許多根本不發)。這一步是香菜成敗的分水嶺。

- **前一天催芽** — 種子放碗裡加清水浸 24 小時;撈出後用手指搓、讓外殼有裂縫;濾乾備用

- **準備盆土** — 土填 8 分滿、表面耙平稍壓實

- **撒播種子** — 均勻撒在土面,盡量不要堆疊;每顆間距大約 2–3 公分

- **覆土 0.5–1 公分** — 不要蓋太深(超過 1.5cm 發芽率下降)

- **輕澆水** — 用細孔澆花壺輕澆,避免種子被沖走;澆到土面濕透、盆底剛好有水出為止

- **放在半日照、通風處** — 避免全天強日照;氣溫 15–22°C 最理想

- **保濕等芽** — 發芽前每天確認土面是否乾燥,稍乾就噴霧補水;7–10 天見芽

- **疏苗** — 苗長到 5cm 高後,疏苗留間距 5–8cm/株,拔掉的小苗可直接吃(撒在湯上)

## 📋 完整時程表 · 播種到採收

**播種 → 催芽 → 發芽 → 施肥 → 採收 → 抽苔警報**,一張表涵蓋整個秋冬季。**照做就對,不用觀察植物狀態**。

> 

**⚠️ 開工前先看月曆 / 節氣:挑對時間,不然種了白種**

香菜**喜冷涼怕高溫**,超過 25°C 生長遲緩、超過 30°C 迅速抽苔,葉子變苦、不能吃。**節氣比月份準**,兩套都列:

| | | | 月份 | | 節氣 | | 狀態 | |

| | ✅ 最佳 | | **10 – 2 月** | | **寒露(10/8)→ 雨水(2/19)** | | 氣溫穩 15–22°C,**手冊所有時程以此為準** | |

| | 🟡 勉強 | | 9 月 / 3 月 | | 白露(9/8) / 驚蟄(3/5) | | 邊緣期;9 月午間仍熱,3 月開始升溫,採收期短 | |

| | ❌ 絕對別播 | | **4 – 8 月** | | **清明(4/4)→ 處暑(8/23)** | | 高溫抽苔;就算發芽,採不到幾次就開花變苦 | |

俗話:**「寒露播,冬至吃,雨水前採完」**。如果現在是春夏,**不要播** — 先備盆土和種子,等寒露(10 月上旬)再下手。

## 🌾 施肥

**本 handbook 預設固體緩釋肥,不推液肥給新手** — 不用稀釋、不用算濃度、撒一次吃 3 個月、難燒根。

*施肥時機(Day 30 / Day 60)已統整在上面「📋 完整時程表」,這個 section 專講 **用多少 · 用哪款 · 怎麼撒**。*

> 

**📐 不同盆大小 · 速查(好康多 1 號)**

| | 盆 | | 顆數 | |

| | 3 吋圓盆(Ø9cm) | | **3 粒** | |

| | 5 吋圓盆(Ø15cm) | | **5 粒** | |

| | 7 吋圓盆(Ø21cm) | | **8 粒** | |

**原則**:寧可比這個少,不可比這個多([iGarden 園藝文摘官方指南](https://blog.igarden.com.tw/2013/01/hicontrol-fertilizer.html))。香菜是一年生短期蔬菜,不用大肥 — 夠用就好。

**🛒 買哪一款**

- **好康多 1 號**(NPK 14-11-13) — 萬用款,葉菜 / 香草都適用

- 350g 小包約 NT$100,一包可用好幾季

- 全聯園藝區、花市、iGarden 網購都有

- 替代品:魔肥、奧綠肥(類似緩釋顆粒都 OK)

> 

**📱 記憶外包 — 播種當天把整季施肥做完**

不要依賴記憶,也不要等到「感覺植物缺肥」才施。播種當天打開行事曆設兩個提醒:

- 播種日 + 30 天 → 「撒 5 粒好康多(20cm 圓盆)、澆水」

- 播種日 + 60 天 → 「再撒 5 粒」

整季決定一次做完。這是照護者 handbook 的核心哲學 — 決定外包到行事曆 / 便利貼 / 物理空間,不是記在腦裡。

## ✂️ 採收

**採收方法(兩種,擇一)**

- **外側摘葉法**(適合想持續採):從外圍較老葉片開始採,留中心嫩芽繼續生長 — 可以持續採 2–3 輪

- **割樁法**(適合一次大量):整株剪到距土面 3–5 cm,留根樁 — 會從根部再發新芽;採量大,適合烹飪用量多的時候

**採收節奏**

- **首採時機**:株高達 20 cm+ 且葉片茂密(約播種後 40–50 天)

- 秋冬氣溫穩,每 2–3 週可採一輪,總共 2–3 輪

- **不要等到抽苔才採**:一旦看到細長花莖,立刻大量採收,抽苔後葉片迅速變苦

**採後保鮮**

- 洗淨、甩乾水分、用廚房紙巾包好放冷藏 → 可保 5–7 天

- 或採後直接加進湯、麵;香氣最強的是剛採下那幾分鐘

> 

**🌸 抽苔後怎麼辦**

抽苔是香菜的自然生命周期,不是種失敗。

- 花莖冒出 → 立刻大量採葉(最後一次大收穫)

- 白色小花:可觀賞 1–2 週,也可放任開花吸引蜜蜂

- 花謝後結種球 → 待外皮轉褐色(完全熟)再採下 → 陰乾保存 → 秋天(10 月)拿出來催芽再播

- 這個循環:一次買種子,理論上可以自己留種不用再買

## 🛡 失敗點與除錯

| | 症狀 | | 可能原因 | | 對策 | |

| | 播了兩週沒發芽 | | 沒催芽 / 硬殼未搓破 / 土太乾 | | 沒催芽的重播一批;同時檢查土面保濕情況 | |

| | 發芽率極低(不到 3 成) | | 種子太老 / 夏季過熱 | | 買新批種子;確認播種月份在 10–2 月 | |

| | 幼苗細長倒伏 | | 光線不足 | | 移到更明亮的位置;疏苗讓每株有空間 | |

| | 葉片黃化、軟塌 | | 積水 / 爛根 | | 檢查排水孔是否堵塞;減少澆水頻率 | |

| | 播種後 1 個月就抽苔 | | 播種月份錯了(春夏播) | | 抽苔不可逆;立刻採收,等 10 月重播 | |

| | 葉片薄、顏色淡黃 | | 缺肥 / 日照不足 | | 確認施肥 Day 30 有做;移至光線較好位置 | |

| | 有葉片被蟲啃 | | 蚜蟲、薊馬 | | 用水強力沖葉背;香菜是要吃的,不推農藥 | |

## 🧠 失智照護應用 Notes

### 輕度

> 

**📍 輕度:她還能主導,你提醒觸發**

她還能理解多步驟指令、走到窗邊盆栽、執行完整動作序列。香菜的催芽步驟是本手冊少數可以**讓長輩主導完成**的儀式感動作。

- **催芽搓種子**:把種子和小碗遞給她,說「幫我把這個泡水」— 這個動作跟她以前洗菜、備料的動作連結

- 隔天你在旁邊,她搓種子 — 這個觸覺動作很適合手部靈活度尚在的輕度階段

- 播種時她可以自己撒種、自己覆土、自己澆水 — 完整的「做了一件事」的成就感

- 疏苗時拔出的小苗讓她拿去廚房洗、加進今晚的湯 — **「我種的、我採的、我放進去的」**整個鏈結

- **目標**:最小化你的介入,讓她的自主性成為照護本身

這個階段的價值:**香菜的氣味是幾十年廚房記憶的觸發器**。搓種子那一刻的香氣,可能帶出她自己種菜、煮飯的記憶。

### 中度

> 

**📍 中度:香氣是最好的引路人**

她的多步驟動作開始卡頓,但**香菜腺體的氣味極強** — 即使不記得「香菜是什麼」,那個熟悉的廚房氣味仍能引起情緒反應和動作意願。

- **摘葉時先讓她聞**:你摘一片葉子揉一下放到她手邊,說「聞聞看」— 不要求她說出名字

- 言語指令**單一動作**:「幫我洗一下這個」、「把這個放進去」,不說「先洗、再切、再放」

- **洗葉動作**:把剛採的香菜放在她手邊的水盆,「幫我沖一下」— 這個反覆沖洗的動作很適合中度

- 把洗好的香菜放在她面前的碗,不需要要求她做下一步 — 她參與了這個儀式就夠了

- **盆栽位置固定**,放在她日常動線顯眼處;窗邊的綠色和氣味是被動感知,不需要她主動記得

這個階段的價值:**把「廚房氣味」帶進她的每天,讓嗅覺記憶做語言做不到的事**。

### 重度

> 

**📍 重度:不要求動作,要求存在**

語言和主動動作有限。香菜的價值在這個階段**從「活動」轉為「感官陪伴」**。嗅覺是人類最後退化的感知之一 — 重度失智者通常仍能對氣味有反應。

- 帶她坐在盆栽旁邊,你摘葉子給她聞;不要求她做任何事

- 把一枝採下的香菜放在她手裡讓她握著 — 觸覺 + 嗅覺同時

- 揉碎一片葉子放在她手背上輕輕塗抹 — 氣味直接接觸皮膚

- 把香菜加進她今天的湯或麵,讓熟悉的味道在餐桌上出現

- **每天同樣時段、同樣動作** — 節奏感比「完成度」更重要;她不需要知道這是香菜

這個階段的價值:**氣味可以在沒有語言、沒有記憶的情況下,喚起一種說不清楚的安心感**。這是香菜在本手冊所有植物裡嗅覺刺激最強的一個 — 重度階段尤其珍貴。
