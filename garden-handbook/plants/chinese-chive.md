## 💡 為什麼值得種韭菜

- **種下去幾乎不死** — 多年生宿根植物,一次種可以採好幾年,冬天半休眠、春天自己醒

- **越剪越長** — 採收後 2–3 週長回來,一年可採 6–8 次,採了就是賺到

- **台式料理的靈魂食材** — 水餃、韭菜盒子、炒蛋、炒蝦仁,家裡種一盆就有

- **香氣是記憶觸發器** — 剪下韭菜那股味道,很多台灣長輩聞到就會想起媽媽的廚房

- **剪韭菜動作極簡單** — 拿剪刀沿根部一刀,失智長輩還能做;比摘葉菜更容易

- **多年生 = 環境穩定** — 盆子固定放,長輩每天看到同一盆,不用換盆換位置,對失智照護友好

## 🛒 取得方式

> 

**🗓️ Step 0 · 先看月曆再動手**

**最佳播種:3 月(驚蟄-春分)或 9–10 月(白露-寒露)**。韭菜雖然四季能活,但種子在高溫夏天發芽率低、寒流冬天生長幾乎停擺。若現在是 1–2 月寒流期,**先備盆土,等 3 月驚蟄(3/5)後再下種**。分株則全年皆可,只要氣溫不極端。

> 

**📦 推薦的新手包 · 三步照順序來**

- **先備盆與土(這一步要比種子/苗先到)** — 花市買 **長條盆 50×20cm 以上 + 培養土**(合計 NT$150–200),填土 8 分滿、澆一次水讓土服貼。

- **再取得種苗** — 優先找**朋友/鄰居分株**(免費,成活最快);其次花市買小盆韭菜苗(NT$30–60)回來分株;最後才考慮蝦皮買種子(NT$30–50,但等發芽要耐心)。

- **到貨/備齊當天立刻種** — 分株:一叢 3–5 根,間距 10 cm 插進土、澆透;種子:薄撒一層、覆土 0.5 cm、澆透

⚠️ **順序不能顛倒**:分株苗離開母叢後根部暴露失水,如果到家才開始找盆買土,苗在袋子裡悶一天就萎軟。總預算 **NT$250 有找**(分株路線最省)。

**三種取得管道比較**

| | 來源 | | 費用 | | 等待 | | 推薦度 | |

| | **朋友 / 鄰居分株** | | 免費 | | 2–3 週可採 | | ⭐⭐⭐ 首選 | |

| | **花市盆栽苗分株** | | NT$30–60 | | 2–3 週可採 | | ⭐⭐⭐ 首選 | |

| | **蝦皮 / 農會種子** | | NT$30–50 | | 60–70 天 | | ⭐⭐ 適合想體驗發芽的人 | |

分株比種子快 6–8 週,**新手強烈建議分株起步**。

## 🏺 盆器與土壤

**盆器**

- **長條盆**(50×20cm 以上) — 韭菜叢生,長條盆空間利用率最高

- 深度 **20 cm 以上**(根系需要空間)

- 盆底**必須有排水孔**(積水爛根)

- 圓盆也行,挑 **7 吋(Ø21cm)以上**

**土壤**

- **肥沃排水佳的培養土** — 一般培養土即可,可混入少量珍珠石改善排水

- 種前先在底層拌入一把**有機基肥**(雞糞肥、堆肥)打底 — 韭菜耗肥,底肥要足

- 不要用太重的黑土或砂質太多的土 — 前者積水,後者保肥差

*施肥細節獨立成下面的「🌾 施肥」section。*

## 🌳 種植步驟

**分株(推薦)**

- **準備盆土** — 長條盆填培養土 + 底部拌有機基肥,填 8 分滿、澆一次讓土服貼

- **分株** — 把韭菜叢輕輕分開,每份保留 3–5 根根系

- **插進土** — 間距約 10 cm,根部埋進土 3–5 cm,露出葉子部分

- **澆透水** — 澆到盆底有水流出

- **放半日照以上** — 陽台或戶外皆可,全日照更好

**種子播種**

- **準備盆土**(同上)

- **撒種** — 種子薄薄撒一層,不要重疊

- **覆土** — 蓋 0.5 cm 薄土

- **輕澆水** — 用噴壺或細水柱,不要沖移種子

- **保持濕潤** — 發芽前每天檢查土表,乾了就噴水

> 

**✂️ 採收關鍵:留根部 2–3 cm**

韭菜採收要**從根部 2–3 cm 處剪**,保留生長點。剪太低傷生長點、剪太高下次長得慢。用剪刀比刀乾淨俐落。

## 📋 完整時程表 · 播種到採收

**播種/分株 → 發根 → 施肥 → 採收 → 年度循環**,一張表涵蓋整年。**照做就對,不用每天觀察**。

> 

**⚠️ 開工前先看月曆 / 節氣:挑對時間效果差很多**

韭菜雖然四季能活,但播種、換盆時機差很多。**節氣比月份準**(反映實際氣溫日照),兩套都列:

- 350g 小包約 NT$100,一包吃好幾年

- 全聯園藝區、花市、iGarden 網購都有

- 替代品:魔肥、奧綠肥(類似緩釋顆粒肥都 OK)

> 

**📱 記憶外包 — 每次採收後立刻設下次採收提醒**

韭菜的施肥不是固定天數,而是「採一次補一次」。每次採完立刻:

- 撒好康多、澆水(當下馬上做)

- Google 行事曆設 **今日 + 21 天** 提醒「韭菜可以再採了」

**採完立刻補肥+設提醒,兩個動作打包做完**。這是照護者 handbook 的核心哲學 —— 決定外包到行事曆,不靠觀察植物。

## ✂️ 採收

**基本採收**

- 葉高達 **20 cm 以上**就可以採

- **從根部 2–3 cm 處剪割** — 保留生長點,才能再長回來

- 用剪刀一刀橫剪,比刀切乾淨俐落

- 剪完後 **2–3 週**可再採;採後**立刻補肥**(見施肥 section)

- 一年可採 **6–8 次**

**春天韭菜花**

- 春天(3–4 月)韭菜會抽出花苔 — 這就是**韭菜花**

- 花苔嫩時可採下,炒蛋、涼拌都好吃

- 若想集中讓葉片生長,趁花苔還嫩時整支剪掉,能量回歸葉片

- 讓它開花結籽後葉片會變細弱 — 通常不划算

**分株換盆(每 2–3 年一次)**

- 叢太密、葉片越長越細 → 到了分株時機

- 春(3 月)或秋(9–10 月)進行,避開夏天高溫

- 把整叢挖出,輕輕分開,每份 3–5 根,重新植入新土

- 分株後如同新植,從 Day 1 重新計時程

| | 葉片越長越細、越長越短 | | 缺肥(最常見) | | 採收後一定補肥;連補兩次看改善 | |

## 🧠 失智照護應用 Notes

### 輕度

> 

**📍 輕度:水餃日的食材她自己準備**

韭菜與水餃、韭菜盒子強綁。包水餃是很多台灣長輩做了幾十年的事。韭菜一盆種在陽台,就是一條「有意義的事情可以做」的線索。

- **水壺位置固定**,放在盆旁邊,她一看就知道要澆

- 口頭觸發就夠:「韭菜長高了,可以採了」—— 她自己走去剪、洗、切(如果她還能拿剪刀/刀)

- 以韭菜為中心辦一次家庭活動 —— 一起包水餃、一起吃,讓採收有出口

- 切韭菜的節奏感、香氣、濕潤觸感都是台式廚房的感官記憶

- **目標**:維持她「我還是那個會做韭菜水餃的人」的自我感,不要剝奪

這個階段的價值:**用一個家常食材,接回她一輩子的廚房身份**。

### 中度

> 

**📍 中度:你剪她洗她分**

她能執行單一動作,但「自己走去陽台 + 拿剪刀 + 剪 + 回來洗」這種多步驟會卡住。拆解動作、你做前置、讓她做中間段。

- **你先剪好一把,拿進廚房交給她洗** — 洗菜是高度自動化的動作,肌肉記憶強

- 洗完讓她「用手撕段」—— 不需要刀,動作安全,手感好

- 包水餃時讓她幫忙拌餡 — 攪拌動作她可能有肌肉記憶,聞到韭菜香就啟動

- 言語指令**一次一個動作**:「幫我洗一下這些韭菜」而不是「先洗再切再拌」

- **盆栽位置要在她日常動線顯眼處** — 放她看得到的地方,不要藏在 6 樓讓她自己找

這個階段的價值:**把她的動作外包給物理空間安排**,不靠她記得要做什麼。

### 重度

> 

**📍 重度:香氣與記憶**

不要求動作。韭菜的價值從「活動」轉為「感官陪伴」—— 那股台式廚房氣味是幾十年的記憶。

- 剪一小把放到她面前,讓她**聞韭菜香氣** — 嗅覺記憶是失智晚期保存最久的感官通道之一

- 做韭菜水餃時讓**廚房氣味瀰漫**,帶她坐在旁邊,不要求任何動作

- 把新鮮韭菜放她手上讓她握著,感受葉片的**濕潤觸感和重量**

- 吃水餃時送一顆到嘴邊,熟悉的**韭菜味道可能觸發微笑或話語**

- 每天相同時段帶她到盆栽旁,**節奏感比完成度重要**

這個階段的價值:**她不需要「做」,她需要「參與」這個家常儀式**。韭菜的香氣、台式廚房的聲音 —— 比任何動作更珍貴。
