# 🥬 茼蒿 家庭種植指南

> 類別:**葉菜** · 難度:低(種子撒播就活) · 季節:播種 10–2 月(夏天不發芽)

## 🎯 5 秒總覽(5W1H)

- **👥 Who**:新手友善 · 兒童可參與
- **🌱 What**:茼蒿種子(當年採收) 1 包 · 長條盆 50cm · 培養土 5L
- **📅 When**:播種 10–2 月(夏天不發芽)
- **📍 Where**:花市 買 / 陽台 / 窗邊 / 戶外 種
- **🔧 How**:10/11/12/1/2 月 下手,照「完整時程表」做

## 基本資訊

| 欄位 | 值 |
|--|--|
| 難度 | 低(種子撒播就活) |
| 季節 | 播種 10–2 月(夏天不發芽) |

## 🛒 購物清單

- **繁殖材料**:茼蒿種子(當年採收) 1 包 (花市, NT$30–50)
- **盆器**:長條盆 50cm × 1 個 (NT$150–250)
- **土壤**:一般培養土 5L (NT$50–100)
- **肥料**:好康多 1 號 350g (多盆共用)

## 💡 為什麼冬天必種茼蒿

- **冬天火鍋王** — 涼季才好吃(夏天苦),11 月 – 隔年 2 月是盛產期

- **生長快** — 發芽 5–10 天、採收 25–50 天,比地瓜葉還快

- **可多次採收** — 摘 5–6 葉留底部 4–5 葉,能一直再發

- **香氣濃** — 葉子一揉就有菊花/胡椒味,嗅覺刺激強

- **種子超便宜** — 一包幾十塊可種整季

## 🛒 取得方式

> 

**🗓️ Step 0 · 先看月曆再下手**

**現在要是 10 月 – 2 月(寒露 → 雨水)才開始。** 茼蒿**喜涼怕熱**,發芽最適溫度 15–20°C,夏天高溫整批不發芽或長出來就抽苔開花。如果現在是春夏,**等到寒露(10/8)前後**再下手,現在先備盆土和種子。

> 

**📦 推薦的新手包 · 種子最划算**

- **種子** — 花市/農會/蝦皮搜尋「茼蒿種子」,一包 NT$30–50 可種一季

- **盆與土** — 長方盆 or 淺圓盆(深 15 cm 夠)+ 培養土(合計 NT$150–200)

- **先備盆、再催芽、再播** — 見下面步驟

總預算 **NT$200 有找**,一包種子種好幾年。

**種子來源**

- **花市 / 農會 / 五金行**:農友種苗、興農等品牌,包裝日期越新越好

- **蝦皮**:搜尋「茼蒿種子」,選**當年採收**的(舊種子發芽率低)

- **朋友家分**:茼蒿開花後自己會留種,一株能留很多

> 

**⚠️ 小心:「山茼蒿」不是「茼蒿」**

市場上的「山茼蒿」(昭和草)是另一種植物,用扦插不是種子。這個 entry 講的是一般**葉用茼蒿**(菊科,種子栽培)。

*資料來源:[農業部 · 茼蒿如何種植](https://kmweb.moa.gov.tw/knowledge_view.php?id=8363)、[風物日誌 · 茼蒿種植指南](https://pqlife.tw/blog/438)*

## 🏺 盆器與土壤

**盆器**

- 深 **15 cm 以上**(根淺不用太深)

- **長方盆最適合** — 撒播面積大,一盆種一排

- 盆底**排水孔必備**

**土壤**

- 一般培養土 + 少量有機肥當基肥

- **肥沃、排水佳** — 茼蒿是肥料吃重的作物,土肥長得好

- pH 6.0–7.0(一般培養土都符合)

*施肥細節獨立成下面的「🌾 施肥」section。*

## 🌳 種植步驟

> 

**🧪 催芽技巧(提高發芽整齊度)**

茼蒿種子發芽率中等,**播種前做這兩步**會讓發芽更齊、更快:

- 種子浸泡**清水 1 天**(讓種皮吸水)

- 撈出放**冰箱冷藏 1 天**(不是冷凍!模擬冬天低溫)

趕時間可以跳過,但發芽會參差不齊。

- **盆填土** — 培養土填 8 分滿,澆一次水讓土服貼

- **撒播種子** — 均勻撒在土表(間距隨意,待會兒疏苗)

- **覆土** — 薄薄蓋 **1–2 cm** 厚土

- **澆水** — 用噴霧或蓮蓬頭灑水(澆太重會沖走種子)

- **放半日照或全日照** — 秋冬陽光溫和,光越多越好

- **保持土壤微濕** — 5–10 天就會冒芽

**疏苗(關鍵!)**

撒播出來會長得密密麻麻,**不疏苗就會互相搶養分,全部長得瘦弱**:

- **長出 2 片本葉時** → 第一次疏苗,間距 **3 cm**

- **長出 4–5 片本葉時** → 第二次疏苗,間距 **10–15 cm**

- 拔起來的幼苗**可以當「茼蒿芽」吃**(嫩脆有香氣)

## 📋 完整時程表 · 播種到採收

**播種 → 發芽 → 疏苗 → 採收**,秋冬茼蒿生長週期短、可多次採。

> 

**⚠️ 開工前先看月曆 / 節氣:挑對時間,不然整批不發**

茼蒿**喜涼怕熱**,發芽最適 15–20°C,跟蒜苗一樣是「秋冬檔」:

| | | | 月份 | | 節氣 | | 狀態 | |

| | ✅ 最佳 | | **10 月中 – 2 月** | | **寒露(10/8)→ 雨水(2/19)** | | 發芽率高、生長穩、最好吃 | |

| | 🟡 勉強 | | 9 月 / 3 月 | | 白露 / 驚蟄 | | 邊緣期,發芽慢 | |

| | ❌ 絕對別種 | | **4 – 8 月** | | **清明(4/4)→ 立秋(8/7)** | | 高溫不發芽,發了也會抽苔變苦 | |

| | 天數 | | 階段 | | 植物狀態 | | 你的動作 | |

| | **Day -2 – -1** | | 催芽 | | 種子準備中 | | 泡水 1 天 + 冰箱冷藏 1 天 | |

| | **Day 1** | | 播種 | | 種子撒土 | | 覆土 1–2 cm,噴霧澆水;❌ 不施肥 | |

| | Day 5–10 | | 發芽期 | | 冒出小綠芽 | | 保持土微濕,別澆太重 | |

| | **Day 10–14 ✂️** | | **第一次疏苗** | | 2 片本葉 | | ✅ 間距 3 cm,拔掉多的(幼苗可吃) | |

| | **Day 15 🌾** | | **第一次施肥** | | 穩定長葉 | | ✅ 撒 **12–15 粒** 好康多 1 號(50cm 長方盆) | |

| | **Day 20 ✂️** | | **第二次疏苗** | | 4–5 片本葉 | | ✅ 間距 10–15 cm,拔掉的當嫩菜吃 | |

| | **Day 25–30 ✂️**
*(9–11 月播種)* | | **首次可採收** | | 株高 20 cm、葉 ~10 | | ✅ 摘 5–6 葉、留底部 4–5 葉 | |

| | **Day 30–50**
*(12 月 – 2 月播種)* | | 冬季稍慢 | | 葉量較慢長 | | 耐心等,冬菠菜/冬茼蒿最甜 | |

| | Day 30–90 | | 多輪採收期 | | 側芽不斷長 | | ✅ 每 1–2 週採一次 | |

| | **春天氣溫升(清明後)** | | **抽苔警戒** | | 開始冒花苞 | | 立刻整株拔收;或留 1–2 株自留種子 | |

> 

**📱 播種當天就要做的 2 件事**

- Google 行事曆設 3 個提醒:**今日 + 10 天**(第一次疏苗)、**今日 + 25 天**(第一次採)、**隔年 3 月**(留意抽苔)

- 便利貼寫今日日期 + 種子品牌,貼盆邊或廚房

## 🌾 施肥

**茼蒿是吃肥重的葉菜 → 氮(N)要重**,但週期短(30–50 天採完),施肥不用多次。

> 

**📐 用量(好康多 1 號)**

- 小盆(Ø15cm):**5 粒**

- 中盆(Ø21cm):**8 粒**

- **長方盆 50×20cm:15 粒**(最常見)

**時機**:Day 15 第二次疏苗後一次性施,撒完澆水。整個生長期通常這一次就夠。

**什麼時候不用施**

- 土本身有混基肥(有機肥顆粒)→ 可以不用再追

- 只打算種一輪就重播 → 可以省略

**想加強葉量**

- Day 30 第一次採收後,如果想多輪採,可以再撒 **一半量(7 粒)**

- 或**薄液肥**(花寶 1 號稀釋再減半)每 10–14 天一次

通用施肥原則看 [📚 通用園藝知識 → 🌾 施肥](#)。

## ✂️ 採收

**多次採葉(推薦)**

- 株高 **20 cm、葉數 ~10 片**時開始採

- **摘頂端嫩葉 5–6 片**,留底部 4–5 葉繼續長

- 側芽會從葉腋冒出,2 週後可再採

- 一盆可採 **4–6 輪**,整個冬天有菜

**整株拔收**

- 春天氣溫升高冒花苞時 → 立刻整株拔(不然會變苦)

- 或想讓盆重新規劃時整批拔

> 

**🍲 採下來馬上能用**

- **火鍋首選** — 涮幾秒就起鍋,甜味最足

- **燙熟加醬油麻油** — 最簡單的吃法

- **炒蛋、炒肉絲** — 香氣釋放

- 留種子:最後 1–2 株不採,等開花結子,隔年自留種

## 🛡 失敗點與除錯

| | 症狀 | | 可能原因 | | 對策 | |

| | **整批不發芽** | | 溫度太高(>25°C)/ 種子太舊 | | 等氣溫降再播;買當年種子 | |

| | 發芽稀疏 | | 覆土太厚 / 種子太老 | | 覆土 1–2 cm 就夠;催芽步驟加回去 | |

| | 幼苗倒伏 | | 立枯病(土太濕 + 通風差) | | 減澆水,搬到通風處,拔掉病株 | |

| | 葉子瘦小 | | 沒疏苗,擁擠 | | 按時間點疏苗(Day 10 + Day 20) | |

| | 抽苔開花 | | 氣溫太高(>25°C) | | 整株拔收;下次避開春末夏天 | |

| | 葉黃脫落 | | 缺肥 / 缺水 / 老化 | | 補薄液肥;免洗筷檢測 | |

| | 蚜蟲爬葉背 | | 冬天乾燥常見 | | 水柱沖走;或用肥皂水(稀釋 1:50)噴 | |

## 🧠 失智照護應用 Notes

### 輕度

> 

**📍 輕度:火鍋記憶的延伸**

茼蒿的香氣對台灣長輩來說 **= 冬天火鍋**,種、採、吃是同一條記憶線。

- 她自己撒種、疏苗、摘葉 — 每個動作都簡單清晰

- 摘葉時**讓她聞葉子的香氣**,問「這像什麼?」喚起火鍋回憶

- 採完一起煮火鍋,「這是我種的茼蒿」的儀式感

- 冬至、除夕這類節氣一定煮,**把種植綁定到節日**

### 中度

> 

**📍 中度:摘葉是單一動作**

撒種/疏苗太複雜,但**「摘葉」是完美的單一動作**。

- 你負責播種、疏苗、澆水;**請她負責「摘葉」**

- 採收時你剪下幾株放她面前的砧板,請她**一片片摘下嫩葉**放碗裡

- 葉香在她手上停留 —— 嗅覺跟視覺雙重刺激

- 煮火鍋時讓她下菜,熟悉動作不會丟

### 重度

> 

**📍 重度:香氣陪伴**

動作不求,**香氣是主軸**。茼蒿香氣濃郁,能穿透嗅覺退化。

- 採一把讓她**用手揉葉子** —— 揉的瞬間香氣爆開

- 放一小盆茼蒿在她坐的地方旁邊,路過隨時能聞

- 煮火鍋時讓她在廚房待著,香氣 = 家 = 冬天

- 吃時剪細,滑進嘴裡,味道是最強的記憶線索
