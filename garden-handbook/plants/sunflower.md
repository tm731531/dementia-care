## 💡 為什麼向日葵是家庭種植的最佳入門花卉

- **種子撒下去幾乎不會死** — 發芽率極高,播種到發芽只要 5–10 天,是所有花卉裡面進場門檻最低的

- **花真的會跟著太陽轉** — 向日性(heliotropism)發生在幼苗期,每天早上朝東傍晚朝西,是活生生的自然課

- **從種子到大花有 90 天的共同目標** — 每天都能看到生長變化,給照顧者和長輩一個「明天繼續期待」的理由

- **矮性品種完全適合陽台盆植** — 「玩具熊」「迷你笑臉」等品種 40–80 cm,盆植不需要地栽空間

- **花謝可以採葵花子** — 從播種到採種一個完整生命週期,長輩親眼見證

- **跨代互動神器** — 小孩和長輩可以一起種,花大顏色鮮,拍照就有成就感

## 🛒 取得方式

> 

**🗓️ Step 0 · 先看月曆再動手**

**現在要是 3–6 月才播種。** 秋冬(9 月後)日照時間不夠,向日葵需要每天 6 小時以上全日照,光照不足就徒長、不開花。如果現在是秋冬,**等到隔年 3 月驚蟄後**再下手;現在就先備好盆土,把品種選好。

> 

**📦 推薦的新手包 · 三步照順序來**

- **先備盆與土(這一步要比種子先到)** — 家附近花市買 **直徑 30 cm 以上塑膠盆 + 一般培養土**(合計 NT$150–200),盆要有排水孔,土填 8 分滿、先澆一次水讓土服貼。

- **再買種子** — [蝦皮搜尋「向日葵 矮性 種子」](https://shopee.tw/search?keyword=%E5%90%91%E6%97%A5%E8%91%B5+%E7%9F%AE%E6%80%A7+%E7%A8%AE%E5%AD%90),或花市直接買一包。NT$30–80 一包(約 15–30 粒),多家包郵。

- **到貨當天立刻播** — 種子直接戳進土 1–2 公分深 → 澆透 → 放全日照陽台

⚠️ **品種選矮性、盆植型**(見下面品種說明),不要買高性種(2 m+),陽台放不了。總預算 **NT$250 有找**。

**品種選擇 — 陽台種必看**

| | 類型 | | 株高 | | 適合場地 | | 推薦品種 | |

| | **矮性盆植種** ✅ 新手首選 | | 40–80 cm | | 陽台盆栽 | | 「玩具熊」「迷你笑臉」「中矮性向日葵」 | |

| | 切花用中性種 🟡 | | 1–1.5 m | | 大型陽台 / 庭院 | | 「向日葵切花品種」 | |

**其他取得管道**

- **全聯、農改場展售**:清明前後常有種子包,現場可問品種株高

- **朋友分籽**:去年種過、花謝留種的人很多,開口要幾粒就好

- **自留種**:花謝後讓花盤乾在植株上,剝開就是下一代的籽,免費

## 🏺 盆器與土壤

**盆器**

- 矮性種:直徑 **25–30 公分**、深 25 公分以上(根系垂直向下)

- 中高性種:直徑 **35 公分以上**或深盆,否則頭重腳輕易倒

- 盆底**必須有排水孔** — 積水是向日葵最常見的死因

- 建議**重一點的陶盆或水泥盆**:向日葵長大會頭重,颱風天塑膠輕盆容易翻倒

**土壤**

- 一般培養土即可,或混入 20–30% 珍珠石改善排水

- 向日葵**耐貧瘠但不耐積水** — 寧可土差一點,不要積水

- pH 6.0–7.5 都可,台灣一般培養土直接用沒問題

## 🌳 種植步驟

- **選盆定位** — 找到陽台或窗邊每天至少 6 小時照得到太陽的位置,先把盆放到位,後面不要再搬(向日葵定植後不喜歡移動)

- **填土** — 培養土填至盆緣下 3 公分,用手輕壓讓土密實一點

- **播種** — 每個盆播 3–5 粒,種子尖端朝下或隨機都可,戳進土約 1–2 公分深,種子間距 10 公分以上

- **第一次澆水澆透** — 澆到盆底有水流出,之後保持土面微濕到發芽

- **等發芽(5–10 天)** — 放全日照處;發芽前土面不要乾透

- **疏苗(Day 14 左右)** — 每盆留 1–2 株最壯的,其餘剪掉(不要拔,拔傷根)

> 

**⚠️ 向日葵不喜歡移盆**

根系主根深而脆,定植後移動容易傷根、導致生長停滯。一開始就找好位置、選好盆,**播下去就是它的家**。

## 📋 完整時程表 · 播種到採種

**播種 → 發芽 → 生長 → 開花 → 採種**,一張表涵蓋整個生命週期。**照做就對**。

> 

**⚠️ 開工前先看月曆 / 節氣:光照不對,花開不了**

向日葵**需要全日照**,秋冬日照不夠就徒長、不開花,浪費整季。**節氣比月份準**(台灣農業推廣標準),兩套都列:

| | | | 月份 | | 節氣 | | 狀態 | |

| | ✅ 最佳 | | **3 – 6 月** | | **驚蟄(3/5)→ 夏至(6/21)** | | 日照長、氣溫升,生長旺盛,**手冊所有時程以此為準** | |

| | 🟡 勉強 | | 2 / 7 / 8 月 | | 雨水(2/19) / 小暑(7/7) / 大暑(7/22) | | 2 月偏冷、7–8 月酷暑可能熱傷,成功率降低 | |

| | ❌ 絕對別播 | | **9 – 1 月** | | **白露(9/8)→ 大寒(1/20)** | | 日照不足,植株徒長無力、開花率極低 | |

## 🌾 施肥

**本 handbook 預設固體緩釋肥,不推液肥給新手** — 撒一次吃 3 個月、不用稀釋、難燒根。向日葵施肥需求**中等**,磷鉀比氮更重要(促花,不要只長葉)。

- 想更促花可選 NPK 含磷(P)較高的款式,但好康多 1 號已夠用

- 350g 小包約 NT$100,全聯園藝區、花市、蝦皮都有

- 替代品:魔肥、奧綠肥(類似緩釋顆粒)

> 

**⚠️ 開花期停肥,不要再加**

花苞出現後(Day 60 左右)就不要再施肥。這時候過多氮肥反而讓植株猛長葉子、花盤縮小。**開花後:只澆水,不施肥**。

## ✂️ 採收與採種

**賞花**

- 花期約 2–3 週,每天欣賞就是最好的「採收」

- 可以剪一支插花瓶:用乾淨剪刀斜剪莖,立刻插入水,放室內陰涼處可賞 5–7 天

- 矮性種通常會從側枝繼續冒花苞,一棵可以開 3–5 朵不等

**採種(葵花子)**

- 花謝後**不要剪掉花盤**,讓它留在植株上自然乾燥(約 30 天)

- 花盤背面變黃、種子用手指可以輕鬆剝落,就是採種時機

- 剝下的種子鋪開在報紙或托盤上,室內通風處再晾 3–5 天

- **明年播種用**:放紙袋密封,寫好品種和日期,放冰箱保鮮層保存

- **炒來吃**:洗淨後加鹽水煮 5 分鐘,撈起瀝乾,烤箱 150°C 烤 20 分鐘,或鍋炒到殼略焦

> 

**🌱 自留種是最好的教育**

讓長輩親手把種子收進袋子、貼上標籤、放進冰箱 —— **這個動作就是「我讓它繼續活下去」的具象化**。明年春天拿出來再播,是跨越一整年的連結。

## 🛡 失敗點與除錯

| | 症狀 | | 可能原因 | | 對策 | |

| | 種下去 10 天沒發芽 | | 種子太舊/土太濕/太深 | | 檢查種子是否壞掉(泡水看沉不沉);重播,深度只要 1–2 cm | |

| | 苗又高又細、容易倒 | | 光照不足(徒長) | | 立刻搬到更曬的地方;補竹竿支撐。這不是肥的問題 | |

| | 葉片發黃、有褐斑 | | 過澆/真菌/積水 | | 停澆 3–5 天讓土乾;若爛根就換盆換土 | |

| | 側枝不開花 | | 正常現象 | | 矮性品種側枝花朵比主莖小,是正常的 | |

## 🧠 失智照護應用 Notes

### 輕度

> 

**📍 輕度:90 天共同目標 + 向日性觀察**

她還能理解「今天的花轉到哪個方向了?」這種問題,能自己走到陽台觀察,能說出感覺。向日葵的向日性是每天都有的話題觸發點。

- **每天早晨一起去陽台看花朝哪個方向** — 「今天還是朝東」這句話就夠了

- 讓她負責澆水:水壺位置固定,你只說「該澆花了」

- 採種時她能自己剝種子、放袋子、貼標籤 — 整個動作序列她都可以完成

- **播種那天拍照**:記錄「我們一起種下的這粒種子」,是 90 天後看到大花的對照

- 目標:讓她感覺**「這盆花是我種的」**,不是「孩子幫我種的」

這個階段的價值:**她還是那個有能力照顧生命的人**。向日葵是最直觀的「我種的東西活了」的成就感。

### 中度

> 

**📍 中度:大花視覺錨點 + 每天固定儀式**

認知能力下降後,鮮豔的大花是最強的注意力錨點。向日葵開花時那朵大黃花幾乎不可能被忽視 — 這是刻意設計的環境介入。

- 把盆放在**她每天必經的動線上**(客廳窗邊,不是 6 樓陽台),讓花自己吸引她的目光

- 澆水改成:你把水壺裝好放在盆旁邊,只說一句「幫我澆一下」

- 言語指令**單一動作一次**,不要說「先去陽台再澆再回來」

- 花期時**每天同樣時間帶她去看花** — 節奏感比每次看到不同的事重要

- 可以讓她做的:用手指輕觸花盤、感受花瓣的觸感、聞花的氣味

這個階段的價值:**大花是記憶錨點**。「那朵黃色的大花」比任何語言描述都更容易被記住。照顧者說「去看花」,她知道是哪朵。

### 重度

> 

**📍 重度:感官存在,不要求動作**

不要求她「做什麼」。向日葵在這個階段的價值是**感官刺激和情緒調節** — 鮮豔的黃色、花盤的觸感、葵花子的氣味。

- 帶她坐在花旁邊,你說「你看,這朵開了」 — 不要求回應

- 讓她**用手輕觸花盤**(花盤表面粗糙有質感,觸覺刺激強)

- 採種時把葵花子放在她手掌上讓她握著 — 重量感和溫度都有意義

- 如果她有剝殼的動作記憶,可以給她帶殼葵花子讓她剝着玩(監督下)

- 每天**同樣路徑、同樣時間**,不在乎「完成度」,在乎「節奏和陪伴」

這個階段的價值:**她不需要「做」,她需要「在」**。那朵大黃花、那粒種子的重量、葵花子的香氣 — 這些感官印記比認知更持久。90 天的共同目標,即使她記不住過程,那朵花開了的那一刻,她感受得到。
