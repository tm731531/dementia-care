## 💡 為什麼新手適合從地瓜葉開始

- **幾乎怎麼種都活** — 扦插成功率極高,是最皮厚的可食植物之一

- **長得快** — 夏天 3–5 天就可以採一把

- **一次買,吃整季** — 採收後不斷冒新葉

- **莖葉都可食** — 連梗都能炒

- **吃水量大** — 夏天要每天澆,**每天都有理由去陽台**(對失智照護超重要)

## 🛒 取得方式

> 

**🗓️ Step 0 · 先看月曆再動手**

**現在要是 4–9 月才開始。** 冬天(11 月 – 2 月)扦插**基本全死** — 低於 15°C 地瓜葉停止生長,新枝條沒體力發根就凍死。如果現在是冬天,**等到隔年 4 月清明後**再下手,現在就先備盆土、看蝦皮賣家(冬天很多不賣枝條)。

> 

**📦 推薦的新手包 · 三步照順序來**

- **先備盆與土(這一步要比枝條先到)** — 家附近花市買 **25cm+ 塑膠盆 + 一般培養土**(合計 NT$150–200),搬回家放好定位、土填 8 分滿、先澆一次水讓土服貼。

- **再下單枝條** — [蝦皮搜尋「地瓜葉 枝條」](https://shopee.tw/search?keyword=%E5%9C%B0%E7%93%9C%E8%91%89%20%E6%9E%9D%E6%A2%9D)(一批 20 條,NT$80–150,多家包郵)

- **到貨當天立刻插** — 剪 3–4 節一段 → 插進預備好的盆 → 澆透 → 放半日照處

⚠️ **順序不能顛倒**:枝條離開母株後水分流失快,如果到家才開始找盆買土,枝條在袋子裡悶一天就半死。總預算 **NT$300 有找**,不用懂園藝 —— 但**月份要對**(見上面 Step 0)。

**其他取得管道**

- **市場買帶莖的地瓜葉**:吃掉葉子,留莖插 → 免費起步

- **朋友家分**:種開後枝條多到用不完,可以跟種的人要

以上三種路徑成功率都 **80% 以上**,比塊莖發芽穩得多。

> 

**⚠️ 不推薦新手:地瓜塊莖發芽**

超市地瓜常被噴抑芽劑。曾有案例:2 月試了兩個月始終不發芽,改用扦插才成功。

## 🏺 盆器與土壤

**盆器**

- 直徑 **25 公分以上**、深 20 公分以上

- 越大越好(根系會擴散)

- 盆底**必須有排水孔**(不然夏天爛根)

**土壤**

- 一般培養土即可

- **排水要好** — 如果用黑土可以混入一些珍珠石或粗砂,避免積水

- 講究的可以先養土(加有機肥擺幾天讓土活化)

*施肥細節獨立成下面的「🌾 施肥」section。*

## 🌳 種植步驟

- **選盆** — 見上面「盆器與土壤」

- **剪枝條** — 每條留 3–4 個節點,頂端葉子留 2–3 片,下面剝掉

- **插進土** — 下方 2–3 個節點埋進土,露出 1–2 片葉子

- **第一次澆水澆透** — 澆到盆底有水流出

- **放在半日照以上的地方** — 斜照 OK,全日照更好

## 📋 完整時程表 · 扦插到採收

**扦插 → 發根 → 施肥 → 採收 → 越冬**,一張表涵蓋整年。**照做就對,不用觀察植物狀態**。

> 

**⚠️ 開工前先看月曆 / 節氣:挑對時間,不然全死**

地瓜葉**喜熱怕冷**,低於 15°C 停止生長,新插枝條**沒體力發根就凍死**。**節氣比月份準**(24 節氣反映溫度和日照,月份只是數字),兩套都列:

| | | | 月份 | | 節氣 | | 狀態 | |

| | ✅ 最佳 | | **4 – 8 月** | | **清明(4/4)→ 立秋(8/7)** | | 氣溫穩、生長旺,**手冊所有時程以此為準** | |

| | 🟡 勉強 | | 3 / 9 / 10 月 | | 春分(3/20) / 白露(9/8) / 寒露(10/8) | | 邊緣期,成活率降低 | |

| | ❌ 絕對別插 | | **11 – 2 月** | | **立冬(11/7)→ 雨水(2/19)** | | 成活率接近零(Tom 第一次冬天種,整批死透) | |

俗話:**「清明插、立秋前還能採」**。如果現在是冬天,**不要插** —— 先備盆土,等清明(4 月上旬)再下手。蝦皮賣家冬天很多停賣枝條,也是風向球。

**本 handbook 預設固體緩釋肥,不推液肥給新手** —— 不用稀釋、不用算濃度、撒一次吃 3 個月、難燒根。

## ✂️ 採收

- 採**頂端 5–10 公分的嫩葉莖**

- 留根部與一部分莖 → 會從節點繼續長

- **越採越多** — 定期採收刺激生長

- 一週至少採 1–2 次

## 🌱 繁殖 · 一盆變多盆(擴盆 SOP)

**地瓜葉長開後,*本盆就是最好的枝源*** —— 不用再買、不用等運送、成活率比蝦皮買的更高(本地、當天扦插、零損耗)。這是地瓜葉最大的隱藏福利。

**📅 什麼時候可以擴盆?看狀態不看日期**

- ✅ 藤蔓長到 **20–30 cm**、3–4 個明顯節點

- ✅ 葉色深綠健康、新葉持續冒

- ✅ 通常是**Day 30+**(氣溫穩的話可能 Day 25 就夠)

- ❌ Day 7–14 別剪 — 母株根剛穩,抽枝會萎

- ❌ 開花期別剪 — 能量分流

- ❌ 冬天別剪(11–2 月)— 對應地瓜葉本身的季節警告

> 

**⏰ 剪前先準備好新盆(原則跟新手包一樣)**

枝條離體**水分流失很快**,剪下來要 30 分鐘內插進土。所以新盆必須**在剪之前就備好**:清位置 → 盆填土 → 先澆一次水讓土服貼。講究的可以提前 1–2 週養土(加一點有機肥拌勻潤濕放著),但地瓜葉不挑,一般培養土就行。

**🔪 擴盆 SOP(剪到下盆一氣呵成)**

- **挑長藤** — 20–30 cm、看得見 3–4 個節

- **剪下來立刻處理** — 不放桌上忘記,不泡水拖延

- **頂端留 2–3 片葉**,下面葉子全剝(節點露出來會發根)

- **下方 2–3 節插進新盆土**,露出頂端葉

- **澆透**(盆底有水流出)

- **放半日照 3–5 天緩苗**,之後搬到正常位置

成活率 **80%+**,比蝦皮買的還穩。

**🏡 階梯擴盆策略(推薦)**

與其一次種很多盆,**每個月新增一盆**,做錯開成熟期:

| | 批次 | | 時機 | | 目的 | |

| | 第一批 | | Day 1 買枝條起步 | | 確認會活 | |

| | 第二批 | | 第一批 Day 30+ 擴盆 | | 擴產 / 備援 | |

| | 第三批 | | 第二批 Day 30+ 再擴 | | 錯開成熟期,連續採收 | |

| | 長期 | | 每個月新加一盆 | | 老盆淘汰也有新盆接棒 | |

> 

**📌 階梯的好處**

- **連續採收** — 不會出現「採光了要等一個月」的空窗

- **備援** — 某盆掛掉(蟲害、意外、長輩誤拔)還有其他盆

- **家庭動線** — 不同樓層/位置都有盆 = 每個地方都有理由去看看

- **送禮** — 枝條太多可以剪給鄰居朋友,推廣的同時也減少工作量

*同樣概念也適用九層塔、薄荷等藤蔓/分枝型植物,**凡是能扦插的就能自我繁殖**,不用每次買苗。*

## 🧠 失智照護應用 Notes

### 輕度

> 

**📍 輕度:她還能主導,你提醒觸發**

她還能理解指令、走到陽台、執行多步驟動作。地瓜葉的 ritual 可以做到**最完整**。

- **水壺位置固定**,擺在陽台入口她一看就知道

- 你只需要口頭觸發:「該澆花了」、「今天可以採來晚餐吃」

- 採收時她能自己剪、自己拿回廚房、自己分菜剝老葉

- **目標**:最小化你需要介入的動作。她的自主性最珍貴,不要剝奪

這個階段的價值:**維持她「我還是那個當媽的人」的自我感**。

### 中度

> 

**📍 中度:動作還在,每步要你引導**

她能執行單一動作,但「自己走去陽台 + 拿水壺 + 澆 + 回來」這種多步驟會卡住。

- **水壺事先裝滿水,擺在盆旁邊** —— 不要讓她找水龍頭(失敗點)

- 盆栽位置要在**她日常動線顯眼處**(放 5 樓陽台,不要放 6 樓讓她自己上樓)

- 言語指令**單一動作一次**:「幫我澆一下」而不是「先澆水再剝葉」

- 採收時**你剪好、放她面前的碗裡,請她剝老葉、去硬梗**。這個動作跟剝蒜頭同類型

- **不要讓她看到你先澆**,她會覺得「澆過了就不動」

這個階段的價值:**把她該做的決定外包到物理空間**(呼應 [blog A](https://blog.tomting.com/2026/04/22/dementia-care-three-misunderstandings/) 那條規則)。

### 重度

> 

**📍 重度:不要求動作,要求存在**

她的自主動作有限、溝通困難。地瓜葉的價值**從「活動」轉為「感官陪伴」**。

- 帶她坐在盆旁邊,你澆給她看

- 讓她**摸葉子**(絨毛觸感)、**聞土跟葉的味道**(嗅覺刺激)

- 你採的時候她在旁邊,把葉子放到她手上讓她握著,不要求她做什麼

- 剝老葉動作如果她還能做就給她做,不行就放在她面前讓她看

- 每天重複同樣時段、同樣路徑,**節奏感比「完成度」重要**

這個階段的價值:**她不需要「做」,她需要「參與」這個儀式**。地瓜葉的綠、葉子的軟、泥土的味道 —— 這些比動作更珍貴。


## 📓 實測紀錄

>
