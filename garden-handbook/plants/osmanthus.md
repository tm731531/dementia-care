## 🛒 購物清單

- **繁殖材料**:桂花盆苗(有花苞更好) 1 株 (花市, NT$200–500)

## 💡 為什麼有失智長輩的家庭值得種桂花

- **嗅覺是失智最後失去的感官** — 桂花香濃烈而特殊,即使重度失智仍能感應,其他植物難以替代

- **童年記憶鎖點** — 對許多長輩而言,桂花香連結著家鄉、老屋、中秋記憶;一聞就能喚起具體的情節片段

- **花可食用** — 採花泡茶、做桂花糕、釀桂花酒,整條「採花 → 加工 → 吃到嘴」的流程是完整的感官記憶

- **秋天花期 = 好天氣陽台時間** — 9–11 月氣溫宜人,正是帶長輩坐到陽台曬太陽的好時機,桂花同步開放

- **常綠低維護** — 非花期照養簡單,不像草本容易枯萎,長輩每天都能看到「它還在那裡」

- **品種豐富可選** — 金桂橙黃、銀桂白色、丹桂深橙、四季桂偶開,依偏好挑一棵

## 🛒 取得方式

> 

**🗓️ Step 0 · 先看月曆再動手**

**定植最佳時機:春(3–4 月)或秋(9–10 月)**。桂花是木本灌木,根系建立需要 3–6 個月,盛夏酷熱(7–8 月)新苗容易熱休克,寒流期(12–2 月)新苗難發根。

如果現在是盛夏或寒冬:先備好盆土,等氣溫回到 18–28°C 再種苗。

> 

**📦 推薦的新手包 · 三步照順序來**

- **先備盆與土(這一步要比苗先到)** — 花市買 **30–40cm 深盆 + 微酸性培養土(或蘭花土混通用土 1:1)**(合計 NT$200–400),搬回定位、土填 8 分滿、澆透

- **再買苗** — 花市購買盆栽苗 **NT$200–500**,挑葉色深綠、枝條充實的;買的時候順口問:「是哪個品種?」金桂/銀桂香氣最濃

- **到家當天立刻定植** — 脫盆、不破壞根球,放進新盆,填土壓實,澆透,放半日照處緩苗一週

⚠️ **順序不能顛倒**:苗到了沒有盆,放著悶一天根會受損。總預算 **NT$700 有找**。

**其他取得管道**

| | 管道 | | 費用 | | 適合時機 | |

| | 花市攤位 | | NT$200–500 | | 春秋兩季最多貨 | |

| | 蝦皮花卉 | | NT$150–400 | | 全年,注意運送對苗的衝擊 | |

| | 朋友家扦插 | | 免費 | | 春末(4–5 月)半熟枝扦插,需配生根粉 | |

| | 農業試驗所展售 | | NT$100–200 | | 不定期,品種標示清楚 | |

> 

**⚠️ 品種選購建議**

新手建議選**金桂或銀桂**——花期集中(9–11 月)、香氣最濃。**四季桂**雖然偶爾全年開,但每次花量少、香氣較淡,不適合作為「桂花香引動記憶」的主角。丹桂橙紅色花好看但香氣略遜。

## 🏺 盆器與土壤

**盆器**

- 直徑 **30 公分以上**、深 30 公分以上(桂花根系垂直向下)

- 木本灌木根系大,**每 2–3 年需換大一號盆**

- 盆底**必須有排水孔** — 積水是桂花黃葉第一殺手

- 陶盆或塑膠盆皆可;陶盆透氣但重,頂樓/陽台考慮承重

**土壤**

- **微酸性**(pH 5.5–6.5)— 桂花在中性或偏鹼的土會缺鐵,導致新葉黃化

- 推薦配方:通用培養土 2 份 + 蘭花樹皮 1 份(提高透氣排水)

- 或直接買標示「酸性植物用土」的培養土

- 不可用純黑土 — 保水太強容易爛根

*施肥細節(花前補磷鉀、什麼時候停肥)獨立成下面的「🌾 施肥」section。*

## 🌳 種植步驟

- **選盆備土** — 見上面「盆器與土壤」,土填 8 分滿,先澆透水讓土服貼

- **脫盆不傷根** — 輕捏原塑膠盆四周,倒扣後輕拍底部,整個根球完整取出,**不要扯根**

- **放入新盆** — 根球放進新盆中央,苗的地際線(根部以上第一段莖)比盆沿低 2–3 cm

- **填土壓實** — 四周填土,用手輕壓趕走氣泡,澆透直到盆底出水

- **緩苗期(第 1–2 週)** — 移到**半日照**、避風處;不施肥;土微濕即可(不要讓它很渴,也不要積水)

- **緩苗後移到全日照** — 葉片開始長新芽代表根系穩定,此時可移到全日照位置並開始正常照顧

> 

**🌬️ 關鍵:通風是桂花的命脈**

桂花怕悶熱不通風。室內光線再好都不適合長期放桂花 — 空氣流通不足容易引發介殼蟲和煤煙病。**陽台、頂樓、院子,有風才放**。

## 🌾 施肥

**桂花施肥分兩個階段:生長期用均衡肥(氮磷鉀平均),花前 6–8 週換高磷鉀肥促花芽**。用錯階段要麼只長葉不開花,要麼花芽分化差、花量少。

*施肥時機(Day 30 / 8–9 月花前 / 冬眠停肥)已統整在上面「📋 完整時程表」,這個 section 講 **用多少 · 用哪款 · 兩段肥如何切換**。*

> 

**📐 生長期 · 速查(好康多 1 號,每 3 個月一次)**

| | 盆 | | 顆數 | |

| | 5 吋圓盆(Ø15cm) | | **5 粒** | |

| | 7 吋圓盆(Ø21cm) | | **8 粒** | |

| | **30cm 圓盆**(一般桂花起始盆) | | **10 粒** | |

| | 40cm 大盆 | | **14 粒** | |

**原則**:寧可比這個少,不可比這個多。施完一定要澆水讓肥溶進土。

> 

**🌸 花前 6–8 週(約 8–9 月) · 換高磷鉀肥**

這是桂花施肥的核心訣竅。**磷(P)促進花芽分化,鉀(K)充實花苞**。如果這階段還在施氮肥,植物把能量拿去長葉,花芽就少。

- 換施 **花寶 2 號**(NPK 10-30-20,高磷高鉀)液肥,每 10–14 天一次

- 或換固體肥 **好康多 2 號**(花果型,高磷鉀)

- 直到花苞明顯可見(約 9 月)後減量或停施

> 

**📱 記憶外包 — 每年 8 月下旬固定換肥**

不用記,設一次重複行事曆:

- 每年 **8 月 20–25 日** →「換花寶 2 號,停好康多 1 號」

- 每年 **11 月 15 日** →「停所有肥,準備越冬」

- 每年 **3 月 10 日** →「重新撒好康多 1 號,開始生長期施肥」

**三個日期設完,桂花年度施肥就不用再動腦**。

## 🌸 花期與採花

**花期辨識**

- 主花期:**9–11 月(寒露 10/8 前後達到高峰)**

- 春花:部分品種 4–5 月小批開放,花量少但可採

- 花苞出現時:**葉腋長出密集小顆粒**,由淡綠轉淡黃/白/橙(依品種)

- 開花後香氣持續 5–10 天,過了就快速凋謝

**採花方式**

- 最佳採收時機:**花剛全開、香氣最濃時**(早晨露水乾後)

- 方法:以細密篩網或盤子放在樹下,**輕搖枝條讓花自然落下**,或用手指輕撥

- 採下後立即用廚房紙巾輕吸水分,攤開通風 1–2 小時

- 不要用水洗(花朵嬌嫩,水洗會破壞香氣)

**採後用途**

| | 用途 | | 做法 | | 備註 | |

| | **桂花茶** | | 新鮮或乾燥桂花 1 小匙 + 熱水(80°C),泡 2–3 分鐘 | | 可加入綠茶或烏龍底茶 | |

| | **桂花糖漿** | | 花 + 等量砂糖,玻璃罐靜置 3–5 天出汁 | | 加冰水或蜂蜜飲用 | |

| | **桂花糕** | | 新鮮花混入糯米粉或芋頭泥蒸製 | | 中秋節應景,長輩參與感強 | |

| | **桂花釀** | | 花 + 米酒 + 冰糖,密封瓶浸泡 3 個月 | | 適合有體力動手的輕度失智長輩 | |

| | **乾燥保存** | | 花攤在廚房紙巾上陰乾 3–5 天,密封罐冷藏 | | 可保存 6 個月,隨時泡茶 | |

> 

**⚠️ 花期不要修枝、不要施高氮肥**

花期修枝會直接剪掉花苞;高氮肥讓植物長葉而非開花。花期管理就兩件事:**採花 + 澆水,其他都不做**。

## 🛡 失敗點與除錯

| | 症狀 | | 可能原因 | | 對策 | |

| | **種了好幾年都不開花** | | 氮肥過多 / 光線不足 / 花前未補磷鉀 | | 8–9 月換花寶 2 號;確保全日照至少 4 小時;停高氮肥 | |

| | 新葉**黃化**(老葉正常) | | 土偏鹼缺鐵 / 根系積水 | | 補硫酸亞鐵;確認盆底排水;換微酸性土 | |

| | 葉片**出現黑色粉末** | | 煤煙病(通風不足或介殼蟲引起) | | 改善通風;移到室外;用肥皂水噴介殼蟲 | |

| | 葉背有**白色/褐色小顆粒** | | 介殼蟲 | | 舊牙刷刷除 + 95% 酒精棉棒擦;嚴重換土重植 | |

| | 花苞長出來但**提早掉落** | | 水分不足 / 溫差過大 | | 花期保持土微濕;避免突然移盆 | |

| | 冬天**大量掉葉** | | 換季正常代謝(常綠但仍換葉) | | 不用處理,等春天新芽 | |

## 🧠 失智照護應用 Notes

### 輕度

> 

**📍 輕度:她能主導整條「採花→泡茶→喝茶」流程**

輕度失智長輩仍能執行多步驟活動。桂花的照護價值在這個階段最完整 — 從看花苞到採花到做成可以吃的東西,每個環節都有意義。

- 每天早上帶她去陽台確認花況 — **「今天開了沒?」是一個簡單但有方向感的問句**

- 花期時讓她主導採花,你只是拿著容器跟著 — 她的「我在採花」自我感比採了多少更重要

- 採完一起做桂花茶:她量花、你燒水、一起喝 — **「我們家的花泡的茶」有別於任何外購的情感重量**

- 如果她說「這個香跟以前XX的味道一樣」,順著記憶走,問她那時候的事 — 桂花香是很強的自傳式記憶鑰匙

- 目標:維持她「我在照顧植物、我在為家人做事」的主體感

### 中度

> 

**📍 中度:讓她參與採花,但你拆解每個動作**

她能執行「用手輕搖枝條」或「把花撥進盤子」這種單一動作,但「自己走去陽台 + 找對那棵 + 採 + 回來」這種多步驟會卡住。

- 你帶她走到桂花旁,把盤子放好,**手把手示範一次:「就這樣輕輕搖」**

- 一次只給一個指令:「幫我搖這枝」—— 不是「你去採花」

- 她可能不記得昨天採過,但香氣會讓她當下感到愉快 — **每次體驗都是新的,但對你來說是同一件事**

- 泡茶時讓她拿杯子、讓她聞一聞 — 嗅覺這條路在中度仍然通暢

- 準備好一個固定的「桂花角」:盆 + 採花盤放同一個地方,不要移動 — 她靠空間記憶多過語言記憶

這個階段的價值:**嗅覺觸發的情緒記憶比語言指令更可靠。香氣先到,再說話。**

### 重度

> 

**📍 重度:不要求動作,要求存在與氣味**

她的主動動作有限。桂花在這個階段的核心價值是:**嗅覺是失智最後失去的感官**。桂花香可以穿透語言功能退化的屏障。

- 花期時摘幾朵放在小碟子上,**擺在她鼻子可以聞到的高度** — 不需要任何語言說明

- 把她的手放在花旁邊讓她自然感受到香氣與溫度 — 不強迫她做什麼

- 如果她有任何反應(眼神改變、嘴角動、說一個字),那就夠了 — **「她還在那裡」**

- 每年秋天桂花期,你可以告訴她:「媽,今年的桂花開了」— 即使她不回應,這句話值得說

- 採下來的花用來泡茶,你端給她,讓香氣充滿房間 — 這是最低操作成本、最高情感回報的使用方式

這個階段的價值:**香氣不需要認知,只需要存在。桂花是這個家秋天還在運轉的證明。**
