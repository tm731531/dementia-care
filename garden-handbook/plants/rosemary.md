## 🛒 購物清單

## 💡 為什麼迷迭香值得種

- **種了幾乎不用管** — 耐旱、耐貧土,忘記澆水也不容易死,是香草裡最好養的一種

- **多年生,種一次享用 10 年** — 不用每年重種,植株越大越茂盛

- **香氣療癒** — 隨手搓一下葉子,松木清香瞬間釋放;嗅覺刺激對記憶力和情緒有幫助

- **廚房直接用** — 烤雞、烤馬鈴薯、橄欖油浸漬、迷迭香茶,隨採隨用

- **外觀美、不突兀** — 灰藍小花、針形葉,放陽台窗邊像一盆景觀植物,不像「菜」

- **對照地瓜葉,幾乎反向** — 地瓜葉每天要澆水;迷迭香叫你**少澆水**,對照顧者負擔更低

## 🛒 取得方式

> 

**🗓️ Step 0 · 先看月曆再動手**

**四季都可以開始,但秋天(9–11 月)扦插最穩。** 盛夏(7–8 月)悶熱潮濕,扦插容易腐爛;梅雨季(5–6 月)連日高濕也要注意通風。春秋最輕鬆,如果現在是盛夏,買苗成活率比扦插高,直接定植更快。

> 

**📦 推薦的新手包 · 三步照順序來**

- **先備盆與土(這一步要比苗先到)** — 花市買 **5–7 吋透氣盆(素燒陶盆或塑膠盆帶排水孔)** + **疏水培養土**(或培養土混珍珠石 3:1),合計 NT$150–250,填好土、定位放好。

- **再取得苗或枝條** — 家附近花市、傳統市場常見盆苗 NT$50–120;[蝦皮搜尋「迷迭香 插枝」](https://shopee.tw/search?keyword=%E8%BF%B7%E8%BF%AD%E9%A6%99+%E6%8F%92%E6%9E%9D)也有現成扦插條。

- **到貨/購苗當天立刻定植或插穗** — 盆苗直接脫盆種入;扦插條修剪後插進土,澆透一次。

⚠️ **順序不能顛倒**:土如果還沒準備好,苗放著乾等一天就乾掉。總預算 **NT$250 有找**。盆苗比扦插快 4–6 週採收,新手優先選盆苗。

**其他取得管道**

| | 蝦皮購苗 | | 低 | | NT$50–100+運費 | | 選有根的苗,避免只寄插枝 | |

> 

**⚠️ 不推薦新手:種子播種**

迷迭香種子發芽率低(30–50%)、生長極慢,從種子到成株需 6 個月以上。扦插或盆苗快上 3–4 倍,沒有特別理由不走種子路線。

## 🏺 盆器與土壤

**盆器**

- 直徑 **15–20 公分(5–7 吋)**以上;多年生後可換 8–10 吋大盆

- **素燒陶盆最佳** — 透氣排水,根不容易悶濕爛掉;塑膠盆也 OK,但要確保排水孔夠大

- **排水孔是命** — 沒有排水孔或積水托盤長期積水,迷迭香根系三天就爛

**土壤**

- **排水優先於肥力** — 迷迭香原產地中海,天然就是砂礫山坡,喜歡「窮土」反而不喜歡過肥

- 推薦配方:一般培養土 + 珍珠石 = **7:3**(讓澆水後快速乾燥)

- 或直接買「多肉仙人掌專用土」也完全適合

- 禁止用黏土/黑泥土單獨種植 — 保水太強,根部長期潮濕會爛

## 🌳 種植步驟

**路線 A · 盆苗定植(推薦)**

- **準備盆土** — 見上方「盆器與土壤」,土填 8 分滿

- **脫盆** — 輕拍盆側讓土鬆,捏住莖基部倒扣,完整帶出根球

- **植入新盆** — 根球置中,填補四周土,覆土到根球頂端即可(不要埋莖)

- **第一次澆透** — 澆到盆底出水;之後等土表**完全乾燥**再澆下一次

- **放全日照位置** — 每日至少 4 小時直射陽光;日照越多長越旺

> 

**💧 澆水判斷法**

用免洗筷插進土 3 公分深,拔出**完全乾燥才澆**。感覺到土濕就不用澆。迷迭香**寧乾勿濕** — 一次缺水可以救,長期積水必死。

**路線 B · 扦插(有現成枝條時用)**

- 剪**10–15 公分健壯嫩枝**,斜剪增加接觸面積

- 摘除下方 1/3 的葉子,裸露莖節

- (選做)切口沾**發根粉**,可提高發根率

- 插入**乾燥砂質土**或珍珠石,深度約 3–5 公分

- 澆水一次,之後土表稍乾才澆;放半日照 2–3 週等發根

- 輕拉有阻力 = 發根成功,可移到全日照

## 📋 完整時程表 · 扦插到採收

**定植/扦插 → 發根穩固 → 施肥 → 首次採枝 → 多年採收**,照這個節奏走。**照做就對,不用每天觀察**。

**⚠️ 開工前先看月曆 / 節氣:挑對季節更輕鬆**

迷迭香四季可種,但**高溫悶濕**時期扦插容易腐爛,需特別注意通風。節氣對應台灣中低海拔:

## 🌾 施肥

**迷迭香耐貧土 — 施肥要「少量」而非「多量」**。過肥反而讓葉子香氣變淡、植株徒長脆弱。固體緩釋肥最適合,不用稀釋、難燒根、撒一次吃 3 個月。

## ✂️ 採收

**採收時機**

- 定植後 **2–3 個月**,新梢長到 10 公分以上就可以剪

- 每次採 **頂端 5–10 公分嫩枝**,保留下方至少 2/3 讓植株再發

- 開花前採收香氣最濃;若讓它開花,採後要強剪一次促新梢

**怎麼用**

| | 料理 | | 用法 | |

| | 烤雞/烤肉 | | 新鮮枝條塞進雞腔,或連枝烤盤鋪底 | |

| | 烤馬鈴薯 / 根莖類 | | 橄欖油 + 迷迭香葉 + 鹽,拌勻後烤 | |

| | 迷迭香橄欖油 | | 新鮮枝條泡橄欖油 2 週,飄香可口 | |

| | 迷迭香茶 | | 新鮮或乾燥葉 2–3 枝 + 熱水,燜 5 分鐘 | |

| | 麵包/佛卡夏 | | 葉子揉入麵團或撒表面烘烤 | |

> 

**🌿 採後保存**

新鮮枝條插入半杯清水,室溫可放 1 週;用廚房紙巾包起放冰箱可撐 2 週。要長期保存就**晾乾或烤箱低溫(50°C)烘乾**,裝密封罐放陰涼處,香氣持續 6 個月以上。

## 🛡 失敗點與除錯

| | 症狀 | | 可能原因 | | 對策 | |

| | **莖基部變黑軟爛** | | 過度澆水、積水、排水差 — 迷迭香最常見死因 | | 立刻停止澆水;剪除腐爛部分;換疏水土;確認盆底排水孔暢通 | |

| | **扦插 3 週沒發根** | | 土太濕、日照不足、溫度過低 | | 確認土乾才澆;移至更亮位置;秋冬扦插耐心等 4–6 週 | |

| | **葉子枯黃脫落** | | ① 過濕爛根 ② 過乾缺水 ③ 盛夏高溫悶熱 | | ① 停澆+換土 ② 補水 ③ 移至通風半遮陰處 | |

| | **葉色淡、徒長** | | 日照不足 | | 移到更亮的位置;迷迭香全日照才最旺 | |

| | **葉片有白粉** | | 白粉病(悶濕環境) | | 加強通風;剪除受害枝;避免葉片噴水 | |

| | **有紅蜘蛛/介殼蟲** | | 乾燥環境、通風差 | | 清水高壓沖葉背;嚴重時用稀釋苦楝油噴灑 | |

| | **香氣變淡** | | 施肥過多 / 日照不足 | | 減少施肥、移到更亮位置 | |

> 

**🚨 迷迭香致命弱點:爛根**

迷迭香幾乎不會缺水死,**但積水會在 3–5 天內爛根**。台灣夏天間歇大雨 + 悶熱最危險。戶外要確認盆底不積水;若用花盆托盤,每次澆水後 30 分鐘倒掉托盤積水。

## 🧠 失智照護應用 Notes

### 輕度

> 

**📍 輕度:嗅覺記憶連結,讓她說故事**

迷迭香強烈的松木香氣和地中海料理氣味,容易勾起食物、旅行、廚房的遠期記憶 — 失智初期這些記憶往往比近期記憶還保存得好。

- 每天澆水(3–5 天一次,頻率低)或修剪時讓她**搓幾片葉子,問她「這個味道讓你想到什麼?」**

- 採收時讓她**主導「今天要用來做什麼菜」** — 維持她還是廚房決策者的身分感

- 一起做迷迭香茶或迷迭香橄欖油浸漬 — 多步驟活動,她可以自主完成,你在旁陪伴

- 把迷迭香盆放在廚房窗台 — 進廚房就能看到、摸到、聞到,嗅覺提醒自然發生

- **目標**:嗅覺刺激觸發遠期記憶對話,每次澆水都是一段小聊天的起點

這個階段的價值:**一株迷迭香讓她每隔幾天就有一個「做了點什麼」的感受**,維持廚房主人翁感。

### 中度

> 

**📍 中度:動作簡化為「搓葉子聞香」**

多步驟指令已難以跟上,但搓葉聞香這個動作只要一步,而且嗅覺路徑直達邊緣系統 — 即使語言理解退化,氣味情緒仍有效。

- **把迷迭香盆放在她日常動線顯眼處** — 她路過自然伸手摸,不需要你提醒

- 引導動作只需說**「聞一下,好香」**然後你先示範搓一下葉子遞給她聞 — 不要說多步驟指令

- 你在澆水時讓她坐旁邊,把剛剪下的一小枝放到她手上 — 讓她握著、聞著

- **迷迭香茶降階版**:你泡好端給她,讓她用雙手捧著杯子聞香氣 — 只要「聞」這個動作即可

- 注意:這個階段不追求「她記住植物叫什麼」,追求**「每次接觸後情緒是平靜的」**

這個階段的價值:**嗅覺是最後退化的感官之一;迷迭香的強烈香氣讓她每次都有「這個我認識」的熟悉感**。

### 重度

> 

**📍 重度:不要求認識,要求感受到香氣**

語言和動作理解已高度退化,但嗅覺仍有反應 — 迷迭香香氣可能讓她微笑、停下來、轉頭,這就足夠了。

- 每天或每隔幾天,**你摘一小枝搓幾下,湊近她鼻子 5–10 秒** — 觀察她的表情變化

- 把盆栽靠近她常坐的位置;自然風吹過偶爾帶香氣,不需要特定動作

- 採下幾枝枝條放在她手上,讓她**握著** — 觸覺 + 嗅覺雙重刺激

- 不要求她說話、不要求她認識「迷迭香」這個詞 — 目標是**3 秒的表情反應**

- 配合迷迭香茶的溫熱蒸氣,讓她靠近聞香氣 — 熱蒸氣擴散力更強

這個階段的價值:**她不需要知道這是什麼植物,她只需要聞到這個香氣然後有一點點平靜**。迷迭香的芳香療法背景讓這個應用有根基 — 地中海香草氣味在失智照護中是有文獻記錄的嗅覺刺激工具。
