## 🛒 購物清單

- **繁殖材料**:紅蘿蔔種子 1 包 (花市, NT$30–50)

## 💡 為什麼值得種紅蘿蔔

- **「尋寶」時刻是紅蘿蔔獨有的** — 拔起前完全不知道長多大、長幾條,出土那一秒是純驚喜,失智長輩最愛這種「打開才知道」

- **橘色根部強視覺刺激** — 橘色在綠葉中極度顯眼,辨識不費力,適合視力退化的長輩

- **台灣冬天越冬才最甜** — 低溫讓澱粉轉化成糖,12–1 月拔起來生吃就甜,不用加工就有成就感

- **播種動作簡單** — 直接把種子灑在土上,不用育苗、不用換盆,減少操作複雜度

- **葉子本身可食** — 嫩葉可炒可湯,等根長好的期間就有中間收穫

- **熟悉的廚房食材** — 紅蘿蔔湯是台灣家家戶戶的記憶,拔起來「自己種的拿去煮湯」的情感連結強

## 🛒 取得方式

> 

**🗓️ Step 0 · 先看月曆再動手**

**只有 9–1 月(白露–大寒)才下手。** 夏天(4–8 月)氣溫過高,根部無法膨大,只長葉不長根,播了等於白種。如果現在是春夏,**先備好深盆與鬆土,等 9 月白露後再播種**。蝦皮上夏天也找不到品質好的紅蘿蔔種子(農家不補貨),這也是風向球。

> 

**📦 推薦的新手包 · 三步照順序來**

- **先備盆與土(這一步要比種子先到)** — 買 **深度 30 cm 以上的深盆**(圓盆直徑 25 cm+ 或長方盆 50×20 cm 深度要夠)+ **蔬菜培養土或細培養土**,揀掉明顯石頭或大塊硬土(石頭是讓根歪斜分叉的主因),土填至 8 分滿,預先澆透備用。合計 NT$200–350。

- **再買種子** — 蝦皮搜尋「紅蘿蔔種子 五寸人參」或「紅蘿蔔 種子 台灣」,9–11 月農業節期間常有促銷,一包 50–100 粒 NT$50–100,花市也有

- **備齊當天立刻播種** — 土面輕劃淺溝(深約 0.5 cm)→ 均勻灑種子(每 2–3 cm 一粒)→ 薄薄蓋一層細土 → 用噴霧瓶噴濕(不用大量澆水,避免種子被沖跑)→ 放全日照陽台

⚠️ **順序不能顛倒**:種子很輕,土沒備好就播,一澆水就全部被沖到角落堆一起。總預算 **NT$350–450 有找**,關鍵是**深盆 + 鬆土 + 月份對**。

> 

**⚠️ 新手陷阱:盆深不夠**

很多人用淺盆(深度 10–15 cm)種紅蘿蔔,結果根長到底部被迫橫向彎曲,形狀扭曲又短小。**深度至少 30 cm**,根才有空間向下生長。寧可買大盆貴一點,不要等 80 天拔起來才後悔。

## 🏺 盆器與土壤

**盆器 — 深度是關鍵**

- **深度 30 cm 以上**(絕對最低標準 — 根要向下生長,淺了就歪)

- 直徑或寬度 20 cm 以上(疏苗後每株間距 10 cm,一盆放 4–6 株)

- 長方盆 50×20 cm、深 30 cm 是性價比最好的選擇

- 盆底**必須有排水孔**,冬季也可能積水爛根

- 材質不拘,陶盆排水佳但重,塑膠盆輕便,二者都 OK

**土壤 — 鬆軟無石才能長直**

- **細培養土或蔬菜專用培養土**優先(顆粒細,根容易穿透)

- 填土前**手動揀出石頭、土塊**(1 公分以上的都要撿),這是唯一最重要的步驟

- 可混入少量細砂或珍珠石增加透氣排水(比例約 7:3)

- 不要用重黏土(根穿不進去,容易分叉)

- 不要用未腐熟有機肥直接混土(讓根分叉長怪形)

## 🌳 種植步驟

- **備深盆、鬆土、撿石頭** — 見上面「盆器與土壤」,這步沒做好後面都沒用

- **土面整平、稍微壓實** — 用手掌輕拍平,不要留空洞(種子會掉進去埋太深)

- **劃淺溝播種** — 食指深度 0.5 cm 劃出播種溝,每 2–3 cm 放 1 粒種子;或直接均勻灑在土面

- **薄蓋細土** — 覆蓋約 0.3–0.5 cm 細土,輕壓讓種子跟土接觸

- **噴霧瓶噴濕** — 第一次澆水用噴霧,不要用澆水壺直接沖(種子輕,一沖就堆一起)

- **保持土表濕潤** — 發芽前每天噴 1–2 次,土表看到乾就補水

- **放全日照位置** — 日照越充足根越甜越直

> 

**💡 疏苗是最重要的技巧(新手最容易跳過)**

紅蘿蔔必須疏苗兩次,否則根擠在一起全部長不大:

- **第一次疏苗(Day 10–14)**:發芽後留下壯苗,每 3 cm 留 1 株,弱小或擁擠的直接拔掉

- **第二次疏苗(Day 25–30)**:本葉長出後,每 10 cm 留 1 株,這才是最終間距

拔掉的嫩苗不要丟,根雖小但葉嫩,可以加入沙拉或炒蛋,讓疏苗這個「捨棄」的動作變成小收穫。

## 📋 完整時程表 · 播種到拔收

**播種 → 發芽 → 疏苗 × 2 → 施肥 → 採收**,一張表涵蓋整個生長週期。**紅蘿蔔要耐心等,80–100 天才看到成果**。

> 

**⚠️ 開工前先看月曆 / 節氣:夏天種等於浪費**

紅蘿蔔**喜涼不喜熱**,需要低溫才能讓根部充分膨大、轉化糖分。夏天高溫只長葉片,地下根不會發育。**節氣比月份準**:

| | | | 月份 | | 節氣 | | 狀態 | |

| | ✅ 最佳 | | **10 – 12 月** | | **寒露(10/8)→ 大雪(12/7)** | | 低溫適合根部發育,台灣冬季紅蘿蔔最甜,**手冊時程以此為準** | |

| | 🟡 勉強 | | 9 月 / 1 月 | | 白露(9/8) / 小寒(1/6) | | 9 月氣溫稍高但可試;1 月下旬後低溫有限,生長期拉長 | |

| | ❌ 絕對別種 | | **4 – 8 月** | | **清明(4/4)→ 立秋(8/7)** | | 根部無法膨大,只長葉片,80 天後拔出來只有細根鬚 | |

台語俗諺:**「冬天的蘿蔔最甜,夏天的蘿蔔最空」**。如果現在是夏天,**不要播** —— 先備深盆鬆土,等白露(9 月上旬)再下手。

## 🌾 施肥

**本 handbook 預設固體緩釋肥,不推液肥給新手** —— 不用稀釋、不用算濃度、撒一次吃 3 個月、難燒根。

> 

**📐 不同盆大小 · 速查(好康多 1 號)**

| | 盆 | | 顆數 | | 備註 | |

| | 3 吋圓盆(Ø9cm) | | **3 粒** | | 太小不建議種紅蘿蔔(根深) | |

| | **長方盆 50×20cm 深 30cm** | | **15 粒** | | 紅蘿蔔最推薦規格 | |

**原則**:寧可少不可多;根莖類植物對肥料很敏感,過肥容易讓根部分叉或形狀怪異。

**🛒 買哪一款**

- **好康多 1 號**(NPK 14-11-13)— 萬用款,適合根莖類蔬菜

- 350g 小包約 NT$100,一包可用多年

- 全聯園藝區、花市、iGarden 網購都有

- 替代品:魔肥、奧綠肥(類似緩釋顆粒肥都 OK)

> 

**⚠️ 根莖類施肥特別注意**

- **不要在播種時就施肥** — 新根遇到高濃度肥會燒傷,等到 Day 30 植株站穩再施

- **氮肥不能過多** — 氮過多讓葉子瘋長、根部反而細小,好康多 1 號 NPK 比例已均衡

- **未腐熟有機肥絕對不能用** — 雞糞、廚餘直接混土,根部接觸後必然分叉或腐爛

> 

**📱 記憶外包 — 播種當天就把施肥排程做完**

- 播種日 + 30 天 → 「撒好康多(用量看便利貼)+ 澆水」

## ✂️ 採收

**判斷何時可以拔**

- **根莖基部微微露出土面** — 這是最可靠的信號,橘色頂端從土裡探出頭

- 播種後 **80–100 天**可以試拔第一批

- 葉片略微倒伏、葉色轉深綠也是熟成信號

- 不確定的話先拔一株確認大小,不影響其他株

**拔收動作**

- 一手壓住盆土邊緣(防止盆整個翻),另一手**抓住葉基部**(不是葉片)

- **緩慢垂直向上拔**,不要旋轉,根部直的才不會斷

- 如果很緊,先用筷子在旁邊挖鬆土再拔

- **出土那一秒是尋寶時刻** —— 讓長輩自己拔,不要幫他拔

> 

**🥕 採收後可做的簡單事**

- **生吃測甜度** — 冬天越冬後的紅蘿蔔切一小片直接吃,比超市的甜

- **葉子別丟** — 嫩葉可以炒蛋或加湯,是第二收穫

- **削皮切塊** — 熟悉的廚房動作,適合讓長輩一起做(簡單重複的動作)

- **加進當天晚餐** — 自己種的拿去煮湯,成就感立即完整

## 🛡 失敗點與除錯

| | 症狀 | | 可能原因 | | 對策 | |

| | 10–14 天還沒發芽 | | 土太乾 / 播種太深 / 季節不對 | | 確認土表每天保濕;種子不要埋超過 0.5 cm;夏天種必然不發芽 | |

| | 拔出來根細如鉛筆 | | 疏苗沒做 / 盆太淺 / 夏天種 | | 下次做兩次疏苗;換深盆;確認月份(秋冬才種) | |

| | 根部分叉 / 歪斜 | | 土裡有石頭或硬塊 / 過肥 | | 播種前更仔細鬆土揀石;肥料減量 | |

| | 根部頂端發青(綠色) | | 頂端露出土面、陽光直射 | | 露出時往根基周圍補土覆蓋(培土) | |

| | 葉色淡黃 | | 缺氮 / 缺水 / 日照不足 | | 先確認日照與澆水;若足夠則補 3–5 粒好康多 | |

## 🧠 失智照護應用 Notes

### 輕度

> 

**📍 輕度:她能主導整個種植節奏**

她還能理解多步驟流程、記得今天要去看看盆、感受到等待的意義。紅蘿蔔的「長等待 + 拔收驚喜」節奏最適合這個階段使用。

- **讓她決定播種密度** — 「這樣放幾顆妳覺得夠了?」讓她有主導感

- 每天「去看一下有沒有發芽」是她自己可以主導的 ritual,水壺和盆固定擺在動線上

- 疏苗時她能自己拔,你提示「這棵太近要拔掉,給旁邊那棵空間長大」就夠

- 施肥時她能自己數顆粒、自己撒,**這個動作的精細度剛好匹配輕度認知能力**

- **目標**:讓她感受完整的農作節奏 — 種、等、照顧、收穫,不跳過任何一步

這個階段的價值:**「我還在種東西、還在照顧生命」的自我感是最好的認知刺激**。

### 中度

> 

**📍 中度:拆解到單一動作,每次一個**

她能執行你引導的單一動作,但記不得「今天要做什麼」,不會自己主動去陽台。你的角色是把每個步驟「準備好,放在她手邊」。

- **噴霧瓶事先裝好水放在盆旁** — 不要讓她找水龍頭,那個路徑太長

- 每天的言語指引只說一個動作:「幫我噴一下土,怕乾了」,不說「去陽台噴水然後確認葉子」

- 疏苗時**你先標出要拔哪株(例:插一根牙籤在旁邊)**,請她拔掉標記的那棵

- 採收日帶她到陽台,把手放在根基:「抓這裡,慢慢往上拔」—— 出土那秒讓她自己完成,**不要提前幫她拔**

- 拔出來的紅蘿蔔放在她手上,讓她握著感受重量和涼意,不急著拿走

這個階段的價值:**採收的驚喜感在中度仍然完整存在** —— 認知退化不影響情緒記憶,那一秒「哇」的感覺她還能體驗。

### 重度

> 

**📍 重度:不要求動作,要求感官接觸**

她的自主動作有限,指令理解困難。紅蘿蔔的價值從「活動」變成**感官刺激 + 情緒連結**。

- 帶她坐在盆旁邊,**你播種或澆水,她在旁邊看**,不要求她動作

- 把嫩葉放到她手上,讓她感受羽狀葉的觸感(有點毛、有點涼);把葉子湊近讓她聞土味

- 採收時你拔起來後,**把橘色的根放在她手上** — 橘色鮮豔、表面光滑涼沉,是強感官刺激

- 削皮的聲音、切塊的聲音、「今天煮湯用我們種的蘿蔔」的話語 —— 這些聲音和語境可能觸動深層記憶

- 每次同樣的路線、同樣的時段,**節奏感比「完成任何事」更重要**

這個階段的價值:**她不需要「種」紅蘿蔔,她需要「在」那個種紅蘿蔔的場景裡**。橘色的根、土的氣味、熟悉的「晚餐要用的蘿蔔」—— 這些比動作更能到達深處。
