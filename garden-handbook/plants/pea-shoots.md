# 🌱 豌豆苗 家庭種植指南

> 類別:**葉菜** · 難度:極低(豆子泡水就發芽) · 季節:播種 10–1 月(平地);2–9 月僅高山適合

## 🎯 5 秒總覽(5W1H)

- **👥 Who**:新手友善 · 兒童可參與 · 懶人友善
- **🌱 What**:乾豌豆(廚房 或 五穀雜糧行,非烘焙) 1 把 · 淺盆 15-20cm 或 長條盆 · 培養土 3L
- **📅 When**:播種 10–1 月(平地);2–9 月僅高山適合
- **📍 Where**:市場 買 / 陽台 / 窗邊 / 戶外 種
- **🔧 How**:10/11/12/1 月 下手,照「完整時程表」做

## 🛒 購物清單

- **繁殖材料**:乾豌豆(廚房 或 五穀雜糧行,非烘焙) 1 把 (市場, NT$20–50) ✨ 廚房乾豌豆可直接用
- **盆器**:淺盆 15-20cm 或 長條盆 × 1 個 (NT$80–200)
- **土壤**:一般培養土 3L (NT$50–100)
- **肥料**:好康多 1 號 350g(週期短可省) (多盆共用)

## 💡 為什麼冬天要種豌豆苗

- **採收超快** — 芽菜模式 **7–12 天**就能吃,土栽 30–40 天首採

- **多次採收** — 留 2 片葉再剪,同盆可採 **3–5 輪**

- **種子便宜** — 一包 NT$30–80 可種整冬

- **可食用豌豆當種子** — 廚房乾豌豆、超市豌豆都能發(非烘焙過的)

- **失智照護** — 泡豆、撒種、採葉,每步都簡單

- **兩種模式**:水耕芽菜快速吃 / 土栽長期採,可同時跑

## 🛒 取得方式

> 

**🗓️ Step 0 · 先看月曆再動手**

**平地要是 10 月 – 1 月才種。** 豌豆**喜涼怕熱**,溫度超過 25°C 會長不好、容易病蟲害。如果現在是春夏,**等到寒露(10/8)前後**再下手。高山地區例外,可 2–9 月種。

> 

**📦 推薦的新手包 · 可以從廚房出發**

- **豌豆種子** — 花市/農會買「豌豆」或「甜豌豆」種子(NT$30–80)。或**直接用廚房的乾豌豆**(五穀雜糧行買的那種,非烘焙過)

- **盆與土** — 淺盆/長方盆(深 15 cm 即可)+ 培養土

- **泡豆 → 播種 → 等**

總預算 **NT$150 有找**。廚房乾豌豆 = 0 採購成本。

> 

**🚀 想更快?試水耕芽菜模式(7 天可吃)**

只想吃「豌豆芽菜」不種「豌豆苗」的話,完全可以跳過盆土:

- 豆子泡水 12–24 小時

- 鋪在塑膠濾水籃或布巾上,**每天沖水 1–2 次**

- **4–7 天即可食用**(日本料理店的「豆苗」就是這個)

本 entry 主要講**土栽豌豆苗**(能多次採、植株大),芽菜模式只是順帶介紹。

**選豆子的原則**

- **沉在水裡的是好豆** — 泡水時漂浮的是空殼或壞豆,挑掉

- **非烘焙處理** — 零食豆(翡翠豆/脆綠豆)是烤過的,不能發芽

- 可用品種:一般豌豆、甜豌豆、荷蘭豆、豌豆仁(冷凍生的可以,冷凍熟的不行)

*資料來源:[農業知識入口網 · 豌豆芽生產](https://kmweb.moa.gov.tw/subject/subject.php?id=60168)、[臺中農改場 · 豌豆小學堂](https://www.tcdares.gov.tw/ws.php?id=12740)*

## 🏺 盆器與土壤

**盆器**

- 深 **15 cm 以上**(根不深,長方盆或淺圓盆都 OK)

- 一盆撒一整排,密一點沒關係(豌豆苗本來就種密)

- 盆底**排水孔必備**

**土壤**

- 一般培養土即可

- 豌豆是**豆科**,本身根瘤菌會固氮,**對土肥要求不高**

- 排水好就行,積水會爛根

*施肥細節獨立成下面的「🌾 施肥」section。*

## 🌳 種植步驟

> 

**💧 泡豆關鍵步驟(千萬別跳)**

- 豆子丟進清水裡,**漂浮的挑掉**(壞豆或空殼)

- 沉下去的**泡 12–24 小時**(冬天取長,夏天取短)

- 泡到豆子吸飽水、外皮微皺即可

**不泡的話**:發芽慢、不整齊。泡過的發芽率 80% 以上。

- **盆填土** — 培養土填 8 分滿,先澆一次水讓土服貼

- **撒泡好的豆子** — 密集撒在土表(間距 1–2 cm,可以密),每顆豆微壓進土

- **覆土** — 蓋 **1–1.5 cm** 薄土

- **澆水** — 蓮蓬頭或噴霧澆透(不要沖走豆子)

- **放半日照到全日照** — 秋冬光越多越好

- **保持土壤微濕** — 不下雨的話每天澆一次(豌豆比茼蒿更喜濕)

> 

**💧 不疏苗,讓它們擠在一起**

豌豆苗**故意種密**,擠一起彼此支撐,葉子細嫩好吃。跟茼蒿相反,**不用疏苗**。如果有特別弱的可以拔掉。

## 📋 完整時程表 · 播種到採收

**泡豆 → 撒種 → 採苗 → 多輪採**,冬天最快的可食植物之一。

> 

**⚠️ 開工前先看月曆 / 節氣:挑對時間**

豌豆**喜涼怕熱**,25°C 以上長不好:

| | | | 月份(平地) | | 節氣 | | 狀態 | |

| | ✅ 最佳 | | **10 – 1 月** | | **寒露(10/8)→ 大寒(1/20)** | | 發芽快、生長穩 | |

| | 🟡 勉強 | | 9 / 2 月 | | 白露 / 雨水 | | 邊緣期 | |

| | ❌ 別種 | | **3 – 8 月** | | **驚蟄 → 立秋** | | 高溫長不好、病蟲害多(高山例外) | |

| | 天數 | | 階段 | | 植物狀態 | | 你的動作 | |

| | **Day -1** | | 泡豆 | | 豆子吸水 | | 清水浸泡 12–24 小時,挑掉浮豆 | |

| | **Day 1** | | 播種 | | 撒豆覆土 | | 間距 1–2 cm,覆土 1–1.5 cm,澆透;❌ 不施肥 | |

| | Day 3–7 | | 發芽 | | 冒出嫩綠尖芽 | | 保持土微濕 | |

| | Day 7–14 | | 快速生長 | | 長出 2–3 片本葉 | | 每天澆一次水 | |

| | Day 15–29 | | 直立期 | | 植株 10–15 cm | | 繼續澆水 | |

| | **Day 30–40 ✂️** | | **首次採收** | | 株高 15–20 cm,葉繁茂 | | ✅ 剪頂端 5–8 cm,**留底部 2 片葉**讓它再發 | |

| | **Day 45–55 ✂️** | | **第二次採** | | 側芽長出 | | ✅ 同樣剪頂端,留 2 葉 | |

| | Day 55+ | | 多輪採收 | | 越採越細 | | ✅ 每 10 天一次,第 3–5 輪後品質下降 | |

| | **春天氣溫升** | | **結束期** | | 葉變小變硬 | | 整盆拔收,重新泡豆再來一輪 | |

> 

**📱 播種當天就要做的 2 件事**

- Google 行事曆設 **今日 + 30 天**(第一次採)和 **之後每 10 天**的重複提醒

- 便利貼寫今日日期,貼盆邊 — 讓全家都看得到「這盆是哪天種的」

*芽菜(水耕)模式只要 4–7 天,見上面「取得方式」的 🚀 callout。*

## 🌾 施肥

**豌豆是豆科,根瘤菌自己會固氮,對肥料要求低**。多數情況下不用追肥就能多輪採。

> 

**📐 用量(好康多 1 號)**

- 小盆(Ø15cm):**3 粒**

- 中盆(Ø21cm):**5 粒**

- 長方盆 50×20cm:**10 粒**

**時機**:Day 20 前後撒一次就夠。或完全不施也可以活。

**可以完全不施肥的情況**

- 只打算採 1–2 輪就重播 → 不施

- 土本身有混基肥 → 不用補

- 只做芽菜(水耕 7–12 天)→ 完全不需施肥

**想多輪採更久**

- 第一次採後撒 **半量**(5 粒)或薄液肥一次

- 磷鉀比氮重要(豆科自己會固氮),花寶 **2 號**(磷重)比 1 號更適合豆科

通用施肥原則看 [📚 通用園藝知識 → 🌾 施肥](#)。

## ✂️ 採收

**多輪採法(推薦)**

- 株高 **15–20 cm** 開始採

- 剪頂端嫩尖 **5–8 cm**,**底部留 2 片真葉**

- 側芽會從留葉的葉腋冒出,10–15 天可再採

- 可採 **3–5 輪**,之後品質下降就整盆拔

**一次性拔收**

- 株高 20 cm 時整株拔起,留根會斷絕多輪採

- 適合一次煮很多、想重開新盤的情況

> 

**🍜 採下來馬上能用**

- **清炒** — 熱鍋大火,30 秒起鍋,口感嫩脆

- **拌沙拉** — 生的豌豆苗清甜,冬天冷盤首選

- **下拉麵 / 湯** — 起鍋前灑一把,顏色好看

- **蒜炒豌豆苗** — 配家裡同時種的蒜苗,全部自產

## 🛡 失敗點與除錯

| | 症狀 | | 可能原因 | | 對策 | |

| | **整批不發芽** | | 豆子是烘焙過 / 太老 / 沒泡水 | | 改用生豌豆;泡水 12–24 小時 | |

| | 發芽率很低 | | 選豆沒挑掉浮豆 / 溫度太高 | | 泡水時挑沉豆;等氣溫降 | |

| | 幼苗倒伏 | | 立枯病(土太濕)/ 種太密 | | 減澆水;拔掉病株 | |

| | 葉子發白變薄 | | 日照不足 | | 移到光線好的位置 | |

| | 葉黃 | | 缺水 / 缺磷 | | 免洗筷檢測;補花寶 2 號 | |

| | **蚜蟲大爆發** | | 豌豆最常見的蟲害 | | 水柱沖走;肥皂水(1:50)噴;嚴重就整盆廢 | |

| | 白粉病(葉面白粉) | | 通風不良 + 冬季潮濕 | | 搬通風處;嚴重拔病葉,間距加大 | |

## 🧠 失智照護應用 Notes

### 輕度

> 

**📍 輕度:泡豆是簡單儀式**

泡豆、挑豆、撒豆,都是**單一動作、結果可見**的事。

- 她**自己挑豆** — 漂浮 vs 沉下的區別簡單明確

- 隔天她自己撒種、覆土、澆水

- 採收頻率高(每 10 天一次),**持續有成就感**不用等太久

- 採下來自己洗、煮、吃,完整閉環

### 中度

> 

**📍 中度:剪葉是熟悉動作**

泡豆、撒種她可能跟不上,但**採收時的「剪」**她還能做。

- 你負責播種、照顧;**採收時請她剪**(剪刀 + 明確的剪點)

- 剪好的豌豆苗放她面前,請她**分揀、挑壞葉**

- 豌豆苗清甜,涼拌生食她很好接受

- 頻繁的採收節奏 — **每 10 天一次的儀式感**

### 重度

> 

**📍 重度:觸感與視覺**

動作不強求,**嫩綠的視覺 + 清涼觸感**是主軸。

- 採一把放她手上 — **葉片細軟**,跟地瓜葉的厚實感不同,觸覺變化

- 讓她用手**輕輕搓**葉子,清淡的豆腥味是新的嗅覺體驗

- 煮清淡湯讓她喝,豌豆苗湯溫和

- 擺一盆在她坐的地方邊 — 密密麻麻的嫩綠本身是視覺刺激
