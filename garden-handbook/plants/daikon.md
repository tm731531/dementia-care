# 🥬 白蘿蔔 家庭種植指南

> 類別:**根莖** · 難度:低-中(根莖需深盆,疏苗要確實) · 季節:播種 9–1 月(秋冬最甜)

## 🛒 購物清單

- **繁殖材料**:白蘿蔔種子 1 包 (花市, NT$30–50)
- **盆器**:深盆 25cm+ 或 深長條盆 × 1 個 (NT$150–300)
- **土壤**:鬆軟培養土 8L (NT$100–150)
- **肥料**:好康多 1 號 350g (多盆共用)

## 💡 為什麼值得在陽台種白蘿蔔

- **葉 + 根都可食** — 蘿蔔葉醃雪裡紅、炒蛋;根做蘿蔔湯、蘿蔔糕、蘿蔔乾,一株兩吃

- **「連根拔起」的儀式感無可取代** — 採收那一刻把藏在土裡兩個月的成果一次挖出,會真的嚇到

- **台灣年節記憶** — 蘿蔔湯、蘿蔔糕是多數長輩童年的家常味,種出來等於召喚一段記憶

- **深盆 25 cm 即可** — 比紅蘿蔔淺,一般花市深盆就夠,不需要特殊設備

- **冬蘿蔔最甜** — 秋冬晝夜溫差大,澱粉轉糖,市售蘿蔔品質不如自種冬收的

- **疏苗的菜苗可以即時吃** — 疏苗拔下的嫩苗可以當芽菜拌沙拉,不浪費

## 🛒 取得方式

> 

**🗓️ Step 0 · 先看月曆再動手**

**現在要是 9–1 月才開始。** 白蘿蔔是**涼季作物**,超過 25°C 容易抽苔(開花)、根部長不大。夏天(5–8 月)播種幾乎必失敗 — 根還沒膨大就先抽苔、質地變硬發苦。如果現在是春夏,**等到白露(9/8)後再下手**,現在就先備盆土、找種子來源。

> 

**📦 推薦的新手包 · 三步照順序來**

- **先備盆與土(這一步要比種子先到)** — 花市買 **直徑 30 cm、深 30 cm 以上的深圓盆**或 **50×30cm 長方深盆**,一般培養土混少量珍珠石(約 2:1),合計 NT$200–300。土填 8 分滿、澆一次水讓土服貼定位。

- **再買種子** — **花市或蝦皮搜尋「白蘿蔔 種子」**(選「夏進白蘿蔔」或「日本白玉蘿蔔」等短根品種,NT$50–80/包,一包夠種 2–3 盆)

- **備齊當天立刻播種** — 每穴點播 2–3 粒種子,深度約 1–1.5 cm(種子直徑的 3 倍),穴距 15–20 cm;澆透;放全日照處

⚠️ **順序不能顛倒**:白蘿蔔**不耐移植**,必須直接播種在最終的盆裡,不可先育苗再移植(根會彎曲或死)。總預算 **NT$350 有找**,但**季節要對**(見上面 Step 0)。

**其他取得管道**

- **花市種子攤**:9–10 月會出現當季蘿蔔種子,可以現場問品種適不適合盆栽

- **農會超市 / 種苗行**:台南、彰化等農業縣市的農會賣場常有本土品種種子

⚠️ **不推薦直接用超市蘿蔔再生**:超市蘿蔔是採收後才販售,頭部插水雖然會發葉但無法再長出新的根,只能欣賞葉子,無法收成蘿蔔。

## 🏺 盆器與土壤

**盆器**

- **深度至少 25–30 公分**(根部需要往下生長空間,太淺蘿蔔會「頭大身短」或彎曲)

- 直徑 **30 cm 圓盆**可種 1 株;長方盆 **50×30 cm** 可種 2–3 株(株距 15 cm)

- 盆底**必須有排水孔**(蘿蔔怕積水,根部浸水會腐爛)

- 選**深色塑膠盆**比陶盆省重量,陽台承重比較不緊張

**土壤**

- **疏鬆、排水良好**最重要 — 蘿蔔是根莖,土太硬或太黏會讓根歪曲

- 推薦:培養土 2 份 + 珍珠石 1 份混合

- 避免使用未完全腐熟的有機肥直接接觸種子(會燒芽)

- 填土時不要壓實,讓根有鬆軟的環境往下鑽

*施肥細節獨立成下面的「🌾 施肥」section。*

## 🌳 種植步驟

- **確認盆深** — 至少 25 cm,不夠就換盆,不要將就

- **填土** — 培養土混珍珠石,填到盆沿下 3–4 cm,澆水讓土下沉定位後再補平

- **點播種子** — 每穴 2–3 粒,穴距 15–20 cm,深度約 1–1.5 cm,輕輕蓋土不壓實

- **第一次澆水** — 澆到盆底滲水,之後土表 1–2 cm 乾了再澆

- **放在全日照處** — 南向陽台最佳;日照不足根部不會膨大

> 

**✂️ 疏苗兩次 — 這步不做蘿蔔長不大**

每穴播多粒種子,必須透過疏苗讓最後只剩 1 株。擠在一起的蘿蔔根部互相競爭,全部都長不好。

- **第一次疏苗(子葉展開,約 Day 7–10)**:每穴留 2 株,拔掉最弱的

- **第二次疏苗(本葉 3 枚,約 Day 18–25)**:每穴只留 1 株,留最強壯的

拔下的嫩苗不要丟 — 洗淨可以當芽菜拌沙拉,帶點辛辣味,很好吃。

## 📋 完整時程表 · 播種到採收

**播種 → 發芽 → 疏苗 → 施肥 → 根部膨大 → 採收**,一張表涵蓋整季。**照做就對**。

> 

**⚠️ 開工前先看月曆 / 節氣:挑對時間,不然只長葉不長根**

白蘿蔔**喜涼怕熱**,超過 25°C 就抽苔開花,根部停止膨大。**節氣比月份準**:

## 🌾 施肥

**本 handbook 預設固體緩釋肥,不推液肥給新手** —— 不用稀釋、不用算濃度、撒一次吃 3 個月、難燒根。

*施肥時機(Day 30)已統整在上面「📋 完整時程表」。這個 section 專講 **用多少 · 用哪款 · 怎麼撒**。*

> 

**📐 不同盆大小 · 速查(好康多 1 號)**

| | 盆 | | 顆數 | |

| | 7 吋圓盆(Ø21cm)· 種 1 株 | | **8 粒** | |

| | 9 吋圓盆(Ø27cm)· 種 1 株 | | **10 粒** | |

| | **長方盆 50×30cm** · 種 2–3 株 | | **15 粒** | |

**原則**:寧可比這個少,不可比這個多。白蘿蔔過肥會促進葉生長而非根膨大。

**🛒 買哪一款**

- **好康多 1 號**(NPK 14-11-13)— 萬用款,根莖葉菜都適用

- 350g 小包約 NT$100,一包用好幾季

- 全聯園藝區、花市、iGarden 網購都有

- 替代品:魔肥、奧綠肥(類似緩釋顆粒肥都 OK)

> 

**⚠️ 白蘿蔔施肥的特殊注意**

- **疏苗後才施第一次肥**(Day 30):疏苗前施肥是在浪費;苗太多根本不會長大

- **不要追第二次肥**:白蘿蔔生長期約 60–90 天,一次緩釋肥夠了;追肥反而讓葉子狂長、根不膨大

- **氮肥不要過多**:好康多均衡肥 OK;純氮肥(如尿素)會讓葉瘋長但根又小又苦

**⚠️ 出事才看:一眼判斷 3 條(日常不用觀察)**

## ✂️ 採收

**根部採收**

- **判斷時機**:根頂露出土面 3–5 cm、葉叢茂密直挺,約播種後 60–90 天。冬天多等兩週更甜

- **採收方法**:抓住葉基部,**垂直往上連根拔起**(不要左右搖晃,會斷根)

- 拔出後立刻用剪刀**在距根頂 3–5 cm 處剪斷葉子**,延長根部保存期限

- 一盆只有 1–3 株,一次採完,不會再長新根

**蘿蔔葉的處理 — 別丟!兩種家常吃法**

> 

**🥬 蘿蔔葉兩用**

- **當天炒蛋**:嫩葉洗淨切段,加蒜炒蛋,口感類似菠菜,帶微辛。採收當天最鮮

- **曬蘿蔔葉乾(雪裡紅)**:葉子洗淨切碎,撒鹽搓揉(每斤葉 10g 鹽),晾半天出水後擰乾,放冰箱 2–3 天發酵成雪裡紅。可炒蛋、包餃子、加豆腐乾炒

**根部保存與料理**

- **冰箱冷藏**:帶皮不切,廚房紙巾包裹放蔬菜盒,可放 1–2 週

- **蘿蔔湯**:台式家常首選。切塊加排骨或小魚乾慢燉,湯頭清甜

- **蘿蔔乾(菜脯)**:切條鹽漬出水後陰乾 7–10 天。台灣年節必備,菜脯蛋、菜脯雞湯都靠它

- **醃蘿蔔**:切片,鹽 + 糖 + 醋醃隔夜即可吃,脆爽下飯

## 🛡 失敗點與除錯

| | 症狀 | | 可能原因 | | 對策 | |

| | 種子不發芽 | | 覆土太深 / 土太乾 / 種子過期 | | 深度不超過 1.5 cm;發芽前保持土面濕潤;買新種子 | |

| | 只長葉不長根 | | 播種時機太熱 / 日照不足 / 氮肥過多 | | 確認季節對(9–1 月);移到全日照處;停肥 | |

| | 蘿蔔根彎曲或分叉 | | 土太硬太黏 / 石頭障礙物 / 未確實疏苗 | | 種前把土打鬆混珍珠石;確認每穴只剩 1 株 | |

| | 根部中空(切開是空心) | | 採收太晚 / 生長期缺水 | | 第 70–80 天就要採,不要拖到 100 天以上;生長期水分穩定 | |

| | 提前抽苔開花 | | 春天回暖後才播 / 突然冷熱劇變 | | 嚴格在白露–小寒(9–1 月)播;抽苔後根已老化不好吃,只能採蘿蔔嬰享用 | |

| | 葉叢被蟲啃洞 | | 蚜蟲 / 小菜蛾 / 夜盜蟲 | | 早晨手抓或水沖;嚴重時套防蟲網。蘿蔔葉本來就多蟲,採收前再處理一次葉子即可 | |

| | 根頂發綠變苦 | | 根頂長期露出土面照到陽光 | | 發現根頂露出時用土稍微蓋一下(不是必須,但可防苦) | |

## 🧠 失智照護應用 Notes

### 輕度

> 

**📍 輕度:用蘿蔔召喚記憶,讓她主導廚房**

她記得蘿蔔湯、菜脯蛋是什麼。種蘿蔔可以喚起整個家常記憶脈絡 —— 「從前都是這樣種的」「阿嬤會做菜脯蛋」。

- 每天早上帶她去陽台看一下盆,問「葉子有沒有更大?」—— 觀察生長是低認知負擔的成就感

- 疏苗時讓她來拔:「這株太擠了,幫我拔掉這株」。單一指令、動作清楚

- 採收當天讓她**自己連根拔起** —— 這個動作的驚喜感是你無法複製的。讓她拔、讓她嚇一跳、讓她說「哇這個好大」

- 採收後一起洗蘿蔔、切蘿蔔 —— 觸覺(涼硬的根)+ 熟悉的廚房動作,雙重刺激

- **目標**:讓她覺得「我還是那個會做菜的人」,不是被照顧的人

這個階段的價值:**蘿蔔是有形的成果,不是抽象的「活動」。她能看到、摸到、吃到 —— 這三點是許多認知活動辦不到的**。

### 中度

> 

**📍 中度:化繁為簡,留下「她還能做的那個動作」**

她無法記得「上週種的蘿蔔」,但**當下看到蘿蔔葉,她的肌肉記憶還在**。照護者的工作是把對的東西放在她面前。

- **把盆放在她日常動線的視線高度**(不要放角落):她走過就會看到、伸手就會摸到葉子

- 澆水由你準備好水壺、放在盆旁邊,她看到水壺自然會澆 —— 不需要說話指令

- 採收時**你先鬆土,她來拔**:拔出來的瞬間表情是值得保留的記憶

- 蘿蔔葉讓她來**摘洗老葉**:重複性動作,她的手知道怎麼做

- 切蘿蔔如果她還能做 → 大塊不要求形狀;如果不行 → 你切好放她面前讓她**擺入鍋**

- **語言觸發用食物名稱**:「蘿蔔湯」「菜脯蛋」比「我們來照顧植物」更能喚起回應

這個階段的價值:**不要期待她「記得種蘿蔔這件事」;期待她「在拔蘿蔔那一秒是快樂的」**。那一秒就夠了。

### 重度

> 

**📍 重度:不要求動作,讓蘿蔔陪著她**

主動動作已非重點。蘿蔔對重度失智者的價值在於**感官層次** —— 冰涼的重量感、泥土的氣味、白色的光澤。

- 採收後把剛拔的蘿蔔放到她手心讓她握著 —— 涼的、重的、滑滑的觸感,比任何玩具都真實

- 用手引導她摸蘿蔔葉(柔軟、微粗糙)和根部(光滑、涼感):觸覺刺激

- 蘿蔔湯煮好端到她面前,讓蒸氣和香氣過來 —— 嗅覺記憶是失智最後消失的感官

- 不問她「好不好吃」「記不記得」——**只說「蘿蔔湯,你以前最喜歡喝」,然後陪著她**

- 蘿蔔白色的視覺特別清楚:放在木製托盤上,讓她看得到

這個階段的價值:**她不需要「參與種植」。她需要「蘿蔔這個東西出現在她的世界裡」**。那個形狀、那個重量、那個氣味 —— 可能帶出你永遠不會知道的那段記憶。
