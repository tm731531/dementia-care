# 🌱 蒜苗 家庭種植指南

> 類別:**葉菜** · 難度:極低(廚房剝好的蒜直接種) · 季節:播種 10–2 月(清明後太熱休眠)

## 🎯 5 秒總覽(5W1H)

- **👥 Who**:新手友善 · 懶人友善(注意:寵物勿食(貓狗有毒))
- **🌱 What**:蒜瓣(廚房剝好的,非冷凍) 10-15 瓣 · 15-20cm 深盆 · 培養土 3L
- **📅 When**:播種 10–2 月(清明後太熱休眠)
- **📍 Where**:家裡 買 / 陽台 / 窗邊 / 戶外 種
- **🔧 How**:10/11/12/1/2 月 下手,照「完整時程表」做

## 🛒 購物清單

- **繁殖材料**:蒜瓣(廚房剝好的,非冷凍) 10-15 瓣 (家裡, NT$0–0) ✨ 廚房就有,0 採購
- **盆器**:15-20cm 深盆 × 1 個 (NT$80–150)
- **土壤**:一般培養土 3L (NT$50–100)
- **肥料**:好康多 1 號 350g(週期短可省略) (多盆共用)

## 💡 為什麼秋冬首選從蒜苗開始

- **0 門檻** — 家裡廚房剝好的蒜瓣直接種,不用買種子/苗

- **秋冬專屬** — 日均溫 25°C 以下才發芽,正好補地瓜葉的空檔

- **快速成就** — 1 週發芽,30 天就能採蒜苗

- **多輪採收** — 剪葉留根,同一瓣可採 2–3 輪

- **失智照護** — 剝蒜、切蒜是家常動作,參與感強

## 🛒 取得方式

> 

**🗓️ Step 0 · 先看月曆再動手**

**現在要是 10 月 – 2 月(寒露 → 雨水)才開始。** 蒜頭**喜涼怕熱**,日均溫 25°C 以上會休眠不發芽,夏天種下去整批爛在土裡。如果現在是春夏,**等到立冬(11/7)前後**再下手,先把蒜頭準備好、冰箱冷藏(不是冷凍!)2 週打破休眠。

> 

**📦 推薦的新手包 · 零採購開工**

- **蒜瓣** — **家裡廚房剝好的就行**(不是冷凍過的),或市場 / 全聯買整顆(NT$30–50)

- **盆與土** — 15–20cm 深盆 + 培養土(NT$100 有找)

- **選對月份插下去** — 尖端朝上,覆土 2cm

總預算 **NT$100 有找**(蒜瓣用廚房的完全免費)。0 門檻,不用學園藝知識。

**蒜瓣來源選擇**

| | 來源 | | 能用嗎 | | 備註 | |

| | ✅ 家裡剝好的蒜 | | 可以 | | 只要不是冷凍過的。蒜膜(薄皮)不用特別保留 | |

| | ✅ 超市 / 市場整顆蒜 | | 可以 | | 發芽率最高,有皮保護 | |

| | ✅ 已經冒小芽的蒜 | | 最佳 | | 休眠已打破,種下去幾天就長 | |

| | ❌ 冷凍過的蒜 | | 不行 | | 低溫破壞發芽力,100% 失敗 | |

| | ❌ 軟爛、發黑的蒜 | | 不行 | | 已經腐敗 | |

| | ⚠️ 進口蒜 | | 看情況 | | 部分國家進口會噴抑芽劑,發芽率低 | |

> 

**💡 加分技巧:冷藏打破休眠**

種之前把蒜頭放冰箱**冷藏室 2 週**(不是冷凍!),模擬冬天低溫讓它「覺得該發芽了」,播種後發芽率和整齊度都高。趕時間可以跳過這步。

## 🏺 盆器與土壤

**盆器**

- 深 **15–20 cm**(蒜頭根系淺,不用深盆)

- 長方盆適合(一次種 10–20 瓣),圓盆也可

- 盆底**必須有排水孔**(蒜最怕爛根)

**土壤**

- 一般培養土即可

- **排水要好** — 蒜最怕積水,可混珍珠石/粗砂

- pH 中性偏酸(6.0–6.8),一般培養土都符合

*施肥細節獨立成下面的「🌾 施肥」section。*

## 🌳 種植步驟

- **備蒜瓣** — 把蒜頭剝成獨立一瓣一瓣,不用去膜(薄皮不影響發芽)

- **盆填土** — 培養土填 8 分滿,先澆一次水讓土服貼

- **插蒜瓣** — **尖端朝上**!間距 3–5 cm,覆土 2 cm,底部留 1 cm 土

- **澆透** — 澆到盆底有水流出

- **放半日照或明亮散光處** — 秋冬陽光不烈,全日照也 OK

> 

**⚠️ 尖端朝下 = 整批白種**

新手最常見的坑:把蒜瓣**方向搞反**。尖端(尖的那頭)朝上,底部(圓的那頭,有根痕)朝下。反了就長不出來。

## 📋 完整時程表 · 播種到採收

**播種 → 發芽 → 採蒜苗 → 採蒜頭**,一張表涵蓋整個週期。

> 

**⚠️ 開工前先看月曆 / 節氣:挑對時間,不然整批休眠不發**

蒜頭**喜涼怕熱**,日均溫 25°C 以上就不發芽,跟地瓜葉完全相反:

| | | | 月份 | | 節氣 | | 狀態 | |

| | ✅ 最佳 | | **10 – 2 月** | | **寒露(10/8)→ 雨水(2/19)** | | 涼爽、日照適中,發芽率 80%+ | |

| | 🟡 勉強 | | 3 / 9 月 | | 春分 / 白露 | | 邊緣期,發芽慢但能活 | |

| | ❌ 絕對別種 | | **4 – 8 月** | | **清明(4/4)→ 立秋(8/7)** | | 高溫休眠,整批爛在土裡 | |

**口訣:地瓜葉和蒜苗剛好輪替** —— 清明拔蒜種地瓜葉,立冬拔地瓜葉種蒜苗,全年有收成。

| | 天數 | | 階段 | | 植物狀態 | | 你的動作 | |

| | **Day 1** | | 播種 | | 蒜瓣插下去 | | 澆透;❌ 不施肥 | |

| | Day 2–6 | | 潛伏期 | | 外觀沒變化 | | 土乾才澆(免洗筷法) | |

| | Day 7–14 | | 發芽期 | | 冒出綠色尖芽 | | 繼續澆水 | |

| | Day 15–29 | | 生長期 | | 葉長 5–15 cm | | 土乾才澆 | |

| | **Day 30 ✂️** | | **首次採蒜苗** | | 葉長 15–20 cm | | ✅ 剪葉,留 2–3 cm 會再長 | |

| | Day 30–60 | | 多輪採收 | | 葉再長 | | ✅ 每 2–3 週再剪一次(共 2–3 輪) | |

| | Day 60–120 | | 開始分化 | | 莖變粗、底下長蒜頭 | | 正常照顧 | |

| | **春夏(清明後)** | | **休眠警戒** | | 葉枯黃 | | ✅ 拔起整顆蒜頭(約 Day 150–180)或直接廢盆重來 | |

> 

**📱 播種當天就要做的 2 件事**

- Google 行事曆設 2 個提醒:**今日 + 30 天**(第一次採蒜苗)、**今日 + 150 天**(看要不要拔蒜頭)

- 便利貼寫今日日期 + 蒜瓣來源(廚房 / 市場品牌),貼盆邊 —— 比對不同來源的發芽率

## 🌾 施肥

**蒜苗生長期短(30 天就採),多數情況下不用追肥** —— 蒜瓣本身自帶養分,30 天採完就拔盆。如果想多輪採收或等蒜頭成熟,才需要施。

> 

**📐 用量(好康多 1 號)**

- 小盆(Ø15cm,種 3–5 瓣):**3–5 粒**

- 中盆(Ø21cm,種 8–10 瓣):**6–8 粒**

- 長方盆 50×20cm(種 15–20 瓣):**12–15 粒**

播種後**Day 30 第一次採蒜苗時**撒,撒完澆水啟動緩釋。不採的話可以不施。

**不需要施肥的情況**

- 只種 30 天採一次蒜苗就拔 → 不用施

- 蒜瓣本身夠大(市場買整顆的),養分自給 → 不用施

**建議施肥的情況**

- 想多輪採葉(2–3 輪)→ 第一次採後撒一次緩釋肥

- 想等整顆蒜頭成熟(Day 150+)→ 中途撒 1–2 次

通用施肥原則看 [📚 通用園藝知識 → 🌾 施肥](#)。

## ✂️ 採收

**採蒜苗(主要用途)**

- 葉長 **15–20 cm** 時剪(通常 Day 30)

- 從離土面 **2–3 cm 處剪**,不要剪到生長點

- 剪完 2–3 週會再長,可採 **2–3 輪**

- 一次剪一點,炒菜、下麵、蔥花替代品都好用

**採整顆蒜頭(進階,需耐心)**

- Day 150–180(約 5–6 個月後)

- 葉子**枯黃 2/3** 時拔起

- 風乾 1–2 週後可長期保存

- 需要的溫度條件高,**家庭小盆通常蒜頭不大**,採蒜苗比較實際

> 

**🍳 採下來馬上能用**

蒜苗就是**青蒜**,切小段炒肉、煮湯、做炒飯、替代青蔥,風味濃郁。剪下來清水沖一下直接切就行。

## 🛡 失敗點與除錯

| | 症狀 | | 可能原因 | | 對策 | |

| | **整批不發芽** | | 蒜瓣冷凍過 / 溫度太高(>25°C) | | 確認蒜來源非冷凍;等氣溫降再試 | |

| | 只發一半 | | 蒜瓣品質不均 / 冷藏催芽不足 | | 可以的話冰箱冷藏 2 週再種 | |

| | 冒芽後軟爛 | | 土太濕 / 排水不良 | | 減少澆水;檢查盆底排水孔 | |

| | 葉子細弱 | | 日照不足 / 種太密 | | 移到光亮處;間距至少 3 cm | |

| | 葉子發黃 | | 缺水 / 根部悶 | | 免洗筷法判斷;嚴重拔起換盆 | |

| | 春天後葉枯 | | 正常 — 高溫休眠 | | 可拔起看蒜頭;或直接廢盆秋天重種 | |

## 🧠 失智照護應用 Notes

### 輕度

> 

**📍 輕度:剝蒜播種是熟悉動作**

她能自主剝蒜、插土、澆水,整個流程是「家常延伸」不是「學新東西」。

- 她**自己剝蒜瓣**(跟平常煮菜剝蒜是同個動作)

- **她自己插土**,你只幫忙「尖端朝上」提醒

- 每次剪蒜苗時讓她參與,**「自己種的蒜苗」的自我感**

- 蒜苗切段炒菜時讓她切 —— 熟悉的刀、熟悉的蒜味

### 中度

> 

**📍 中度:單一動作她還能做**

她剝蒜的動作沒丟,但「插入土 + 方向對 + 覆土」這種多步驟會卡住。

- 你事先把土整好、挖好洞,**請她「把蒜瓣放進洞裡」**(單一動作)

- 剪蒜苗時**你剪、放她面前的碗,請她挑掉老葉**

- 澆水:水壺裝好擺盆旁邊,她知道動作是「倒下去」

- 蒜味會勾起她對廚房的記憶,**聞蒜也是參與**

### 重度

> 

**📍 重度:感官參與,不求動作**

動作不強求,但**蒜的味道是強記憶**,剪下的蒜苗放她手上讓她聞。

- 剪一把蒜苗放她手上 —— **嗅覺刺激**是她最後失去的感官之一

- 你剝蒜她看,節奏感比動作重要

- 煮蒜苗湯時讓她在廚房待著,味道喚醒食慾

- 每天固定時段剪一點,儀式比完成度重要
