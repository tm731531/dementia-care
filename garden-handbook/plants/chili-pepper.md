## 💡 為什麼辣椒是家庭陽台果菜的首選

- **台灣氣候天生適合** — 春夏高溫多日照，辣椒如魚得水，種了很難不活

- **一株採好幾個月** — 春天種下，夏秋持續結果，每週都有東西可採

- **紅綠同株、視覺張力強** — 未熟是綠，熟了是紅，同一株上顏色並存；失智長輩每天都看得到變化

- **採收是「尋寶」動作** — 找出哪顆紅了可以採，比葉菜更有成就感

- **不需要搭架** — 多數品種 40–60 公分自立，陽台盆栽不用工程

- **品種彈性大** — 不吃辣選糯米椒或青龍椒（幾乎不辣），全家都能參與採收

## 🛒 取得方式

> 

**🗓️ Step 0 · 先看月曆再動手**

**4–9 月才是辣椒的時間。** 12–2 月播種發芽極慢（低溫下種子發芽率低、幼苗站不住），新手幾乎必失敗。如果現在是冬天，**等到隔年 3 月驚蟄後、或 4 月清明後直接買苗**再下手。買苗比播種快 6–8 週，新手首選。

> 

**📦 推薦的新手包 · 三步照順序來**

- **先備盆與土（這一步要比苗先到）** — 花市買 **直徑 25–30 公分深盆 + 疏水性培養土（或蔬菜專用土）**，合計 NT$150–250，搬回定位、填土 8 分滿、先澆一次讓土服貼。

- **再買苗** — **花市 / 農會超市 / 蝦皮** 搜尋「辣椒苗」或「朝天椒苗」，4 月起大量有貨，一株 NT$30–80，買 2–3 株備份。品種首選：**朝天椒**（最容易、最多貨）或**糯米椒**（不辣、家人敢採）。

- **苗到當天立刻定植** — 輕輕取出苗、放入盆、填土壓實、澆透到盆底出水、放全日照戶外

⚠️ **順序不能顛倒**：苗買回沒盆土接，根系在袋裡壓一天就受損。總預算 **NT$400 有找**，比自己播種省 6 週等待。

**品種速查 — 新手怎麼選**

| | 品種 | | 辣度 | | 特色 | | 新手適合度 | |

| | **朝天椒** | | 辣 | | 最常見、最耐種、結果量大 | | ⭐⭐⭐ 首選 | |

| | **糯米椒** | | 幾乎不辣 | | 肉厚、台灣家常菜常用、老人小孩都能嚐 | | ⭐⭐⭐ 首選 | |

| | **青龍椒** | | 微辣 | | 外形像大辣椒、視覺衝擊、採收感強 | | ⭐⭐ 適合 | |

| | **雞心椒** | | 大辣 | | 傳統台灣辣椒、香氣足 | | ⭐⭐ 適合 | |

**其他取得管道**

- **播種自育**（進階）：3–4 月或 8–9 月，溫度 20–30°C 發芽率高，10–14 天出芽，但要多等 6–8 週才能定植。適合想掌控品種的人。

- **農會 / 社區菜市場**：有時有現成苗，比蝦皮更便宜也能看苗況

> 

**⚠️ 播種新手常踩的坑**

種子發芽需要溫度穩定在 20–30°C。台灣 3 月前氣溫波動大，很多人播下去等兩週沒動靜就放棄。其實是溫度不夠，不是種子壞了。**買苗直接跳過這個等待，新手最推。**

## 🏺 盆器與土壤

**盆器**

- 直徑 **25–30 公分以上**、深 25 公分以上（辣椒根系比地瓜葉深）

- 越深越好 — 結果期需要穩固根系撐住果實重量

- 盆底**必須有排水孔**，積水是辣椒最大殺手

- 黑色盆吸熱，台灣夏天陽台建議用白色或淺色盆（避免根部過熱）

**土壤**

- **疏鬆、排水好的培養土**，可選蔬菜專用土或一般培養土混珍珠石（比例 7:3）

- 辣椒怕積水爛根，排水能力比地力更重要

- 定植前可拌入少量有機基肥（雞糞堆肥），讓土先有底肥

- pH 偏好 6.0–6.8（微酸），一般培養土通常在這個範圍，不用特別調

**底部排水層（可選但推薦）**

- 盆底鋪 3–5 公分碎石或粗砂，改善排水

- 或舖幾片碎陶片蓋住排水孔，防土粒流出同時保持排水

*施肥細節獨立成下面的「🌾 施肥」section。*

## 🌳 種植步驟

- **備盆** — 盆底鋪排水層，填土 8 分滿，先澆水讓土沉降

- **定植苗** — 從育苗盆輕輕取出，保持根團完整，放入定植穴，深度與苗原本土面齊平

- **填土壓實** — 從四周填土，輕壓讓根與土密合，不留空氣包

- **第一次澆水澆透** — 澆到盆底有水流出，確認排水正常

- **放全日照位置** — 每天至少 6 小時直射陽光，果實才會結；**光線不足只長葉不開花**

> 

**💡 定植後的 1 週緩苗期**

剛定植的苗需要 5–7 天適應新環境（「緩苗」），可能暫時葉片下垂，這是正常的。**這週不要施肥、保持土壤微濕即可**，讓根系先站穩。

## 📋 完整時程表 · 定植到採收

**定植 → 緩苗 → 開花 → 結果 → 首次採收 → 持續採收 → 冬季低迷**，一張表涵蓋整個生命週期。**照做就對，不用觀察植物狀態。**

> 

**⚠️ 開工前先看月曆 / 節氣：辣椒是「喜熱的植物」，季節錯了一切白費**

## ✂️ 採收

**何時可採？**

- **青椒（未熟）**：果實飽滿、表皮有光澤，即可採 —— 通常定植後 90–120 天

- **紅椒（成熟）**：完全轉紅後採，辣度與香氣最足，視覺衝擊最強

- **糯米椒 / 青龍椒**：偏向採青，飽滿亮澤即採，不用等紅

**怎麼採？**

- **用剪刀剪**，不要用手扭斷（扭斷容易傷到主枝）

- 保留蒂頭 1 公分，讓剪口不會接近植物枝幹

- **越採越多** — 採收刺激新花苞形成，不採反而讓植株「滿足」停止結果

- 盛產期可每 1–2 週採一輪，每輪可採 5–20 顆（視株數）

**紅綠辨認技巧（失智照護特別版）**

- **「紅色 = 可以採了」** — 用顏色當信號，簡單清晰

- **「綠色 = 再等一下」** — 兩個規則，不需要其他判斷

- 讓長輩每天去「巡視」有沒有新紅椒，這是最天然的日常任務

## 🛡 失敗點與除錯

| | 症狀 | | 可能原因 | | 對策 | |

| | 開花但花掉落、不結果 | | 光線不足、溫度過高（＞35°C）、氮肥過多、濕度低 | | 移到更日照處；開花期停氮肥加磷鉀；輕搖莖幫助授粉；盆土澆水避免過乾 | |

| | 只長葉不開花 | | 氮肥太多、日照不足 | | 停好康多 1–2 個月；施一次花寶 2 號；確保全日照 | |

| | 葉片發黃（整株） | | 缺肥、根部積水、根系不健康 | | 確認排水；若排水 OK 就補施肥 | |

| | 葉片捲曲、有蚜蟲 | | 蚜蟲群聚 | | 手指搓掉或用水柱沖；可噴稀釋洗碗精水（1:200）；避農藥（要吃的） | |

| | 果實表面有凹陷斑點 | | 鈣缺乏（果腐病初期） | | 給水要均勻、不要讓土過乾後猛澆；可施鈣鎂液肥一次 | |

| | 幼苗定植後整株塌軟 | | 正常緩苗反應（移植衝擊） | | 等 3–5 天，保持土微濕，不施肥，多數會自行恢復 | |

| | 冬天完全停止生長 | | 低溫（正常） | | 移室內避霜；停肥；等春天 | |

## 🧠 失智照護應用 Notes

### 輕度

> 

**📍 輕度：紅色是每天的小驚喜，採收是有成就感的任務**

她還能自己去陽台、自己判斷「這顆紅了沒」、自己剪下來。辣椒的採收邏輯比地瓜葉更有意思 —— 有目標（找紅的）、有判斷（夠紅了嗎）、有成果（採下來給廚房用）。

- **建立每日任務**：早上澆水順便看有沒有紅椒可採，這是完整的「出任務」儀式

- **讓她拿剪刀**：她還能精確操作剪刀，讓她主導採收，你在旁邊看即可

- **紅色是最強的視覺錨點**：綠葉叢中一顆紅椒，即使輕度認知退化也能鎖定目標

- **香氣召喚記憶**：採收時讓她聞辣椒香味，連結台灣料理記憶（炒豆干、麻婆豆腐）

- **目標**：讓她覺得「家裡有個東西需要我去照顧」，維持自我效能感

這個階段的價值：**「我還是那個會種菜、會採菜、會給廚房送菜的人」**。

### 中度

> 

**📍 中度：簡化判斷、具體化任務、用顏色代替語言**

她可能記不住「去陽台採辣椒」這件事，但你說「妳去看看有沒有紅的」她能懂並執行。辣椒的「紅色 = 可採」規則比任何語言指令都簡單。

- **單一指令**：「妳去幫我看有沒有紅椒可以摘」—— 一句話、一個動作、不要連說兩件事

- **水壺事先擺好盆旁邊**：不要讓她去找水龍頭，減少失敗點（找不到 → 放棄 → 挫折）

- **剪刀放固定位置**：就在盆旁邊一個小籃子裡，不需要找

- **收穫即時回饋**：她採回來的辣椒立刻用（晚餐炒菜）—— 讓她看到「我採的辣椒今天上桌了」

- **不辣品種更安全**：如果家裡有中度失智長輩，建議選糯米椒，她可以直接咬一口確認（不辣不危險）

這個階段的價值：**把複雜的判斷簡化成「看顏色」一個規則，她能做到、你不用緊盯。**

### 重度

> 

**📍 重度：不要求採收，紅色本身就是刺激**

她可能無法完成採收動作，但**紅色辣椒是視覺上最強的感官刺激之一**。在綠葉叢中一顆顆的紅椒，會讓她的目光停留、甚至自然伸手觸摸。這就夠了。

- 帶她坐在辣椒盆旁邊，讓她**看紅椒**（強烈視覺對比，視覺皮質仍有反應）

- 把一顆紅椒放到她手上 —— 讓她**握著感受重量和光滑表皮**，不要求她做什麼

- 採下來的辣椒讓她**聞味道**（台灣家庭廚房的嗅覺記憶，可能喚起深層情緒記憶）

- 你澆水時讓她在旁邊**看著水滲入土** —— 流水聲 + 土濕的氣味，感官豐富

- 每天同樣時段、同樣動作，**節奏感比任何動作完成度都重要**

這個階段的價值：**紅椒不是食物，是感官錨點。它的顏色、重量、氣味 —— 這些在失智晚期仍能抵達她。**
