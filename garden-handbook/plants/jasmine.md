## 💡 為什麼適合家庭種茉莉花

- **香氣最強的家庭花卉之一** — 夏夜開花,整個陽台都是香味,不需要靠近才聞得到

- **多年生,一株種好可以連續開 5 年以上** — 不像葉菜年年重種,買一次就有了

- **開花期長達 5 個月(5–9 月)** — 每天都有新花,每天都有事做

- **採花有儀式感** — 每晚或清晨新花開放,固定時間採花是很好的日常錨點

- **香氣做茉莉花茶、泡水、裝飾** — 採下來的花立刻有用處,不只是觀賞

- **失智照護核心應用** — 茉莉花香對長輩有強烈嗅覺記憶(夏夜、童年、家鄉),即使嗅覺退化也常能感應

## 🛒 取得方式

> 

**🗓️ Step 0 · 先看月曆再動手**

**現在要是 3–7 月(春夏)才最適合買苗定植。** 秋天(9–10 月)可以買苗但當年不會開幾朵;冬天寒流期(12 月 – 2 月)盡量別定植 — 新苗抗寒力弱,容易葉黃脫落甚至死苗。如果現在是冬天,**等到隔年 3 月驚蟄後**再下手,現在可以先備盆土、認識品種。

> 

**📦 推薦的新手包 · 三步照順序來**

- **先備盆與土(這一步要比苗先到)** — 花市買 **6–8 吋深盆(直徑 18–25cm)+ 排水良好的培養土**,土裡混 10–20% 珍珠石幫助排水(合計 NT$150–250)。盆土填 8 分滿,澆一次水讓土服貼。

- **再買苗** — 花市、農會、蝦皮搜尋「茉莉花 盆栽苗」(NT$80–150 一盆),選葉片深綠有光澤、沒有黃葉爛根的。單瓣茉莉花香比雙瓣略清淡但更耐種;雙瓣(洋茉莉)花大香濃但稍難)。

- **到貨當天立刻定植** — 從原盆輕輕脫出,不要過度鬆土(保留原根球),移入新盆,填土壓實,澆透到盆底出水,放半日照以上處適應 1 週。

⚠️ **順序不能顛倒**:苗到家了才找盆,根球在袋子裡悶幾小時就受傷。總預算 **NT$350 有找**,一次種好連開好幾年。

## 🏺 盆器與土壤

**盆器**

- 直徑 **18–25 公分(6–8 吋)**,深 20 公分以上

- 茉莉花根系喜歡「溫暖 + 透氣」— 陶盆 / 赤陶盆比塑膠盆透氣,但夏天乾得快要多澆,新手用塑膠盆也 OK

- 盆底**必須有排水孔** — 積水爛根是茉莉花最常見的死法

- 多年生植株 2–3 年後根系塞滿就換大一號

**土壤**

- 一般培養土 + 10–20% 珍珠石(或粗河砂)— 混合後排水加分

- 偏酸性(pH 6–7)最佳;台灣一般培養土多在此範圍不需調整

- 不建議用純黑土(保水過強,夏天悶熱容易爛根)

- 2–3 年換盆時可以補新土 + 有機堆肥(見施肥 section)

## 🌳 種植步驟

- **選盆備土** — 見上面「盆器與土壤」。土填 8 分滿,先澆一次水服貼。

- **脫盆定植** — 苗輕輕從原盆脫出,保留根球不要搖散。放進新盆中央,根部與土面齊平(不要埋太深)。

- **填土壓實** — 四周填土,用手指輕壓讓土跟根系接觸,避免空洞。

- **第一次澆水澆透** — 澆到盆底有水流出,讓根系與新土充分接觸。

- **放半遮陰處適應 7 天** — 定植後第一週避開正午強烈直射(新苗脫水壓力大),1 週後移到全日照 / 半日照正常位置。

- **春末到夏天看花開** — 定植後 3–4 週發根穩定,春末開始現蕾,5 月後盛花期。

> 

**✂️ 冬季修剪是關鍵一步**

茉莉花的花長在**當年新枝**的頂端 — 冬天(11 月 – 2 月,落葉或葉疏時)進行一次重剪,剪掉當年開過花的老枝 1/3–1/2,能促使春天長出更多新枝,新枝越多花越多。**不修剪的茉莉花兩三年後越開越少**,很多人不知道這個原因。

修剪細節在「✂️ 採收與修剪」section。

**定植 → 發根 → 施肥 → 開花 → 冬剪 → 隔年更旺**,多年生植株的一年節奏。

> 

**⚠️ 開工前先看月曆 / 節氣:茉莉花喜熱,季節對了才香**

## ✂️ 採花與冬季修剪

**採花:每晚新花,每晚採**

- 茉莉花**花苞在晚上開放**(傍晚 18–22 時最盛),清晨也能採還沒謝的花

- 採**剛開或半開的花苞** — 最香、保存最久;完全開、花心露出的香氣已過了一半

- 用手輕輕摘下花朵,或用剪刀連小段花梗剪;不要扯傷枝條

- 採下的花放淺碟子、泡水碗、或直接散放:香氣會持續 6–12 小時

- **摘花不傷植株** — 花謝後不採也沒關係,只是消耗養分

**採花用途**

| | 用法 | | 做法 | | 適合對象 | |

| | 🍵 茉莉花茶(快速版) | | 新鮮花放進已泡好的綠茶碗裡靜置 5 分鐘,撈出花,喝茶 | | 長輩 / 全家 | |

| | 💧 茉莉花水 | | 花放入水瓶,冷藏 4 小時,淡香甜水(不加糖) | | 長輩 / 照顧者 | |

| | 🛁 香氛 | | 花放小碟子擺床頭、浴室、客廳;散發香氣 4–8 小時 | | 重度失智者(嗅覺刺激) | |

| | 📿 裝飾 / 儀式 | | 花插頭髮、夾書、泡澡水、壓花 | | 輕中度(參與感) | |

**冬季修剪:多年生茉莉花的關鍵一刀**

> 

**🗓️ 時機:11 月 – 2 月(立冬後 / 落葉或葉疏時)**

茉莉花**花開在當年新枝頂端**。若不修剪,每年枝條越長越老,花越開越少,最後只有頂端零星幾朵。

- 剪掉當年開過花的枝條 **1/3–1/2**(留 2–3 個飽滿芽點在枝條上)

- 枯枝、交叉枝、向內長的枝優先剪掉

- 剪完像個禿頭是正常的 — 春天新芽會長出來

- 剪下來的嫩枝可以扦插繁殖(半木質化 10cm 段,插入濕土)

**扦插繁殖(有剪多餘枝條時)**

- 選**當年生半木質化枝條**(彎折有點硬、不是全嫩綠也不是全木質)— 春末到夏天都可以取

- 剪 10–12cm 一段,留頂端 2–3 片葉,下端斜剪

- 插入濕潤培養土 3–4cm,放半日照處;3–4 週發根

- 春夏扦插成功率 70–80%;秋天降至 40–50%

## 🛡 失敗點與除錯

| | 症狀 | | 可能原因 | | 對策 | |

## 🧠 失智照護應用 Notes

### 輕度

> 

**📍 輕度:嗅覺記憶觸發 + 儀式主導**

她還能主導多步驟動作。茉莉花的儀式可以做到最完整 — 澆水、看花苞、每晚採花、拿進廚房泡茉莉花茶或茉莉水,全鏈條都能做。

- **嗅覺記憶觸發**:請她聞花的時候,自然問「這個香味讓你想到什麼?」— 茉莉花香常能觸發夏夜、外婆院子、廟會、年輕時愛喝茉莉花茶等記憶

- **每晚採花是固定錨點**:傍晚 18 時固定去陽台採花,時間感 + 目的感都很強;比「去陽台散步」更有意義

- **讓她主導分花用途**:「今天的花要泡茶還是放床頭?」— 小決策維持自主感

- **茉莉花茶一起泡**:採花 → 泡茶 → 一起喝,完整的社交儀式,不只是照護任務

- **目標**:她還是那個「家裡種茉莉花的人」,照顧者是配角

這個階段的價值:**香氣記憶比視覺記憶衰退慢**,茉莉花是少數能持續觸發長期記憶的家庭植物。

### 中度

> 

**📍 中度:簡化動作 + 香氣引路**

她能執行單一動作,但多步驟的「去陽台 → 認出花苞 → 採下來 → 拿回廚房」需要拆解引導。

- **香氣引路**:傍晚帶她去陽台,讓她先聞花香再說話 — 嗅覺直接連結邊緣系統,有時比言語更能讓她當下「到位」

- **預先準備小碗放旁邊**:你已經採好幾朵放碗裡,讓她拿著聞;她不需要完成採花動作

- **一句指令一個動作**:「把這朵花放到碗裡」而不是「幫我採花拿進去泡茶」

- **水已經裝好放桌上**:請她把花放進水杯,不需要她找水、找杯(失敗點)

- **香氣不要問記憶**:不要說「這是什麼花?你還記得嗎?」— 記憶失敗會讓她沮喪;說「香不香?」就夠

這個階段的價值:**香氣是語言以外的溝通頻道**。即使話說不清楚,她對茉莉花香的反應(微笑、深呼吸、放鬆)是真實的情緒回應。

### 重度

> 

**📍 重度:不求動作,香氣就是照護**

她的自主動作有限、語言溝通困難。茉莉花在重度階段的價值**從「活動」變成「感官陪伴」**。

- **採幾朵花放她手上**:讓她握著,不要求她做任何事;花的溫度和質感本身就是觸覺刺激

- **放花在枕頭旁邊**:入睡前幾朵茉莉花放床頭小碟;香氣持續 6–8 小時,可以伴她整晚

- **做茉莉水讓她喝**:花泡冷水 4 小時,淡淡的甜香水;比一般水更能讓她願意喝

- **你澆花她在旁邊**:帶她坐在茉莉花盆旁,你澆水、她在旁邊,不要求互動;存在就夠

- **固定香氣建立安全感**:每天同一時間同一地點聞茉莉花,重複的感官刺激對重度失智者有穩定作用

這個階段的價值:**嗅覺是最後退化的感官之一**。即使她忘記了你的名字、忘記了自己的房間、甚至忘記了白天剛發生的事,茉莉花的香氣還是能讓她當下感到「這裡是安全的,是家的味道」。
