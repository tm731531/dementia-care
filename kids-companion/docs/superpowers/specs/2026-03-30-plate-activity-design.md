# 餐盤設計活動 設計文件

## 目標

新增「餐盤設計」活動（`page-plate`），讓 2–6 歲兒童運用食物原型活動所學，自由或任務式地組合食材裝盤，學習營養均衡概念。這是「從認知到產出」的創造型活動。

## 架構

```
進入活動
  → toddler/small：自由裝盤模式
  → middle：任務模式
  → large：分格餐盤模式
  → 裝完按「開動！」→ 回饋畫面 → 完成
```

## 資料來源

直接複用現有食物資料，不需新增：
- `FOOD_CATEGORIES`（6 類）
- `FOODS`（110 種食材）
- `IMG['food-*']`（108 張圖片）

## 年齡分級

### toddler — 自由裝盤 + 角色回饋

**玩法：**
- 畫面上方：圓形餐盤（空的）
- 畫面下方：食物選擇區，顯示 12 種隨機食物（從全部 110 種中抽取，每類至少 1 種），用圖片+名稱顯示
- 點擊食物 → 食物出現在餐盤上（隨機擺放位置，帶小動畫）
- 最多放 5 樣食物
- 放完或按「開動！」→ 進入回饋

**回饋：**
- 🦊 角色出現，語音說「好好吃！🦊 吃得好開心！」
- 不做營養評價，純正面鼓勵

**選項數/操作：** 點擊（不拖拉）

### small — 自由裝盤 + 角色回饋（進階）

**玩法：**
- 同 toddler，但食物選擇區顯示 16 種
- 最多放 6 樣食物
- 點擊操作

**回饋：**
- 🦊 角色 + 語音
- 多一句營養提示：「你選了蔬菜和水果，🦊 的眼睛會變得好厲害！」
- 根據選到的食物類別數給不同台詞：
  - 1~2 類：「好好吃！下次可以試試不同的食物喔！」
  - 3~4 類：「哇！好均衡！🦊 變得好有力氣！」
  - 5~6 類：「太厲害了！營養超級均衡！🦊 開心地轉圈圈！」

### middle — 任務模式 + 簡單回饋

**玩法：**
- 系統隨機出一道任務，例如：
  - 「請幫 🦊 準備一份有蔬菜和蛋白質的午餐」
  - 「🦊 想吃有水果和五穀的點心」
  - 「請選出三種不同類別的食物給 🦊」
- 每道任務指定 2~3 個必須包含的食物類別
- 食物選擇區顯示 16 種食物（含必要類別 + 干擾項）
- 孩子選 3~5 樣食物
- 按「開動！」→ 檢查是否滿足任務條件

**任務池（8 題，隨機抽 1 題）：**

| 任務文字 | 必要類別 |
|---------|---------|
| 幫 🦊 準備有蔬菜和蛋白質的午餐 | veggie, protein |
| 🦊 想吃有水果和奶類的點心 | fruit, dairy |
| 選三種不同類別的食物給 🦊 | 任意 3 類不同 |
| 幫 🦊 準備有五穀和蔬菜的晚餐 | grain, veggie |
| 🦊 需要堅果和水果補充能量 | nut, fruit |
| 準備一份有蛋白質和五穀的便當 | protein, grain |
| 幫 🦊 選有蔬菜和水果的沙拉 | veggie, fruit |
| 🦊 想要均衡的一餐，至少選四種不同類 | 任意 4 類不同 |

**回饋：**
- 成功：「太棒了！你選了 N 類食物，好均衡！」+ 語音
- 失敗：「差一點點！再加一樣 [缺少的類別] 就完美了！」→ 回到選擇畫面繼續選

### large — 分格餐盤 + 星星評分

**玩法：**
- 餐盤分成 6 格，每格標註一個食物類別（🥦蔬菜、🍎水果、🍚五穀、🥚蛋白質、🥛奶類、🥜堅果）
- 每格下方顯示 3 個該類別的食物選項（隨機抽取）
- 孩子每格選 1 樣食物放入
- 可以跳過某些格（不強制全填）
- 按「開動！」→ 評分

**評分：**
- 填 1~2 格：⭐（1 星）「不錯的開始！」
- 填 3~4 格：⭐⭐（2 星）「吃得真均衡！」
- 填 5~6 格：⭐⭐⭐（3 星）「完美營養餐！」+ `launchConfettiSmall()`

**語音：** 只朗讀評語，不念選項

## UI 結構

### 自由裝盤（toddler/small）

```
[← 返回]  🍽️ 餐盤設計

  ┌─────────────┐
  │   圓形餐盤    │  ← 食物出現在這裡
  │  （白底圓盤）  │
  └─────────────┘

  [🥦花椰菜] [🍎蘋果] [🥚雞蛋] ...  ← 可捲動的食物列

       [ 開動！🍴 ]
```

### 任務模式（middle）

```
[← 返回]  🍽️ 餐盤設計

  「幫 🦊 準備有蔬菜和蛋白質的午餐」

  ┌─────────────┐
  │   圓形餐盤    │
  └─────────────┘

  [花椰菜] [蘋果] [雞蛋] [牛奶] ...

       [ 開動！🍴 ]
```

### 分格餐盤（large）

```
[← 返回]  🍽️ 餐盤設計

  ┌───┬───┬───┐
  │🥦 │🍎 │🍚 │
  │蔬菜│水果│五穀│
  ├───┼───┼───┤
  │🥚 │🥛 │🥜 │
  │蛋白│奶類│堅果│
  └───┴───┴───┘
  每格: [選項1] [選項2] [選項3]

       [ 開動！🍴 ]
```

## 語音規則

- toddler：選食物時自動念食物名稱；回饋語音完整朗讀
- small：選食物時念名稱；回饋語音完整朗讀
- middle：進入時朗讀任務文字；回饋朗讀
- large：不自動朗讀選項；只朗讀評語

## 遊戲化整合

- 完成活動：`completeActivity('page-plate')` → +1 星星
- 貼紙：🍽️
- 首頁探索 tab 加入活動卡片

## 程式架構

| 區塊 | 錨點 |
|------|------|
| HTML 頁面 | `#SECTION:PAGE-PLATE` |

不需要獨立 DATA section（複用 FOODS）。不需要新圖片。

## 不在範圍內

- 真實營養計算（卡路里、克數）
- 食譜建議
- 餐盤儲存/匯出
- 拖拉操作（toddler/small 用點擊，large 用格內選擇）
