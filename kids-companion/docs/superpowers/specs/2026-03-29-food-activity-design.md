# 食物原型活動 設計文件

## 目標

新增「食物原型」活動（`page-food`），讓 2–6 歲兒童透過瀏覽與測驗認識天然食材，了解食物分類與對身體的益處。

## 架構

與動物園活動（`page-zoo`）完全相同的三層架構：

```
分類貨架（food-shelf）
  → 瀏覽卡片（food-browse）：逐張翻閱，語音念名稱
    → 測驗（food-quiz）：每 5 張觸發一題（中班/大班）
      → 完成畫面（food-complete）
```

## 每張卡片資訊

```
[圖片]
中文名稱（大字）
英文名稱（小字）
簡短描述（1句，適合大人讀給小孩聽）
身體功能提示（例：讓眼睛更厲害 👀）
```

## 年齡差異

| 年齡 | 行為 |
|------|------|
| toddler | 瀏覽，自動朗讀中文名稱，2 個選項測驗 |
| small | 瀏覽，朗讀名稱，2–3 個選項測驗 |
| middle | 每 5 張觸發測驗（這是哪類食物？），3–4 選項 |
| large | 每 5 張觸發測驗，4 選項，只念題目不念選項 |

## 測驗題型

- **toddler/small**：「這是什麼？」→ 選正確食物名稱
- **middle/large**：「這是哪一類食物？」→ 選正確分類（蔬菜/水果/五穀/魚肉蛋豆/奶類/堅果）

## 圖片來源

Wikipedia Commons，與動物園相同方法：
- Wikipedia thumbnail API（200px）
- 轉 base64，存入 `index.html` 末尾 `#SECTION:IMAGES-FOOD` 區塊
- 使用 `Object.assign(IMG, { 'food-apple': '...', ... })`
- 無法取得圖片時 fallback 為 emoji

## 遊戲化整合

- 完成活動：`completeActivity('page-food')` → +1 星星
- 貼紙：🍎（蘋果）
- 首頁探索 tab 加入活動卡片

## 程式架構（參照 page-zoo）

| 區塊 | 錨點 |
|------|------|
| HTML 頁面 | `#SECTION:PAGE-FOOD` |
| 食物資料 | `#SECTION:PAGE-FOOD-DATA` |
| JS 邏輯 | `#SECTION:PAGE-FOOD-JS` |
| 圖片資料 | `#SECTION:IMAGES-FOOD`（檔案末尾） |

## 不在範圍內

- 加工食品（餅乾、糖果、飲料）
- 食譜或烹飪步驟
- 熱量/營養數字（太複雜，非此年齡目標）
