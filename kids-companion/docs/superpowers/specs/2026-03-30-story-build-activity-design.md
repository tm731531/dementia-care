# 故事拼組活動 設計文件

## 目標

新增「故事拼組」活動（`page-story-build`），讓 2–6 歲兒童從圖卡庫選擇角色、場景、物件等素材，排列成自己的故事，系統用語音念出來。這是「開放式創造」型活動，重點在創作體驗而非對錯。

## 架構

```
進入活動
  → toddler/small：自由排列模式（選 3~4 張卡）
  → middle：欄位式（誰→去哪→做什麼→遇到誰）
  → large：欄位式 + 多一欄「然後…」
  → 按「說故事！」→ 語音念出 → 回饋
```

## 素材來源

全部複用現有資料，不需新增：

| 類型 | 來源 | 用途 |
|------|------|------|
| 角色 | 動物園動物（從 `ANIMALS` 各分類隨機取） | 「誰」「遇到誰」 |
| 場景 | `SCENE_THEMES`（草地/海邊/森林） | 「去哪」 |
| 食物 | `FOODS`（從各分類隨機取） | 自由模式素材 / large「然後…」 |
| 動作 | 內建動作詞陣列 | 「做什麼」 |
| 圖片 | `IMG['zoo-*']`、`IMG['food-*']` | 圖卡顯示 |

## 年齡分級

### toddler — 自由排列 + 純鼓勵

**玩法：**
- 畫面上方：故事軌道（3 個空格，水平排列）
- 畫面下方：8 張圖卡（隨機混合動物、場景 emoji、食物）
- 點擊圖卡 → 填入下一個空格（帶 pop 動畫）
- 點擊已填的空格 → 移除
- 填滿 3 格後出現「說故事！」按鈕

**語音輸出：** 依序念出 3 張卡的名稱，中間頓 600ms
例如：「兔子… 草地… 蘋果」

**回饋：** 🦊「好有趣的故事！」純正面鼓勵

### small — 自由排列 + 數量回饋

**玩法：**
- 故事軌道 4 個空格
- 10 張圖卡可選
- 同 toddler 操作

**語音輸出：** 念出 4 張卡名稱

**回饋：** 🦊 + 「你用了 N 種不同的東西！好豐富！」（計算不同類型：角色/場景/食物）

### middle — 欄位式 + 類型回饋

**玩法：**
- 4 個欄位，各有標題和 3 個選項：
  - 「誰」→ 3 隻動物（圖片+名字）
  - 「去了」→ 3 個場景（emoji+名字）
  - 「做什麼」→ 3 個動作（emoji+文字）
  - 「遇到誰」→ 3 隻動物（圖片+名字，不重複第一欄）
- 每欄點一個選項 → 高亮選中
- 4 欄都選完後出現「說故事！」

**語音輸出：** 組成完整句子
「[角色] 去了 [場景] [動作]，遇到 [角色2]」
例如：「兔子 去了 草地 散步，遇到 獅子」

**回饋：** 「好棒的故事！你用了角色、場景和動作，好完整！」

### large — 欄位式 + 第五欄 + 星星

**玩法：**
- 5 個欄位，各有 4 個選項：
  - 「誰」→ 4 隻動物
  - 「去了」→ 3 個場景 + 1 個額外場景 emoji
  - 「做什麼」→ 4 個動作
  - 「遇到誰」→ 4 隻動物
  - 「然後…」→ 4 個結局（混合食物和物件）
- 5 欄都選完後出現「說故事！」

**語音輸出：**
「[角色] 去了 [場景] [動作]，遇到 [角色2]，然後 [結局]」
例如：「大象 去了 森林 探險，遇到 貓頭鷹，然後 一起吃蘋果」

**回饋：**
- 填完全部 5 欄：⭐⭐⭐ + 「完美的故事！太有創意了！」
- 填 4 欄：⭐⭐ + 「很棒的故事！」
- 填 3 欄以下：⭐ + 「不錯的開始！」

## UI 結構

### 自由模式（toddler/small）

```
[← 返回]  📖 故事拼組

  [ ? ] [ ? ] [ ? ] ([ ? ])  ← 故事軌道（3 或 4 格）

  [🐰兔子] [🌿草地] [🍎蘋果] [🦊狐狸] ...  ← 圖卡列

       [ 說故事！📖 ]
```

### 欄位模式（middle/large）

```
[← 返回]  📖 故事拼組

  誰？        去了哪裡？    做什麼？      遇到誰？
  [🐰兔子]   [🌿草地]    [🚶散步]     [🦁獅子]
  [🦊狐狸]   [🏖️海邊]    [🎵唱歌]     [🐬海豚]
  [🐘大象]   [🌲森林]    [💃跳舞]     [🦉貓頭鷹]

       [ 說故事！📖 ]
```

## 語音規則

- toddler：選卡片時自動念名稱；故事念完後念回饋
- small：選卡片時念名稱；故事念完後念回饋
- middle：不自動念選項；按「說故事」後念完整句子 + 回饋
- large：不自動念選項；念完整句子 + 回饋

## 遊戲化整合

- 完成活動：`completeActivity('page-story-build')` → +1 星星
- 貼紙：📖
- 首頁探索 tab 加入活動卡片
