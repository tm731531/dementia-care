# 數學樂園活動 設計文件

## 目標

新增「數學樂園」活動（`page-math`），讓 2–6 歲兒童透過兩個子模式（算術、時鐘）學習基礎數學概念。進入活動後選擇模式，各自獨立完成。

## 架構

```
模式選擇（math-shelf）
  → 🔢 算術模式（math-arithmetic）
    → 多輪出題 → 完成畫面（math-complete）
  → 🕐 時鐘模式（math-clock）
    → 多輪出題 → 完成畫面（math-complete）
```

## 算術子模式

### 年齡分級

| 年齡 | 題型 | 數字範圍 | 選項數 | 輪數 |
|------|------|----------|--------|------|
| toddler | 數 emoji 動物「一共幾隻？」 | 1–5 | 2 | 4 |
| small | 圖示輔助加法「🐱🐱 + 🐱 = ?」 | 1–10 | 3 | 5 |
| middle | 數字算式加減「8 - 3 = ?」 | 1–20 | 4 | 6 |
| large | 連續算式「5 + 3 - 2 = ?」 | 1–30 | 4 | 8 |

### 出題邏輯

- **toddler**：隨機選 1–5 隻動物 emoji 排列在畫面上，問「一共幾隻？」，2 個選項
- **small**：顯示加法視覺題，左邊 N 隻 + 右邊 M 隻，問「一共幾隻？」，3 個選項。N+M ≤ 10
- **middle**：純數字加法或減法算式。加法 a+b，a,b ∈ [1,10]，答案 ≤ 20。減法 a-b，a ∈ [3,20]，b < a，答案 ≥ 0
- **large**：連續加減算式 a ± b ± c。a ∈ [1,15]，b,c ∈ [1,10]，中間結果和最終答案都 ≥ 0 且 ≤ 30

### 選項生成

正確答案 + (選項數-1) 個干擾項。干擾項 = 正確答案 ± 1~3（不重複、不為負數）。

## 時鐘子模式

### 年齡分級

| 年齡 | 可出題範圍 | 選項數 | 輪數 |
|------|-----------|--------|------|
| toddler | 整點（1:00–12:00） | 2 | 4 |
| small | 整點（1:00–12:00） | 2 | 5 |
| middle | 整點 + 半點（X:00, X:30） | 3 | 6 |
| large | 整點 + 半點 + 刻鐘（X:00, X:15, X:30, X:45） | 4 | 8 |

### 時鐘盤面

使用純 CSS + inline SVG 繪製：
- 圓形錶盤（白底、灰邊框）
- 12 個小時刻度（粗線）+ 數字 1–12
- 時針（短粗）+ 分針（長細）
- JS 計算指針角度：
  - 時針角度 = (hour % 12) × 30 + minute × 0.5
  - 分針角度 = minute × 6

### 題目格式

- 顯示時鐘盤面 + 問「現在幾點？」
- toddler/small：選項顯示「3 點」格式
- middle：選項顯示「3 點」或「3 點半」格式
- large：選項顯示「3 點」「3 點 15 分」「3 點半」「3 點 45 分」格式

### 選項生成

正確答案 + 干擾項（同類型的隨機時間，不重複）。

## 語音

- toddler/small：自動朗讀題目和選項
- middle：朗讀題目，hover 念選項
- large：只朗讀題目

## 答題回饋

與其他活動相同：
- 答對：`playCorrectSound()` + 鼓勵語（「太棒了！」）+ 綠色高亮
- 答錯：`playWrongSound()` + 溫和提示（「再試一次！」）+ 紅色高亮
- toddler 答錯：直接高亮正確答案
- small 答錯：高亮正確答案
- middle：第 2 次錯才高亮
- large：只提示再試

## UI 結構

### 模式選擇畫面（math-shelf）

兩張大卡片，水平排列：
```
┌──────────┐  ┌──────────┐
│    🔢    │  │    🕐    │
│   算術    │  │   時鐘    │
└──────────┘  └──────────┘
```

### 算術出題畫面（math-arithmetic）

```
[← 返回]  🔢 算術  [3/6]

   🐱🐱🐱 + 🐱🐱 = ?
    （或純數字算式）

 [  3  ]  [  5  ]  [  4  ]
```

### 時鐘出題畫面（math-clock）

```
[← 返回]  🕐 時鐘  [2/6]

    ┌───────┐
    │  12   │
    │9  ⌚ 3│
    │  6    │
    └───────┘
    現在幾點？

 [3 點] [6 點] [9 點]
```

## 遊戲化整合

- 完成任一子模式：`completeActivity('page-math')` → +1 星星
- 貼紙：🔢（數字）
- 首頁探索 tab 加入活動卡片

## 程式架構

| 區塊 | 錨點 |
|------|------|
| HTML 頁面 | `#SECTION:PAGE-MATH` |
| JS 邏輯 | `#SECTION:PAGE-MATH-JS` |

不需要獨立的 DATA section（資料量小，直接寫在 JS 裡）。
不需要圖片（時鐘用 CSS/SVG，算術用 emoji）。

## 不在範圍內

- 乘法、除法
- 分數、小數
- 真實圖片（全部用 emoji 和 CSS 繪製）
- 計時挑戰（速度壓力不適合此年齡）
