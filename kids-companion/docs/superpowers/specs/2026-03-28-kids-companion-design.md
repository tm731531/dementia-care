# 小朋友學習樂園 — 設計規格

## 概述

給 3–6 歲兒童在平板使用的學習互動 app。寓教於樂，親子兩用，中英文可切換。
所有功能在單一 `index.html`，完全離線可用。

---

## 視覺風格

**溫暖自然**：溫柔不過度刺激，像繪本的感覺。

| 用途 | 色碼 |
|------|------|
| 主色（橘） | `#E8724A` |
| 背景（米白） | `#FFF8F0` |
| 輔色（草綠） | `#98D8C8` |
| 暖黃 | `#FFB347` |
| 卡片白 | `#FFFFFF` |
| 深字色 | `#333333` |
| 淺字色 | `#888888` |

- 圓角：20px（卡片）、50px（按鈕）
- 陰影：`0 4px 16px rgba(0,0,0,0.08)`
- 字型：Nunito → 蘋方 → 微軟正黑體（圓體優先）
- 基礎字級：22px；按鈕：28px；遊戲目標字：80–120px

## 活動規格

### 1. 認中文字（page-chinese）
- junior：3 選項，筆畫少的象形字（山、日、月、水、火、木）
- senior：6 選項，形近字組干擾（形/邢/刑/型）
- 語音：朗讀目標字名稱及選項

### 2. 英文字母（page-abc）
- junior：認 A–J，每次顯示一個字母配大圖，3 選項
- senior：認 A–Z，從 4 選項中選正確字母，配單字發音
- 語音：字母發音 + 單字（A is for Apple）

### 3. 數數（page-counting）
- junior：1–5 個物品，3 選項數字
- senior：1–20，5 選項；進階有簡單加法（2+3=?）
- 語音：「數數看，有幾個？」逐一念出數字

### 4. 詞彙學習（page-vocab）
- 顯示 emoji 圖片，選出正確名稱（中文 or 英文依語言設定）
- junior：3 選項，常見物品（蘋果、狗、車子）
- senior：6 選項，較少見（草莓、企鵝、飛機）
- 語音：讀出物品名稱

### 5. 顏色形狀（page-colors-shapes）
- 兩種子模式：顏色模式 / 形狀模式（交替出現）
- junior：4 種基本色、3 種形狀，高對比
- senior：8 種顏色（含相近色）、6 種形狀
- 語音：「這是什麼顏色？」、「找出一樣的形狀」

### 6. 音樂節奏（page-music）
- junior：點 emoji 樂器發出音效（🥁🔔🎺🎸），自由探索
- senior：聽節奏樣式，照樣打拍子（Simon Says 形式）
- 音效：使用 Web Audio API 產生簡單音調

### 7. 拼圖（page-puzzle）
- 使用 CSS grid 切割圖片（emoji 大圖）
- 拖拉拼塊到正確位置（pointer events）
- junior：2×2（4 塊）
- senior：3×3（9 塊）

### 8. 記憶翻牌（page-memory）
- 牌面朝下，翻兩張找相同 emoji
- junior：2×3（6 張，3 對）
- senior：3×4（12 張，6 對）
- 配對成功保持翻開，不計時

### 9. 排序分類（page-sort）
- 兩種子模式：大小排序 / 類別分類（交替出現）
- junior：大小排序 3 個物品（差距大）
- senior：分類 4–5 個物品到 2–3 個類別（拖拉放入）

### 10. 互動繪本（page-story）
- 書架頁顯示可選的故事（2×N 格）
- 每本書：多頁，每頁有背景圖（emoji 組合）、文字、語音
- 點擊畫面中的物件有聲音/動畫反應
- 左右滑動或點箭頭翻頁
- 內建故事資料格式：
```javascript
const STORIES = [
  {
    id: 'monkey-king',
    title: { zh: '孫悟空大鬧天宮', en: 'The Monkey King' },
    cover: '🐒',
    pages: [
      {
        bg: ['☁️','🏔️','🌳'],
        text: { zh: '很久很久以前，有一隻石頭裡跑出來的猴子...', en: 'Long long ago, a monkey was born from a magic stone...' },
        objects: [
          { emoji: '🐒', sound: '哇！我出生了！', x: 50, y: 60 }
        ]
      }
    ]
  }
];
```
- junior：語音自動朗讀每一頁文字
- senior：點播放鍵才朗讀

## 遊戲化系統

### 角色選擇（初次啟動）
- 全屏歡迎畫面，選一個角色：🐱 🐶 🐼 🦊 🐧 🐸
- 選定後角色出現在首頁 header，陪伴整個 app
- 之後可在設定中更換

### 星星系統
- 每完成一個活動得 1 顆星
- 首頁顯示今日進度（例：⭐⭐⭐☆☆）
- 集滿 5 顆：全屏慶祝動畫（五彩紙屑 + 角色跳舞）
- 每天重置

### 貼紙書
- 每個活動對應一張專屬貼紙
- 首次完成解鎖，點首頁 header 的 🎴 按鈕查看貼紙書頁
- 已解鎖：彩色顯示；未解鎖：灰色輪廓（共 10 格）

### 連續天數
- 比對 lastPlayDate 與今天日期
- 進 app 時顯示「你連續玩了 N 天！」
- 中斷後重置為 1

### 慶祝動畫
- 答對時：小型五彩紙屑從頂部飄落 + 角色跳一下
- 完成活動：大型慶祝動畫 + 星星飛進進度條
- 集滿 5 星：全屏派對動畫（3 秒）

### 家長給讚
- 遊戲中右下角小按鈕 👏（半透明，不干擾遊戲）
- 按下後：全屏出現「爸爸/媽媽說你好棒！❤️」3 秒

### 神秘寶箱
- 當天完成 3 個（以上）活動後，首頁出現寶箱動畫
- 點擊開箱：顯示「今日小知識」（一張圖 + 一句話 + 語音）
- 小知識內容：內嵌在 HTML 裡的 `TRIVIA` 陣列，例如：
  - 🦒「長頸鹿的脖子有 1.8 公尺！比你還高很多喔」
  - 🌙「月亮不會自己發光，它反射太陽的光」
  - 🐬「海豚睡覺時只睡一半的腦」
  - 🌱「向日葵白天會轉頭追著太陽走」

### 搖一搖
- 偵測 devicemotion（加速度 > 15）
- 觸發「今日驚喜」：隨機跳到一個活動，搭配動畫

## 資料轉移機制

localStorage 只存在當前裝置，換裝置即遺失。設定面板提供匯出/匯入功能讓家長轉移孩子的進度。

### 匯出
- 設定面板底部「匯出進度」按鈕
- 將完整 APP 狀態序列化為 JSON，觸發瀏覽器下載
- 檔名：`kids-companion-backup-YYYYMMDD.json`
- 格式：
```json
{
  "version": 1,
  "exportedAt": "2026-03-28T14:00:00",
  "data": {
    "language": "zh",
    "ageGroup": "junior",
    "character": "🦊",
    "stars": 3,
    "stickers": ["page-chinese", "page-memory"],
    "streak": 5,
    "lastPlayDate": "2026-03-28",
    "completedToday": ["page-chinese"]
  }
}
```

### 匯入
- 設定面板底部「匯入進度」按鈕
- 點擊後開啟系統檔案選擇器（`<input type="file" accept=".json">`）
- 讀取 JSON，驗證 `version` 欄位存在且為數字
- 驗證通過：覆蓋 localStorage，顯示「進度已還原！」並重新載入
- 驗證失敗（格式錯誤）：顯示「檔案格式不對，請選正確的備份檔」

### 注意事項
- 匯入會**完全覆蓋**現有進度，操作前顯示確認提示
- 不支援跨版本合併，只做整包覆蓋
- 匯出/匯入操作均為純前端，不傳送任何資料到外部
