# UI/UX 設計規格

## 設計理念
為 2–6 歲兒童打造「好玩、好看、不害怕」的平板學習體驗。
溫暖自然的繪本風格，所有設計以「孩子不會感到壓力或恐懼」為核心。

---

## 圖示原則 — 小動物為主，清晰優先

### 核心原則
- **小動物優先**：所有圖示、獎勵、引導元素，優先使用動物形象
- **清晰可辨**：圖示在 48px 以上必須一眼辨識，不用細節複雜的圖形
- **情感連結**：動物帶來親切感，降低陌生感和焦慮

### 應用範圍

| 用途 | 建議動物 emoji | 說明 |
|------|----------------|------|
| 角色選擇 | 🐱 🐶 🐼 🦊 🐧 🐸 | 6 種清晰可辨的動物 |
| 活動圖示 | 🐘🦁🐬🐦🐰🐻 | 各活動搭配一種動物 |
| 答對獎勵 | 🐥（小雞跳舞） | 固定用小雞做慶祝動畫 |
| 神秘寶箱 | 🐿️（松鼠開箱） | 松鼠挖出寶物的意象 |
| 休息提醒 | 🦉（貓頭鷹） | 「貓頭鷹說要休息囉」 |
| 今日小知識 | 動物配對主題 | 每則知識搭配該動物 emoji |

### 各活動對應動物圖示

| 活動 | 動物圖示 | 理由 |
|------|----------|------|
| 認中文字 | 🐼 | 貓熊 = 中文/台灣印象 |
| 英文字母 | 🦜 | 鸚鵡學說話 |
| 數數 | 🐘 | 大象記憶力好，數數聯想 |
| 詞彙學習 | 🦁 | 獅子大聲說名字 |
| 顏色形狀 | 🦋 | 蝴蝶色彩繽紛 |
| 音樂節奏 | 🐸 | 青蛙跳跳打節拍 |
| 拼圖 | 🐢 | 烏龜慢慢拼，不急 |
| 記憶翻牌 | 🐬 | 海豚記憶力強 |
| 排序分類 | 🐝 | 蜜蜂分工整理 |
| 互動繪本 | 🦉 | 貓頭鷹代表智慧與故事 |

### 尺寸規範
- **活動卡片圖示**：64px（clear at 2× on retina）
- **遊戲內引導圖示**：80px 以上
- **角色 header**：48px
- **完成慶祝動畫**：120px（全屏時放大至 160px）

### 禁止
- 不使用細節複雜的圖形（蟲類、恐怖動物）
- 不用動作模糊的圖示
- 不用純幾何圖形取代動物（活動卡片一律用動物 emoji）

## 配色方案（溫暖自然）

| 用途 | 色碼 | 說明 |
|------|------|------|
| 主色 | `#E8724A` | 暖橘，主要互動元素 |
| 背景 | `#FFF8F0` | 米白，護眼不刺激 |
| 輔色綠 | `#98D8C8` | 草綠，輔助強調 |
| 輔色黃 | `#FFB347` | 暖黃，獎勵元素 |
| 卡片 | `#FFFFFF` | 白卡，內容承載 |
| 深字 | `#333333` | 主要文字 |
| 淺字 | `#888888` | 輔助文字 |
| 答對 | `#4CAF82` | 柔和綠，正向回饋 |
| 答錯 | `#E8724A` | 暖橘搖晃，溫和提示 |

### 護眼原則
- 禁用高飽和霓虹色（`#FF0000`、`#00FF00`）
- 背景不用純白，使用米白 `#FFF8F0`
- 答錯不用紅色閃爍，改用橘色輕微搖晃
- 禁止任何快速閃爍（頻率不超過每秒 3 次）

## 字體規格

- **字型**：Nunito → 蘋方 → 微軟正黑體（圓體優先，親切感）
- **基礎字級**：22px
- **按鈕文字**：28px
- **遊戲目標字**：依年齡組 80–140px
- **行高**：1.8（寬鬆，易閱讀）

---

## 按鈕規格

- 最小高度：72px（適合小手指點選）
- 圓角：50px（圓潤感）
- 按鈕間距：至少 20px（避免誤觸）
- 按下效果：縮放 0.92 + 輕微陰影消失（0.15s）
- 觸控區域：最小 60×60px

## 頁面佈局

### 首頁
```
┌────────────────────────────────────────┐
│ 🦊 早安！   ⭐⭐⭐☆☆   🎴  中文  ⚙️  │
├────────────────────────────────────────┤
│  📚 語言  │  🎨 創作  │  🧠 思維  │ 📖 │
├────────────────────────────────────────┤
│                                        │
│   [認中文字]    [英文字母]              │
│                                        │
│   [數數]        [詞彙學習]             │
│                                        │
└────────────────────────────────────────┘
```

- Header：角色 + 星星進度 + 貼紙書入口 + 語言切換 + 設定
- Tab 切換：4 個分類（語言/創作/思維/故事）
- 活動卡片：2 欄排列，圖示 + 名稱，圓角白卡

### 設定面板（齒輪，滑入式）
- 班別選擇：幼幼班 / 小班 / 中班 / 大班（4 個大按鈕）
- 陪伴者稱謂：爸爸 / 媽媽 / 老師 / 阿姨 / 醫師 / 陪伴者
- 語言切換：中文 / English
- 聲音開關
- 使用時長提醒開關
- 匯出進度 / 匯入進度
- 更換角色

### 遊戲頁面（通用結構）
```
┌────────────────────────────────────────┐
│ ← 回首頁              遊戲名稱   👏     │
├───────────────┬────────────────────────┤
│               │                        │
│   題目 / 目標  │   選項區域             │
│   （大字）     │   （大按鈕 grid）      │
│               │                        │
└───────────────┴────────────────────────┘
```
- 左側：題目文字（大字體）
- 右側：選項 grid（2–4 欄，依選項數調整）
- 右下角：家長給讚按鈕 👏（半透明）
- 手機 / 直向平板：上下排列

## 角色選擇頁（初次啟動）

全屏顯示，6 個角色大卡：
```
🐱  🐶  🐼
🦊  🐧  🐸
```
選完後動畫：角色飛到 header，歡迎語音「嗨！我是你的好朋友，我們一起玩吧！」

## 動畫規格

| 動畫 | 時長 | 觸發 |
|------|------|------|
| 答對（小型）五彩紙屑 | 1.5s | 每次答對 |
| 角色跳一下 | 0.5s | 每次答對 |
| 完成活動慶祝 | 3s | 完成一個活動 |
| 集滿星星派對 | 3s | 第 5 顆星 |
| 貼紙解鎖飛入 | 1s | 完成活動後 |
| 頁面進入 | 0.2s | 任何頁面切換 |
| 答錯搖晃 | 0.4s | 點到錯誤選項 |

五彩紙屑規格：從畫面頂部緩緩飄落（非爆炸），顏色用暖色系，速度適中。

## 聲音設計

| 音效 | 描述 |
|------|------|
| 答對音 | 輕快「叮！」，0.3s |
| 答錯音 | 低沉溫和「啵」，0.3s |
| 完成音 | 短促歡快旋律，1s |
| 貼紙解鎖 | 輕柔「叮咚」，0.5s |
| 頁面切換 | 無（靜音） |
| 集滿星星 | 歡樂旋律，2s |

**禁止**：突然大音量、尖銳蜂鳴、倒數計時聲、警報聲。

## 使用時長提醒 UI

### 5 分鐘警告橫幅
頁面頂部滑入，高度 48px，暖橘底色：
```
👀 再玩 5 分鐘就要讓眼睛休息一下喔！   [×]
```

### 休息時間全屏
角色動畫 + 大字：
```
      🦊（伸懶腰動畫）

   眼睛累了，我們去喝水伸懶腰！

  [好，休息 🌿]    [繼續玩 10 分鐘]
```
