## 活動規格

### 1. 認中文字（page-chinese）
| 等級 | 內容 |
|------|------|
| 幼幼班 | 2 選項，象形字（山🏔️、日☀️、月🌙、水💧、火🔥、木🌳） |
| 小班 | 2–3 選項，常見筆畫少的字（大、小、上、下、手、口） |
| 中班 | 3–4 選項，國小預備字彙，加入同部首字 |
| 大班 | 4–6 選項，形近字干擾（形/邢/刑/型） |
語音：朗讀目標字名稱及各選項

---

### 2. 英文字母（page-abc）
| 等級 | 內容 |
|------|------|
| 幼幼班 | 認 A–E，超大字母配動物圖，2 選項 |
| 小班 | 認 A–J，字母配圖，3 選項 |
| 中班 | 認 A–Z，4 選項，配單字發音（A is for Apple） |
| 大班 | 認 A–Z，4 選項，加入大小寫配對 |
語音：字母發音 + 單字（英文模式）

---

### 3. 數數（page-counting）
| 等級 | 內容 |
|------|------|
| 幼幼班 | 1–3 個物品，2 選項 |
| 小班 | 1–5 個物品，3 選項 |
| 中班 | 1–10 個，4 選項 |
| 大班 | 1–20，5 選項；進階有簡單加法（2+3=?） |
語音：「數數看，有幾個？」逐一念出數字

---

### 4. 詞彙學習（page-vocab）
| 等級 | 內容 |
|------|------|
| 幼幼班 | 2 選項，最基本物品（奶瓶、球、床） |
| 小班 | 3 選項，常見物品（蘋果、狗、車子） |
| 中班 | 4 選項，中等常見（草莓、公車、雨傘） |
| 大班 | 6 選項，較少見（企鵝、飛機、望遠鏡） |
顯示 emoji 圖片，選出正確名稱（依語言設定顯示中/英文）

---

### 5. 顏色形狀（page-colors-shapes）
兩種子模式交替：顏色模式 / 形狀模式

| 等級 | 顏色 | 形狀 |
|------|------|------|
| 幼幼班 | 3 種基本色（紅黃藍），2 選項 | 圓形/方形，2 選項 |
| 小班 | 4 種基本色，高對比，3 選項 | 3 種形狀 |
| 中班 | 6 種顏色，中對比，3–4 選項 | 4 種形狀 |
| 大班 | 8 種（含相近色），4 選項 | 6 種形狀 |

---

### 6. 音樂節奏（page-music）
| 等級 | 內容 |
|------|------|
| 幼幼班 | 點大型樂器 emoji 發音，純自由探索 |
| 小班 | 同上，加入「跟我一樣」模仿單個音效 |
| 中班 | 聽 2–3 個音的節奏樣式，照樣打拍子 |
| 大班 | 聽 3–5 個音的節奏樣式（Simon Says） |
音效：Web Audio API 產生簡單音調（無需外部音檔）

---

### 7. 拼圖（page-puzzle）
| 等級 | 規格 | 操作 |
|------|------|------|
| 幼幼班 | 2×2（4 塊） | 點選（非拖拉） |
| 小班 | 2×2（4 塊） | 拖拉 |
| 中班 | 2×3（6 塊） | 拖拉 |
| 大班 | 3×3（9 塊） | 拖拉 |
CSS grid 切割 emoji 大圖；pointer events 支援觸控

---

### 8. 記憶翻牌（page-memory）
| 等級 | 規格 |
|------|------|
| 幼幼班 | 2×2（4 張，2 對） |
| 小班 | 2×3（6 張，3 對） |
| 中班 | 3×3（9 張，圖案提示輔助） |
| 大班 | 3×4（12 張，6 對） |
牌面朝下，翻兩張找相同 emoji，配對成功保持翻開，不計時

---

### 9. 排序分類（page-sort）
兩種子模式交替：大小排序 / 類別分類

| 等級 | 大小排序 | 類別分類 |
|------|----------|----------|
| 幼幼班 | 2 個（明顯差距），點選 | — |
| 小班 | 3 個（差距大），點選 | — |
| 中班 | 3–4 個，拖拉 | 2 類別，拖拉放入 |
| 大班 | 4 個，拖拉 | 3 類別，4–5 個物品 |

---

### 10. 互動繪本（page-story）
**書架頁**：2 欄 grid 顯示可選故事，有封面 emoji + 書名

**閱讀頁**：
- 背景 emoji 組合構成場景
- 文字置底（大字，語音朗讀）
- 左右滑動或點箭頭翻頁
- 點畫面物件：觸發音效 + 小動畫

**年齡組差異**：
- toddler/small：自動朗讀每頁文字
- middle/large：點播放鍵才朗讀

**內建故事**（2 本）：
1. 孫悟空大鬧天宮（西遊記兒童版）
2. 曹沖稱象（三國 / 歷史故事兒童版）

**故事資料格式**（可擴充）：
```javascript
const STORIES = [
  {
    id: 'monkey-king',
    title: { zh: '孫悟空大鬧天宮', en: 'The Monkey King' },
    cover: '🐒',
    pages: [
      {
        bg: ['☁️','🏔️','🌳'],
        text: { zh: '很久很久以前...', en: 'Long long ago...' },
        objects: [{ emoji: '🐒', sound: '哇！我出生了！', x: 50, y: 60 }]
      }
    ]
  }
];
```

## 遊戲化系統

### 角色選擇
- 初次開啟全屏顯示，選一個：🐱 🐶 🐼 🦊 🐧 🐸
- 角色出現在首頁 header，設定可更換

### 星星系統
- 完成一個活動 +1 顆星，首頁進度條顯示（X/5）
- 集滿 5 顆：全屏慶祝動畫（3 秒），每天重置

### 貼紙書
- 每個活動對應一張貼紙（共 10 格）
- 首次完成解鎖，彩色；未解鎖灰色輪廓
- 點首頁 🎴 按鈕查看

### 連續天數
- 比對 `lastPlayDate` 與今天，連續進 app 累加 `streak`
- 進 app 時顯示「你連續玩了 N 天！🔥」

### 慶祝動畫
- 答對：小型五彩紙屑飄落 + 角色跳一下
- 完成活動：全屏慶祝 + 星星飛進進度條
- 集滿 5 星：全屏派對動畫（3 秒）

### 家長給讚
- 遊戲中右下角半透明 👏 按鈕
- 按下：全屏顯示「爸爸/媽媽說你好棒！❤️」3 秒

### 神秘寶箱
- 當天完成 3 個以上活動後首頁出現寶箱
- 點擊：顯示「今日小知識」（emoji 圖 + 一句話 + 語音）
- 內容來源：`TRIVIA` 陣列（內嵌在 HTML）

### 搖一搖
- 偵測 devicemotion（加速度 > 15）
- 隨機跳到一個活動，搭配「今日驚喜！」動畫

## 使用時長提醒

| 班別 | 建議上限 |
|------|----------|
| 幼幼班 / 小班 | 30 分鐘 |
| 中班 / 大班 | 45 分鐘 |

- 剩 5 分鐘：頂部橫幅「再玩 5 分鐘就要休息囉 👀」
- 達上限：全屏休息畫面，家長可延長 10 分鐘
- 設定面板可關閉

## 資料匯出 / 匯入

- **匯出**：下載 `kids-companion-backup-YYYYMMDD.json`（完整 APP 狀態）
- **匯入**：上傳 JSON，驗證 `version` 欄位，確認後整包覆蓋 localStorage
- 純前端操作，資料不離開裝置
