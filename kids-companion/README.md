# 小朋友學習樂園 — 使用手冊

> 一位工程師爸爸為 2023 年出生的女兒打造的學習遊戲。
> 想做什麼就做什麼，用最純粹的前端技術，不裝 app、不看廣告、不傳資料。

純前端、零廣告、完全離線可用。

---

## 快速開始

1. 用瀏覽器開啟 `index.html`（或前往部署網址）
2. 第一次使用：選一個小動物角色
3. 在首頁上方選擇孩子的班別
4. 點選喜歡的活動，開始玩！

不需要帳號、不需要安裝、不需要網路。

## 班別設定

請依孩子實際就讀班別設定，app 會自動調整難度和語音輔助：

| 班別 | 年齡 | 特色 |
|------|------|------|
| 幼幼班 | 約 2–3 歲 | 2 個選項、超大字體、語音重複朗讀所有選項 |
| 小班 | 約 3–4 歲 | 2–3 個選項、語音朗讀所有選項 |
| 中班 | 約 4–5 歲 | 3–4 個選項、可用手指拖拉、語音朗讀題目 |
| 大班 | 約 5–6 歲 | 4–6 個選項、挑戰難度、語音只念題目 |

## 25+ 種活動

### 📚 語言類
| 活動 | 學習重點 |
|------|----------|
| 認中文字 | 從象形字到形近字，依班別調整難度 |
| 英文字母 | A–Z 認識與發音，配合圖片記憶 |
| 數數 | 1–20 數量認知，大班有簡單加法 |
| 詞彙學習 | 看 emoji 圖片，選出正確名稱（中/英文） |

### 🎨 創作類
| 活動 | 學習重點 |
|------|----------|
| 顏色形狀 | 認識顏色和幾何形狀，交替出題 |
| 音樂節奏 | 點樂器發音（幼幼/小班），跟著節拍打鼓（中/大班） |

### 🧠 思維類
| 活動 | 學習重點 |
|------|----------|
| 拼圖 | 拖拉拼塊完成圖片，訓練空間概念 |
| 記憶翻牌 | 翻牌找相同圖案，訓練短期記憶 |
| 排序分類 | 大小排序 + 類別分類，訓練邏輯概念 |

### 📖 故事類
| 活動 | 說明 |
|------|------|
| 互動繪本 | 經典故事兒童版，點物件有音效和動畫 |
| 童謠朗讀 | 經典兒歌，可循環播放搭配動作提示 |

### 🔍 探索類（新！）
| 活動 | 說明 |
|------|----------|
| 動物園 | 120 種動物圖鑑，依分類瀏覽 + 測驗 |
| 食物原型 | 110 種天然食材，6 大類分類 + 測驗 |
| 數學樂園 | 算術（加減法）+ 時鐘認讀，兩個子模式 |
| 餐盤設計 | 從六大類食物自由/任務裝盤，學營養均衡 |
| 場景佈置 | 選場景（草地/海邊/森林），拖放素材自由創作 |
| 故事拼組 | 選圖卡排列成故事，系統語音念出來 |

## 遊戲化功能

| 功能 | 說明 |
|------|------|
| 集星星 ⭐ | 每完成一個活動得 1 顆星，集滿 5 顆有全屏慶祝動畫 |
| 貼紙書 🎴 | 每個活動完成後解鎖一張貼紙 |

## 使用時間建議

| 班別 | 建議每次時間 |
|------|-------------|
| 幼幼班 / 小班 | 30 分鐘 |
| 中班 / 大班 | 45 分鐘 |

## 備份與還原

**備份（換裝置前）：**
1. 點右上角 ⚙️ → 「匯出進度」
2. 儲存下載的 JSON 檔案

**還原（新裝置上）：**
1. 點右上角 ⚙️ → 「匯入進度」
2. 選擇剛才儲存的 JSON 檔案

npx serve -l 8003 .

## 設計理念

- 溫暖自然的繪本風格，暖橘 + 米白 + 草綠
- 護眼配色，不用高飽和霓虹色
- 答錯溫柔提示「再試一次」，不懲罰、不計時
- 所有音效音量統一，不突然大聲嚇到孩子
- 觸控友善，按鈕最小 60×60px
- 中英文可切換，語音自動跟著切
