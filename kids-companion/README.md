# 小朋友學習樂園 — 使用手冊

為 2–6 歲幼兒設計的平板學習 app，寓教於樂，親子兩用。

---

## 快速開始

1. 開啟瀏覽器，前往網址（例如：`http://192.168.0.48:8003`）
2. 第一次使用：選一個小動物角色
3. 在首頁上方選擇孩子的班別
4. 點選喜歡的活動，開始玩！

---

## 首頁說明

| 元素 | 功能 |
|------|------|

| Tab | 活動 |

## 班別設定

請依孩子實際就讀班別設定，app 會自動調整難度和語音輔助：

| 班別 | 年齡 | 特色 |
|------|------|------|
| 幼幼班 | 約 2–3 歲 | 2 個選項、超大字體、語音重複朗讀所有選項 |
| 小班 | 約 3–4 歲 | 2–3 個選項、語音朗讀所有選項 |
| 中班 | 約 4–5 歲 | 3–4 個選項、可用手指拖拉、語音朗讀題目 |
| 大班 | 約 5–6 歲 | 4–6 個選項、挑戰難度、語音只念題目 |

## 活動介紹

### 📚 語言類

| 活動 | 學習重點 |
|------|----------|
| 認中文字 | 從象形字到形近字，依班別調整難度 |
| 英文字母 | A–Z 認識與發音，配合圖片記憶 |
| 數數 | 1–20 數量認知，大班有簡單加法 |
| 詞彙學習 | 看 emoji 圖片，選出正確名稱（中/英文） |

### 🎨 創作類

| 活動 | 學習重點 |
|------|----------|
| 顏色形狀 | 認識顏色和幾何形狀，交替出題 |
| 音樂節奏 | 點樂器發音（幼幼/小班），跟著節拍打鼓（中/大班） |

### 🧠 思維類

| 活動 | 學習重點 |
|------|----------|
| 拼圖 | 拖拉拼塊完成圖片（幼幼班點選），訓練空間概念 |
| 記憶翻牌 | 翻牌找相同圖案，訓練短期記憶 |
| 排序分類 | 大小排序 + 類別分類，訓練邏輯概念 |

### 📖 故事類

| 故事 | 說明 |
|------|------|
| 孫悟空大鬧天宮 | 西遊記兒童版，點物件有聲音反應 |
| 曹沖稱象 | 三國歷史故事兒童版 |

點畫面中的物件會有音效和動畫！幼幼/小班自動朗讀，中/大班點播放鍵。

## 遊戲化功能

### 集星星 ⭐
每完成一個活動得 1 顆星。集滿 5 顆星有全屏慶祝動畫！

### 貼紙書 🎴
每個活動完成後解鎖一張貼紙，點 header 的 🎴 查看收藏。

### 連續天數 🔥
每天開 app 累計連續天數，養成學習好習慣！

### 神秘寶箱 🎁
當天完成 3 個以上活動，首頁出現寶箱，開啟有今日小知識！

### 家長給讚 👏
遊戲中右下角有個 👏 按鈕，按下去孩子畫面會出現大大的鼓勵！

### 搖一搖 📳
搖晃平板會隨機跳到一個「今日驚喜活動」，孩子超喜歡！

## 使用時間建議

| 班別 | 建議每次時間 |
|------|-------------|
| 幼幼班 / 小班 | 30 分鐘 |
| 中班 / 大班 | 45 分鐘 |

app 會在剩 5 分鐘時溫和提醒，到時間顯示休息畫面。
家長可點「繼續玩」延長 10 分鐘，或點「好，休息」回首頁。

## 換裝置 / 備份進度

**備份（換裝置前）：**
1. 點右上角 ⚙️ → 「匯出進度」
2. 儲存下載的 `kids-companion-backup-日期.json` 檔案

**還原（新裝置上）：**
1. 點右上角 ⚙️ → 「匯入進度」
2. 選擇剛才儲存的 JSON 檔案
3. 確認後所有進度還原

## 技術安裝說明

### 環境需求
- 任何現代瀏覽器（Chrome、Safari、Firefox）
- Node.js（用於本機伺服器）

### 本機執行
```bash
cd kids-companion
serve -l 8003 .
```

開啟瀏覽器前往 `http://localhost:8003`

### 開機自動啟動
參考 `dementia-companion` 的 systemd 設定，建立類似的服務，埠號改為 8003。

## 設計理念

- 溫暖自然的繪本風格，暖橘 + 米白 + 草綠
- 護眼配色，不用高飽和霓虹色
- 答錯溫柔提示，不懲罰，不計時
- 所有音效音量統一，不突然大聲嚇到孩子
- 觸控友善，按鈕最小 60×60px
