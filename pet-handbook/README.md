# 🐾 寵物手冊

一本**家中寵物生活手冊**,按物種分類(貓/狗/兔/鳥/倉鼠/魚),每個物種下有多個 topics 涵蓋親訓、飼料、醫療、訓練等主題。

## 核心定位:家中寵物也是一種療法

寵物陪伴對失智長輩、小孩、照顧者自己都有療癒意義。這是 [dementia-care repo](https://github.com/tm731531/dementia-care)「**讓整個家忙起來**」策略的一部分,跟 [garden-handbook](../garden-handbook/) 平行。

家裡有生命在生長(植物)、有動物陪伴(寵物)、有人的照顧互動 —— 這組合本身就是照護。

## 這本手冊涵蓋什麼

**按物種分類**,第一層選你家養什麼:
- 🐱 貓(目前有 6 個 topics)
- 🐶 狗(待擴充)
- 🐰 兔(待擴充)
- 🐦 鳥(待擴充)
- 🐠 魚(待擴充)
- 🐢 烏龜(待擴充)

### 📚 通用知識頁(不分物種)

**核心 section**:❤ **失智照護 × 寵物選擇指南** —— 不同物種在失智照護 context 的適配度 matrix、選擇原則(輕度/中度/重度)、已有寵物時的家庭應對。

> 例如:大型犬每天要長時間運動,中度失智後長輩體力跟不上 —— 這類判斷是 handbook 給照顧者的價值核心。

### 目前在 🐱 貓底下的 topics

- 🤝 **親訓新貓** — 新貓來家的 4 階段 SOP
- 👥 **多貓介紹 SOP** — 15 步驟完整流程
- 🍲 **飼料挑選原則** — 看成分 5 步驟 + 可接受/放棄品牌清單
- 🌾 **貓草種植** — 土耕 vs 水耕實測結論
- 💉 **疫苗接種時程** — 從 8 週齡到成年
- ❄️ **冬天保暖** — 家中 + 外出注意事項

另有 📚 **通用知識頁**:罐頭碳水乾物質計算公式、飼料判讀、飲水公式、疫苗代號對照(目前主要適用於貓)。
