# 🗺️ 週末帶小孩去哪 · 北台灣親子景點推薦工具

每到週末為了「該帶小孩去哪玩」煩惱嗎?這個工具把北台灣 20+ 個常見親子景點分類好,輸入 4 個條件就推薦給你。

## 線上看
https://tm731531.github.io/dementia-care/kids-weekend/

## 怎麼用
1. **首頁 Wizard 4 題**:今天天氣?季節?孩子年齡?出門範圍?
2. **看推薦清單**:每筆景點有室內外標籤、適合年齡、好處、注意事項
3. **點 📍 Google Maps**:直接導航,不用再 google
4. **❤️ 加入口袋**:好景點存起來下次再去
5. **📝 寫我家評語**:「2025/3 去過,小孩很喜歡」(localStorage 本機保留)

## 涵蓋範圍
- **台北市**:科教館、美術館、動物園、四四南村、士林官邸 等
- **新北市**:十三行 / 八里左岸、土城公園、林口三井、烏來、深坑老街 等
- **桃園市**:大溪老街、小烏來、可口可樂博物館 等
- **宜蘭**(假日小旅行):傳藝中心、羅東林場、頭城海邊 等

每筆景點標註:**室內 / 戶外 / 室內外**、**適合年齡**、**適合季節**、**晴天 / 雨天**、**類型**(教育、玩水、動物、公園、文化、海邊、山林、購物)。

## 資料寫死,改起來簡單
20 個景點全部寫在 `index.html` 的 `PLACES` array。新增景點 = 複製一筆改內容 + 推 GitHub。

## 隱私
- **0 CDN**:沒外部字型 / 沒分析 / 沒追蹤
- **所有評語跟收藏存本機**(localStorage)
- 點 Google Maps 按鈕後 = 你主動連 google.com,該行為走你裝置
