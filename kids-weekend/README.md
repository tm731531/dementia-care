# 🗺️ 週末帶小孩去哪

全台 800+ 親子景點 + 路線規劃工具。為「週末該帶小孩去哪玩?」煩惱時,3 題答完就有推薦,還能組多點路線一鍵丟 Google Maps 導航。

**線上開** → https://tm731531.github.io/dementia-care/kids-weekend/

**不用註冊**、**沒有廣告**、**資料只在你手機**。打開就能用,離線也跑得動。

---

## 30 秒上手

1. 打開網頁 → **3 題 wizard**:你家在哪?今天天氣?出門範圍?
2. 結果頁:景點按距離排序,每筆有 emoji + 區域 + 標籤 + 簡介
3. 點 **📍 Google Maps** = 直接導航
4. 看到喜歡的 → ❤️ **加入口袋**;不再去 → 🚫 永久封存

進階:**規劃多點路線**(家 → IKEA → 五股 → 觀音山)→ 一鍵丟 Google Maps 多段導航,可傳到手機。

## 主要功能

### 🎯 推薦核心
- **3 題 wizard**:住哪 / 天氣 / 範圍 → 即時篩選
- **GPS 自動定位** + 反查地名(顯示「📍 新北土城」不是經緯度)
- **範圍滑桿 0-400 km**(雙滑桿:最近/最遠),拉的時候即時顯示「邊緣對應的區域」(15 km ≈ 台北中正、新北樹林)

### 🏷️ 標籤系統
- 全部 63 個標籤(室內/戶外/廟宇/公園/觀光工廠/...)三態點選:**必選 / 排除 / 取消**
- 標籤計數 `(當前/全資料庫)`,看出哪些標籤有結果
- 「🤧 避開人多景點」sticky checkbox(怕感冒 / 太太 friendly)

### 🛣️ 路線規劃
- **設目的地 + 沿路規劃**:選一個目的地 → 顯示「家 → 目的地」沿路 corridor 內景點
- **加 waypoints**:看到順路的「➕ 加入路線」自動排序
- **手動換序**:桌機拖拉 ⋮⋮ / 手機按 ↑↓
- **🗺️ 一鍵 Google Maps**:組好的路線丟到 Google Maps app,多段導航

### 🗺️ 區域分組
- 「依區域分組」勾起來 → 結果按 area 分群(板橋一群、中和一群...),群依距家排序
- 一目瞭然「板橋一日:林家花園 + 大遠百 + 親子館」這種混排規劃

### 🔍 搜尋
- 「找特定景點」直接打名字(九份 / 奇美 / 觀音山 ...)
- 即時候選清單,直接設目的地或加路線

### 💾 收藏 + 個人化
- ❤️ **加入口袋**(收藏清單)
- 📝 **寫我家評語**(只存 localStorage,不上雲端)
- ✅ **標去過**(看一眼就知道哪些已造訪)
- 🚫 **不再去**(永久封存,不再出現在推薦)
- ➕ **自加景點**(私房景點寫進來,也能算距離 + 一起篩)

## 隱私 — 講人話

- **不用註冊**,沒帳號密碼,打開就用
- **沒有廣告**,沒有彈跳視窗,不會看到你正在 google 什麼
- **你的東西只在你的手機**:收藏、評語、規劃的路線,通通在你裝置的瀏覽器裡,不會上傳給任何人
- 點「📍 Google Maps」 = 你主動連去 google.com,那次連線是**你的手機**直接打給 google,跟這個工具無關
- GPS 定位也是,只在當下用,不會存伺服器

(技術細節給工程師朋友看:純前端、零外部 CDN、localStorage 持久化、HTTPS 部署)
