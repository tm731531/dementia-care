❤️ 我的口袋

收藏的景點。每筆可寫「我家評語」(例如:「2025 年去過,小孩很喜歡親水池」),會存在 localStorage,本機保留。

📖 關於 / 編輯資料

📌 這個工具是什麼

週末帶小孩出門前的決策工具。每次煩惱「今天天氣這樣 / 這個季節 / 小孩這個年紀,該去哪?」時打開來看。方法 + 資料 各半 — 方法(Wizard 推薦邏輯)寫死在程式,資料(20+ 景點)放在程式開頭可以改。

📝 怎麼加新景點

  1. 用文字編輯器打開 kids-weekend/index.html
  2. 找到 const PLACES = [ 那行(用 Ctrl+F 搜「PLACES」)
  3. 複製一筆景點,改裡面的內容,記得 每筆 object 後面要有逗號,最後一筆可不要
  4. 儲存,推到 GitHub 自動部署

景點欄位說明

  • name:景點名(用來 Google Maps 搜尋,要打全名)
  • emoji:1 個 emoji
  • area:行政區(會顯示在卡片上)
  • region:'taipei' / 'newtaipei' / 'taoyuan' / 'yilan' / 'other'
  • indoor / outdoor:true / false(可同時 true,代表室內外都有)
  • ages:['0-2', '2-3', '3-6'] 任選
  • seasons:['spring', 'summer', 'autumn', 'winter'] 任選
  • weathers:['sunny', 'rainy'] — 都可選的話兩個都填
  • types:類型 tag,自由文字陣列(會出現在篩選下拉)
  • note:好處 / 簡介 1-2 句
  • warning:注意事項(可省略)

📍 Google Maps 連結

每個景點的 Google Maps 按鈕用 name + area 組成搜尋字串,直接打開 Google Maps 搜尋。手機點會直接進 Google Maps app。不用記座標,改名稱就好。

🛡 隱私

  • 0 CDN:沒外部字型 / 沒分析 / 沒追蹤
  • 所有資料本機:收藏跟評語都在 localStorage,沒上雲
  • 點 Google Maps 按鈕後 = 你主動連 google.com,該行為走你裝置(跟手冊本身的隱私分開)
