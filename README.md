# Dementia Care Tools

為失智症照護設計的工具集合。每個子資料夾是一個獨立工具，可單獨使用。

## 工具列表

| 工具 | 說明 | 狀態 |
|------|------|------|
| [dementia-companion](./dementia-companion/) | 認知訓練小遊戲，適合照護者陪同使用 | ✅ 可用 |

## 開發指引

- 每個工具放在獨立子資料夾
- 每個工具有自己的 `README.md`（使用手冊）
- 優先純前端、無需安裝（HTML + CSS + JS）

## 部署

各工具請參考各自 `README.md` 的「安裝說明」章節。
