# 陪伴小幫手 — 失智者陪伴互動工具

## 專案簡介
這是一個為失智症長輩設計的網頁互動工具，由照護者陪同使用。
目標是提供簡單、溫暖、無壓力的互動體驗。

## 技術架構
- **單一檔案應用**：所有功能集中在 `index.html`，無需伺服器或建置工具
- **純前端**：HTML + CSS + JS，無任何外部依賴
- **離線可用**：所有功能完全離線運作
- **語音回饋**：使用瀏覽器內建 Web Speech API，繁體中文語音

## 功能模組
| 模組 | 說明 |
|------|------|
| 認字遊戲 | 小學程度常用字辨識，相近字組干擾 |
| 形狀配對遊戲 | 找出與目標相同的形狀與顏色（紅色系） |
| 顏色辨識遊戲 | 從高對比色塊中找出指定顏色 |
| 認數字遊戲 | 從紅色系按鈕中找出目標數字 |
| 每日問候 | 時間問候、心情互動、每日一句 |

## 設計原則
- 純黑背景 + 純白卡片，最大對比度（適合低視力長輩）
- 大字體（基礎 20px）、大按鈕
- 橫向佈局為主，單頁不捲動（100vh + overflow hidden）
- 所有文字皆為繁體中文
- 答對給予正向鼓勵（文字 + 語音），答錯溫柔提示，不扣分不計時
- 觸控友善，最小點擊區域 48x48px
- 遊戲中已使用的題目/顏色不重複出現

## 目標使用者
- **主要使用者**：照護者（操作介面、陪伴引導）
- **體驗使用者**：失智症長輩（觀看、互動、遊玩）

## 開發指引
- 保持程式碼簡單易讀
- 不使用任何框架或套件
- 所有互動回饋需正面、溫暖
- 修改時注意無障礙設計（高對比、大字體、黑白配色）
- 遊戲顏色以紅色系為主（形狀、數字），顏色辨識用高對比不同色系
- 佈局以橫向為主，避免需要捲動
