# 陪伴小幫手 — 失智者陪伴互動工具

## 專案簡介
這是一個為失智症長輩設計的網頁互動工具，由照護者陪同使用。
目標是提供簡單、溫暖、無壓力的互動體驗，並依長輩認知程度調整難度。

## 技術架構
- **單一檔案應用**：所有功能集中在 `index.html`，無需伺服器或建置工具
- **純前端**：HTML + CSS + JS，無任何外部依賴
- **離線可用**：所有功能完全離線運作
- **語音回饋**：使用瀏覽器內建 Web Speech API，繁體中文語音（rate: 0.7, pitch: 1.1）
- **狀態管理**：全域 `APP` 物件 + `localStorage`（key: `dementiaApp`）

## 難度系統

全域物件 `APP.level`（1–8）控制所有遊戲行為：

| 範圍 | 說明 | 語音行為 |
|------|------|----------|
| 1–2 | 輕度（較難） | 只朗讀題目 |
| 3–6 | 中度 | 朗讀題目 + hover 選項唸出 |
| 7–8 | 重度（最簡） | 朗讀題目 + 所有選項，重複一次 |

遊戲庫（`GAME_LIBRARY`）有 15 個遊戲，每個有 `min`/`max` 難度範圍。
`selectGames(level)` 從符合範圍的遊戲中隨機抽 10 個顯示在首頁。

| 模組 | 難度範圍 | 說明 |

| 連連看 | 1–5 | 圖片配文字，輕度 5 對，重度 3 對 |
| 顏色辨識 | 2–7 | 找指定顏色，輕度相近色，重度高對比 4 選項 |
| 形狀配對 | 2–7 | 找相同形狀，輕度 6 種，重度 3 種 |
| 看圖認物 | 2–7 | 圖片選名稱，輕度少見物品 6 選項，重度常見 3 選項 |
| 認數字 | 2–7 | 找目標數字，輕度 1–20 六選項，重度 1–5 三選項 |
| 大小排序 | 2–7 | 排序物品，輕度 4 個小差距，重度 3 個大差距 |

| 身體部位 | 6–8 | 語音指示點選身體部位 |
| 每日問候 | — | 時間問候、心情互動、每日一句（首頁常駐） |

## 設計原則
- 純黑背景 + 純白卡片，最大對比度（適合低視力長輩）
- 大字體（基礎 20px）、大按鈕（最小 64px）
- 頁面可捲動（非固定 100vh），適合手機使用
- 所有文字皆為繁體中文
- 答對給予正向鼓勵（文字 + 語音），答錯溫柔提示，不扣分不計時
- 觸控友善，最小點擊區域 48x48px
- 遊戲中已使用的題目/顏色不重複出現

## 目標使用者
- **主要使用者**：照護者（操作介面、陪伴引導）
- **體驗使用者**：失智症長輩（觀看、互動、遊玩）

## 開發指引
- 保持程式碼簡單易讀，不使用任何框架或套件
- 所有功能維持在單一 `index.html`，嚴禁分拆成多個 JS/CSS 檔
- 修改一處功能時，注意不要影響其他遊戲（避免改 A 壞 B）
- 所有互動回饋需正面、溫暖
- 新遊戲必須接入 `APP.level` 難度系統（autoReadQuestion / bindOptionHover）
- 完成頁面用 `.complete-screen` + `.classList.add('show')` 模式
