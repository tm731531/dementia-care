## 1. 首頁選單

**目的**：讓照護者快速選擇要進行的活動。

**功能描述**：
- 顯示當前日期與時間問候語（早安/午安/晚安）
- 五個大型按鈕橫向排列連結到各功能頁
- 按鈕包含圖示 + 文字標題 + 簡短描述
- 頂部彩色邊框區分不同功能

**互動**：
- 點擊按鈕 → 切換到對應功能頁

## 2. 認字遊戲

**目的**：辨識小學程度常用字，訓練文字辨認能力。

**遊戲規則**：
1. 左側顯示一個目標字（120px 大字，紅色）
2. 右側顯示 6 個選項（3x2 grid），含 1 個正確 + 5 個同組干擾字
3. 點擊正確的字 → 答對，進入下一題
4. 點擊錯誤 → 溫柔提示，可繼續嘗試
5. 完成 8 輪後 → 顯示完成畫面

**字組設計（12 組，外形相近）**：
- 大/太/天/犬/夫/木
- 日/目/田/白/百/自
- 人/入/八/十/千/土
- 王/玉/主/生/牛/午
- 上/下/山/出/中/水
- 手/毛/父/母/女/子
- 花/草/苗/葉/林/森
- 口/回/四/因/國/園
- 門/問/間/開/關/聞
- 馬/鳥/魚/羊/兔/象
- 風/雨/雪/雲/雷/電
- 吃/喝/叫/吹/唱/哭

**不重複機制**：
- 每輪使用不同的字組（`usedGroups` 追蹤）
- 8 輪共使用 8 個不同字組

## 3. 形狀配對遊戲

**目的**：辨識顏色與形狀，簡單的認知訓練。

**遊戲規則**：
1. 左側顯示一個「目標」形狀（特定紅色系顏色 + 形狀）
2. 右側顯示 4 個選項（4 欄橫向排列）
3. 點擊與目標相同的那個 → 正確，進入下一題
4. 點擊錯誤 → 溫柔提示，可繼續嘗試
5. 完成 8 輪後 → 顯示完成畫面

**形狀種類（6 種）**：
- 圓形 ● / 方形 ■ / 三角形 ▲ / 星形 ★ / 菱形 ◆ / 心形 ♥

**顏色種類（8 種紅色系）**：
- 大紅 `#FF0000` / 暗紅 `#8B0000` / 橘紅 `#FF4500` / 玫瑰紅 `#FF007F`
- 酒紅 `#722F37` / 珊瑚紅 `#FF6F61` / 磚紅 `#CB4154` / 桃紅 `#FF1493`

**不重複機制**：
- 每輪目標顏色不重複（`usedColors` 追蹤）
- 同一輪選項中每個顏色都不同

## 4. 顏色辨識遊戲

**目的**：辨識基本顏色名稱與色塊的對應。

**遊戲規則**：
1. 左側顯示顏色名稱文字（如「紅色」，以該顏色顯示）
2. 右側顯示 6 個色塊（3x2 grid）
3. 點擊與名稱對應的色塊 → 正確，進入下一題
4. 點擊錯誤 → 溫柔提示，可繼續嘗試
5. 完成 8 輪後 → 顯示完成畫面

**顏色種類（12 種高對比）**：
- 紅 `#FF0000` / 藍 `#0000FF` / 黃 `#FFD700` / 綠 `#009900`
- 橘 `#FF6600` / 紫 `#7700CC` / 粉紅 `#FF69B4` / 咖啡 `#8B4513`
- 黑 `#000000` / 白 `#FFFFFF` / 灰 `#888888` / 天藍 `#00BFFF`

**不重複機制**：
- 每輪目標顏色不重複（`usedAsTarget` 追蹤）
- 同一輪選項中每個顏色都不同

## 5. 認數字遊戲

**目的**：辨識阿拉伯數字，訓練數字認知能力。

**遊戲規則**：
1. 左側顯示一個目標數字（120px 大字，紅色）
2. 右側顯示 6 個紅色系按鈕（3x2 grid），每個不同深淺的紅色底
3. 點擊正確數字 → 答對，進入下一題
4. 點擊錯誤 → 溫柔提示，可繼續嘗試
5. 完成 8 輪後 → 顯示完成畫面

**數字範圍**：1～20

**不重複機制**：
- 每輪目標數字不重複（`usedNumbers` 追蹤）
- 同一輪選項中數字不重複

## 6. 每日問候

**目的**：建立日常儀式感，簡單的情緒互動。

**功能描述**（三區塊橫向排列）：

### 時間問候（左）
- 根據時段顯示：☀️ 早安（5-11）/ 🌤️ 午安（12-17）/ 🌙 晚安（18-4）
- 顯示完整日期：YYYY年MM月DD日 星期X
- 時間每 30 秒更新

### 心情互動（中）
- 「今天心情如何？」
- 三個選項：😊 開心 / 😐 還好 / 😢 不太好
- 選擇後顯示對應溫暖回應

### 每日一句（右）
- 從預設清單隨機顯示一句鼓勵的話
- 範例：
  - 「每一天都是新的開始 🌅」
  - 「有你在身邊真好 💕」
  - 「慢慢來，不急 🌿」
  - 「今天也要開心喔 😊」
  - 「你很重要，大家都愛你 ❤️」

## 規劃中功能

### 看圖認物
- 顯示 emoji 圖片（水果/動物/日用品）
- 從文字選項中選出正確名稱
- 訓練物品辨識與詞彙連結

### 物品分類
- 畫面顯示多個物品 emoji
- 拖曳或點擊分類到正確類別（食物/動物/交通工具等）
- 訓練分類與歸納能力

### 時鐘認讀
- 用 Canvas 畫指針鐘面
- 從選項中選出正確時間
- 訓練時間感知

### 找不同
- 兩組相似的 emoji 排列
- 找出哪個不一樣
- 訓練注意力與觀察力

### 簡單塗鴉
- 觸控畫布，自由畫畫
- 選顏色、調粗細、清除重畫
- 放鬆娛樂用途

### 呼吸練習
- 動畫圓圈引導吸氣（放大）/ 吐氣（縮小）
- 配合語音提示
- 幫助情緒穩定
