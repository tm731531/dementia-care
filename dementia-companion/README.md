# 陪伴小幫手 — 使用手冊

為失智症長輩設計的認知訓練工具，由照護者陪同使用。

---

## 快速開始

1. 開啟瀏覽器，前往工具網址（例如：`http://192.168.0.93:8002`）
2. 依長輩的認知程度，調整頁面頂端的**難度滑桿**
3. 從首頁選擇一個遊戲，開始陪伴互動

---

## 首頁說明

### 難度滑桿

頁面最上方有一條滑桿，範圍 **1（輕度）到 8（重度）**：

| 範圍 | 說明 | 適合對象 |
|------|------|----------|
| 1–2 | 輕度 | 輕微認知退化，仍可獨立完成較有挑戰的任務 |
| 3–6 | 中度 | 中等認知退化，需照護者從旁引導 |
| 7–8 | 重度 | 較嚴重退化，需要大量語音提示與簡化選項 |

調整滑桿後，首頁會自動從遊戲庫中選出 **10 個適合的遊戲**顯示。設定會記住，下次開啟自動套用。

### 語音輔助

系統依難度自動調整語音說明：

- **重度（7–8）**：進入每道題目，自動朗讀題目與所有選項，並重複一次
- **中度（3–6）**：自動朗讀題目；手指滑過選項時即時念出該選項
- **輕度（1–2）**：只朗讀題目

---

## 遊戲介紹

### 認知訓練類

| 遊戲 | 適合難度 | 說明 |
|------|----------|------|
| 翻牌配對 | 1–3（輕） | 翻開兩張牌，找出相同的圖案，訓練短期記憶 |
| 認字遊戲 | 1–4 | 辨識中文字，輕度出形近干擾字，重度出象形常用字 |
| 找不同 | 1–5 | 兩排圖案中找出不一樣的那一個 |
| 時鐘認讀 | 1–5 | 看時鐘選出正確時間，訓練時間定向感 |
| 連連看 | 1–5 | 圖片配文字，訓練配對與語意記憶 |
| 顏色辨識 | 2–7 | 找出指定顏色的色塊，重度用高對比色 |
| 形狀配對 | 2–7 | 找出與目標相同的形狀，重度只用 3 種形狀 |
| 看圖認物 | 2–7 | 看圖片選出正確名稱，重度用常見物品 |
| 認數字 | 2–7 | 從選項中找出指定數字 |
| 大小排序 | 2–7 | 把物品從小排到大，訓練比較概念 |
| 物品分類 | 2–7 | 把物品放進正確類別（食物、動物…） |
| 數數練習 | 3–8 | 數一數畫面上有幾個圖案，重度數 1–5 個 |
| 情緒辨識 | 3–8 | 看臉部表情選出情緒名稱，重度只有2選項 |

### 舒緩工具

| 遊戲 | 適合難度 | 說明 |
|------|----------|------|
| 呼吸練習 | 5–8（重） | 動畫圓圈引導慢慢吸氣吐氣，適合焦躁時使用 |
| 身體部位 | 6–8（重） | 語音指示點選身體部位，訓練身體意識 |

---

## 使用建議

- 每次使用 **15–30 分鐘**為宜，避免過度疲勞
- 答錯時不扣分、不計時，以**鼓勵**為主
- 照護者可陪同朗讀選項，加強互動
- 若長輩對某遊戲特別有反應，可多停留

## 技術安裝說明

### 環境需求

- 任何現代瀏覽器（Chrome、Firefox、Safari）
- Node.js（用於本機伺服器）

### 安裝伺服器（一次性設定）

```bash
npm install -g serve
```

### 設定開機自動啟動（Linux systemd）

建立服務檔 `~/.config/systemd/user/dementia-companion.service`：

```ini
[Unit]
Description=Dementia Companion Web App
After=network.target

[Service]
Type=simple
WorkingDirectory=/home/<使用者>/Desktop/dementia-care/dementia-companion
ExecStart=/path/to/serve -l 8002 .
Restart=always
RestartSec=5

[Install]
WantedBy=default.target
```

啟用服務：

```bash
systemctl --user daemon-reload
systemctl --user enable dementia-companion
systemctl --user start dementia-companion
loginctl enable-linger <使用者名稱>
```

### 本機手動執行

```bash
cd dementia-companion
serve -l 8002 .
```

開啟瀏覽器前往 `http://localhost:8002`

---

## 離線使用

本工具完全離線可用，無需網際網路連線。所有語音使用瀏覽器內建 Web Speech API（繁體中文）。
