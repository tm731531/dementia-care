# 陪伴小幫手 — 失智症照護互動工具

> 一位很 I 的軟體工程師，因為陪伴失智家人時想話題太消耗腦力，所以做了這個工具。
> 打開就能用，不用想話題，遊戲自動引導互動。希望幫助所有同樣辛苦的照護者。

純前端、零廣告、完全離線可用。

> **💡 也試試 [v2 試驗版](../dementia-companion-v2/)**：同樣 15 個遊戲，但首頁改成「今天一起做這個好嗎？」的推薦式設計，附 45 條給照護者的陪伴指南跟今日摘要分享。v1、v2 並存沒替換，可以對照比較。

---

## 快速開始

1. 用瀏覽器開啟 `index.html`（或前往部署網址）
2. 依長輩的認知程度，調整頁面頂端的**難度滑桿**（1 輕度 → 8 重度）
3. 從首頁選擇一個遊戲，開始陪伴互動

不需要帳號、不需要安裝、不需要網路。

---

## 難度系統

頁面最上方有一條滑桿，範圍 **1（輕度）到 8（重度）**：

| 範圍 | 說明 | 適合對象 |
|------|------|----------|
| 1–2 | 輕度 | 輕微認知退化，仍可獨立完成較有挑戰的任務 |
| 3–6 | 中度 | 中等認知退化，需照護者從旁引導 |
| 7–8 | 重度 | 較嚴重退化，需要大量語音提示與簡化選項 |

調整後，首頁自動從 15 個遊戲中選出 **10 個適合的**顯示。設定會記住，下次自動套用。

### 語音輔助（自動依難度調整）

- **重度（7–8）**：自動朗讀題目與所有選項，重複一次
- **中度（3–6）**：朗讀題目，手指滑過選項即時念出
- **輕度（1–2）**：只朗讀題目

---

## 15 種遊戲

### 認知訓練（13 種）

| 遊戲 | 難度範圍 | 訓練重點 |
|------|----------|----------|
| 翻牌配對 | 1–3 | 短期記憶 — 翻開兩張牌找相同圖案 |
| 認字遊戲 | 1–4 | 文字辨識 — 輕度形近字干擾，重度象形常用字 |
| 找不同 | 1–5 | 視覺注意力 — 兩排圖案中找出不一樣的 |
| 時鐘認讀 | 1–5 | 時間定向感 — 輕度任意分鐘，重度整點 |
| 連連看 | 1–5 | 語意記憶 — 圖片配文字，輕度 5 對，重度 3 對 |
| 顏色辨識 | 2–7 | 色彩認知 — 輕度相近色，重度高對比 |
| 形狀配對 | 2–7 | 圖形辨識 — 輕度 6 種形狀，重度 3 種 |
| 看圖認物 | 2–7 | 物品辨識 — 輕度少見物品 6 選項，重度常見 3 選項 |
| 認數字 | 2–7 | 數字概念 — 輕度 1–20，重度 1–5 |
| 大小排序 | 2–7 | 比較邏輯 — 把物品從小排到大 |
| 物品分類 | 2–7 | 分類概念 — 輕度 4 類，重度 2 類 |
| 數數練習 | 3–8 | 計數能力 — 數畫面上有幾個圖案 |
| 情緒辨識 | 3–8 | 社交認知 — 看表情選情緒名稱 |

### 舒緩工具（2 種）

| 遊戲 | 難度範圍 | 說明 |
|------|----------|------|
| 呼吸練習 | 5–8 | 動畫圓圈引導慢慢吸氣吐氣，適合焦躁時使用 |
| 身體部位 | 6–8 | 語音指示點選身體部位，訓練身體意識 |

### 每日問候（常駐首頁）

時間問候、心情互動、每日一句，每天開啟都有不同的溫暖問候。

## 使用建議

- 每次使用 **15–30 分鐘**為宜，避免過度疲勞
- 答錯不扣分、不計時，以**鼓勵**為主
- 照護者可陪同朗讀選項，加強互動
- 若長輩對某遊戲特別有反應，可多停留
- 純黑背景 + 大字體設計，適合低視力長輩

## 備份與還原

進度存在瀏覽器中，換裝置時：

**備份：** 開發者工具 → Console → `localStorage.getItem('dementiaApp')` 複製

**還原：** 新裝置 Console → `localStorage.setItem('dementiaApp', '...')` 貼上

## 技術安裝

### 最簡單的方式
直接用瀏覽器開 `index.html`，雙擊就能用。

### 本機伺服器
```bash
npx serve -l 8002 .
```
開啟 `http://localhost:8002`

### 開機自動啟動（Linux systemd）

```bash
# 建立服務
mkdir -p ~/.config/systemd/user
cat > ~/.config/systemd/user/dementia-companion.service << 'EOF'
[Unit]
Description=Dementia Companion Web App
After=network.target

[Service]
Type=simple
WorkingDirectory=%h/Desktop/dementia-care/dementia-companion
ExecStart=npx serve -l 8002 .
Restart=always
RestartSec=5

[Install]
WantedBy=default.target
EOF

# 啟用
systemctl --user daemon-reload
systemctl --user enable --now dementia-companion
loginctl enable-linger $USER
```

## 設計理念

- **高對比度**：純黑背景 + 純白卡片，適合低視力長輩
- **大字大按鈕**：基礎字體 20px，按鈕最小 64px
- **溫暖正向**：答錯說「再試一次」，不說「錯了」
- **零壓力**：不計分、不計時、不排名
- **完全離線**：不傳任何資料到外部，隱私安全
