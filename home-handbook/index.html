給剛確診失智家人的家屬。這不是教科書、不是長照 3.0 的 CMS 評估,是一位工程師在照顧媽媽的過程中踩過坑、改過家、換過擺設累積的筆記。希望你不用重蹈一樣的 6 個月。

🚨 先做這 3 件(P0)

如果這週只能改 3 件事,照這個順序:
1🚪 廁所門拆掉改布簾

問題:她找不到廁所,會在陽台或廚房上廁所。

為什麼:失智中期視空間能力退化,關閉的門 = 「這裡不是目的地」的認知訊號。布簾透光 + 半掩,讓廁所永遠「看得到」。

怎麼做:把原木門拿掉,用伸縮桿 + 棉布簾(淺色、半透明),或直接拆門框。成本 NT$300-500。

2🎨 馬桶圈換成對比色

問題:白色馬桶 + 白色馬桶圈 + 白色牆,她看不到座位位置。

為什麼:失智後期 contrast sensitivity(對比敏感度)下降,同色背景會「消失」。

怎麼做:買有顏色的馬桶圈(藍、紅、深黃最好),或貼對比色膠帶。成本 NT$150-500。

3🪟 鐵窗 + 紗網 + 隔音板材(三層組合)

問題:她會從窗戶丟東西下樓(衣服、杯子、食物),還會敲窗、拍牆造成鄰居投訴。

為什麼:失智中期判斷力退化 + 重複行為 + 想「整理」空間。

怎麼做 — 三層組合(這是不得已的選擇,原本想換氣密窗,但踩坑):

  • ① 鐵窗 — 扛破壞力。她會嚴重敲打,一般窗戶撐不住。
  • ② 紗網(外層)+ 大量塑膠束帶固定 — 擋丟東西的密度,束帶比綁繩牢固、不易剪斷。
  • ③ 隔音板材(內層)— 擋「音波攻擊」(她敲窗時的噪音會讓全家跟鄰居都崩潰)。可用吸音棉、蛋盒型泡綿、或隔音牆板。

成本:鐵窗已有 → 紗網束帶 NT$300-800,隔音板材 NT$500-2000。

為什麼不換氣密窗(踩過的坑):原本想直接換氣密窗解決所有問題,但發現:
  1. 施工期間她會嚴重妨礙工班(抓師傅、搶工具、情緒大起)
  2. 裝好後她的破壞力強大 — 一般氣密窗撐不住持續拍打
  3. 要扛住「人的攻擊力」的氣密窗,材質費用超過 NT$5 萬/窗,且市面不好找
三層組合的 CP 值遠勝氣密窗。
為什麼這 3 件優先:都是「會造成家人立即困擾或鄰居投訴」的安全問題。廁所失禁會壞地板、丟東西會砸到人、馬桶看不到會在地上。其他 8 項可以慢慢做,但這 3 項本週就要完成。

4⏰ 定時器播音樂 + 燈光

問題:她會在奇怪時段「叫醒」自己做事(凌晨 3 點打掃)。

怎麼做:用傳統機械式定時器或智慧插座,設定早上 7:00 自動開電視新聞台、傍晚 17:30 自動開柔和音樂、晚上 21:00 自動關大燈只留夜燈。外部光線訊號 + 聲音訊號幫她錨定一天的節奏。

為什麼:失智者 circadian rhythm(日夜節律)容易紊亂,規律的環境刺激比口頭提醒有效。

5🔄 家具擺設改迴圈行走路線

問題:她在房間裡來回踱步,撞到桌角。

怎麼做:把沙發、餐桌、茶几擺成讓她可以「繞圈走」的路線(客廳 → 餐廳 → 走廊 → 臥室 → 回客廳)。不要把路堵死,也不要有尖角死路。

為什麼:重複行走是失智中期常見行為,用來消耗體力。如果空間太擁擠會撞傷,或她會挫折。

6🌱 三樣「可玩的東西」輪替

問題:她在家沒事做,盯電視又會打瞌睡到晚上睡不著。

怎麼做:準備 3 樣可以「每天做一點」的東西 — 豆芽、蒜頭、豆子(裝在淺盤)。每天給她看一個(「今天看看豆芽發多少了」),讓她有「參與」照料活物的感覺。

為什麼:活物 + 可觀察變化 + 可參與澆水 = 每天有「錨點」。參考 家庭園藝手冊 的「新手 3 株推薦」。

7🎬 小遊戲 + YT 播放

問題:她坐著沒事、又起不了身。

怎麼做:桌上永遠擺一個「小遊戲」(拼圖、串珠、疊杯),配 YouTube 播她年輕時喜歡的節目(老歌、台語劇、懷舊廣告)。看著電視時手上不空。

參考 陪伴小幫手 v2 的 15 個認知遊戲(純離線可用)。

8🔐 廚房門鎖改方便開式

問題:她半夜進廚房煮東西/打翻瓦斯。

怎麼做:廚房門鎖換成跟廁所一樣的「外部方便開、內部可反鎖」式。或直接裝一般推拉門,不要卡死她但限制頻繁進入。

不要用完全鎖死的方式 — 這會讓她焦慮打門,且緊急狀況進不去。

9🔇 大門用隔音棉遮蔽

問題:她會衝去應門、探頭、跟送貨員對話混亂,或家人開關門時她以為要走。

怎麼做:大門內側貼隔音棉(蛋盒型泡綿就行)+ 裝簡單門擋防她自己打開。門鈴關靜音,改手機 app 通知。

失智中期 frontal 腦區退化,對聲音刺激會過度反應。降噪 = 降焦躁。

10🚴 一間房改運動間

問題:她體力有但沒地方發洩,在家踱步變干擾。

怎麼做:騰出一間房間(客房、儲藏室),放室內腳踏車(2-10 分鐘為宜)、瑜珈墊、氣球、豆袋。讓她每天有個「運動儀式」的地方。

失智合併肌少症很常見,刻意製造運動機會延緩退化,也消耗體力讓她晚上好睡。

11💨 定時器設電風扇

問題:她在特定房間待太久,空氣不流通會想睡。

怎麼做:在她常待的空間(運動間、客廳、臥室)放電風扇配定時器,設定每小時吹 10 分鐘。

失智者對溫度變化敏感度下降,自己不會覺得悶熱,但體溫上升會加速疲勞/譫妄。空氣流通是被低估的環境因子。

☀️ 早上 SOP(9 點前完成)

我接手早餐班後的標準流程:

  1. 換燈 — 全家開燈(小夜燈關,只留老媽房間小桌、客廳小桌)。強烈光線訊號告訴身體「白天開始了」。
  2. 開電視新聞台 — 聲音背景,讓她知道不是獨處。
  3. 早餐:固定配方 → 不用每天想菜單。實際組成會隨季節微調:
    • 蛋 × 2(固定,取代原本的豆漿 — 飽足感 + 蛋白質更穩定)
    • 燕麥片:冬天多一匙、夏天少一匙(看天氣加減)
    • 香蕉 + 核桃(補鉀 + Omega-3)
    • 飲水溫度隨天氣:冬天熱水、夏天溫水;失智者對溫度感覺遲鈍,絕對不要涼水(腸胃差)
  4. 吃藥 — 早上的那一格。
  5. 放一個蛋在調理器給中午班(從煮蛋器中取一個)。中午別人來照顧時已經有現成蛋可以用。
  6. 做活動 — 丟垃圾 / 擦桌子 / 洗碗 / 澆花 / 刷牙 / 腳踏車運動(2–10 min)。有活動身體就不會 11 點就想睡。
為什麼這麼機械化:失智者的 executive function(計畫執行功能)退化,每天早上重新「決定要做什麼」對照護者跟她都很累。同樣的 6 步、同樣的順序、同樣的食物 — 穩定性本身就是治療。

🌙 睡前 SOP

  1. 換燈 — 全家關燈,進入小夜燈模式:廚房 + 廁所前燈 + 老媽房小夜燈 + 客廳小夜燈。外部光線訊號切換「晚上」。
  2. 關電視。
  3. 吃藥 — 晚上的格 + 睡前的格。
  4. 做活動 — 掃地 / 刷牙 / 泡腳(精神好才做,水溫 35–38°C,5–10 分鐘,泡的時候跟她對話讓她累,不好就跳過)。
  5. 入房睡覺(幫蓋被子,調整睡眠方向)。
  6. 沙發放滿重物 — 逼她半夜醒不能在客廳睡回沙發,只能回房間。重物可用:整堆衣物、抱枕、硬紙箱、購物袋。早上起來再清理。這不是「故意為難」,是她自己半夜起來迷路找地方躺時,沒有舒服的備案 → 自然回房。
  7. 收尾:
    • 放蛋入煮蛋器,設定 P3 定時 9.5 小時(明天早上有得吃)
    • 便當盒放入便當袋(明天外出用)
    • 洗杯子、補水、補鐵水壺、熱水瓶(杯子放在鐵櫃上 — 固定位置她才找得到)
睡前泡腳的原則:目的不是「泡乾淨」,是讓她流點汗、累一點、慢下來。水溫不要超過 38°C(失智者體溫調節差)。邊泡邊聊她年輕時的事(reminiscence therapy),她會慢慢放鬆。

🎯 分散注意力 3 招(最重要)

這一整節比上面 11 項改造還重要。不是「怎麼陪伴長輩」的理論,是怎麼讓照護者不用整天站著陪的實戰。

「模擬日照並不是什麼,而是我的想法是要分散她的注意力,讓她的身體有點事情。」(我的想法,不是教科書)

招式 1 · 偷偷製造「要收的工作」

做法:趁她沒看到的時候,故意把她的衣服 5 件上下打亂丟沙發邊、把紅色籃子裡的玩具丟幾件打散在桌上。

結果:她自然會走去收 → 整理 → 疊 → 放回 → 再發現別處也亂了 → 繼續做。

鐵律:不能被她看到你擺。如果她看到你故意弄亂,反而不會去做。失智者保有 self-awareness,看到你故意弄亂會感覺被當小孩、拒絕配合。她必須覺得是自己發現的。

招式 2 · 利用她的身體記憶(procedural memory)

做法:觀察自己媽媽 — 「她是三個小孩的媽媽,每天洗一堆衣服 + 疊。」

驗證:現在給她多少衣服,她都可以五分鐘內疊好(但疊完又沒事做)。

所以:持續製造「該疊的衣服」,她就持續在做她最熟練、最自信的動作。

怎麼找長輩的身體記憶:問自己「她年輕時每天重複做什麼?」— 洗碗、摺衣服、擦桌子、撿菜、捏麵團、打毛線、縫扣子。這些動作比看電視、做拼圖更有意義,因為跟她的自我認同連得上。

招式 3 · 日照中心的祕訣:問背景找蛛絲馬跡

我實地去觀察日照中心後發現:照服員會「一直問每個個案的背景,從中找蛛絲馬跡讓這個長輩多做一點」。

實作:問家人、翻舊相簿、問媽媽本人她記得的。找出她年輕時的身分標記:

  • 她是什麼職業?(老師 → 擺粉筆、課本讓她「備課」)
  • 她擅長什麼?(裁縫 → 放布料、鈕扣讓她分類)
  • 她的家鄉在哪?(播那裡的民謠)
  • 她養過什麼?(看動物圖片會有反應)

把這些轉換成每天家裡擺放的物件,不是硬塞給她「玩」。

🚶 外出時段 · 黃昏症候群

2026-01 學到的教訓:原本晚餐後帶她外出散步,結果「很難帶回來」。她想繼續走、情緒上來、拒絕回家。改中午帶外出,晚上的長照只負責給吃,就解決了。

為什麼叫「黃昏症候群」

Sundowning 是失智症的典型症狀:下午 3 點過後到傍晚,長輩情緒會變得焦躁、疑神疑鬼、想離家、拒絕配合。成因包括:

  • Circadian rhythm 紊亂 — 失智影響生理時鐘
  • 光線減弱 — 視覺輸入減少加劇 disorientation
  • 下午體力耗盡 — 累了反而控制不住情緒
  • 過去生活慣性 — 很多長輩年輕時下午要接小孩、買菜,身體記得但說不清為什麼焦慮

⚠️ 踩過的坑

🚫 坑 1:想「一次改好」結果停工

剛確診的家屬很容易列出 30 項要改的東西,然後被預算+時間壓倒,結果一項都沒做。先做 P0 三件,其他分週做。

🚫 坑 2:期望她「還能自己處理」

例如「廁所門拆了她會不會尷尬?」— 失智中期的她其實已經不在意這種事,家屬才在意。改造以她的實際功能為準,不是她剛確診時的狀態。

🚫 坑 3:被她看到你故意擺東西

分散注意力 3 招的鐵律。如果她看到你故意弄亂,反而不會去收。這需要照護者偷偷做,像魔術師一樣。

🚫 坑 4:太晚發現黃昏症候群

前 3 個月我每天晚餐後帶她散步,她越走越情緒大,一度以為是我帶的方式不對。其實是她根本不該下午出門。試了一週改中午,就再也不躁。

🚫 坑 5:把「讓她忙起來」當成「認知訓練」

做家事 ≠ 給她考題。折衣服、擦桌子、撿菜這些家事的價值在於身體動起來 + 連接她的身分認同,不是在測試她記憶力。每天不要問「這個你記得嗎?」— 她不記得會沮喪。
