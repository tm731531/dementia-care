🏫 4 種托育類型

費用、報名方式、抽籤規則完全不同。選錯類型,所有後續動作都白搞。

📊 一張表看完

類型 適用年齡 費用(每月) 報名方式 抽籤規則 等候時間
公托 / 公幼 0-6 最低(月費 2000-4000,含補助後) 各縣市網站 區域順位 + 抽籤 1-3 年(0-2 歲尤其競爭)
非營利幼兒園 2-6 第 1 胎 3000(統一) 新北 kid123 / 各縣市統一網站 統一電腦抽籤 中等
準公共 0-6 第 1 胎 3000 上限 各園自辦(現場/Email/網站) 各園自辦公開抽籤 中等
私立(自費) 0-6 8000-50000+(無上限) 各園自辦 名額制(可加錢插隊) 即時(有錢就有位)

🏛 公托 / 公幼

政府辦的公立托嬰中心 / 公立幼兒園。費用最低,但僧多粥少,要靠抽籤。

特點

  • 👍 費用最低(月費 2000-4000,含政府補助後)
  • 👍 師資穩定(教保員多為公務體系,流動率低)
  • 👍 標準化(教學內容、餐食、安全規範統一管理)
  • 👎 名額極少,公托(0-2 歲)等候 1-3 年很正常
  • 👎 區域順位:沒設籍在該區可能根本沒抽籤資格
  • 👎 教學偏「保守」:沒有蒙氏/華德福之類的特色

怎麼報

  • 各縣市教育局 / 社會局網站(各地不同 — 看 📚 縣市資源)
  • 0-2 歲公托:**孩子出生就要去登記**,等候名單通常 1-3 年
  • 2-6 歲公幼:每年 4-5 月一波,跟非營利同期

🏘 非營利幼兒園(只有 2-6 歲)

政府委託民間單位辦的幼兒園。費用跟準公共一樣低,但**報名規則跟公幼一樣統一**(新北統一用 kid123.ntpc.edu.tw)。

特點

  • 👍 費用低(第 1 胎月費 3000 上限,跟準公共相同)
  • 👍 報名統一(各縣市一個網站搞定多家)
  • 👍 備取保留期長(新北非營利保留至下年留園直升前一日,**最長**)
  • 👍 線上報到(不用臨櫃)
  • 👎 名額少,抽籤競爭激烈
  • 👎 不收 0-2 歲(如果有 0-2 歲幼兒只能找公托或私托)

怎麼報

  1. 找該縣市的統一報名網站(新北:kid123.ntpc.edu.tw)
  2. 提前註冊帳號(不要等到報名日才註冊)
  3. 缺額公告當天確認可報名園所
  4. 網路報名 → 資格審查 → 補件 → 抽籤 → 線上報到

🏢 準公共

私立但跟政府簽合約,讓家長享有「跟非營利一樣的低費用」。涵蓋 0-6 歲(包含 0-2 歲托嬰)。

特點

  • 👍 費用低(第 1 胎月費 3000 上限)
  • 👍 名額較多(多家私園加入,選擇多)
  • 👍 含 0-2 歲托嬰(準公共托嬰中心)
  • 👍 教學風格較多元(各園自由發展特色)
  • 👎 各園自辦報名(時程、地點、文件、規則全不一樣 — 累)
  • 👎 備取保留期差異大(短的 5 天,長的 4 個月,要看清楚)
  • 👎 5 年合約到期可能退場(變回完全私立,費用爆漲)

怎麼報

  1. 找各縣市「準公共幼兒園/托嬰中心」清單(教育部 / 衛福部官網有)
  2. 篩選步行範圍內的園所
  3. 逐家查報名期間 + 報名方式(現場/Email/網站,各家不同)
  4. **注意:準公共報名期間常與非營利重疊**(都是 4-6 月),要排好優先序

💸 私立(自費)

完全自由訂價。費用從每月 8000(基本)到 50000+(雙語/特色園)都有。沒有政府補助。

特點

  • 👍 有錢就有位(不用抽籤,不用等)
  • 👍 教學風格多元(蒙氏、華德福、雙語、森林、體育)
  • 👍 可以中途轉入(不限年度開學)
  • 👎 費用高(年費可能是準公共的 5-15 倍)
  • 👎 隱形費用多(註冊費、書本費、制服費、活動費、延托費 — 累計可能再加 30%)
  • 👎 退費規則嚴(中途離園可能要付違約金)
  • 👎 師資流動率參差(有些園所離職率高,孩子適應期長)

什麼情況考慮私立

  • 沒抽中任何公幼/非營利/準公共,等不到下一年
  • 家庭預算寬裕,要特定教學風格
  • 想要某些特殊服務(全美語、雙師制、小班)
  • 想要中途插班(其他類型只在 8 月開學)

📐 怎麼決定優先順序

狀況建議優先序
預算緊 + 雙薪 + 0-2 歲幼兒公托 → 準公共托嬰 → 私立托嬰
預算緊 + 雙薪 + 2-6 歲幼兒非營利 + 公幼(同期報) → 準公共 → 私立(備援)
預算寬 + 想要特定教學直接看私立(蒙氏/華德福/雙語) → 準公共補位
住偏遠地區查最近的「準公共」+ 「私立」(公幼/非營利通常市區才多)
想全家搬家配合園所公托/公幼是首選(便宜 + 穩定),其他類型不用為它搬

⚠️ 不要只報一家

即使最想要 A 園,也要報 B、C、D 當保險。抽籤是運氣,沒抽中要等下一年(8 月開學前可能補不到位)。同時報 3-5 家是常態。

🎯 5W1H 評估方法論

每個家庭適合的園所不同。這 6 個維度逐一回答,不是想哪個算哪個。

👤 Who · 你家是誰

「孩子幾歲」不是 Who 的全部。Who 是整個家庭結構:

  • 家庭結構:雙薪 / 單薪 / 單親 / 多代同堂(有失智長輩參與接送嗎?)
  • 預算:每月教育預算上限是多少?(影響類型選擇)
  • 接送角色:誰接、誰送、有沒有備援?
  • 孩子個性:好動 / 內向 / 過敏體質 / 有特殊需求
  • 家裡其他成員:兄弟姊妹是否同園(可能有「在園生弟妹」優先順位)

📌 隱形維度:祖父母

如果有失智長輩在家,需要評估:長輩是否能參與接送(認知功能)、長輩日照中心位置是否在接送動線上、長輩走失風險是否影響你的行程彈性。看 👨‍👩‍👧 三明治世代。

🏫 What · 選哪一類

看 🏫 4 種類型。決策邏輯:

  1. 先看預算(私立 vs 補助類)
  2. 再看年齡(0-2 vs 2-6)
  3. 再看時間(等得起公托嗎?)
  4. 最後看教學風格偏好

📅 When · 什麼時候動

類型關鍵時機
公托(0-2 歲)孩子出生就排隊。等候 1-3 年正常
非營利 + 公幼(2-6)每年 4 月底缺額公告 → 5 月初報名 → 5 月中抽籤
準公共各園不同,集中在 5-6 月
私立全年隨時(看名額),但 7-8 月開學潮位置最少

過了報名截止就是 GG。各園截止日不同,排不開的話只能放棄某些園。看 📅 時程地圖。

📍 Where · 距離與動線

步行 20 分鐘原則

  • 地圖顯示 15 分鐘 → 實際 20 分鐘起跳(小孩腳步慢)
  • 下雨 / 颱風 / 寒流 → 30+ 分鐘或無法步行
  • 不要評估「我成人快走的時間」

接送動線評估

  • 從家到園所(早上)
  • 從園所到工作地點(早上接續)
  • 從工作地點回園所(下午接)
  • 從園所回家(下午接續)
  • 有失智長輩家庭:加上「日照中心位置」「醫院位置」

📌 試走一遍

真的去走一次。算上紅綠燈、推車、抱孩子。把這個時間 +10 分鐘當作「能接受的上限」。超過就不要勉強(會崩潰)。

💡 Why · 教學風格

不要只看招生簡章上的標榜。問:

  • 典型一天的時間表是什麼?(自由探索 vs 結構式上課的比例)
  • 戶外活動多久一次?(每天 / 每週?)
  • 有沒有特殊課程?(雙語?才藝?)
  • 怎麼處理孩子情緒?(這題回答得好不好,反映教保理念)
  • 家長如何參與?(只看 Line 群還是真的有家長日)
常見教學風格簡介
  • 蒙特梭利(Montessori):孩子自選工作、混齡、教具導向。強調獨立。
  • 華德福(Waldorf):節奏、季節、自然材質、不接觸電子產品。慢學派。
  • 瑞吉歐(Reggio Emilia):專案式學習、藝術中心、強調對話。
  • 主題式(主流):每月或每季一個主題,各種活動圍繞主題。
  • 遊戲式 / 開放教育:大量自由遊戲時間。
  • 雙語 / 全美語:重視語言環境。注意是「外師全程」還是「中師翻譯式」。

教學風格沒有絕對好壞 — 看孩子個性。內向慢熟的孩子可能不適合大團體競爭式環境。

🔧 How · 報名流程

  1. 準備期(報名前 2-4 週):備齊文件、排清單、跟家人達成共識
  2. 報名期(各園不同):準時遞表 / 上傳
  3. 等候期(報名後 1-2 週):資格審查、補件
  4. 抽籤日(各園不同):當天公告結果
  5. 報到期(抽中後 1-3 天):**準時報到!過了就放棄正取**
  6. 備取等候期(看各園公告,5/14 ~ 10/31 都有):備取保留期長的有機會

👶 0-2 歲(托嬰)的時程不同

公托不是「每年 4-6 月一波」,而是滾動式登記。出生就排,等候名單通常 1-3 年。

什麼時候做什麼

  • 孕期 / 出生:確認戶籍區,查該區有哪些公托 + 準公共托嬰
  • 出生 0-3 月:辦戶籍 → 立刻去公托登記排隊
  • 出生 6 個月後:確認自己預計幾月送托(影響可登記月份)
  • 送托前 2 個月:看排到第幾位 → 決定送公托還是準公共/私立
  • 送托前 1 個月:報名選定園所 + 安排試讀(很重要,試讀看孩子適應度)

🎒 4-6 歲(大班/中班/續讀)

  • 大部分孩子 2 歲入園後就留園直升到大班畢業
  • **留園直升登記**通常 1-2 月,不要錯過(錯過要重抽籤)
  • 想換園的話,看新生缺額(4-5 月公告)
  • 準備上小學:大班下學期(隔年 4-5 月)看小學學區

📌 提前該做的事(現在開始)

⚠️ 預防接種紀錄影本

這是所有園所都需要的文件,但很多家長到報到才發現黃卡找不到 → 衛生所補發要 1-2 週。現在就先確認在哪、影印 5 份。看 📄 文件 SOP。

📄 文件 SOP

各園需要的文件大同小異。提前備齊,錄取後不用臨時找。

📋 報名階段(一般生)

大部分準公共/私立的一般生報名只需現場填表,不需提前帶文件。但到場時帶以下「正本」:

  • ✅ 孩子健保卡 正本(確認出生年月日)
  • ✅ 家長身分證 正本(確認家長身份)
  • ✅ 原子筆

💻 非營利網路報名

  • ✅ 提前在統一網站註冊帳號(別等到報名日才註冊)
  • ✅ 戶口名簿掃描檔(系統查調為主,失敗時上傳)
  • ✅ 確認手機號碼可收簡訊(補件、抽籤結果都靠簡訊)
  • ✅ 跨縣市新搬遷者:**必須上傳戶口名簿**(設籍未滿期者)

📦 錄取後報到(備齊全套)

各園要求略有差異,建議提前準備全套 3-5 份,錄取後電話確認該園實際需求。

⚠️ 黃卡找不到怎辦?

  1. 先翻媽媽手冊和產檢資料夾(常被夾在裡面)
  2. 找不到去戶籍地衛生所補發,要帶孩子健保卡
  3. 補發要 1-2 週,不要等到報到前才辦
  4. 有些縣市可線上查接種紀錄(健保署「健康存摺」)

🎲 抽籤機制

大部分公幼/非營利/準公共都靠抽籤。備取保留期短的等於沒中,這個細節千萬要看。

🔄 標準流程

  1. 報名 — 各園規定的報名期間提交資料
  2. 資格審查 — 園方/教育局審核資料、補件通知
  3. 抽籤 — 報名人數 > 名額 才會抽。報名人數 ≤ 名額 直接全錄取
  4. 正取 + 備取公告 — 抽到正取的依序遞補,備取按抽籤順序排隊
  5. 正取生報到 — 截止日內到場/線上報到。**過了就放棄正取資格**
  6. 備取生遞補 — 正取沒報到 → 備取依序通知 → 直到滿額或備取保留期截止

⚠️ 順位 vs 抽籤的差別

同一園所內可能有多個順位(例:第 2 順位是教職員子女,第 3 順位是社區住戶,第 4 順位才是一般生)。順位是先後順序,前順位優先填滿,輪到一般生才開始抽籤。

典型順位範例(各園不同)

  1. 第 1 順位:特殊身份(身障、原民、低收)
  2. 第 2 順位:教職員子女、在園生兄弟姊妹
  3. 第 3 順位:社區/鄰里居民、簽約合作機構員工
  4. 第 4 順位:一般生(沒在前 3 順位)

問清楚自己屬於第幾順位。報名前看簡章「順位」段,搞清楚自己被排在哪一順。

🚨 備取保留期 — 最容易踩的地雷

❌ 備取保留 5 天 = 跟沒中差不多

假設你是備取第 5 名,正取通常 1-2 個月內陸續報到。如果園所規定「備取保留只到報到日後 7 天」,那等於正取沒人放棄你就 GG。備取保留期短的園所,沒抽中正取就要當作沒中。

備取保留期長短意義

備取保留期對你的意義
5-14 天⚠️ 等於只看正取。沒中正取 → 看下家
1 個月🟡 部分機會。前 3 名備取可期待
2-3 個月🟢 真的有機會。8 月開學前都可能輪到
到下年留園直升前🟢🟢 最長(新北非營利的標準)。整年都有機會

📊 同時報多家的策略

  1. 3-5 家是常態,不要只押寶一家
  2. 確認時程不衝突(報名期間、抽籤日、報到日不要撞)
  3. 排優先順序(離家最近 / 教學風格最合 / 費用最低)
  4. 抽中複數家時:選最想要的,其他家**主動放棄正取**(讓其他備取家庭有機會,不要佔位)
  5. 都沒抽中:看哪家備取保留期長,等遞補。也準備私立 plan B

🤝 留園直升

已經在園的孩子,通常隔年可以「直升」上一個年級,不用重抽。直升登記時間各園不同(通常 1-2 月),錯過就要重新抽籤。

⚠️ 常見地雷

這些是「會全盤失敗」的前提條件。看完再開始動作,不要踩過才知道。

🚨 過了報名截止 = 等下一年

大部分園所一年只開一次報名期。錯過就要等到隔年(整整 12 個月)。截止日記到 Google 行事曆 + 提醒前 3 天。

🚨 沒準時報到 = 放棄正取

抽中正取後,各園規定的報到期限(通常 2-3 天)內必須完成報到。**逾期視同放棄,直接讓備取遞補**。如果正在處理多家,務必把每家報到截止日列出來。

🚨 備取保留期太短 ≈ 沒中

備取保留只到 5-14 天的園所,實質上等於只看正取。**評估這家園所時,要當作「沒抽中正取就是沒上」**。

⚠️ 預防接種卡找不到 → 補發 1-2 週

健兒手冊黃卡是入園必備。錄取後才找會發現要去衛生所補發,可能來不及。**現在就翻出來確認,影印 5 份**。

⚠️ 距離評估太樂觀

Google Maps 顯示 15 分鐘 = 大人快走。實際帶孩子(小腳步、推車、抱抱、停下來看路邊東西)是 25-30 分。再加上下雨/颱風日 → 不可行。**選園前親自走一趟**。

⚠️ 報名表要先去園所拿

部分準公共要「現場索取報名表」,不能下載。指定日期才能拿。查清楚什麼時候可以拿,不要報名截止前一天才發現沒表格。

⚠️ 多家報名期重疊

5 月初是地獄週 — 非營利網路報名(統一)+ 多家準公共現場報名(各別)同期跑。**先做行事曆**,把每家的「報名期間 + 報到日 + 抽籤日」全標出來,排出可行的時序。

⚠️ 順位搞錯

很多家長以為自己是第 2 順位「鄰里居民」,結果簡章寫的「鄰里」只算特定路段或里。**逐字看簡章「優先順位」段**,不確定就打電話問。

⚠️ 私立隱形費用

月費 12000 看起來合理,但加上註冊費 5000 / 學期 + 書本費 3000 + 制服 2000 + 活動費 1500/月 + 延托費 200/天 → 實際年費可能多 30%。問清楚「全年總費用」,不要只看月費。

⚠️ 教學風格名稱含糊

招牌寫「蒙特梭利」「華德福」不一定是真的(這些名稱沒法律保護,任何園所都能掛)。**問實際做法**:教具是否齊全、老師是否經過認證、混齡比例多少。

⚠️ 雙語/全美語的真實程度

「全美語」可能是「外師上 1 小時、其他時間中文」,也可能是「全天英文,午睡才中文」。**實際聽老師對話**,不是看招生海報。

⚠️ 試讀沒做

部分園所提供試讀(半天、一天、一週)。**如果有提供一定要試**。孩子能不能適應、跟老師合不合,只有試了才知道。

📌 5 月初的地獄週(假設你住新北)

同一週內可能要處理:

  • 非營利統一網路報名(5/初開放,5/7 截止)
  • 常春藤 / 天翔現場報名(各別 5/3 / 5/4 開始)
  • 跑去廣福拿報名表(5/11 才能拿)
  • 準備全部園所要的文件

**4 月底就把整個 5 月的時序排好**,不要 5/1 才開始想。

👨‍👩‍👧 三明治世代家庭專區

同時照顧失智長輩 + 幼兒的家庭。**接送動線、認知資源、緊急應變** 三件事要一起想。

📌 為什麼這一頁

主流幼兒園手冊只考慮「夫妻 + 孩子」三人結構。但很多家庭實際是「失智長輩 + 夫妻 + 0-6 歲孩子」四到五人結構,選園的考量完全不同。這一頁專門寫這類家庭。

🧭 接送動線整合

單純夫妻只要算「家 ↔ 園所 ↔ 工作」三點。三明治家庭加上:

  • 長輩日照中心(早送 / 下午接)
  • 長輩看診(每週固定 / 不定期)
  • 居服員到府時段(白天某幾小時)
  • 長輩固定散步路線(若仍可外出)

能做的事

  • 幼兒園選在「日照中心 → 工作」的動線上(一趟解決兩個送)
  • 避開「長輩固定外出時段」的接送(長輩可能跟孫見到產生情緒波動)
  • 接送時段固定 → 居服員時段安排不衝突

👵 祖父母接孫的可行性評估

失智早期長輩可能還能接送孫,但要評估:

  • 認知功能:能否記得時間、地址、孩子名字
  • 方向感:單獨往返園所是否會迷路
  • 應變能力:孩子哭鬧、跌倒、突然需要回家時的處理
  • 體力:推車、抱孩子、走樓梯是否吃力
  • **最重要**:長輩自己願不願意(被指派照顧責任會增加壓力)

建議

  • **輕度失智**:可以「陪伴接送」(成人陪去成人陪回),不單獨負責
  • **中度失智**:不建議單獨接送,但可以一起去當「長輩跟孫互動的活動」
  • **重度失智**:不適合接送,但可以在家等接回來時跟孫說話、看孩子畫畫

長輩跟孫互動本身對失智是正向刺激(嗅覺/視覺/觸覺豐富)。重點是不增加長輩負擔,讓互動是樂趣不是責任。

🆘 緊急狀況預演

三明治家庭最怕「兩邊同時出狀況」。預先想清楚:

  • 長輩半夜送醫,白天孩子怎麼接?(備援接送名單:配偶 / 鄰居 / 教保員家送 / 暫托)
  • 孩子在園所發燒要接,你正陪長輩看診?(備援名單)
  • 長輩走失,你在園所路上?(分流原則:孩子是當下最緊急,長輩有警察)
  • 夫妻有一人發燒/出差,單一照顧者撐多久?(同時照顧 2 個失能者的極限是 24-48 小時)

📌 備援名單

至少 2 個人能緊急接孩子(寫在園所聯絡卡上),並讓他們知道。配偶 / 親戚 / 鄰居 / 朋友 / 居服員 / 教保員都可以。

**長照 + 兒童托育** 服務可以疊用,讓主照顧者真的有休息時間。

🎯 選園 checklist(失智家庭專屬)

在 📋 一般 checklist 之外,額外要看:

  • ✅ 接送動線是否經過日照中心 / 長輩會去的地方
  • ✅ 接送時段是否錯開居服員到府時間
  • ✅ 是否能接受祖父母陪伴接送(部分園所要求只能法定監護人)
  • ✅ 是否提供緊急接送的彈性(臨時需要早接 / 晚送)
  • ✅ 教保員是否願意配合多人接送名單(備援人選)
  • ✅ 園所是否會主動聯繫家中其他成員(若主照顧者聯繫不上,會否找備援)

📌 跟園所先講白

第一次面談就坦白家裡的狀況(有失智長輩、可能有突發狀況)。願意配合的園所會欣賞你的誠實,不願意的提早知道也比較好。

📝 Case A · 新北土城雙薪 2 歲女兒(2026 報名)

家庭背景

  • 新北土城,離捷運海山站約 10 分鐘步行
  • 雙薪家庭,孩子 2 歲女兒(2024 年 9 月前出生)
  • 住址設籍青雲里(該里在園所順位上幾乎都是「一般生」)
  • 有失智長輩在家(由居服員協助)

篩選標準

  • 步行 30 分鐘內(實測,不只 Google Maps)
  • 2 歲班有名額
  • 非營利優先(費用最低 + 統一報名 + 備取保留期最長)
  • 準公共補位(費用相同,各園自辦)

找到的 11 家(步行 + 2 歲班 + 公費類)

類型數量備註
非營利2中正、明德(統一網路報名 4/30-5/7,5/19 抽籤)
準公共(20 分內)3肯恩土城、樹德、天翔
準公共(28-32 分)4征洋、奇士華、常春藤、廣福、有一天
準公共(40+ 分)2全球、詠樂(較遠,當保險)
排除2凱樂(99 分)、頂埔(152 分)路程不實際

策略

  1. **非營利**先報(統一網路,費力低)
  2. **步行 20 分內**的 3 家(肯恩、樹德、天翔)優先實地參觀
  3. **5 月初地獄週**:同步處理非營利 + 常春藤 + 天翔(報名期重疊)
  4. **5 月中**:跑廣福取報名表(5/11 後才開放索取)
  5. **備取保留期**:廣福(6/30)、征洋(7/31)、土城名仁(10/31)是長期備取的安全網

學到的事

  • 「鄰里順位」要逐字看:常春藤的「第 3 順位是清水里」,青雲里就不算,直接是一般生
  • 備取保留期極短的園所要謹慎:常春藤備取只到 5/14,等於沒中正取就 GG
  • 名額多 ≠ 容易抽中:天翔 24 名,但報名也多,實際抽中率要看當年資料
  • 多家報名要做時序表:同一週可能跑 4-5 家,沒排表會錯過

整理日期 2026-04-26,各園資料以該年度簡章為準。隔年資料可能調整。

👶 托嬰專區(0-2 歲)

托嬰跟幼兒園邏輯完全不同。0-2 歲的關鍵是「替家裡分擔基本照顧」,不是教育。重點看老師怎麼帶、怎麼餵、怎麼管。

📌 跟幼兒園(2-6 歲)的根本差異

  • 師生比 1:5(法規)— 比幼兒園嚴格(2-3 歲是 1:8)
  • 不教東西 — 重點是吃飽、睡好、不哭、安全。發展刺激是附加
  • 每日狀態回報 — 大便幾次、睡幾小時、喝幾 cc 全要記
  • 家長端配合多 — 副食品要家裡先試吃、午睡作息要對齊
  • 送托心理門檻高 — 第一次送 0-2 歲幼兒,新手家長不適應期 1-3 個月

🍼 1. 帶小孩方法論(教保理念)

每家托嬰風格差很多。問下面這些問題,聽實際做法不是聽口號。

哭聲處置

  • 立刻抱起(回應派):0-1 歲哭通常是需求未滿足,先處理生理
  • 觀察 30 秒(混合派):非生理哭(無聊/想抱)讓他學自我安撫
  • 哭聲控制法(訓練派):制定哭多久才回應的時間表

沒有絕對對錯,但要跟家裡作法一致。家裡立刻抱、托嬰讓他哭,孩子會混亂。

副食品引入

  • 4-6 個月開始(主流):米精 → 米糊 → 蔬菜泥 → 水果泥 → 蛋黃 → 肉泥
  • BLW(寶寶主導離乳):6 個月直接吃手指食物,讓寶寶自己抓
  • **禁忌**:1 歲前不給蜂蜜(肉毒桿菌)、不給未消毒鮮奶

睡眠訓練

  • 不訓練(自然派):隨需求餵 / 抱睡 / 自然睡
  • 溫和訓練(費伯法 / 抱起放下法):漸進式延長獨處時間
  • 哭出來訓練:讓寶寶自己學會入睡(爭議大)

體罰 / 罰站

  • 0-2 歲禁止任何體罰(法規 + 倫理),這條沒得談
  • 看老師怎麼處理「打人」「咬人」「搶玩具」的孩子,有沒有 SOP

面試問題:「孩子哭超過 30 分鐘,老師怎麼處理?」「孩子打另一個孩子,你會怎麼做?」回答得越具體越可信,越含糊越要警戒。

📹 2. 監視器 / 透明度

0-2 歲不會說話,家長無法從孩子口中知道一天發生什麼。監視器是最低保障。

方案透明度備註
無監視器❌ 零放棄這家
只有出入口監視⚠️ 低看不到教室內,不夠
教室全區監視 + 法規保存 30 天🟡 標準有狀況可調閱(但要寫申請)
家長可遠端查看(手機 App)🟢 高較罕見,有的話加分
每日推照片 / 影片給家長🟢 高主動推送,看得到孩子實況

要問的問題

  • 監視器涵蓋哪些區域?(教室 / 寢室 / 用餐區 / 廁所內 — 廁所內通常沒有,但廁所外要有)
  • 錄影保存幾天?有狀況怎麼調閱?
  • 家長能不能主動看?還是只在出事才能看?
  • 有家長手機 App 嗎?照片頻率多高?

🏠 3. 空間大小

0-2 歲多數時間在地板上爬、玩、睡。空間夠不夠很關鍵。

  • 法規:每幼兒室內活動空間至少 2 平方公尺(0.6 坪)
  • 實際扣掉:嬰兒床 + 圍欄 + 教具櫃 + 桌椅後,真正能爬能玩的空間是多少?
  • 戶外活動空間:有自己的戶外?還是要去公園?雨天怎麼活動?
  • 分齡分區:0-1 歲(不會走)和 1-2 歲(會走會爬)有沒有分開,還是混在一起?(混區風險:大寶寶不小心踩到小寶寶)
  • 午睡區:每個孩子有專屬嬰兒床嗎?床距夠不夠(預防交叉傳染)?

🍱 4. 餐點 / 飲食

0-1 歲

  • 配方奶:用哪個牌子?家長能不能指定品牌?
  • 配方奶溫度 / 沖泡比例:園所有 SOP 嗎?
  • 瓶餵方式:抱著餵還是放著餵?(影響孩子安全感 + 中耳炎風險)
  • 母乳冷藏:家長送母乳到園,有沒有冰箱?如何分裝、加熱?

1-2 歲(副食品)

  • 菜單透明度:有沒有公開週菜單?食材來源?
  • 過敏處理:家裡有過敏紀錄,園所怎麼處理(完全不給 vs 替換餐)?
  • 家長端配合:新食材要不要家裡先試 3 天?
  • 餵食方式:幾點餵 / 餵多少 / 不吃怎麼處理(強迫 vs 換時段 vs 跳過)

共通

  • 餐具消毒:每個孩子專屬餐具?每次消毒方式?
  • 飲水:1 歲後可以喝水,水從哪來、水溫多高?
  • 輔食器具:果汁機 / 食物剪 / 食物剪刀清洗頻率?

👩‍🏫 5. 老師師資(0-2 歲特殊)

0-2 歲老師的證照種類比幼兒園更多元,要看清楚。

證照意義適合年齡
保母證照(技術士)主要訓練嬰幼兒生活照顧0-2 歲(主流)
教保員(高中職幼保科以上)幼教學門出身,理論強2-6 歲
教保員 + 嬰幼兒組大學幼教加修 0-2 歲0-6 歲
幼兒園教師證(大學幼教系 + 教檢)專業師資,薪資較高2-6 歲
護理師少數園所配置,處理生病各年齡

要問的問題

  • 0-1 歲班的老師有幾個是有保母證的?
  • 有沒有護理師駐園?(發燒、跌倒、餵藥誰負責)
  • 老師平均待多久?(高流動率對 0-2 歲影響特別大,孩子需要建立依附)
  • **主帶老師**(primary caregiver)制度?同一個孩子是否由同一位老師主帶,讓孩子能建立穩定關係

⏰ 6. 延托(extended care)

標準時間:7:30-17:30。超過要延托。雙薪家庭幾乎一定會用。

要問清楚

  • 延托到幾點?(18:30 / 19:00 / 19:30 — 各家不同)
  • 計費方式:每小時(150-300 元)/ 每天(50-100 元)/ 月票
  • 單次 vs 月租:偶爾用 vs 每天用,哪個划算
  • 提前通知時間:當天打電話算晚還是 OK?
  • 晚餐:延托時間有沒有提供餐點(有的有有的沒)
  • 緊急狀況:你 18:00 突然開會,能不能臨時延?
  • 逾時罰款:超過延托時間還沒接,每分鐘怎麼算(很多園 5 分鐘就罰)

三明治世代特別注意:長輩臨時送醫的話,延托是救命用。問清楚「臨時延托」的彈性。如果園所說「一律不能臨時延」,你家可能不適合這家。

📅 7. 活動管理 / 一天的時間表

0-1 歲(沒結構日,只有作息)

0-1 歲不需要「課程」,需要的是穩定作息。

  • 早上:到園 → 玩 → 喝奶 → 小睡
  • 中午:午餐 → 午睡(2-3 小時)
  • 下午:醒 → 喝奶 → 玩 → 點心 → 接送

「玩」是觀察 / 抓握 / 跟老師互動,不是上課。如果園所說「我們有教 0-1 歲認字」, 那是行銷話術。

1-2 歲(開始有結構)

  • 9:30-10:00 主題活動(15-20 分鐘上限,1-2 歲注意力短)
  • 10:00-10:30 戶外或大肢體
  • 10:30-11:00 點心 + 自由玩
  • 11:00-11:30 午餐
  • 11:30-14:30 午睡
  • 下午類似上午,但少一個主題活動

要問的問題

  • **戶外活動**多久一次?雨天替代方案?
  • **主題活動**怎麼設計?是否每月有不同主題?
  • **音樂 / 律動 / 美勞**頻率?
  • **蒙氏教具 / 感統教具**有嗎?用在什麼時段?
  • **電子設備**:1-2 歲會看影片嗎?(有的家長很在意)

🤝 8. 活動配合方式 · 家長端的功課

0-2 歲送托後家長還要做很多。**了解清楚再簽約**,不然變成雙方互怨。

每天

  • 聯絡簿:紙本還是 App?家長要不要每天回?(有的園所要)
  • 用品補充:尿布、濕紙巾、口水巾、奶粉、奶瓶 — 用完要立刻補
  • 衣物備品:換洗衣 2-3 套,弄髒要立刻補
  • 奶粉 / 母乳:每天送新鮮的還是 1 週送一批

每週 / 每月

  • 家庭日 / 親子活動:多久一次?強制參加嗎?
  • 新食材試吃:園所要試新食材時,家裡要先試 3 天觀察過敏
  • 親師會:多久一次?重要時段
  • 體適能 / 健康檢查:園方主辦還是家長帶去?

突發

  • 生病請假:發燒幾度要接回?(通常 38°C)
  • 傳染病停課:腸病毒 / 流感 — 全班停或個案隔離?
  • 送醫:園方先送哪家醫院?家長到場時間預期?

📌 0-2 歲送托心理建設

第一個月會崩潰,這是正常的:

  • 孩子哭著去 / 哭著回(分離焦慮高峰)
  • 常生病(免疫系統碰到群體病菌的洗禮,3-6 個月才穩)
  • 作息打亂、夜醒增加
  • 家長自責、懷疑送托是不是錯誤

這些大部分會過去。如果 3 個月後還是無法適應 → 評估是園所不合還是孩子真的不適合送托(極少數)。
