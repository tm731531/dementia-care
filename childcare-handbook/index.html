🏫 4 種托育類型

費用、報名方式、抽籤規則完全不同。選錯類型,所有後續動作都白搞。

⚠️ 重要前提:0-2 歲跟 2-6 歲是兩套法律

「0-2 歲機構式托嬰」歸衛福部管(兒少法),「2-6 歲幼兒園」歸教育部管(幼照法)。兩套主管機關、兩套類型、兩套收費規則。所以下面拆兩張表看,不要互相套用。

🏛 公托 / 公幼

政府辦的公立托嬰中心 / 公立幼兒園。費用最低,但僧多粥少,要靠抽籤。

特點

  • 👍 費用最低(月費 2000-4000,含政府補助後)
  • 👍 師資穩定(教保員多為公務體系,流動率低)— 但代價是:若分配到風格不合的老師,公務體系幾乎不能換班 / 換園,私立 / 準公共反而還有彈性
  • 👍 標準化(教學內容、餐食、安全規範統一管理)
  • 👎 名額極少,公托(0-2 歲)等候 1-3 年很正常
  • 👎 區域順位:沒設籍在該區可能根本沒抽籤資格
  • 👎 教學偏「保守」:沒有蒙氏/華德福之類的特色

怎麼報

  • 各縣市教育局 / 社會局網站(各地不同 — 看 📚 縣市資源)
  • 0-2 歲公托:孩子出生就要去登記,等候名單通常 1-3 年
  • 2-6 歲公幼:每年 4-5 月一波,跟非營利同期

🏘 非營利幼兒園(只有 2-6 歲)

政府委託符合資格的非營利法人辦的幼兒園。法源是《幼兒教育及照顧法》第 9 條 + 《非營利幼兒園實施辦法》。契約期 4 學年,連續 3 年考評通過可申請續約。

承辦資格(法定)

  • 學校財團法人 / 醫療法人
  • 幼兒教保工會組織 / 職工福利委員會(已完成法人登記)
  • 章程載明兒童福祉或教保服務的財團法人 / 非營利社團法人

特點

  • 👍 費用低(第 1 胎月費約 2,000 元,各園略有差異,比準公共更低)
  • 👍 報名統一(各縣市一個網站搞定多家)
  • 👍 備取保留期長(新北非營利保留至下年留園直升前一日,最長)
  • 👍 線上報到(不用臨櫃)
  • 👎 名額少,抽籤競爭激烈
  • 👎 不收 0-2 歲(0-2 歲只能找公托 / 準公共托嬰 / 私托 / 保母)

怎麼報

  1. 找該縣市的統一報名網站(新北:kid123.ntpc.edu.tw)
  2. 提前註冊帳號(不要等到報名日才註冊)
  3. 缺額公告當天確認可報名園所
  4. 網路報名 → 資格審查 → 補件 → 抽籤 → 線上報到

🏢 準公共

📌 「準公共三兄弟」一句話釐清

「準公共」有 3 種,補助金額和規則完全不同,看名字會混:

  • 準公共托嬰中心(0-未滿 3 歲)— 衛福部 / 補助 13,000+ / 家長實付 5,000-12,000
  • 準公共保母(0-未滿 3 歲)— 衛福部 / 補助 13,000+ / 家長實付 5,000-12,000
  • 準公共幼兒園(2-6 歲)— 教育部 / 家長月費上限 3,000(第 1 胎)

準公共 · 兩種版本對比

準公共托嬰(0-2 歲)準公共幼兒園(2-6 歲)
主管機關衛福部社家署教育部
家長費用沒有 3000 統一上限,各中心收費 - 政府補助 13,000 後家長自付 5,000-12,000第 1 胎 3,000 / 第 2 胎 2,000 / 第 3 胎 1,000(法定上限)
政府補助來源中央托育補助 13,000 + 縣市加碼免學費 + 家長部分負擔
合約期程視縣市,常 1-3 年第 3 期程 113-116 / 7(每期程約 3 學年)

特點

  • 👍 費用比私立低很多(0-2 歲補助後可省一半;2-6 歲第 1 胎只 3,000)
  • 👍 名額較多(多家私園加入,選擇多)
  • 👍 有 0-2 歲托嬰版本(這是非營利沒有的)
  • 👍 教學風格較多元(各園自由發展特色)
  • 👎 各園自辦報名(時程、地點、文件、規則全不一樣 — 累)
  • 👎 備取保留期差異大(短的 5 天,長的 4 個月,要看清楚)
  • 👎 合約期到期可能退場(看下方警告)

⚠️ 準公共合約期到期 = 隨時可能爆價

準公共不是永久合約,而是期程制(第 3 期程 113/8 - 116/7)。期滿:

  • 不自動續約 — 需地方政府跟園所重新協商
  • 評鑑未通過者無法續約 — 直接從清單消失
  • 不續約後:園所恢復為一般私立,費用立刻回到 8,000-15,000+ 月費
  • 家長端:每年 7-8 月新學年前注意園所公告;到期前一學期(1-2 月)主動詢問是否續約

怎麼報

  1. 找各縣市「準公共幼兒園/托嬰中心」清單(教育部 / 衛福部官網有)
  2. 篩選步行範圍內的園所
  3. 逐家查報名期間 + 報名方式(現場/Email/網站,各家不同)
  4. 注意:準公共報名期間常與非營利重疊(都是 4-6 月),要排好優先序
  5. 確認該園所合約期到哪一年(全國教保資訊網查得到)

💸 私立(自費)

完全自由訂價。費用從每月 8000(基本)到 50000+(雙語/特色園)都有。沒有政府補助。

特點

  • 👍 不用抽籤,看名額即可申請(隨時可入,不限開學)
  • 👍 教學風格多元(蒙氏、華德福、雙語、森林、體育)
  • 👍 可以中途轉入(不限年度開學)
  • 👎 費用高(年費可能是準公共的 5-15 倍)
  • 👎 隱形費用多(註冊費、書本費、制服費、活動費、延托費 — 累計可能再加 30%)
  • 👎 退費規則嚴(中途離園可能要付違約金)
  • 👎 師資流動率參差(有些園所離職率高,孩子適應期長)

什麼情況考慮私立

  • 沒抽中任何公幼/非營利/準公共,等不到下一年
  • 家庭預算寬裕,要特定教學風格
  • 想要某些特殊服務(全美語、雙師制、小班)
  • 想要中途插班(其他類型只在 8 月開學)

📐 怎麼決定優先順序

狀況建議優先序
預算緊 + 雙薪 + 0-2 歲幼兒公托 → 準公共托嬰 → 私立托嬰
預算緊 + 雙薪 + 2-6 歲幼兒非營利 + 公幼(同期報) → 準公共 → 私立(備援)
預算寬 + 想要特定教學直接看私立(蒙氏/華德福/雙語) → 準公共補位
住偏遠地區查最近的「準公共」+ 「私立」(公幼/非營利通常市區才多)
想全家搬家配合園所公托/公幼是首選(便宜 + 穩定),其他類型不用為它搬

⚠️ 不要只報一家

即使最想要 A 園,也要報 B、C、D 當保險。抽籤是運氣,沒抽中要等下一年(8 月開學前可能補不到位)。同時報 3-5 家是常態。

🎯 5W1H 評估方法論

每個家庭適合的園所不同。這 6 個維度逐一回答,不是想哪個算哪個。

👤 Who · 你家是誰

「孩子幾歲」不是 Who 的全部。Who 是整個家庭結構:

  • 家庭結構:雙薪 / 單薪 / 單親 / 多代同堂(有失智長輩參與接送嗎?)
  • 預算:每月教育預算上限是多少?(影響類型選擇)
  • 接送角色:誰接、誰送、有沒有備援?
  • 孩子個性:好動 / 內向 / 過敏體質 / 有特殊需求
  • 家裡其他成員:兄弟姊妹是否同園(可能有「在園生弟妹」優先順位)

📌 隱形維度:祖父母

如果有失智長輩在家,需要評估:長輩是否能參與接送(認知功能)、長輩日照中心位置是否在接送動線上、長輩走失風險是否影響你的行程彈性。看 👨‍👩‍👧 三明治世代。

🏫 What · 選哪一類

看 🏫 4 種類型。決策邏輯:

  1. 先看預算(私立 vs 補助類)
  2. 再看年齡(0-2 vs 2-6)
  3. 再看時間(等得起公托嗎?)
  4. 最後看教學風格偏好

📅 When · 什麼時候動

類型關鍵時機
公托(0-2 歲)孩子出生就排隊。等候 1-3 年正常
非營利 + 公幼(2-6)每年 4 月底缺額公告 → 5 月初報名 → 5 月中抽籤
準公共各園不同,集中在 5-6 月
私立全年隨時(看名額),但 7-8 月開學潮位置最少

過了報名截止就是 GG。各園截止日不同,排不開的話只能放棄某些園。看 📅 時程地圖。

📍 Where · 距離與動線

步行 20 分鐘原則

  • 地圖顯示 15 分鐘 → 實際 20 分鐘起跳(小孩腳步慢)
  • 下雨 / 颱風 / 寒流 → 30+ 分鐘或無法步行
  • 不要評估「我成人快走的時間」

接送動線評估

  • 從家到園所(早上)
  • 從園所到工作地點(早上接續)
  • 從工作地點回園所(下午接)
  • 從園所回家(下午接續)
  • 有失智長輩家庭:加上「日照中心位置」「醫院位置」

📌 試走一遍

真的去走一次。算上紅綠燈、推車、抱孩子。把這個時間 +10 分鐘當作「能接受的上限」。超過就不要勉強(會崩潰)。

💡 Why · 教學風格

不要只看招生簡章上的標榜。問:

  • 典型一天的時間表是什麼?(自由探索 vs 結構式上課的比例)
  • 戶外活動多久一次?(每天 / 每週?)
  • 有沒有特殊課程?(雙語?才藝?)
  • 怎麼處理孩子情緒?(這題回答得好不好,反映教保理念)
  • 家長如何參與?(只看 Line 群還是真的有家長日)
常見教學風格簡介
  • 蒙特梭利(Montessori):孩子自選工作、混齡、教具導向。強調獨立。
  • 華德福(Waldorf):節奏、季節、自然材質、不接觸電子產品。慢學派。
  • 瑞吉歐(Reggio Emilia):專案式學習、藝術中心、強調對話。
  • 主題式(主流):每月或每季一個主題,各種活動圍繞主題。
  • 遊戲式 / 開放教育:大量自由遊戲時間。
  • 雙語 / 全美語:重視語言環境。注意是「外師全程」還是「中師翻譯式」。

教學風格沒有絕對好壞 — 看孩子個性。內向慢熟的孩子可能不適合大團體競爭式環境。

⚠️ 雙語 / 全美語的法規灰色地帶

《幼兒教保及照顧服務實施準則》第 13 條第 4 款原文:有進行外語教學之必要者,應以部分時間融入教保活動課程,並符合幼兒園教保活動課程大綱;不得以全部時間為之,或以部分時間採非融入方式進行教保活動。意思是「全美語幼兒園」可能不符準則第 4 款規定,各縣市查核力度不一 — 有的嚴查、有的睜一眼閉一眼。家長選擇前宜自行評估風險。

  • 教育部立場:外語可以「融入活動」(例如玩遊戲時帶幾句英文),但不能整天英文
  • 市場現況:「全美語」招牌仍廣泛存在,實質遊走灰色地帶
  • 家長判斷方法:直接問「每天有幾小時中文活動?中文教師是誰?」
  • 查證:全教網查該園評鑑 + 裁罰紀錄

⚠️ 注音教育部明確不教,但私園可能偷教

《幼兒教保及照顧服務實施準則》第 13 條規定課程應以「統整方式」進行,不得採分科方式進行;同條第 9 款:不得進行以精熟為目的之讀、寫、算教學。教育部明確反對學前分科教注音、寫字。理由:幼兒眼球和手部肌肉仍在發育。

  • 正規期:小一前 10 週才是注音教學期,不需提前學完
  • 實際:部分私園大班仍有「認字」「注音遊戲」,以「融入」包裝
  • 該不該先學:研究顯示提前死記反而在一年級造成干擾;但若孩子有興趣 + 不強迫,亦非禁止
  • 選園要問:「大班會教注音嗎?寫字嗎?」回答「教」就違規,「不教」要看實際情況

🎲 抽籤機制

大部分公幼/非營利/準公共都靠抽籤。備取保留期短的等於沒中,這個細節千萬要看。

🔄 標準流程

  1. 報名 — 各園規定的報名期間提交資料
  2. 資格審查 — 園方/教育局審核資料、補件通知
  3. 抽籤 — 報名人數 > 名額 才會抽。報名人數 ≤ 名額 直接全錄取
  4. 正取 + 備取公告 — 抽到正取的依序遞補,備取按抽籤順序排隊
  5. 正取生報到 — 截止日內到場/線上報到。過了就放棄正取資格
  6. 備取生遞補 — 正取沒報到 → 備取依序通知 → 直到滿額或備取保留期截止

⚠️ 順位 vs 抽籤的差別

同一園所內可能有多個順位(例:第 2 順位是教職員子女,第 3 順位是社區住戶,第 4 順位才是一般生)。順位是先後順序,前順位優先填滿,輪到一般生才開始抽籤。

典型順位範例(各園不同)

  1. 第 1 順位:特殊身份(身障、原民、低收)
  2. 第 2 順位:教職員子女、在園生兄弟姊妹
  3. 第 3 順位:社區/鄰里居民、簽約合作機構員工
  4. 第 4 順位:一般生(沒在前 3 順位)

問清楚自己屬於第幾順位。報名前看簡章「順位」段,搞清楚自己被排在哪一順。

🚨 備取保留期 — 最容易踩的地雷

❌ 備取保留 5 天 = 跟沒中差不多

假設你是備取第 5 名,正取通常 1-2 個月內陸續報到。如果園所規定「備取保留只到報到日後 7 天」,那等於正取沒人放棄你就 GG。備取保留期短的園所,沒抽中正取就要當作沒中。

📊 同時報多家的策略

  1. 3-5 家是常態,不要只押寶一家
  2. 確認時程不衝突(報名期間、抽籤日、報到日不要撞)
  3. 排優先順序(離家最近 / 教學風格最合 / 費用最低)
  4. 抽中複數家時:選最想要的,其他家主動放棄正取(讓其他備取家庭有機會,不要佔位)
  5. 都沒抽中:看哪家備取保留期長,等遞補。也準備私立 plan B

🤝 留園直升

已經在園的孩子,通常隔年可以「直升」上一個年級,不用重抽。直升登記時間各園不同(通常 1-2 月),錯過就要重新抽籤。

⚠️ 常見地雷

這些是「會全盤失敗」的前提條件。看完再開始動作,不要踩過才知道。

🚨 過了報名截止 = 等下一年

大部分園所一年只開一次報名期。錯過就要等到隔年(整整 12 個月)。截止日記到 Google 行事曆 + 提醒前 3 天。

🚨 沒準時報到 = 放棄正取

抽中正取後,各園規定的報到期限(通常 2-3 天)內必須完成報到。逾期視同放棄,直接讓備取遞補。如果正在處理多家,務必把每家報到截止日列出來。

🚨 備取保留期太短 ≈ 沒中

備取保留只到 5-14 天的園所,實質上等於只看正取。評估這家園所時,要當作「沒抽中正取就是沒上」。

⚠️ 預防接種卡找不到 → 補發 1-2 週

健兒手冊黃卡是入園必備。錄取後才找會發現要去衛生所補發,可能來不及。現在就翻出來確認,影印 5 份。

⚠️ 距離評估太樂觀

Google Maps 顯示 15 分鐘 = 大人快走。實際帶孩子(小腳步、推車、抱抱、停下來看路邊東西)是 25-30 分。再加上下雨/颱風日 → 不可行。選園前親自走一趟。

⚠️ 報名表要先去園所拿

部分準公共要「現場索取報名表」,不能下載。指定日期才能拿。查清楚什麼時候可以拿,不要報名截止前一天才發現沒表格。

⚠️ 多家報名期重疊

5 月初是地獄週 — 非營利網路報名(統一)+ 多家準公共現場報名(各別)同期跑。先做行事曆,把每家的「報名期間 + 報到日 + 抽籤日」全標出來,排出可行的時序。

⚠️ 順位搞錯

很多家長以為自己是第 2 順位「鄰里居民」,結果簡章寫的「鄰里」只算特定路段或里。逐字看簡章「優先順位」段,不確定就打電話問。

⚠️ 私立隱形費用

月費 12,000 看起來合理,但全年實際支出常多 20-40%:

  • 註冊費 5,000 / 學期 × 2
  • 書本 / 教材 3,000 / 學期
  • 制服 / 圍兜 2,000(成長後續買)
  • 活動 / 校外教學 1,500 / 月
  • 延托費 200 / 天 → 月票約 2,000-3,500
  • 學生平安保險 400-1,000 / 年(常被忽略)
  • 才藝 / 課輔費 1,500-3,000 / 月(部分強制)
  • 冷氣費 / 水電費 200-500 / 月(部分私立另收)
  • 家長會費 500-1,000 / 年
  • 節慶禮金(教師節/學期末)合資後約 1,000-2,000 / 年

舉例兩個情境(月費 12,000 私立):

  • A 不用延托 / 不上才藝:月費 12K×12 + 註冊 10K + 書本 6K + 制服 2K + 活動 18K + 保險 0.7K + 家長會 0.75K + 節慶 1.5K = 約 183K / 年(月均 15.3K,比月費多 28%)
  • B 雙薪用延托 + 才藝:A + 延托月票 33K + 才藝 27K + 冷氣 4.2K = 約 247K / 年(月均 20.6K,比月費多 71%)

問清楚「全年總費用」+ 拿合約看清楚,不要只看月費。

⚠️ 教學風格名稱含糊

招牌寫「蒙特梭利」「華德福」不一定是真的(這些名稱沒法律保護,任何園所都能掛)。問實際做法:教具是否齊全、老師是否經過認證、混齡比例多少。

⚠️ 雙語/全美語的真實程度

「全美語」可能是「外師上 1 小時、其他時間中文」,也可能是「全天英文,午睡才中文」。實際聽老師對話,不是看招生海報。

⚠️ 試讀沒做

部分園所提供試讀(半天、一天、一週)。如果有提供一定要試。孩子能不能適應、跟老師合不合,只有試了才知道。

同一週內可能要處理:

  • 非營利統一網路報名(5/初開放,5/7 截止)
  • 常春藤 / 天翔現場報名(各別 5/3 / 5/4 開始)

  • 準備全部園所要的文件

4 月底就把整個 5 月的時序排好,不要 5/1 才開始想。

👨‍👩‍👧 三明治世代家庭專區

同時照顧失智長輩 + 幼兒的家庭。接送動線、認知資源、緊急應變 三件事要一起想。

📌 為什麼這一頁

主流幼兒園手冊只考慮「夫妻 + 孩子」三人結構。但很多家庭實際是「失智長輩 + 夫妻 + 0-6 歲孩子」四到五人結構,選園的考量完全不同。這一頁專門寫這類家庭。

🧭 接送動線整合

單純夫妻只要算「家 ↔ 園所 ↔ 工作」三點。三明治家庭加上:

  • 長輩日照中心(早送 / 下午接)
  • 長輩看診(每週固定 / 不定期)
  • 居服員到府時段(白天某幾小時)
  • 長輩固定散步路線(若仍可外出)

能做的事

  • 幼兒園選在「日照中心 → 工作」的動線上(一趟解決兩個送)
  • 避開「長輩固定外出時段」的接送(長輩可能跟孫見到產生情緒波動)
  • 接送時段固定 → 居服員時段安排不衝突

👵 祖父母接孫的可行性評估

失智早期長輩可能還能接送孫,但要評估:

  • 認知功能:能否記得時間、地址、孩子名字
  • 方向感:單獨往返園所是否會迷路
  • 應變能力:孩子哭鬧、跌倒、突然需要回家時的處理
  • 體力:推車、抱孩子、走樓梯是否吃力
  • 最重要:長輩自己願不願意(被指派照顧責任會增加壓力)

建議

  • 輕度失智:可以「陪伴接送」(成人陪去成人陪回),不單獨負責
  • 中度失智:不建議單獨接送,但可以一起去當「長輩跟孫互動的活動」
  • 重度失智:不適合接送,但可以在家等接回來時跟孫說話、看孩子畫畫

長輩跟孫互動本身對失智是正向刺激(嗅覺/視覺/觸覺豐富)。重點是不增加長輩負擔,讓互動是樂趣不是責任。

📜 法定接送授權(很多家庭忽略)

大多數幼兒園 / 托嬰中心要求事先提交「授權接送人清單」+ 書面授權。祖父母若為固定接送人,必須在入園時就書面授權,而非臨時告知;若有失智長輩單日認知波動,園所如何判斷接送是否安全也需事先約定 SOP。建議入園面談就釐清這點。

🆘 緊急狀況預演

三明治家庭最怕「兩邊同時出狀況」。預先想清楚:

  • 長輩半夜送醫,白天孩子怎麼接?(備援接送名單:配偶 / 鄰居 / 教保員家送 / 暫托)
  • 孩子在園所發燒要接,你正陪長輩看診?(備援名單)
  • 長輩走失,你在園所路上?(分流原則:孩子是當下最緊急,長輩有警察)
  • 夫妻有一人發燒/出差,單一照顧者撐多久?(同時照顧 2 個失能者的極限是 24-48 小時)

📌 備援名單

至少 2 個人能緊急接孩子(寫在園所聯絡卡上),並讓他們知道。配偶 / 親戚 / 鄰居 / 朋友 / 居服員 / 教保員都可以。

🏥 喘息資源整合

長照 + 兒童托育 服務可以疊用,讓主照顧者真的有休息時間。

🎯 選園 checklist(失智家庭專屬)

在 📋 一般 checklist 之外,額外要看:

  • ✅ 接送動線是否經過日照中心 / 長輩會去的地方
  • ✅ 接送時段是否錯開居服員到府時間
  • ✅ 是否能接受祖父母陪伴接送(部分園所要求只能法定監護人)
  • ✅ 是否提供緊急接送的彈性(臨時需要早接 / 晚送)
  • ✅ 教保員是否願意配合多人接送名單(備援人選)
  • ✅ 園所是否會主動聯繫家中其他成員(若主照顧者聯繫不上,會否找備援)

📌 跟園所先講白

第一次面談就坦白家裡的狀況(有失智長輩、可能有突發狀況)。願意配合的園所會欣賞你的誠實,不願意的提早知道也比較好。

🏡 入園後實務 · 簽約到上學的日常

選好園所後的「使用說明書」。試讀怎麼跑、適應期幾天、群組規矩、節慶禮金、生病備援 — 沒人會告訴你,踩過才知道。

📌 入園後 = 雙向合作,不是單向消費

這份手冊前面教你「怎麼選」(評估、要求、判斷),站在消費者角度。入園後角色翻轉 — 老師跟家長變成合作關係,孩子成長得不得益,看雙方配合度。下方第 0 節「家長端配合」是教保員 / 保母端最希望家長知道的事。

🤝 0. 家長端配合 Checklist(教保員 / 保母端最希望家長做的)

從教保員 / 保母真實感受整理。看一遍知道哪些是雷,孩子在園 / 在保母家會更順。

  • ✅ 不送病兒 — 發燒 38°C 退燒未滿 24 小時前不送(不是「感覺好很多」就送)。送病兒會讓老師同時照顧 5 個孩子裡有 1 個需要 100% 注意,其他孩子安全風險上升
  • ✅ 準時接送 — 改時間提前說。臨時 4:30 變 6:30 而不通知,老師需重新安排,影響其他孩子
  • ✅ 早睡配合作息 — 孩子帶著睡眠債入園,午睡睡不著會干擾全班。老師花在哄 1 個睡不著的時間,是從其他孩子借來
  • ✅ 主動告知孩子狀況 — 過敏、特殊習慣、家庭變動(搬家 / 弟妹出生 / 父母吵架),不讓老師猜
  • ✅ 不在群組討論個別問題 — 個別問題用聯絡簿或單獨 LINE,不在全班群組(其他家長不需知道,容易演變成指責)
  • ✅ 不破壞老師權威 — 孩子說「老師說不行」,家長在場時也說「對,老師說的對」(不在孩子面前翻盤)
  • ✅ 試讀期間不外顯焦慮 — 不在門口徘徊、不眼眶紅。離開就離開,老師能處理
  • ✅ 反映問題找老師,不找家長 — 先跟老師談,不在家長私群炒作製造輿論壓力
  • ✅ 下班後 + 週末避免發訊 — 21:00 後 / 7:00 前 / 週末非急事不發。老師有自己的生活
  • ✅ 不要傳孩子受傷照片到群組 — 引發集體恐慌是高頻事故 → 直接跟老師私訊
  • ✅ 節慶禮金量力而為 — 不參與也直接告訴班代「我這次不參與,謝謝」,不要默不作聲
  • ✅ 新食材主動先試 3 天 — 不要等園所要求,自己在家先試,告知結果
  • ✅ 疫苗 / 接種紀錄主動配合查核 — 老師有法定義務看黃卡,不是侵犯隱私

🧪 1. 試讀(試托)完整 SOP

有提供試讀的園所一定要試。試讀是孩子適合不適合、家長感受對不對的最後關卡。

試讀模式

模式常見費用
體驗半天多數準公共/私立多免費
試讀 1-3 天口碑園所免費或 500-1,000 元
試讀一週少數高端私立1,500-3,000 元
無試讀部分公托(名額太搶)—

試讀觀察重點

  • 孩子第一次哭多久才平靜?老師怎麼處理?
  • 老師有沒有叫得出孩子名字?(有的試讀第二天就叫得出 = 加分)
  • 午睡適應 — 孩子能不能在陌生環境睡著?(0-3 歲送托最大斷點 — 午睡哭鬧 = 整個下午都難過)
  • 第一餐吃多少 — 願不願意在陌生環境吃飯?(0-2 歲若拒食是嚴重訊號)
  • 換尿布 / 如廁 — 0-2 歲對換尿布者的接受度;2-3 歲如廁訓練配合度
  • 孩子離開時是哭著要回去還是開心離開?
  • 跟其他孩子互動如何?有沒有被欺負/欺負人?
  • 老師有沒有拍照給家長?
  • 試讀結束後園方是否主動回饋「今天孩子表現」?

試讀後決定

  • 多數園所要求 3 天內確認(過了名額讓給後面的)— 但口碑園所通常願意給 5-7 天,真誠猶豫直接問「我能不能週五前決定?」
  • 不確定可問:「我能再參觀一次嗎?」(願意的園所是好園所)
  • 孩子哭整天但你還是想送 → 給 1 週適應期再判斷;完全拒絕進教室 → 重新評估

🌅 2. 入園適應期 · 漸進入園 SOP

第一週孩子崩潰是常態。不要看到哭就退縮,但也不要硬撐。漸進式入園是業界標準做法。

業界通行 3 階段

  1. 第 1-2 天:家長送到門口後離開(多數園所標準);部分蒙氏 / 小型園才允許陪 10-30 分鐘在教室內
  2. 第 3-5 天:半天接(早上送 → 中午接,不留午睡)
  3. 第 2 週起:全天送,但提早接(下午 3-4 點)— 這是「全天的開始」,不是「完全適應」
  4. 第 3-6 週:正常時段;3-6 週才是真正穩定,別期待 2 週解決

實務提醒:60%+ 準公共 / 私立不允許家長在教室待超過 10-15 分鐘(會讓其他孩子也鬧著要家長)。問了園所說「不行」不代表是壞園所,只是大園子無法個別應對。蒙氏 / 華德福 / 小型園才常允許家長陪。

已適應 vs 還沒適應的訊號

已適應還沒適應(注意)
到園哭一下就跟老師走哭整路、抓住家長不放
離園玩到不想走看到家長立刻撲過來
飲食正常吃不吃午餐、不喝奶
睡眠會午睡整天不睡
晚上正常入睡夜醒、做惡夢

3-6 週內仍未適應 → 跟園所討論;3 個月仍崩潰 → 可能要換園或換策略。

📱 3. 家長 Line 群組潛規則

幾乎所有園所都有家長群組,有大量隱性規則,新手容易踩雷:

  • 發訊時段:多數園所默契是晚上 21:00 後不發(老師休息),早上 7:00 前也避免
  • 群組不適合討論個別孩子問題:個別問題走聯絡簿或老師指定管道(各園不同 — 入園第一週問清楚老師偏好,有的歡迎私訊有的希望走聯絡簿)
  • 不要在群組批評園所:有意見直接跟園長談,群組會被截圖
  • 老師不一定每條都回:不是不重視你,是孩子在身邊脫不開
  • 群組成員確認:有時候所有老師在群組,有時候只有主帶老師 — 影響你發訊措辭
  • 不要轉發外部訊息(廣告、政治、宗教)
  • 家長之間互加:有的家長會建立家長私群討論園所狀況(沒老師),要不要加自己決定

🎁 4. 節慶禮 · 隱性壓力

沒有強制,但有社會壓力。不送被視為冷漠;送太多被視為攀比。中庸最安全。

常見場合 + 行情

場合頻率慣例(每位老師)
教師節(9/28)1 次/年200-500 元小禮 / 卡片 / 全班合資購禮卡
學期末(1 月、6 月)2 次/年同上,常合資
母親節 / 父親節不一定多為孩子手作小卡,家長無壓力
端午 / 中秋不一定應景小禮(粽子/月餅),合資

🚨 公幼特別注意 · 公務員廉政規範

公立幼兒園教保員(含約聘僱)依法令從事公務,適用《公務員廉政倫理規範》:

  • 第 2 條定義「正常社交禮俗標準」:市價 ≤ 3,000 元(同年度同來源累計 ≤ 10,000 元)
  • 第 4 條但書第(三)款:對職務利害關係者,偶發餽贈 ≤ 500 元(或對機關內多數人合計 ≤ 1,000 元)為例外得受贈

家長作為「服務使用者」屬職務利害關係者,單次禮金 500 元是務實上限;現金 / 禮卡仍可能造成老師困擾。公幼建議改送班級共用耗材(繪本、美勞材料),或一起辦班級活動,比個人禮金更合規。私立 / 準公共另依各園工作倫理守則,仍建議低調合資不送現金。

🏠 5. 入園前家訪

部分非營利園所跟口碑私立會主動安排「老師到府家訪」(多在申請入園後、正式入園前)。

家訪做什麼

  • 了解孩子家庭環境(房間、玩具、家庭結構)
  • 看孩子在熟悉環境的反應
  • 家長補充孩子個性、習慣、過敏、特殊需求
  • 建立家長-老師關係(以後溝通有基礎)

判斷

  • 有家訪 = 加分(代表園所重視個別化)
  • 沒家訪也不一定差,但主動詢問是否能安排是合理的
  • 家訪不需要特別布置,孩子日常空間就好

📅 6. 家長日 / 親子日的時段現實

多數園所辦在平日下午或週六上午。雙薪 / 三明治世代家庭幾乎都無法全勤。

選園前要問

  • 家長日 / 親子日多久一次?(每月 / 每季 / 每學期?)
  • 幾點?週幾?
  • 如果有一次無法參加,有沒有補看的方式?(影片回放 / 個別會談)
  • 能不能補上?(影片回放 / 個別會談)
  • 三明治世代:長輩可以代替家長出席嗎?

🤒 7. 病假備援計畫(三明治世代特別)

入園 3-6 個月孩子會頻繁生病(免疫系統洗禮)。這段時間家長必須有「孩子發燒立刻接」的彈性,但雙薪/三明治世代難以全部自己扛。先準備備援名單。

備援選項

  1. 配偶輪班:當天輪流請假
  2. 祖父母 / 親屬:住附近能接的
  3. 居家臨托保母:各縣市社家署的「非預期性緊急托育」,時薪 100-250(政府補助每小時 50-200)。務必透過縣市平台媒合已登記的保母 — 私下找未登記保母屬「無照托育」(保母方違規,但家長不違法),且無補助、無保障
  4. 朋友 / 鄰居互助:平時打好關係
  5. 育嬰留停半天 / 全天:預先規劃好可用天數
  6. 家庭照顧者支持服務:撥 0800-507272(非緊急狀況用,接線員協助轉介,等待時間有時較長)
    急性危機:1925 安心專線(自己心理崩潰)/ 1966 長照(長輩照顧斷檔)

「不送托」的閾值

  • 發燒 38°C 以上 → 不送(各園規定)
  • 急性腹瀉(24 小時內 3 次以上)→ 不送
  • 有水痘 / 腸病毒 / 流感 → 強制隔離,看症狀緩解 + 退燒 24 小時
  • 感冒(無發燒)→ 部分園所收,部分不收,事先問清楚
  • 腸病毒:孩子確診須隔離 5-7 天(到水泡結痂、退燒 24 小時)
  • 流感:確診須隔離至退燒滿 24 小時(通常 3-5 天)
  • 水痘:全部水痘結痂(約 7 天)才可返園
  • 接觸確診者(腸病毒/流感)但本人無症狀:各園規定不同,有的要隔離 3-5 天觀察,有的不管 — 務必問清楚

🎓 8. 大班升小一 · 銜接準備指南

5-6 歲家長最焦慮:注音要不要先學?認字量要多少?寫字要練嗎?生活自理基準在哪?

注音 / 認字 / 寫字

  • 注音:教育部明確「小一前 10 週才正式教,不需提前學」。研究顯示提前死記反而干擾。孩子有興趣 → 用遊戲方式接觸 OK;沒興趣 → 完全不用學
  • 認字:能認自己名字 + 家人名字 + 常見字(出口/入口/廁所/危險)即可
  • 寫字:完全不用練字。手部小肌肉還在發育,過早握筆姿勢不正確很難改
  • 數字:1-20 認得 + 簡單加減 OK,不用會 99 乘法表

生活自理基準(上小學前該會)

項目應達程度
自己穿脫鞋襪5 分鐘內完成(建議先換成魔鬼氈款式,鞋帶式對 5 歲孩子過難)
自己上廁所包含擦屁股、沖水、洗手

自己背書包知道書包要背好,會收
找廁所 / 飲水機新環境能主動找
表達需求會跟老師說「我想上廁所」「我不舒服」
跟陌生小朋友互動能主動加入或被拒絕後不崩潰
等待 / 排隊能等 5-10 分鐘不抓狂
專注 15-20 分鐘能坐著聽故事 / 完成一個活動

這些比注音重要 100 倍。生活自理是小一適應的關鍵,知識零基礎沒關係,生活白癡才是真正吃力。

該不該送注音先修班

  • 不需要(教育部立場)
  • 如果你還是想送 → 選「遊戲式」「不寫字」的;避開「填鴨式」「考試式」
  • 把「先修班學費」存起來,買繪本 / 戶外活動 / 跟朋友玩反而對小一適應更有幫助

🔄 9. 留園直升 vs 換園評估

留園直升

  • 各縣市時間不同,通常 1-2 月辦理
  • 錯過 = 失去直升資格,要重新抽籤(可能抽不到原園)
  • 記在 Google 行事曆 + 學期初提醒

什麼時候考慮換園

  • 孩子持續抗拒(超過 6 個月仍每天哭)
  • 跟特定老師合不來且無法調班
  • 園所師資大量流動(影響穩定性)
  • 搬家(動線改變)
  • 學費調漲不合理
  • 孩子被霸凌或長期被忽略

換園的成本

  • 新園適應期重來(再 1-3 個月)
  • 老朋友斷層
  • 家長作息又要重新校正
  • 新老師對孩子是陌生人,建立信任要 1-3 個月 — 過敏 / 習慣 / 情緒觸發點全要從零傳遞,孩子個人檔案要重建
  • 先評估是「老師問題」還是「園所問題」 — 老師問題可能下學期就解決
  • 霸凌情況:先用通報機制,讓園所處置(問清楚 SOP) — 換園是最後手段,不是第一反應

👩‍🏫 11. 老師流動率影響你孩子的穩定 · 教保員勞動現實

🎯 你希望孩子穩定 → 那要看老師流動率 → 流動率是這樣形成的

家長端的小事,加總起來決定教保員流動率,而流動率直接影響你孩子的穩定。下面講教保員真實工作條件,不是要家長同情,而是讓你看到「為什麼好老師難留」,然後做出能讓他留下來的小事。

真實工作條件(2025-2026 雙北)

  • 實領薪資:助理教保員 28,000-32,000 / 正式教保員 32,000-38,000(扣勞健保後)。準公共老師 114 年起政府規定底薪 34,200(任職未滿 3 年)/ 37,200(滿 3 年)/ 40,200(滿 6 年)
    (來源:雙北人力銀行 2025 行情 + 教育部準公共化政策公告,以各縣市最新為準)
  • 勞健保保護缺口:私立 / 非準公共部分機構規避雇主負擔,以「自行投保」「兼職」掛名;選園時可問:你們教保員勞健保由園所負擔嗎?
  • 輪班與午休:多數公托一班兩位輪早晚班(7:00-15:00 / 10:30-18:30),午休名義有但常用來交班、填記錄、備餐
  • 體力:0-2 歲一個班平均每天「抱起放下」累積 300+ 次,下背傷害是職業病。職業傷害補償申請困難,多數老師自己吞
  • 年終 / 考績:私立和準公共多數無年終;公托有公務員年終但教保員常被列「約聘」排除
  • 離職主因排序:① 家長關係壓力(群組截圖、投訴) → ② 體力負擔 → ③ 薪水 → ④ 升遷有限

家長能做的(讓好老師願意留下來)

  • 用上面「家長端配合 Checklist」的 13 條
  • 群組感謝要點名(讓老師被肯定)
  • 不要傳截圖到第三方
  • 了解「老師沒立即回訊」是因為手上有 5 個孩子,不是態度問題
  • 節慶禮(若送)寫個小卡 + 班級耗材,比現金禮卡讓老師更有溫度

💔 12. 家長最常後悔的決策(園長端真實觀察)

選園後最常聽到的回頭話 — 提前讀過,可避開 90%。

  1. 選了近的,不是合的 — 用步行距離 / 費用篩選,入園才發現孩子跟老師個性不合,但因轉園成本高(等候 / 適應)只能硬撐。孩子氣質 vs 教學風格的適配性比距離重要。看 🏫 幼兒園專區 教學風格章節
  2. 聽媽媽群推薦,沒自己觀察 — 口碑是滯後指標。兩年前口碑最好的園所,主帶老師可能已換過一輪。問口碑時,問的是「最近一年在讀家長」的感受,不是三年前
  3. 沒問清楚延托規則,臨時才發現不能配合 — 選園優先看費用環境師資,延托等到入園後才問,才發現給不了你需要的彈性。選園 Top 10 必問就有「延托規則」這條,務必問
  4. 抽到了就去,沒有停下來想「這家對不對」 — 抽到的沉沒成本心理 → 抽到後不再評估適合度。抽到不代表一定要去。去觀察後覺得不合,放棄才是對孩子負責(讓備取家庭遞補)
  5. 急著做決定,沒等試讀 — 為了「卡位」直接報到,進去才發現不適合。有提供試讀務必試,試讀對 vs 不對的感覺很真
  6. 把園所「漂亮裝潢」當品質指標 — 漂亮地板 / 大遊戲區 ≠ 好教學。老師眼神 + 孩子離園神情比裝潢可信
  7. 太相信「招牌」蒙氏 / 華德福 — 這些名稱沒法律保護,任何園所都能掛。問實際做法:教具齊全嗎?老師認證嗎?混齡比例?(看幼兒園專區)
  8. 低估「孩子常生病」對家長作息的衝擊 — 入園 3-6 月免疫洗禮期,雙薪家庭常崩潰。先準備備援名單(看入園後實務 § 7)
  9. 「觀課老師 ≠ 帶班老師」 — 觀課時看到的常是園長 / 主任 / 資深老師,入園後孩子跟的可能是剛進來的助理員。問清楚「我家孩子的主帶老師會是哪位?她之前帶幾班?」
  10. 選園前沒確認「放學後誰接」 — 多數園所 17:30 接,雙薪家庭根本接不到。發現時開始找安親 / 求公婆,衍生家庭衝突。選園前先確認接送方案
  11. 大班下學期才找安親 / 小一課輔,通常已太晚 — 口碑好的安親班大班上學期就排,甚至更早。大班下學期去問,好位子多半已滿
  12. 選了特色園(蒙氏 / 雙語)但沒評估孩子氣質 — 花 25,000 / 月送蒙氏,才發現孩子需要老師明確指令,自由選工作就在教室遊蕩;送雙語但孩子母語都還沒穩
  13. 觀課時忽略孩子的第一反應 — 孩子一進教室就不想留 / 想走,家長卻因裝潢 / 師資 / 費用說服自己「試了會習慣」。孩子在現場的反應是可靠訊號,不是參考意見

  • 2 歲班有名額
  • 非營利優先(費用最低 + 統一報名 + 備取保留期最長)
  • 準公共補位(費用相同,各園自辦)

類型數量備註

  2. 步行 20 分內的 3 家(肯恩、樹德、天翔)優先實地參觀
  3. 5 月初地獄週:同步處理非營利 + 常春藤 + 天翔(報名期重疊)
  4. 5 月中:跑廣福取報名表(5/11 後才開放索取)

  • 「未超額不抽籤」要看清楚:常春藤 PDF 有設 5/8 抽籤日,只是「未超額才全錄」— 不是完全不抽。意思是即使你以為一定上,也要等到 5/8 才知道

  • 「正取 X 人」要看是哪個班:詠樂寫「正取 10 人」是全園正取數(2 歲班 8 + 3 歲班 2),不是 2 歲班 10 人

  • 多家報名要做時序表:同一週可能跑 4-5 家,沒排表會錯過

👶 托嬰專區(0-2 歲)

托嬰跟幼兒園邏輯完全不同。0-2 歲的關鍵是「替家裡分擔基本照顧」,不是教育。重點看老師怎麼帶、怎麼餵、怎麼管。

📌 跟幼兒園(2-6 歲)的根本差異

  • 師生比 1:5(法規)— 比幼兒園嚴格(2-3 歲是 1:8)
  • 不教東西 — 重點是吃飽、睡好、不哭、安全。發展刺激是附加
  • 每日狀態回報 — 大便幾次、睡幾小時、喝幾 cc 全要記
  • 家長端配合多 — 副食品要家裡先試吃、午睡作息要對齊
  • 送托心理門檻高 — 第一次送 0-2 歲幼兒,新手家長不適應期 1-3 個月

🍼 1. 帶小孩方法論(教保理念)

每家托嬰風格差很多。問下面這些問題,聽實際做法不是聽口號。

哭聲處置

  • 立刻抱起(回應派):0-1 歲哭通常是需求未滿足,先處理生理
  • 觀察 30 秒(混合派):非生理哭(無聊/想抱)讓他學自我安撫
  • 哭聲控制法(訓練派):制定哭多久才回應的時間表

沒有絕對對錯,但要跟家裡作法一致。家裡立刻抱、托嬰讓他哭,孩子會混亂。

副食品引入

  • 4-6 個月開始(主流):米精 → 米糊 → 蔬菜泥 → 水果泥 → 蛋黃 → 肉泥
  • BLW(寶寶主導離乳):6 個月直接吃手指食物,讓寶寶自己抓
  • 禁忌:1 歲前不給蜂蜜(肉毒桿菌)、不給未消毒鮮奶

睡眠訓練

  • 不訓練(自然派):隨需求餵 / 抱睡 / 自然睡
  • 溫和訓練(費伯法 / 抱起放下法):漸進式延長獨處時間
  • 哭出來訓練:讓寶寶自己學會入睡(爭議大)

體罰 / 罰站

  • 0-2 歲禁止任何體罰(法規 + 倫理),這條沒得談
  • 看老師怎麼處理「打人」「咬人」「搶玩具」的孩子,有沒有 SOP

面試問題:「孩子哭超過 30 分鐘,老師怎麼處理?」「孩子打另一個孩子,你會怎麼做?」回答得越具體越可信,越含糊越要警戒。

📹 2. 監視器 / 透明度

0-2 歲不會說話,家長無法從孩子口中知道一天發生什麼。監視器是最低保障。

📌 監視器是「透明度工具」不是「監視老師工具」

家長隨時看監視器,任何老師非語言動作都可能被截圖、斷章取義(例如讓孩子哭幾秒才去抱是觀察期、在旁做記錄沒立刻蹲下)。疑問先跟老師說、不要截圖傳群組 — 否則好老師會因壓力離職,你的孩子受害。

偏鄉小型公托(社區公共托育家園)許多沒有全室監視器,但家長可隨時到訪參觀 + 出入口監視 也是替代標準,不必只用都市標準否定資源有限的公立機構。

方案透明度備註
無監視器❌ 零放棄這家
只有出入口監視⚠️ 低看不到教室內,不夠
教室全區監視 + 法規保存 30 天🟡 標準有狀況可調閱(但要寫申請)
家長可遠端查看(手機 App)🟢 高較罕見,有的話加分
每日推照片 / 影片給家長🟢 高主動推送,看得到孩子實況

要問的問題

  • 監視器涵蓋哪些區域?(教室 / 寢室 / 用餐區 / 廁所內 — 廁所內通常沒有,但廁所外要有)
  • 錄影保存幾天?有狀況怎麼調閱?
  • 家長能不能主動看?還是只在出事才能看?
  • 有家長手機 App 嗎?照片頻率多高?

🏠 3. 空間大小

0-2 歲多數時間在地板上爬、玩、睡。空間夠不夠很關鍵。

  • 法規:每幼兒室內活動空間至少 2 平方公尺(0.6 坪)
  • 實際扣掉:嬰兒床 + 圍欄 + 教具櫃 + 桌椅後,真正能爬能玩的空間是多少?
  • 戶外活動空間:有自己的戶外?還是要去公園?雨天怎麼活動?
  • 分齡分區:0-1 歲(不會走)和 1-2 歲(會走會爬)有沒有分開,還是混在一起?(混區風險:大寶寶不小心踩到小寶寶)
  • 午睡區:每個孩子有專屬嬰兒床嗎?床距夠不夠(預防交叉傳染)?

🍱 4. 餐點 / 飲食

0-1 歲

  • 配方奶:用哪個牌子?家長能不能指定品牌?
  • 配方奶溫度 / 沖泡比例:園所有 SOP 嗎?
  • 瓶餵方式:抱著餵還是放著餵?(影響孩子安全感 + 中耳炎風險)
  • 母乳冷藏:家長送母乳到園,有沒有冰箱?如何分裝、加熱?

1-2 歲(副食品)

  • 菜單透明度:有沒有公開週菜單?食材來源?
  • 過敏處理:家裡有過敏紀錄,園所怎麼處理(完全不給 vs 替換餐)?
  • 家長端配合:新食材要不要家裡先試 3 天?
  • 餵食方式:幾點餵 / 餵多少 / 不吃怎麼處理(強迫 vs 換時段 vs 跳過)

共通

  • 餐具消毒:每個孩子專屬餐具?每次消毒方式?
  • 飲水:1 歲後可以喝水,水從哪來、水溫多高?
  • 輔食器具:果汁機 / 食物剪 / 食物剪刀清洗頻率?

👩‍🏫 5. 老師師資(0-2 歲特殊)

0-2 歲老師的證照種類比幼兒園更多元,要看清楚。

證照意義適合年齡
保母證照(技術士)主要訓練嬰幼兒生活照顧0-2 歲(主流)

教保員 + 嬰幼兒組大學幼教加修 0-2 歲0-6 歲

要問的問題

  • 0-1 歲班的老師有幾個是有保母證的?
  • 有沒有護理師駐園?(發燒、跌倒、餵藥誰負責)
  • 老師平均待多久?(高流動率對 0-2 歲影響特別大,孩子需要建立依附)
  • 主帶老師(primary caregiver)制度?同一個孩子是否由同一位老師主帶,讓孩子能建立穩定關係

⏰ 6. 延托(extended care)

標準時間:7:30-17:30。超過要延托。雙薪家庭幾乎一定會用。

要問清楚

  • 延托到幾點?(18:30 / 19:00 / 19:30 — 各家不同)
  • 計費方式(2025-2026 雙北行情):公幼 / 準公共 100-250 元/天;私立 200-400 元/天;月票公幼 2,000-3,500 / 私立 3,500-4,500;逾時罰每 15 分鐘 100-500 元(私立偏高,有的甚至「超時即追加一日延托費」)
  • 單次 vs 月租:偶爾用 vs 每天用,哪個划算
  • 提前通知時間:當天打電話算晚還是 OK?
  • 晚餐:延托時間有沒有提供餐點(有的有有的沒)
  • 緊急狀況:你 18:00 突然開會,能不能臨時延?
  • 逾時罰款:超過延托時間還沒接,每分鐘怎麼算(很多園 5 分鐘就罰)

三明治世代特別注意:長輩臨時送醫的話,延托是救命用。問清楚「臨時延托」的彈性。如果園所說「一律不能臨時延」,你家可能不適合這家。

📅 7. 活動管理 / 一天的時間表

0-1 歲(沒結構日,只有作息)

0-1 歲不需要「課程」,需要的是穩定作息。

  • 早上:到園 → 玩 → 喝奶 → 小睡
  • 中午:午餐 → 午睡(2-3 小時)
  • 下午:醒 → 喝奶 → 玩 → 點心 → 接送

「玩」是觀察 / 抓握 / 跟老師互動,不是上課。如果園所說「我們有教 0-1 歲認字」, 那是行銷話術。

1-2 歲(開始有結構)

  • 9:30-10:00 主題活動(15-20 分鐘上限,1-2 歲注意力短)
  • 10:00-10:30 戶外或大肢體
  • 10:30-11:00 點心 + 自由玩
  • 11:00-11:30 午餐
  • 11:30-14:30 午睡
  • 下午類似上午,但少一個主題活動

要問的問題

  • 戶外活動多久一次?雨天替代方案?
  • 主題活動怎麼設計?是否每月有不同主題?
  • 音樂 / 律動 / 美勞頻率?
  • 蒙氏教具 / 感統教具有嗎?用在什麼時段?
  • 電子設備:1-2 歲會看影片嗎?(有的家長很在意)

🤝 8. 活動配合方式 · 家長端的功課

0-2 歲送托後家長還要做很多。了解清楚再簽約,不然變成雙方互怨。

每天

  • 聯絡簿:紙本還是 App?家長要不要每天回?(有的園所要)
  • 用品補充:尿布、濕紙巾、口水巾、奶粉、奶瓶 — 用完要立刻補
  • 衣物備品:換洗衣 2-3 套,弄髒要立刻補
  • 奶粉 / 母乳:每天送新鮮的還是 1 週送一批

每週 / 每月

  • 家庭日 / 親子活動:多久一次?強制參加嗎?
  • 新食材試吃:正規做法是「家裡先試 3 天觀察過敏」,但實務上嚴格執行的園所不到 30%(尤其準公共/私立)。多數做法是家長填過敏記錄表 + 新食材通知 Line 群 + 有反應立刻聯繫。如果園所沒要求,家長主動在家先引入並告知結果比較實際,不要拿這條去質問
  • 親師會:多久一次?重要時段
  • 體適能 / 健康檢查:園方主辦還是家長帶去?

突發

  • 生病請假:發燒幾度要接回?(通常 38°C)
  • 傳染病停課:腸病毒 / 流感 — 全班停或個案隔離?
  • 送醫:園方先送哪家醫院?家長到場時間預期?

👶 9. 0-3 個月嬰兒送托特殊性

法規 vs 實務有大落差。法規寫滿 2 個月可送托嬰,但實務上熱門托嬰中心 / 保母通常要求 4-6 個月才收(因為小月齡照顧密度高,風險高)。打電話前先問清楚最低收托月齡,不要查表了才白跑。

3 個月以下送托要注意

  • 找有「0-6 個月新生兒班」的托嬰 — 跟 1-2 歲幼兒混班風險高(被踩到、被搶玩具)
  • 感染風險特別高 — 0-3 個月免疫系統弱,送托後常生病的比例 > 1 歲後送托
  • 母乳家庭規劃:
    • 母乳冷藏保存標準:冷藏 48 小時 / 冷凍 3 個月
    • 確認托嬰有專屬冰箱 + 加熱器(隔水加熱不能微波)
    • 親餵媽媽:詢問中午能否到園親餵?有沒有哺乳室?
  • 配方奶轉換期 — 媽媽親餵 → 進托嬰瓶餵,孩子常拒奶 1-2 週。送托前 2-3 週在家先練習瓶餵,讓孩子接受不同餵食方式
  • 分齡照顧確認:
    • 0-1 歲(不會走)和 1-2 歲(會走會爬)是否分區?
    • 嬰兒床 vs 學步區是否完全隔離?
    • 午睡是否有獨立空間?

💸 10. 0-2 歲家庭月隱形成本全景

送托費 + 補助以外,0-2 歲家庭每月還有以下必要支出(預算規劃請計入):

📌 11. 0-2 歲送托心理建設

第一個月會崩潰,這是正常的:

  • 孩子哭著去 / 哭著回(分離焦慮高峰)
  • 常生病(免疫系統碰到群體病菌的洗禮,3-6 個月才穩)
  • 作息打亂、夜醒增加
  • 家長自責、懷疑送托是不是錯誤

這些大部分會過去。如果 3 個月後還是無法適應 → 評估是園所不合還是孩子真的不適合送托(極少數)。

🏫 幼兒園專區(2-6 歲)

2-6 歲教育階段,跟 0-2 歲托嬰邏輯完全不同。重點不再是「替家裡分擔基本照顧」,而是「孩子第一次進入有結構的群體學習環境」。教學風格、班級文化、老師個性,影響可能延續到小學。

⚠️ 招牌不等於實質 + 假冒辨識

「蒙特梭利」「華德福」「瑞吉歐」這些名稱未設認證資格管制,任何園所都能掛。問實際做法:

  • 蒙氏(最難假冒):教具齊全?(蒙氏教具一套 5-20 萬)老師 AMI / AMS 認證(可上 AMI 官網查)?混齡比例?
  • 華德福:教室無電子產品?天然教具?節慶儀式?
  • 瑞吉歐(最容易假冒):核心是「精神」非可驗證硬指標,任何有「藝術中心」就自稱。問:作品紀錄系統怎麼做?能看前 3 個月孩子作品紀錄嗎?
  • 主題式(品質差距大):問「能看前 3 個月主題紀錄?活動怎麼設計?」 真主題式整月圍繞一脈絡深挖(例如「水」從浮力到狀態變化),假主題式只換月曆封面
  • 遊戲式 / 開放教育(難辨真假):「自由遊戲」vs「放著沒管」外觀近似。看老師在旁做什麼:搭鷹架?還是滑手機?
  • 雙語 / 全美語:每天中文活動幾小時?中文教師是誰?問:外師是 ESL 認證還是只持簽證?(法規:不得全日外語)

🚨 招牌話術:「歡迎隨時參觀」陷阱

最常見話術。實際執行差很多:

  • 真有信心的園所:允許「預約後隨機時段」觀課,你想哪天就哪天
  • 有問題的園所:只能「固定時段」(通常是有準備的活動展示)、「只能在門口看」、或「先打電話確認當日適不適合」
  • 識破方法:打電話問「我下週三上午 10 點可以去看嗎?」如果對方推到「下週五下午才有開放日」 — 就是固定時段話術

⚠️ 招牌話術:「我們老師很有愛」確認偏誤

家長看了一次老師表現很好,就認定這是日常。但「觀課日的老師」跟「週三下午平日的老師」狀態完全不同。問近期(最近 1 年)在讀家長感受(可請園所介紹幾位家長),不要只信你去那天的印象。

📋 2. 觀課 / 面試完整 SOP

觀課前準備

  • 提前預約:電話/網路問可否觀課,通常 1 週前約
  • 選一個「平日上午」:這時段最能看真實狀況(下午通常午睡,週末沒人)
  • 最好帶孩子一起去:看孩子在現場的反應(有沒有想久留 / 立刻想走)
  • 準備問題清單:看 📋 看園所 Checklist 「現場必問 Top 10」

觀課當天看什麼

  1. 進門第一印象:有沒有怪味(消毒水過重 / 食物異味 / 霉味)?孩子的鞋子整齊嗎?
  2. 老師眼神:跟孩子互動時是「眼睛在孩子身上」還是「在看天花板等下班」?
  3. 孩子的狀態:活潑 vs 呆滯?有沒有孩子坐在角落沒人管?哭的孩子有沒有人去處理?
  4. 師生比實況:當下教室真的幾個老師幾個孩子?問:「今天請假幾位老師?」
  5. 教室空間:擁擠嗎?有沒有「危險角落」(尖銳 / 開放電源 / 易倒物品)?
  6. 廁所:乾淨嗎?2-3 歲幼幼班廁所應在教室內(法規)
  7. 廚房(邀請式問法):「可以帶我大概走走廚房旁邊嗎,想感受一下環境?」注意:廚房非法定家長視察空間,園方可婉拒。可改問「廚房最近衛生局稽查結果如何?」並查衛生局網站
  8. 戶外活動空間:有自己的還是要去公園?雨天替代方案?
  9. 離開時孩子的樣子:剛來的孩子哭著要走 vs 離不開,差很大

🎓 6. 大班升小一銜接(深化版)

5-6 歲家長最焦慮的議題。除了「🏡 入園後實務 § 8 大班升小一」的 9 項生活自理,加碼回答家長最在意的 5 個問題:

Q1 · 注音真的不用先學嗎?

  • 教育部立場:小一前 10 週才正式教,提前死記反而干擾
  • 實務:孩子主動有興趣 → 用遊戲方式接觸 OK;沒興趣 → 完全不用學
  • 大班該做的:多看故事書、認自己名字、能聽懂老師指令

Q2 · 認字量要多少才夠?

  • 沒有官方標準。能認得「自己 / 家人名字 + 出口 / 入口 / 廁所 / 危險」常見字即可
  • 不要強迫認字:5-6 歲手部小肌肉還在發育,過早握筆姿勢不正確很難改
  • 替代做法:大量繪本親子共讀,孩子自然會在書本中認字

Q3 · 數學要會什麼?

  • 1-20 認得 + 基本加減(透過實物、玩具、糖果)即可
  • 不需要會 99 乘法表、不需要寫算式
  • 實用替代:逛市場讓孩子算錢、玩桌遊培養數量概念

Q4 · 該不該送注音 / 學科先修班?

  • 教育部明確「不需要」。研究顯示提前死記反而在小一時造成干擾
  • 如果一定要送 → 選「遊戲式」「不寫字」的;避開「填鴨式」「考試式」
  • 先修班費用行情(雙北 2025-2026):
    • 暑期短期遊戲式 4-8 週:3,000-5,000 元(郊區小型班)
    • 學期制注音/數學:8,000-15,000 / 期(主流補習班)
    • 外師雙語先修:15,000-20,000+ / 期
  • 替代方案:把先修班學費換成繪本(40 本) / 戶外活動 / 跟朋友玩 — 對小一適應更有幫助
  • 家長壓力:長輩 / 同儕施壓送先修班時,可用「教育部立場」回應

Q5 · 學區小學選擇怎麼跟幼兒園搭配?

  • 戶籍學區:大班下學期(隔年 4-5 月)就要確認學區小學 + 跨區就學流程
  • 放學時間衝擊:小一通常 12:30-13:30 放學,比幼兒園早接 4 小時 — 雙薪家庭要規劃安親 / 課輔
  • 銜接班 vs 安親班 vs 自學:
    • 銜接班(暑假 4-6 週):有的好有的廢,看師資
    • 安親班(平日下午到 18:00):接送 + 寫作業督促。一般型 5,000-10,000 / 月;英語/才藝型 12,000-20,000 / 月。品質差異極大,要去看,不要只看費用
    • 自學(家長自己帶):需要長輩 / 一方家長能配合,免費但體力負擔大
  • 大班暑假怎麼過:6/30 畢業 → 8/30 開學,中間 2 個月空窗;有的園所「續托」到 8 月,有的家長另找暑期營

📌 幼兒園 vs 安親班 vs 才藝班 怎麼搭

大班 / 小一前後家長常問:

  • 幼兒園 + 才藝:園內才藝(美術 / 音樂 / 體能)通常足夠,不用另送外面
  • 幼兒園 + 安親(大班下學期試用):讓孩子提前適應「下午有結構式時間」
  • 純自由玩:大班下學期已是「準備上小學」階段,純放鬆不練專注力上小一可能跌跤

💡 找幼兒園「比找托嬰更難」的理由

  1. 影響時間長:托嬰最多 1-2 年,幼兒園 4 年 — 選錯成本高 4 倍
  2. 教學風格差異大:托嬰主要是照顧,幼兒園是教育 + 文化,流派之間差很多
  3. 家長對「學到什麼」有期待:托嬰可以無感,幼兒園無感家長會焦慮
  4. 孩子有意見了:2-6 歲孩子會表達喜歡 / 不喜歡,要納入考量
  5. 「銜接小一」隱形壓力:大班開始就要想下一站,很多家長從這時開始焦慮

結論:找幼兒園不要急,多參觀 3-5 家、各家觀課、最好試讀。比找托嬰多投入 2 倍的時間都值得。

👤 1. 保母 4 種類型

不只「來家裡的保母」一種。台灣現行有 4 種正式分類:

① 在宅居家式托育人員(主流,90% 用這種)

  • 定義:在自己家中收托兒童的登記保母
  • 登記要求:依法須向居家托育服務中心辦理登記,取得「居家式托育服務提供者」資格
  • 地點:保母自宅
  • 能否簽準公共:能 — 簽約後家長可領 13,000 元/月補助

② 到府保母 / 駐家保母

  • 定義:保母來你家照顧
  • 登記要求:⚠️ 法規灰色地帶 — 「居家式托育服務提供者登記及管理辦法」是否適用未完全明確,多數以個人勞務合約執業
  • 能否簽準公共:通常不能(政府補助以「送托」為主),家長只能領育兒津貼 5,000
  • 費用:最高,雙北月費 27,000-46,000

③ 臨時托育保母(臨托)

  • 定義:獨立登記的臨托保母,提供「非預期性緊急托育」(家長生病、就醫、突發狀況)
  • 計費:時薪 100-250 元(政府補助每小時 50-200 元),家長實付通常 50-150 元/小時
  • 找法:各縣市居家托育服務中心均有提供臨托媒合 → 不要私下找未登記保母(屬無照托育,無補助無保障)
  • 適用情境:三明治世代家庭長輩臨時送醫、家長出差、孩子突然病退托後緊急照顧

④ 親屬保母(三親等內)

  • 定義:祖父母、叔伯姑舅等三親等內親屬照顧
  • 補助:主要領「育兒津貼」5,000-7,000(跟有無證照無關)
  • 台北市加碼:有結業證書/技術士證者,協力照顧補助 加 2,000-3,000
  • 適用情境:不想送外人 + 有可信任家族成員

🔍 7. 找保母的管道

🚨 灰色市場警示 · 沒登記的保母

台灣有大量「有帶孩子但沒向居家托育服務中心登記」的非正式保母,透過 FB 社團、里辦公室公告、親友轉介流通。這些保母:

  • 不在政府訪視系統 — 出事難追責
  • 無法簽準公共 — 家長無補助,只能領育兒津貼 5,000
  • 無強制保險 — 出意外無保障

判斷有沒有合法登記的方法:請保母提供「居家式托育服務提供者登記字號」→ 到 衛福部托育媒合平台 用名字或地址查 → 查不到 = 未登記。

官方(免費,推薦)

  • 各縣市居家托育服務中心(全國 22 縣市):媒合 + 訪視 + 輔導一條龍
  • 查詢方式:撥 1957 福利專線 / 各縣市社會局網
  • 衛福部托育媒合平台:ncwisweb.sfaa.gov.tw
  • 新北市育兒資訊網:lovebaby.sw.ntpc.gov.tw
  • 台北市:dosw.gov.taipei

民間平台

  • 波比保母 PoBi、Bananny 托育小幫手、OurKids 育兒樂

媒合流程(透過官方中心)

  1. 聯繫居家托育服務中心 — 說明區域、時段、起托日
  2. 中心媒合保母名單 — 自行聯繫面談
  3. 面談 + 試托(通常 1-3 天)— 確認彼此適合
  4. 簽訂托育服務契約(衛福部有定型化契約範本)

🚨 8. 風險點 · 家長要正視

  • 單人作業沒同事監督 — 異常行為無人即時發現(機構式有 2 人以上互相 check)
  • 監視器普及率低 — 多數保母家未裝。但願意配合家長加裝(育兒空間如客廳/餵奶區)的保母,透明度反而比機構更高(機構監視器要申請才能調閱,家長手機可即時看)。監視器是可協商的條件,面談時主動提
  • 保母生病 / 家事突發 → 直接停托 — 沒有人員替補機制,當日可能突然無人可托(對職場影響大)
  • 品質難外部評估 — 沒有同梯爸媽口耳相傳,主要靠面談印象 + 訪視報告
  • 家庭隱私外漏(主要適用到府保母) — 到府保母知悉家庭作息、財物、家庭關係
  • 在宅家庭環境盲區(主要適用在宅保母) — 孩子在保母家 8-10 小時,保母家庭的同住者(配偶 / 公婆吸菸 / 訪客)、環境衛生,家長無法即時掌握 — 試托時要看仔細

確認品質的方法

  • 透過居家托育服務中心查保母的訪視紀錄 + 評鑑結果
  • 要求查看環境安全檢核表(登記申請時即填報)
  • 詢問是否願意裝監視器並開放遠端查看
  • 打聽前任家長評價(可請中心協助)
  • 觀察試托期間保母與孩子互動細節
  • 確認保母是否加入居家托育服務中心(有加入者定期接受訪視)

📋 9. 面談必問清單

  • 有托育人員技術士證嗎?(要看實體證照)
  • 有跟政府簽準公共嗎?(影響補助金額)
  • 同時帶幾個孩子?各幾歲?(法定上限 4 人 / 0-2 歲限 2 人)
  • 家裡有誰?(配偶 / 公婆 / 自己孩子)— 出入頻率影響孩子
  • 有寵物嗎?(過敏家長要問)
  • 抽煙嗎?(直接 NG)
  • 典型一天的時間表?(看作息穩不穩)
  • 哭鬧處理方式?
  • 遇到孩子打人 / 咬人怎麼處理?
  • 過敏 / 特殊飲食配合?
  • 請假 / 休假有多少?替手是誰?
  • 緊急狀況送哪家醫院?
  • 願意給家長看 LINE 群 / 推照片?頻率?
  • 能不能裝 / 開監視器?
  • 跟之前家長有沒有出過糾紛?(問了她敢不敢答,反映誠實度)

💰 補助系統

育兒津貼 vs 托育補助 vs 幼兒園補助 — 三選一,不可並領。很多家長搞錯,選錯每月差 8,000。

🚨 最關鍵的一條:互斥規則

育兒津貼、托育補助、幼兒園補助:三選一,只能領一種。不能同時領、不能交替月份領。送公托 / 準公共托嬰 / 準公共保母 → 一定選托育補助(高出 8,000+)。自己帶 / 親屬帶 / 一般私托 → 只能領育兒津貼。

🌳 三步驟決策樹 · 我能領哪一種?

  1. 問:孩子有送機構 / 簽約保母嗎?
    ❌ 沒有(自己帶 / 親屬帶 / 一般保母 / 一般私托) → 領 育兒津貼 5,000(就停在這)
    ✅ 有 → 進下一題
  2. 問:是「公托 / 準公共」嗎?
    ❌ 不是(一般私托) → 領 育兒津貼 5,000(就停在這)
    ✅ 公托 → 領 托育補助 7,000 + 縣市加碼
    ✅ 準公共 → 領 托育補助 13,000 + 縣市加碼(進下一題)
  3. 問:孩子幾歲?
    0-未滿 3 歲 → 上述托育補助繼續領
    2-未滿 6 歲送幼兒園 → 改領 幼兒園補助(月費上限 3,000 起)

關鍵:三種補助不可疊加。送公托不能再領育兒津貼;送準公共不能再領育兒津貼。選錯類型 = 每月差 8,000 起(台北市差到 16,000)。

⚠️ 例外:身心障礙 / 發展遲緩 / 原住民幼兒另有特殊補助(早療補助、原民加碼等),可疊加於一般育兒津貼或托育補助之上,規則複雜。請洽 1957 福利專線或各縣市社會局 / 早療資源中心。

🚫 常見誤解

  • ❌ 「我送托嬰 + 也想領育兒津貼」 — 不行,二選一
  • ❌ 「我自己帶,但領托育補助」 — 不行,托育補助限「送機構/簽約保母」
  • ❌ 「我送非準公共私托,但想領 13,000」 — 不行,只能領育兒津貼 5,000
  • ❌ 「育嬰留停跟托育補助互斥」 — 111 年起已取消互斥,可以同時領
  • ✅ 「育嬰留停 + 育兒津貼」可以(只要符合各自資格)
  • ✅ 「育嬰留停 + 托育補助」可以(2024 年新規)

  • 育兒津貼:孩子戶籍地公所申請,不會自動發放(很多家長不知)
