## 完成

### 1. 基礎建設(設定頁 / 規劃模式 / 比較籃)

**設定頁(`page-settings`)**
- 規劃模式 toggle(switch UI)
- 家庭模式 select(一般 / 三明治 / 雙薪 / 單親)
- 匯出 JSON 備份(Blob + download)
- 匯入 JSON(file input + FileReader)
- 清空所有資料(double confirm)
- 版本 / 來源 / 隱私說明

**規劃模式**
- `STATE.planningMode` boolean
- 開啟時首頁顯示「本月該做的事」block
- `MONTHLY_ACTIONS[1..12]` 12 月份對應 actions
  - 1-2 月:留園直升登記、文件整理
  - 3-4 月:預防接種卡確認、4 月底非營利缺額公告
  - 5 月:🔴🔴 地獄週(非營利報名 + 多家準公共同期)
  - 6-7 月:備取遞補、私立補位
  - 8 月:開學
  - 9-12 月:適應、檢討
- 月份過了自動更新(`new Date().getMonth() + 1`)

**比較籃(`page-compare`)**
- `STATE.compareList = [{id, name, type, walk, classSize, fee, reg, lottery, retain, notes}]`
- 表單填一筆 → 加入比較籃
- 列表 view + 移除單筆
- 自動產生對照表(≥ 2 筆才顯示)
- 匯出純文字到剪貼簿(`navigator.clipboard` + fallback `document.execCommand`)
- escapeHtml() 防 XSS
- localStorage 持久化

### 2. 托嬰專區獨立頁(`page-infant`)

8 大 section,涵蓋 B 類 4 個方向 + 額外:
1. 🍼 帶小孩方法論 — 哭聲處置、副食品引入、睡眠訓練、體罰
2. 📹 監視器 / 透明度 — 5 級透明度 + 4 個必問問題
3. 🏠 空間大小 — 法規 vs 實際、分齡分區、午睡區
4. 🍱 餐點 — 0-1 歲(配方奶/母乳)+ 1-2 歲(副食品)+ 共通
5. 👩‍🏫 老師師資 — 5 種證照差異(保母證 / 教保員 / 教師證等)
6. ⏰ 延托 — 7 個必問問題 + 三明治世代專屬警告
7. 📅 活動管理 — 0-1 歲只有作息、1-2 歲開始有結構
8. 🤝 活動配合 — 每天 / 每週 / 突發事件

底部有「送托心理建設」block(第一個月會崩潰是正常的)
