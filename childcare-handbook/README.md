## 🎯 這本手冊在解決什麼問題

第一次幫孩子找托嬰/幼兒園的家長常踩到的坑:

- 🧱 **不知道有哪幾種**(公幼/非營利/準公共/私立 差在哪?)
- 🧱 **不知道何時報名**(各家時程不同,過了就等明年)
- 🧱 **不知道現場該看什麼**(去參觀只看到漂亮裝潢)
- 🧱 **不知道抽籤怎麼運作**(備取保留期太短 = 等於沒中)
- 🧱 **不知道文件要什麼時候備**(預防接種卡補發要 2 週)

這本手冊把這些坑寫在使用者「**動作之前**」,讓你不用踩過才知道。

## 👥 誰適合看這本手冊

1. **第一次找托嬰/幼兒園的家長** — 完全沒概念,需要從 0 學起
2. **三明治世代家庭** — 同時照顧失智長輩 + 幼兒,需要評估接送動線、祖父母可行性
3. **準備搬家的家庭** — 跨縣市搬遷時,需要重新評估教育資源
4. **想要系統性決策的家長** — 不靠口耳相傳,靠 checklist + 對照表

## ⚠️ 重要聲明

這本手冊是**方法論**,不是「該選哪家」的建議。每個家庭的:
- 預算
- 接送時間
- 教學風格偏好
- 孩子個性
- 家庭結構(雙薪/單親/多代同堂)

都不同。手冊提供工具,**選擇權永遠在你**。
