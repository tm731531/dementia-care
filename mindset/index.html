失智照護面對新事件時,我腦袋裡跑的判斷框架
6 條主觀經驗 · 可採可不採
一位失智照護者的判斷視角整理 · 2026-04-25 初版

這份手冊不是 SOP、不是規則。是我面對失智照護新狀況時,腦袋裡跑的判斷框架。每一條都可以採用、改寫、或拒絕 — 你家情境不同,選擇用不用都合理。

跟家中照護手冊的關係:home-handbook 是我家的實務經驗(13 項改造、4 段 SOP、踩坑),這份是我寫那些東西時背後的判斷視角。

為什麼分開:home 是<我們家發生過的具體事>,可以照抄(家具擺一擺、買個布簾)。心法是<我怎麼看一個新事件>,本質上是主觀的;每個照護者的視角會不一樣,有些可能你完全不採用 — 那也對。

📐 6 條 frame

🧠 1. 失智不是失能(我這樣分)
我的描述

失能:身體弱、認知正常 → 用輔具解(她想做但身體做不到 → 給她工具)

失智:認知退化、身體可能還可以 → 輔具大多無效(她不知道自己在做什麼 → 給她工具沒用,要從環境本身設計)

對失智 frame 的設計原則:

  • 從環境源頭限制(熱水器溫度鎖,不是叫她別轉燙)
  • 把危險物品搬走(洗劑放她拿不到的位置,不是放著叫她別碰)
  • 用感應器代替她的注意力(漏水偵測,不是叫她注意)
  • 用陪伴 / 同步操作取代事後救援(洗澡有人在,不是裝緊急鈴等她按)
我用這個 frame 的情境
  • 浴室安全:扶手 / 緊急鈴對失智無效(她不會抓 / 不會按),熱水器溫度鎖才是主防線
  • 廚房瓦斯:換電磁爐 + 智慧插座,不是「叫她別開瓦斯」
  • 大門隔音:貼外側不是內側(內側她會破壞)
你不一定適用因為
  • 你家長輩可能同時失智 + 失能 — 兩種設計都要
  • 失智早期可能還停留在「失能 frame 有效」的階段(扶手抓得到、緊急鈴按得來)
  • 程度不同的長輩,這個 frame 的應用比例也不同
📖 對應 home-handbook 第 13 項浴室安全 + 第 8 項全家門鎖
➕ 2. 用加法對抗她自己的減法
我的描述

失智本身是減法 — 她每天都在忘記、能力都在退化。如果照護者去測試她原有能力(「這個你記得嗎?」「你還會不會 X?」「你記不記得我?」)= 強化減法,讓她意識到自己失去了什麼,只會沮喪。

正確方向是加法 — 給她新的環境刺激、新的動作、新的陪伴內容,讓她不需要展示自己仍能做什麼。重複動作她就做、不問她能不能做、不評論她做得好不好、不對比過去。

我用這個 frame 的情境
  • 給她剝蒜頭 / 分豆子時,不問「你還會分嗎?」,只給她做
  • 看到老照片時,不問「這是誰?」,直接說「這是 X 阿姨,以前常來我們家」
  • 她叫不出我的名字時,不糾正,直接接話往下走
你不一定適用因為
  • 你跟長輩的互動模式可能 reminiscence(懷舊治療) 比較有效 — 用老照片喚起記憶,可能正是你家家庭凝聚的方式
  • 你想「保留她的記憶」是有意義的努力,選擇不用這個 frame 也合理
  • 失智早期她還能 actively recall,測試可能還是 OK 的
📖 對應 home-handbook 坑 5「認知訓練 vs 讓她忙起來」 + 招式 2 身體記憶
🚶 3. Restlessness vs Boredom · 失智中期不是「無聊」
我的描述

失智中期常見的是restlessness(停不下來),不是boredom(無聊)。這是 BPSD 行為精神症狀的一種。兩個 frame 看起來像同一回事,但工具設計目的相反:

Frame設計目的寫起來像
Boredom補充樂趣觀察、訓練、學習、reminiscence
Restlessness物理鎮靜動作重複 + 身體記憶 + 不需要認知
我用這個 frame 的情境
  • 三樣手工(剝蒜 / 分豆 / 撥豆芽)— 目的是讓她坐下,不是給她事做
  • 運動間 — 為了消耗體力讓她休息,不是訓練她肌少症
  • 沙發放重物 — 為了引導她回房,不是逼她
你不一定適用因為
  • 你長輩可能還在輕度,真的會無聊(boredom frame 適用)
  • restlessness vs boredom 是 spectrum 不是 binary,中間地帶很大
  • 選 boredom frame(reminiscence、認知訓練、社交活動)也合理 — 是另一個 valid 路徑
📖 對應 home-handbook 第 6 項三樣手工 + 第 7 項小遊戲 + 第 10 項運動間
🔄 4. 動線是功能可及性,不是體力管理
我的描述

失智者不會主動想到「我該去喝水了」「我該去廁所了」。動線複雜或被堵 → 她可能卡在原地、或在不該的地方解決 → 脫水、便秘、失禁 連環發生。

真正目的:讓她有機會「無意間走到」廁所跟飲水區。動線通暢 = 她繞著走時自然會經過這兩個關鍵點 → 自然完成基本生理機能。

所以家具設計的核心問題不是「她會不會撞到」,是「她繞行時會不會經過廁所跟水杯位置」。

我用這個 frame 的情境
  • 雙層迴圈(全屋大迴圈 + 客廳小迴圈)讓她無意間走到廁所/水杯
  • 水杯放在她迴圈會經過的位置,不是分散四處(那是 reminder thinking,失智不適用)
  • 沒尖角不是設計目的,有迴圈才是 — 尖角用防撞角貼上去就解
你不一定適用因為
  • 你家空間結構不一定能做雙層迴圈(套房 / 小坪數可能不行)
  • 你長輩可能還能主動 plan(輕度時) — 那她想到要喝水會自己去,不需要動線經過
  • 「她走到 X」不一定能保證她「做 X」 — 還是要看認知程度
📖 對應 home-handbook 第 5 項雙層迴圈 + 全天候盯緊水杯水量
💪 5. 誠實版 framing > 善意包裝
我的描述

寫照護動作時,「為患者好」跟「為照護者好」常常是同一個動作的兩種解釋。我選誠實版(「為了我能撐下去」),因為:

  • 善意包裝(「為她健康」)其他家屬讀了會被勸退 — 覺得「太奢侈 / 太麻煩 / 我做不到那麼周全」
  • 功利誠實(「少吵我」「我半夜不用起來」「我能睡」)讀完反而想做 — 「對啊那我也要做」

照護是 caregiver-patient 雙人系統,不是 patient 單方。包裝成單方利他會讓系統不可持續(照護者燒完 → 系統倒)。

我用這個 frame 的情境
  • 運動間:讓她耗體力 → 她休息 → 我能安靜(不寫成「為她肌少症」)
  • 沙發放重物:讓她回房睡 → 我半夜不用起來 → 我能睡(不寫成「對她比較好」)
  • 家中照護手冊整本強調「優先保的是你不是她」
你不一定適用因為
  • 你還在「為她好」階段,那也對 — 那是健康的 starting point
  • 文化 / 家庭 dynamic 不同,不一定能對外承認「為了少吵我」
  • 如果你的伴侶 / 長輩看到這份手冊,他們可能不能接受這個 framing — 那就保留原本的
📖 對應 home-handbook 第 10 項運動間 + 照護者自己 section
🕸️ 6. Handbook 是 mesh 不是 list
我的描述

家中改造看起來是 list(13 項清單),但實際是 mesh — 很多 item 的真正效益來自跨 section 配合。單獨做某 item 不會 work。

例子:

  • 動線(#5)+ 水杯位置(全天候 section)= 完整補水機制
  • 沙發重物(睡前 SOP)+ 早上定時器(#4)= 半夜走客廳的完整對策
  • GPS 藏習慣物品(法律 P0)+ 第 12 門縫氣密 = 防走失硬軟體配合

單獨做動線、單獨買水杯、單獨買 GPS、單獨裝感應器,效果都比 paired 做差很多。

我用這個 frame 的情境
  • 讀其他失智資源時,看到單一 item 推薦,我會問:「跟哪個 paired 才完整?」
  • 自己排優先級時,選 item pair 一起做,不選單獨 item
  • 跟其他家屬討論時,會推薦 item 組合,不是單個
你不一定適用因為
  • 你資源有限,可能只能做單一 item — 那也比沒做好,不要因為「找不到 pair」就不做
  • 你的 pair 跟我家的 pair 可能不同(你家動線跟水杯的搭配可能不同)
  • 不是所有 item 都有 pair(例如廁所拆門 = 獨立的 P0,沒有明顯需要 pair 的 item)

⚠️ 重要提醒 · 不要把任何 framework 當絕對

這不是失智照護的真理

這 6 條 frame 是我寫家中照護手冊時用的判斷方式,不是失智照護的真理。照護圈裡有很多其他 framework:

  • Reminiscence Therapy(懷舊治療) — 用過去的記憶喚起情感連結。跟我的加法 vs 減法有衝突。
  • Validation Therapy(認可治療) — 不糾正失智者的話語,接受她的現實。跟我的某些 framework 互補。
  • Montessori Method for Dementia — 用兒童教育原則套用到失智照護。跟我的restlessness vs boredom不一定相同。
  • Person-Centered Care(以人為本照護)— Tom Kitwood 的失智照護理論基礎。

這些跟我列的 6 條,有的衝突、有的互補。不要把任何一個 framework 當絕對。我的 frames 反映我家情境;你家不同,你的 frames 也會不同。
