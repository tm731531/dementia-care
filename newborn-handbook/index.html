🚖 孕中:好孕專車 / 計程車補助(別漏領 NT$5,000-8,800)

💰 這一行不領就沒了 — 新北市「好孕專車」實質補助高達 NT$8,800(88 趟 × 每趟最高 100 元),從產檢開始直到產後一年內都可用,**錯過了不能補申請**。其他縣市制度名稱不同(台北市未提供類似專車補助、桃園市另有「孕婦交通安全座椅補助」、台中 / 高雄等以生育獎勵為主),**先 google「{你戶籍縣市} 好孕專車 / 孕婦計程車補助」**確認。

💡 實戰提醒(Tom 家 2023 經驗)

  • 領了媽媽手冊立刻申請,不要等:晚申請 = 浪費前期產檢趟數(那段每月跑診所車資 ~150-200/趟,自己付滿浪費)
  • 排黃金路線:診所近就用車券,診所遠(如高層次超音波要去大醫院)更要用,**單趟 200-300 元車資刷掉一張券差額 100-200**
  • 產後回診也算:坐月子期間自然產 / 剖腹產回診 + 寶寶打疫苗都可用,**至少能多用 8-12 趟**
  • 每月查餘額:APP 內可看,別拖到產後才發現沒用完(過期作廢)
  • 計程車司機可能不熟:上車前先問「有合作好孕專車嗎」,沒有就換一台

💡 三明治世代提醒:若家有失智長輩(主照顧者也要顧),孕婦自己一個人坐車去醫院的次數會更多(很難每次都有家屬陪同)。**好孕專車對單獨就醫的安全感 + 體力 buffer 都有實質幫助**,別覺得「自己騎車就好」省了這筆。產後身體 4-6 週恢復期,坐車比自己騎車安全太多。

💪 產後恢復:自然產

  1. 休息和康復:分娩後需休息,醫生建議數週內充分休息
  2. 疼痛管理:會陰疼痛(撕裂或側切),醫生開止痛藥按時服
  3. 產後護理:陰道清潔每天溫水 + 溫和肥皂清洗,預防感染
  4. 經血(惡露):子宮廢物 + 血液混合,使用產婦專用衛生巾,**注意量 / 顏色變化異常立即回診**
  5. 恢復運動:產後 1-2 週開始輕度散步,避免劇烈運動

🔪 產後恢復:剖腹產

⚠ 剖腹產是大手術 — 體外傷口 7-10 天恢復,但體內傷口要 2 個月。2 個月內不搬重物 / 不劇烈運動 / 飲食維持高蛋白。
  1. 休息和康復:大手術需充分休息,數週內避免重體力 / 提重物 / 劇烈運動
  2. 疼痛管理:遵循醫生用藥,**不要自行停藥**
  3. 傷口護理 5 點:
    • 保持手術切口乾淨乾燥
    • 溫和清潔(溫水 + 溫和肥皂)
    • 溫和拍乾,不摩擦
    • 避免刺激性化妝品
    • 定期更換繃帶 / 敷料
  4. 避免感染:傷口出現紅腫 / 滲液 / 發燒 → 立即看醫生
  5. 限制重物:康復期間避免舉重 / 重體力
  6. 適當飲食:均衡 + 高蛋白(母乳餵養更要),維生素 + 礦物質
  7. 產後體位練習:逐漸開始,加強腹部 + 核心肌肉
  8. 情感支持:產後憂鬱要找家人 / 朋友 / 心理師說

🤱 出生後:媽媽動作清單

母乳餵養 — 增加母乳的 4 招

  1. 補充水分:水是乳汁主要原料,多喝水 + 多吃蔬果 / 果汁 / 湯
  2. 避免酒精:酒精會減少母乳量
  3. 定期排出母乳(最關鍵):
    • 母乳分泌依需求,需求越多分泌越多
    • 頻繁排出 → 身體自動製造更多
    • 每側乳房擠 15-20 分鐘
    • 新生兒每 2-3 小時餵 1 次
    • 用預擠母乳餵也應每 2-3 小時擠 1 次
    • 每天約擠 8-12 次
  4. 促進母乳分泌的食材:
    • 大蒜、洋蔥、薄荷(會讓母乳味道不同,刺激寶寶想喝更多)
    • 糙米、燕麥等全穀物
    • 嘗試香草 / 茶飲前先諮詢醫師

配方乳選擇

  • 找有認證標章的
  • 選大廠牌(出問題有保障 + 比較不會 stale)
  • 從新生兒配方 → 較大嬰兒(6 個月)→ 幼兒(1 歲)分階段換

👶 寶寶體重問題

小孩體重跟超音波有 10-20% 誤差,出生後 3-5 天又會降生理性體重(約 5-10%)。

WHO / 國健署 早產兒分類(供參考,各醫院實際標準會略不同):

  • **極低體重兒**:< 1,500g(通常進新生兒加護)
  • **低體重兒**:1,500-2,500g(評估觀察,部分需保溫箱)
  • **正常足月**:≥ 2,500g

月中收住標準:多數月子中心要求 ≥ 2,200-2,500g 且體溫穩定,**各家差異大,訂房前確認**。

結論:孕期維持均衡飲食 + 規律產檢,**體重落在足月區間最不麻煩**。具體標準以你的婦產科 / 兒科醫師為準。

水奶 vs 配方奶差異

  • 水奶:即開即喝,衛生 / 方便,但貴 + 開封後保存短
  • 配方奶:粉狀,需熱水沖泡,便宜 + 保存久,但要洗奶瓶 / 控溫
  • 外出建議帶水奶,在家用配方奶
