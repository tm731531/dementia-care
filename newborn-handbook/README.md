## 涵蓋階段

1. **🌱 孕前**:健康檢查 / 懷孕日記 / 後援評估
2. **🤰 孕中**(40 週):
   - 10 種胎教
   - 14 次產檢時程(8w-40w)
   - 飲食指南(含子宮肌瘤患者特別注意)
   - 月中 / 後援規劃決策樹
3. **🎒 產前**:家中物品 + 待產包 4 類 checklist(資料 / 日常 / 媽媽 / 可有可無)
4. **💪 產後**:自然產 5 點 / 剖腹產 8 點恢復重點
5. **👨‍👧 出生後爸爸 5 步政府表單**:戶政 → 區公所 → 公司 → 銀行 → 保險
6. **🤱 媽媽母乳 / 配方乳**:增加母乳 4 招 / 配方乳選擇
7. **🏫 托嬰中心比較**:Tom 家土城 6 家公托/準公對比範例
8. **📚 育兒資源**:教具 / 親子館 / 圖書 / 週末出遊
9. **❓ 疑難雜症**:寶寶體重 / 水奶 vs 配方奶

## 為什麼三明治世代特別需要

家中同時照顧失智長輩 + 育兒新生兒,壓力 stack:
- **後援系統不能 share**:失智者環境變動會 sundowning,不能請後援同住坐月子
- **時間排程要整合**:回診 / 接送 / 餵奶 / 換尿布 / 早晚 SOP 全部撞在一起
- **財務壓力**:長照支出 + 育兒支出 + 可能育嬰留停 → 收入降

每章都有「若家有失智長輩需特別注意」callout。
